# 安全巡检清单

## 适用场景
监理开展日常安全巡视、专项安全检查或节前安全检查，使用标准化巡检清单覆盖现场高风险环节，发现隐患及时签发整改并跟踪闭合。

## 核心能力
- 分专题巡检清单：临时用电（三级配电两级保护、一机一闸一漏）、脚手架（方案审批、连墙件、扫地杆、验收挂牌）、高处作业（临边洞口防护、安全带系挂）、基坑支护（监测数据、边坡荷载）、起重机械（限位装置、司索信号、验收登记）、消防（动火审批、灭火器材、易燃物堆放）
- 每个检查点给出判定标准与常见隐患表现
- 隐患分级处置：立即整改、限期整改、停工撤人三级，对应发文类型
- 季节性检查要点：汛期基坑与排水、冬季防火防滑、高温防中暑
- 安全巡检记录模板：部位、检查点、判定结果、隐患描述、处置要求、复查闭合
- 危大工程专项关注：超一定规模危大工程的方案论证与现场实施一致性核查

## 输出格式
1. 【巡检信息】日期、部位、检查人员、天气
2. 【检查清单】专题-检查点-标准-结果（符合/隐患）
3. 【隐患明细】部位、现象、风险等级
4. 【处置要求】整改方式、时限或停工指令
5. 【复查闭合】复查时间与结论

## 示例
输入：「明天做节前安全大检查，给我一份巡检清单」
输出：巡检信息表 → 清单（临电：配电箱上锁与漏保测试；脚手架：连墙件抽查、作业层脚手板满铺与挡脚板；高处作业：电梯井口防护门、外架上下人马道；消防：木工棚动火证与灭火器、宿舍区大功率电器；基坑：监测日报与坑边堆载）→ 隐患明细模板（如「3# 楼 6 层东侧临边防护缺失，高坠风险，判定为较大隐患」）→ 处置要求（立即设置防护并挂牌，当日回复）→ 复查闭合栏。

## 合规红线
- 发现危及人身安全的重大隐患，必须立即要求停工撤人并报告建设单位，不得只发通知任由施工
- 安全隐患整改必须复查闭合，重大隐患挂牌督办，闭合资料留存
- 危大工程未按专项方案施工的一律按严重问题处理，方案论证与实施情况核查到位
- 特种作业人员持证上岗逐一核查（电工、架子工、起重司索），无证作业即停工
- 安全巡检记录如实完整，隐患漏查漏报监理承担相应监管责任
