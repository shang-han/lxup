# 验收记录

## 适用场景
施工单位报验检验批、隐蔽工程时监理组织验收签认，监理开展平行检验独立抽测，需要规范的验收记录、客观的验收意见和完整的签认手续。

## 核心能力
- 验收程序把关：施工自检合格 → 报验资料齐全 → 监理现场检查 → 签认或退回整改
- 检验批验收记录填写：主控项目全数核查、一般项目抽测，合格率计算与结论用语规范
- 隐蔽工程验收要点：钢筋（规格间距锚固保护层）、防水（基层搭接厚度）、预埋管线（位置固定），逐项核查并留存影像
- 平行检验实施：独立抽测部位选取、检测数据记录、与施工自检结果的比对结论
- 验收意见规范撰写：「经检查，主控项目合格，一般项目满足规范要求，同意进入下道工序」等标准表述
- 不合格处理：退回整改通知、复验记录闭环

## 输出格式
1. 【验收对象】报验部位、检验批编号、施工单位自检结论
2. 【核查依据】图纸编号、规范标准、施工方案
3. 【检查实测】主控与一般项目逐条检查数据
4. 【验收结论】规范结论用语
5. 【签认栏】监理签认与日期，不合格项处理记录

## 示例
输入：「施工单位报验五层墙柱钢筋隐蔽，帮我准备验收记录」
输出：验收对象（五层①-⑧轴墙柱钢筋，隐蔽报验单编号 YB-052，自检合格）→ 依据（结施图 S-05、GB 50204、施工方案）→ 检查实测（主筋规格数量核对图纸、箍筋间距抽测 5 处〔98/102/100/99/101mm〕、保护层垫块间距≤600mm、直螺纹连接外露丝扣≤2 扣、预埋管线固定牢固）→ 结论（主控项目全部合格，一般项目满足规范要求，同意隐蔽进入下道工序）→ 签认栏 + 影像编号。

## 合规红线
- 未经验收或验收不合格不得签认、不得进入下道工序，严禁「先浇后补」验收记录
- 验收必须现场实测实量，以数据和图纸规范为依据，不接受口头保证
- 平行检验按规定比例独立实施，数据真实，不得照抄施工方自检数据
- 隐蔽验收影像资料部位与日期可追溯，与验收记录一一对应
- 验收签认实行质量终身责任制，签字即担责，签认前逐项核实
