# 行业基础

## 行业概况
工程监理是受建设单位委托、依据法律法规与合同对施工质量、安全、进度、投资实施监督管理的第三方，实行总监理工程师负责制。监理的法定职责源于《建筑法》《建设工程质量管理条例》《建设工程安全生产管理条例》，关键工序旁站、隐患报告、验收签认均实行质量终身责任制。监理工作手段以文件为主：巡视、旁站、平行检验、签发监理通知与暂停令、组织验收、审核计量。持证上岗（监理工程师/监理员）是从业门槛，客观公正、留痕履职是职业底线。

## 常用术语
- 旁站：对关键部位、关键工序施工全过程的现场跟班监督
- 巡视：监理人员对施工现场进行的定期或不定期的巡回检查
- 平行检验：监理机构独立于施工单位进行的检测试验活动
- 监理通知单：要求施工单位整改质量或安全问题的正式监理文件
- 工程暂停令：存在重大隐患或违规施工时要求局部或全面停工的指令
- 工程例会：建设、监理、施工等各方参加的定期协调会议
- 危大工程：危险性较大的分部分项工程，需编制专项方案，超一定规模需专家论证
- 见证取样：监理见证下现场取样并送具备资质检测机构检测

## 核心工作流程
事前控制：审核施工组织设计与专项方案 → 审查人员资格与机械验收 → 进场材料报验审核 → 复核测量放线
事中控制：巡视检查 → 关键工序旁站 → 平行检验 → 检验批与隐蔽验收签认 → 发现问题签发通知并跟踪
事后控制：分项分部验收 → 审核工程计量与支付 → 编写监理月报与总结 → 参与竣工验收与资料归档

## 常见问题
Q：施工单位不配合监理怎么办？
A：先口头指出并留记录，不整改即签发监理通知单限期整改；仍不执行的签发暂停令并书面报告建设单位，重大问题按法定义务报告主管部门，全程留痕。

Q：发现质量问题标准处理流程是什么？
A：拍照与实测取证 → 判定等级 → 一般问题发通知单限期整改、严重问题发暂停令 → 审核整改回复 → 现场复查闭合 → 记入台账。

Q：旁站和巡视怎么区分安排？
A：按旁站监理方案对混凝土浇筑、防水、预应力等关键工序全过程旁站；其余部位按计划每日巡视，巡视覆盖全部在施工作面并记录。

Q：监理签字的责任有多大？
A：验收签认、计量审核实行质量终身责任制，签字即承担相应监理责任，必须坚持「未经验收不签、数据不实不签、依据不足不签」。
