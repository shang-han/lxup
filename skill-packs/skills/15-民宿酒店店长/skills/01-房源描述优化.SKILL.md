# 房源描述优化

## 适用场景
民宿主/酒店店长在 OTA 平台（携程/美团/途家/爱彼迎）上架或优化房源时使用。需要写出吸引人的标题、突出差异化卖点、补全设施与周边描述，提升房源点击率与下单转化。

## 核心能力
- 根据房型、户型、景观、设计风格提炼差异化卖点（海景/loft/亲子/投影/智能家居/庭院）
- 撰写高点击房源标题（位置+核心卖点+户型，控制在 20 字内）
- 生成结构化房源描述：亮点、户型设施、适合人群、周边交通
- 设施清单规范化（品牌床品、一次性用品、厨房电器、停车位、Wi-Fi）
- 图片拍摄建议文案，引导「所见即所得」降低落差差评
- 多平台风格适配（携程商务、途家亲子、爱彼迎个性化）

## 输出格式
1. 【房源标题】主标题 + 备选标题
2. 【一句话卖点】核心吸引力
3. 【房源描述】亮点 / 户型设施 / 适合人群 / 周边配套
4. 【设施清单】分类列表
5. 【预订须知】入住时间 / 押金 / 宠物 / 退订规则
6. 【优化建议】图片拍摄与细节补充建议

## 示例
输入：「帮我写个厦门鼓浪屿附近的海景 loft 房源，住 2 人，有投影」
输出：主标题「鼓浪屿码头旁·180°海景loft·巨幕投影·步行 2 分钟到沙滩」→ 一句话卖点「推窗见海，躺在床上看日落」→ 房源描述（loft 户型、品牌床垫、独立卫浴、步行 5 分钟到轮渡码头）→ 设施清单 → 预订须知 → 拍摄建议（补拍夜景落地窗海景与投影使用场景）。

## 注意事项
- 描述须与实物一致，所见即所得，避免照片落差引发差评
- 不夸大不确定设施（如「必海景」实际为侧海景需注明）
- 易争议规则（押金、加人费、宠物、退订）提前写清
- 遵守广告法，不使用「最近」「第一」等绝对化用语，不虚假宣传距离
