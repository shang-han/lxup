# 行业基础

## 行业概况
民宿行业经历爆发式增长后进入品质化阶段，全国民宿超 100 万家。OTA 平台（美团/携程/途家/爱彼迎）是主要客源渠道。核心竞争力：差异化体验 + 口碑运营。收入取决于入住率 × 房价（RevPAR），季节与节假日波动大，淡季考验运营功底。评分口碑直接决定平台排名与流量分配，卫生与服务是复购根基。

## 常用术语
- 入住率：出租房间 / 总房间数
- ADR：平均房价
- RevPAR：每间可售房收入 = 入住率 × ADR
- OTA评分：各平台综合评分
- 好评率：好评占总评价比例
- 退房房（走客房）：客人退房后需清洁整理的房间
- 闪订/即时确认：客人下单无需房东确认即可直接预订
- 连住优惠：多晚连住给予的价格优惠

## 核心工作流程
每日：订单处理 → 入住接待 → 退房查房 → 清洁周转 → 评价回复
每周：保洁排班 → 设施检查 → 平台价格调整 → 竞品监控
每月：经营分析 → 成本控制 → 营销策划 → 口碑维护

## 常见问题
Q：淡季没客人怎么办？
A：降价促销 + 套餐组合（住宿+体验），开发企业团采客户，利用淡季做设施升级与口碑积累。

Q：差评怎么处理？
A：24 小时内回复道歉，解释原因 + 补偿方案，私下沟通争取理解，并落实内部整改防止复发。

Q：怎么提升平台排名？
A：保持闪订确认率与回复速度，提升评分，完善房源信息，合理参与平台活动带动转化。

Q：节假日如何定价？
A：参考同地段历史同期价格阶梯涨价，配套套餐与提前订优惠锁客，明示不可退规则，按预订进度动态微调。
