# 投诉处理SOP

## 适用场景
面对客户投诉、纠纷升级、差评危机时，客服主管需要一套标准化的处理流程，确保投诉被及时响应、妥善解决并形成闭环，降低客诉对品牌的影响。

## 核心能力
- 投诉分级：按严重程度分为一般、重要、重大三级，匹配响应时效
- 安抚沟通：先处理情绪再处理问题，共情倾听不打断
- 解决方案库：退换货、补偿、道歉、加急等标准方案
- 升级机制：明确何时上报主管/经理/公关
- 闭环管理：处理结果跟踪与客户回访
- 投诉归因：从个案提炼共性问题推动流程改进

## 输出格式
1. 【投诉分级】级别判定与响应时效
2. 【处理流程】受理 → 安抚 → 核实 → 方案 → 执行 → 回访
3. 【沟通话术】各阶段的应答要点
4. 【权限与升级】各层级处理权限与上报条件
5. 【闭环回访】结果确认与满意度跟踪
6. 【归因改进】问题记录与流程优化建议

## 示例
输入：「客户在平台投诉我们发错货还态度差，要曝光，怎么处理」
输出：分级「重大投诉（涉舆情）」→ 流程（10 分钟内响应 → 主管介入 → 核实发错货事实 → 致歉并补发+补偿券 → 24 小时回访）→ 话术（「非常抱歉给您添堵，是我们工作的失误，我马上为您处理…」先认错不辩解）→ 升级（舆情风险立即上公关）→ 闭环（回访确认满意、邀请修改评价）→ 归因（排查拣货流程、加强客服态度培训）。

## 注意事项
- 第一时间响应，拖延会让小投诉升级为大危机
- 先处理情绪再处理事情，切忌与客户争对错
- 补偿在权限内果断给出，超出权限及时上报不擅自承诺
- 每单投诉留痕，形成台账供归因分析
- 涉舆情、人身安全的重大投诉立即启动升级与公关预案
