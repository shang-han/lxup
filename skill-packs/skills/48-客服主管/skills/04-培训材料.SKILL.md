# 培训材料

## 适用场景
客服主管需要为新人入职、技能提升、专项整改准备培训材料，包括课件大纲、知识点、情景案例与考核内容，快速提升团队服务能力。

## 核心能力
- 新人培训体系：企业文化、系统操作、服务规范、话术、考核
- 产品知识培训：卖点、参数、常见问题、对比
- 服务技巧：沟通、共情、异议处理、投诉应对
- 情景演练：典型案例模拟与标准应答
- 考核设计：知识点测试与实操评分标准
- 课件结构化：章节、要点、案例、练习分层组织

## 输出格式
1. 【培训主题】目标人群与学习目标
2. 【课程大纲】章节结构与要点
3. 【核心内容】知识点详解
4. 【情景案例】典型案例与标准处理
5. 【实操练习】演练与互动环节
6. 【考核方式】测试题或评分标准

## 示例
输入：「下周一有 5 个新客服入职，帮我做个新人培训方案」
输出：主题「新客服 3 天上岗培训」→ 大纲（Day1 企业文化+系统操作、Day2 产品知识+服务规范、Day3 话术演练+情景模拟+考核）→ 核心内容（服务五步法、禁语清单、退换货政策）→ 情景案例（客户催发货、质量投诉、要求赔偿的标准应答）→ 实操（老带新旁听 + 模拟对话）→ 考核（笔试 80 分及格 + 模拟会话评分达标方可上岗）。

## 注意事项
- 培训内容贴合实际工单场景，避免空泛理论
- 新人培训重在上手速度，先掌握高频场景
- 情景案例来自真实脱敏工单，更有代入感
- 培训后要有考核验证效果，不走过场
- 材料随产品与政策更新定期迭代
