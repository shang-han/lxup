# 行业基础

## 行业概况
客服是企业与客户沟通的核心窗口，渠道涵盖电话、在线客服、社群、邮件等。客服质量直接影响客户留存、复购与品牌口碑，是零售/电商企业的核心竞争力之一。客服主管的核心职责是团队管理、服务质量监控与效能提升，核心考核指标为响应时长、一次性解决率、客户满意度。行业痛点是人员流动性大、情绪压力高，管理上需兼顾制度刚性与人文关怀。

## 常用术语
- 首次响应时长：客户发起咨询到客服首次回复的时间
- 平均处理时长：处理一个工单/会话的平均耗时
- 一次性解决率：一次对话即解决问题、无需二次联系的比例
- CSAT：客户满意度评分
- NPS：净推荐值，衡量客户推荐意愿
- 客诉率：投诉订单占总订单的比例
- 升级单：超出客服权限需上报主管处理的工单

## 核心工作流程
日常：团队排班 → 质量监控 → 疑难升级 → 数据日报
定期：培训计划制定 → 质检报告 → 流程优化 → 系统建议
月度：KPI 考核 → 团队复盘 → 培训总结 → 下月目标

## 常见问题
Q：客服离职率高怎么办？
A：优化排班减少高强度夜班、完善晋升通道、增加情绪疏导与团队关怀、合理设计薪酬激励。

Q：客户情绪激动甚至辱骂怎么办？
A：先倾听再回应、共情理解、不被情绪带偏、提供解决方案、跟进闭环；必要时按流程转交主管处理。

Q：怎么提升一次性解决率？
A：完善话术库与产品知识库、加强培训、授权一线合理处理权限、建立常见问题的速查与标准化流程。

Q：如何科学排班？
A：按时段话务量预测匹配人力，高峰留足余量，班次轮换公平透明，保障员工休息，大促提前增配。
