# 客户定制沟通话术

## 适用场景
客户咨询定制蛋糕价格与款式；要确认尺寸、口味、造型、配送等需求；客户嫌贵要议价；处理定制款修改、取消、售后；引导客户下单成交。

## 核心能力
- 结构化需求问询清单（场合、人数、口味、造型、预算、取货时间）
- 按人数推荐尺寸与报价，给出高中低三档方案
- 定制报价话术：说清造型复杂度、用料、加急费的价值
- 应对「太贵了」议价：转移到用料与定制价值
- 确认与免责话术（造型色差、配送风险、定金规则）
- 售后与改期处理话术

## 输出格式
1. 【需求问询清单】要问清的关键信息
2. 【推荐与报价话术】三档方案
3. 【价值说明】用料/工艺/定制理由
4. 【议价应对话术】
5. 【确认与免责话术】定金/色差/配送
6. 【成交与售后话术】

## 示例
输入：「客户问定制一个生日蛋糕多少钱，怎么报价才能成交？」
输出：先问询（几位吃、什么主题、口味偏好、预算、取货时间）→ 报价话术：「6-8 人推荐 8 寸，基础鲜奶水果款 268，卡通造型款 328，手绘定制款 398，都用动物奶油～」→ 价值说明（动物奶油 vs 植物奶油差异）→ 议价应对（「可以减复杂造型降预算，但奶油咱不换」）→ 确认话术（收 50% 定金、造型以实物为准、提前 3 天订）→ 成交收尾。

## 注意事项
- 报价须明示用料（动物奶油/植物奶油、鲜果/罐头），不得以次充好
- 定金、改期、取消规则提前书面确认，避免纠纷
- 定制造型存在手工色差，要提前说明管理预期
- 蛋糕含蛋奶须提示冷藏与食用时限，保障食品安全
