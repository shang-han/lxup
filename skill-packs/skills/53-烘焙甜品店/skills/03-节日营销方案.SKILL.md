# 节日营销方案

## 适用场景
中秋月饼、圣诞蛋糕、情人节巧克力、母亲节/生日季等节日大促；要推季节限定款；策划预售与礼盒；门店节日氛围布置与引流。

## 核心能力
- 按节日特性选品（中秋月饼、情人节巧克力、圣诞树根蛋糕）并设计礼盒
- 制定预售机制（早鸟价、定金、阶梯优惠）平滑产能高峰
- 设计节日定价与利润款组合
- 产出预热文案（朋友圈、社群、小红书）与倒计时节奏
- 规划产能与备料排期，避免节日爆单断货
- 输出活动复盘指标（预售量、客单价、礼盒占比）

## 输出格式
1. 【活动概览】节日、主题、时间、目标
2. 【限定选品与礼盒】
3. 【预售与定价机制】
4. 【预热节奏与文案】朋友圈/社群/小红书版
5. 【产能备料排期】
6. 【复盘指标】

## 示例
输入：「中秋节想做月饼礼盒，帮我出个营销方案」
输出：主题「手作团圆·中秋限定」→ 选品（流心奶黄/蛋黄莲蓉双拼 + 高端礼盒款）→ 预售（提前 30 天早鸟 85 折、满 5 盒送 1 盒、企业团购阶梯价）→ 预热节奏（倒计时海报 + 试吃短视频 + 朋友圈晒单）→ 文案「今年中秋，把心意揉进手作月饼……」→ 产能排期（预售订单提前锁料、节前两周集中生产）→ 复盘指标。

## 注意事项
- 节日礼盒须明码标价，预售规则与发货时效写清，避免违约纠纷
- 月饼等节令食品须严控保质期与馅料安全，标注生产日期与存储条件
- 文案规避《广告法》禁用词，不得虚假宣称用料
- 节前备料须索证索票，产能不足时勿赶工牺牲品质与卫生
