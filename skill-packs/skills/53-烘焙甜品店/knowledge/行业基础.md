# 行业基础

## 行业概况
烘焙甜品是高频、高毛利的餐饮细分，涵盖面包、蛋糕、西点、定制甜品与节令食品（月饼、粽子）。行业两极分化：连锁品牌靠供应链与规模，独立店靠手工、定制与社区口碑。核心痛点：原料价格波动大（黄油、奶油、面粉）、保质期短损耗高、节日产能潮汐明显。增长靠产品创新、私域会员、节日礼盒与定制服务。

## 专业技能体系
### 硬技能
- 烘焙配方研发与标准化
- 原料成本控制（黄油/奶油/面粉价格波动应对）
- 裱花与装饰技法
- 保质期与损耗管理
- 橱窗陈列设计

### 软技能
- 客户定制沟通
- 节日营销策划

## 常用术语
- 烘焙百分比：以面粉为 100% 基准表达各原料比例的配方表示法
- 出成率：成品重量/投入原料重量，衡量损耗
- 动物奶油/植物奶油：前者乳脂提取、口感好成本高，后者植物油氢化、稳定便宜
- 毛利率：（售价-原料成本）/售价
- 损耗率：报损金额/营业额，现烤业态核心控制指标
- 以销定产：按销售预测安排烤制批次，减少滞销损耗
- 客单价：平均每单金额，靠礼盒、定制、套餐拉升
- 私域：微信社群/个人号沉淀的自有客户池

## 核心工作流程
每日：备料 → 分批次烤制 → 橱窗陈列 → 销售 → 晚间临期促销 → 报损登记与日结
每周：原料盘点与采购 → 损耗分析 → 会员/社群运营 → 新品试做
每月：成本与毛利核算 → 节日营销规划 → 产品结构优化 → 设备保养

## 常见问题
Q：黄油奶油老涨价，怎么控成本？
A：与供应商谈季度锁价或批量囤货，做涨价敏感性测算，必要时推高价值感产品支撑调价，别靠换劣质料压成本。

Q：现烤面包损耗太高怎么办？
A：以销定产按分时段销量定烤制批次，下午少烤勤补，晚间折扣清货，临期做面包干再利用，把损耗率控在 5% 内。

Q：定制蛋糕客户嫌贵怎么谈？
A：先问清需求推三档方案，说清动物奶油与手工造型的价值，议价时减造型复杂度而非换料，提前收定金锁单。

Q：节日爆单怎么不断货又不压货？
A：提前预售锁单（早鸟价+定金），按预售量排产备料，留弹性产能应对散单，节前集中生产平滑高峰。
