# 行业基础

## 行业概况
电商市场规模超 15 万亿元，淘系、京东、拼多多、抖音电商四分天下，内容电商（直播、短视频带货）持续高速增长。运营核心链路为「流量获取 → 转化提升 → 客单价提升 → 复购率」，精细化运营成为主要壁垒。平台规则迭代快、流量成本持续上升，对运营者的数据分析、内容营销与供应链协同能力要求越来越高。

## 常用术语
- UV：独立访客数，去重后的进店人数
- PV：页面浏览量，页面被浏览的总次数
- 转化率：访客到买家的比例，衡量详情页说服力的核心指标
- 客单价：平均每单支付金额，靠关联搭配与满减拉升
- SKU：商品品种数量，含颜色、尺码等规格变体
- GMV：成交总额（通常含未付款与退款订单）
- ROI：投入产出比，付费推广成交金额 / 推广花费
- DSR：店铺动态评分，含描述相符、服务态度、物流速度三项

## 核心工作流程
每日：商品上架与详情页优化 → 推广数据监控与调价 → 客服与评价管理 → 库存与发货跟进
每周：平台活动报名 → 竞品分析 → 推广计划与素材优化 → 选品调整
每月：经营数据复盘 → 库存盘点与补货 → 供应商结算 → 复购与会员策划

## 常见问题
Q：店铺没流量怎么办？
A：先诊断流量来源结构，再对症优化——优化搜索标题关键词抓自然流量，开通付费推广（直通车/千川）测款放量，积极报名平台活动，布局短视频/直播等内容渠道拓增量。

Q：转化率低怎么办？
A：检查详情页、价格、评价、客服四要素：主图与首屏突出核心卖点，价格对标竞品保持竞争力，积累优质买家秀并及时处理差评，提升客服响应速度与话术专业度。

Q：推广 ROI 上不去怎么办？
A：精准定向降低无效点击，多组素材轮播测试提升点击率，优化详情页与价格提升转化率，坚决暂停高消耗低产出的计划与关键词。
