# 竞品分析报告

## 适用场景
电商运营在新品类切入、新品上市、销量下滑诊断或大促备战时，系统了解竞争格局。通过对 2-5 个竞品的选品、价格、流量结构、营销玩法与评价口碑的对比，找到差异化突破口与可落地的应对策略。

## 核心能力
- 竞品识别：直接竞品（同品类同价格带）+ 间接竞品（同人群不同品）+ 标杆竞品（行业 TOP）
- 多维对比：选品结构、价格带、月销量、流量渠道（搜索/推荐/直播/付费）、核心卖点、促销机制
- 评价挖掘：系统梳理竞品差评集中点，定位其短板与我方机会（如竞品差评集中在「发货慢」→ 我方主打「当日发货」）
- 营销动作分析：推广关键词、直播排期、达人合作、活动报名节奏
- SWOT 综合：我方优势 / 劣势 / 机会 / 威胁
- 可落地策略：差异化卖点、价格策略、流量打法、选品补充，分优先级排序

## 输出格式
1. 【分析背景】目的、对象、竞品清单与选取理由
2. 【竞品对比表】价格 / 月销 / 卖点 / 流量结构 / 活动力度多维对比
3. 【评价洞察】竞品好评集中点（行业需求）与差评集中点（机会点）
4. 【SWOT 分析】我方竞争位置总结
5. 【策略建议】3-5 条可执行的应对措施，标注优先级
6. 【跟踪计划】后续监控指标、工具与频率

## 示例
输入：「我们卖咖啡液，分析一下类目 TOP3，找机会」
输出：锁定 3 个竞品（A 主打 5.9 元/条走量、B 主打 8.8 元/条达人种草、C 主打 12 元/条精品产地）→ 对比表（价格/月销/流量结构/卖点）→ 评价洞察：竞品差评集中在「太苦」「包装漏液」→ SWOT：我方优势是低酸口感与高颜值包装 → 策略：卡位「入门精品咖啡液」差异化定位，价格带定 7.9 元打 A 品质差与 B 价格高的空档，主图强调「不苦不涩」→ 制定每月一次的竞品跟踪表。

## 注意事项
- 数据取自公开渠道（平台页面、第三方数据工具），不采用不正当手段获取竞品内部数据
- 避免以单点数据下结论，关键指标看趋势与均值
- 分析价值在策略落地，每条结论都要对应可执行动作
- 竞争格局变化快，建立周期性跟踪机制而非一次性报告
- 保持客观，不主观贬低或神化竞品
