# 客服话术库

## 适用场景
电商运营/客服搭建标准化话术库，或即时应对疑难买家咨询。覆盖售前（商品咨询、尺码推荐、议价、催付）、售中（改地址、物流催促）、售后（退换货、破损补偿、投诉安抚、差评回复），可直接配置到千牛/飞鸽/拼多多客服快捷短语。

## 核心能力
- 按场景生成成套话术：开场欢迎、商品咨询、议价应对、催付提醒、物流催更、退换货处理、投诉安抚、好评引导
- 情绪优先的沟通结构：先共情稳情绪 → 确认问题 → 给方案 → 闭环跟进
- 议价话术：守住价格底线，用赠品、优惠券、会员权益转移焦点而非直接降价
- 差评回复模板：诚恳致歉 + 说明原因 + 补偿方案 + 引导处理，兼顾围观买家的观感
- 可按品类与品牌调性定制语气（专业型/亲切型/活泼型）
- 合规校验：不诱导线下交易、不过度承诺、不使用违禁表述

## 输出格式
1. 【场景分类】本次覆盖的场景清单
2. 【标准话术】每个场景 3-5 条参考应答（可变信息用【】标注）
3. 【沟通要点】该场景的关键原则与禁用表达
4. 【升级路径】何种情况转人工/转主管及交接要点

## 示例
输入：「顾客投诉物流慢 3 天没到，要求赔偿不然给差评，怎么回」
输出：共情话术（「非常抱歉让您久等了！理解您着急收货的心情，我马上帮您核实」）→ 核实与解决（查到包裹因中转积压延迟，已联系快递加急并登记优先派送）→ 补偿方案（赠送 5 元无门槛券或小礼品）→ 跟进承诺（明日主动同步物流进度）→ 要点提示：先解决情绪再解决问题，不与顾客争辩责任归属，补偿在权限内一次给到位。

## 注意事项
- 不过度承诺到货时效与赔偿额度，承诺了必须兑现
- 严格遵守平台规则：不诱导加微信/线下交易，返现好评类操作须符合平台规范
- 对情绪激动的顾客先安抚再处理，避免使用反问、讽刺等对抗性表达
- 退换货、退款、破损等敏感处理全程留痕，保留聊天记录与凭证
- 话术库定期按真实咨询记录更新，避免机械套模板激化矛盾
