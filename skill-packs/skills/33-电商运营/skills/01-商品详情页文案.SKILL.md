# 商品详情页文案

## 适用场景
电商运营/店长上新商品、改版详情页、备战大促时使用。适合淘宝/天猫、京东、拼多多、抖音小店等平台的详情页分屏、主图卖点、SKU 短文案撰写，目标是提升点击率与转化率。

## 核心能力
- 按「目标人群 + 使用场景 + 核心痛点」提炼卖点（USP），区分主卖点与辅助卖点
- 输出主图文案（5 张主图的卖点短句）、标题关键词组合与详情页分屏结构
- 场景化表达：把参数翻译成用户利益点（如「10000mAh」→「出差一整天不用带充电宝」）
- 信任背书设计：销量、好评、资质检测、材质溯源、售后保障模块的排布逻辑
- 适配不同平台风格：淘系偏细节说服、拼多多偏性价比直给、抖音偏情绪种草
- 合规自查：自动规避「最」「第一」「全网最低」等广告法禁用词与功效断言

## 输出格式
1. 【卖点提炼】核心卖点 1 条 + 辅助卖点 3 条（各一句话）
2. 【主图文案】5 张主图的卖点短句（每条不超过 12 字）
3. 【标题建议】含核心词/属性词/长尾词的关键词组合
4. 【详情页分屏】首屏钩子 → 痛点场景 → 卖点论证 → 参数说明 → 信任背书 → 售后保障
5. 【促销信息】活动价、赠品、优惠券的呈现文案
6. 【合规提示】需规避的敏感词及替代说法

## 示例
输入：「卖便携小风扇，目标通勤白领，写详情页文案」
输出：核心卖点「4000mAh 三档风力，通勤一天一充够用」→ 5 张主图文案（「一充用一天」「三档风力可调」「比手机还轻」「静音不扰同事」「USB-C 快充」）→ 详情页分屏：首屏钩子（通勤挤地铁汗流浃背场景）→ 痛点（重、没电、吵）→ 卖点论证（重量对比、续航实测）→ 参数表 → 信任背书（10 万+ 销量 + 7 天无理由）→ 促销文案 → 合规提示（避免「风力最强」等绝对化表述）。

## 注意事项
- 严格遵守《广告法》与平台规则：禁用绝对化用语（最、第一、顶级），功效类宣称（美白、瘦身）需有资质证明
- 不堆砌参数，每个卖点都要翻译成用户能感知的利益
- 首屏与主图第一张决定点击与停留，最强卖点前置
- 文案与平台人群匹配，同一商品不同渠道差异化表达
- 促销信息写清有效期与参与条件，避免「虚假宣传」投诉
