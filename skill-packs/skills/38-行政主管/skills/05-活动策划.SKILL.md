# 活动策划

## 适用场景
行政主管/经理策划公司员工活动：年会、季度团建、员工生日会、节日庆祝（三八/中秋/圣诞）、运动会、家庭日等。输出完整可执行方案，在预算内办出员工参与度与满意度，体现企业文化。

## 核心能力
- 活动类型库：年会盛典、户外团建、主题生日会、节日活动、趣味运动会、家庭开放日
- 完整方案输出：目的 / 主题 / 时间地点 / 参与对象 / 流程 / 预算 / 物料 / 分工 / 宣传 / 预案
- 流程设计：暖场 → 主体环节 → 高潮（抽奖/颁奖/互动）→ 收尾，节奏把控员工体验
- 预算管理：场地、餐饮、物料、奖品、外包费用的明细与控制技巧
- 互动设计：提升参与度的游戏、抽奖、投票机制，避免「领导讲话员工刷手机」
- 执行分工：筹备组分工表、时间推进表、当天执行 checklist

## 输出格式
1. 【活动概览】目的、主题、时间、地点、参与对象
2. 【活动流程】分时段环节安排
3. 【预算明细】分项费用与合计
4. 【物料清单】需准备的物品与奖品
5. 【执行分工】筹备任务与责任人、时间节点
6. 【风险预案】天气、设备、安全等应急方案

## 示例
输入：「策划一场 100 人公司的年会，预算 8 万」
输出：主题「乘风破浪·共创 2027」→ 流程：17:00 签到墙合影 → 18:00 开场视频 + 领导致辞（控制在 15 分钟内）→ 年度表彰 → 节目表演与三轮抽奖穿插 → 晚宴敬酒 → 21:00 结束 → 预算：场地餐饮 4 万 + 舞台灯光 1.5 万 + 奖品 1.8 万 + 物料礼品 0.7 万 → 分工：行政统筹、HR 负责表彰名单、各部门出一个节目 → 预案：设备双备份、奖品提前清点、安排安全负责人。

## 注意事项
- 预算先定大盘再拆细项，奖品与餐饮是员工体验核心，不在这里省
- 领导致辞环节控制时长，活动主角是员工，体验优先于形式
- 户外活动必配天气预案与保险，高风险项目评估安全资质
- 抽奖与评奖规则提前公示，过程公开，避免公平性争议
- 活动后收集满意度反馈并复盘，为下次策划积累数据
