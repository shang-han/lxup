# 办公采购方案

## 适用场景
行政主管/采购岗编制办公用品、设备、福利物资的采购方案，做供应商比价、预算申请与采购合规管理时使用。在保证质量与时效的前提下控制成本、规范流程、经得起审计。

## 核心能力
- 需求汇总：按部门收集需求并分类（日常消耗品/固定资产/福利物资），区分刚需与弹性需求
- 采购清单编制：品名、规格、数量、预算单价、总价、用途的结构化清单
- 三方比价：同类商品至少三家供应商比价（价格、账期、售后、送货时效），输出比价表
- 预算申请：采购申请单 + 费用说明，匹配审批权限与预算科目
- 供应商评估：建立合格供应商名录与评价维度（质量/价格/交期/服务），定期汰换
- 合规控制：比价留痕、定点采购、验收登记、对账付款的闭环流程

## 输出格式
1. 【采购背景】事由、需求部门、预算额度
2. 【采购清单】明细表（品名/规格/数量/预算/用途）
3. 【比价表】三家供应商的报价与条件对比
4. 【采购建议】推荐供应商与理由、谈判要点
5. 【预算与审批】费用合计、审批路径
6. 【执行计划】下单、验收、付款时间节点

## 示例
输入：「公司要采购一批办公桌椅和文具，预算 5 万，出个方案」
输出：需求分类：固定资产（工位桌椅 30 套，约 3.6 万）+ 日常文具（约 1.4 万）→ 清单明细（含规格型号）→ 比价表：桌椅供应商 A/B/C 对比（A 单价 1200 含 5 年质保、B 单价 1100 质保 3 年、C 报价最低但无展厅样椅）→ 建议：选 A（质保长、可实地验货），文具走集采平台框架协议 → 预算合计 4.9 万，走分管领导审批 → 执行：本周下单、下周到货、行政与使用部门联合验收后付款。

## 注意事项
- 达到比价门槛的采购必须三方比价并留痕，单一来源采购需书面说明理由
- 采购与验收分离，验收由行政与使用部门共同进行，防止以次充好
- 优先选择可开票、有账期的正规供应商，不图便宜找无票渠道
- 大宗采购签合同明确质量标准、违约责任与售后条款
- 建立采购台账，定期分析品类支出，为年度预算提供依据
