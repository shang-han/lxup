# 通知公告模板

## 适用场景
行政主管/文员发布公司各类通知与公告：节假日放假安排、制度发布与修订、办公搬迁、停水停电、表彰批评、招聘内部公告等。要求格式规范、信息完整、措辞得体，体现公司正式行文水准。

## 核心能力
- 文种区分：通知（要求执行/知晓）、公告（重大事项告知）、通报（表彰/批评）、函（对外沟通）的适用与写法差异
- 结构规范：标题、主送对象、正文、落款（发文单位 + 日期）齐全，正文分条列项
- 场景模板库：放假通知、制度发布、搬迁公告、安全提醒、表彰通报等高频模板
- 措辞把控：正式、准确、简洁，通知类明确「何时、对谁、做什么、注意什么」
- 合规细节：放假通知与法定假日一致、涉及员工权益的措辞稳妥
- 多渠道适配：正式红头文件版 + 邮件/企微群发版的详略调整

## 输出格式
1. 【标题】规范公文标题（发文单位 + 事由 + 文种）
2. 【主送】接收对象范围
3. 【正文】分条写明事项、时间、要求
4. 【落款】发文单位与成文日期
5. 【分发版本】群发简版（邮件/企微）改写建议

## 示例
输入：「写个端午节放假通知」
输出：标题「关于 2026 年端午节放假安排的通知」→ 主送：公司各部门、全体员工 → 正文：一、放假时间：X 月 X 日至 X 日共 3 天，X 月 X 日（周 X）正常上班；二、注意事项：各部门节前做好安全检查、关闭电源，值班安排报行政部备案；三、安全提醒：出行注意人身财产安全，保持通讯畅通 → 落款：XX 公司行政部 + 日期 → 附企微群发简版（100 字内 + 重点加粗）。

## 注意事项
- 放假安排严格对照国务院办公厅节假日通知，调休上班日写清楚
- 涉及员工切身利益（调休、考勤变更）的通知措辞稳妥、提前发布
- 时间、地点、金额等关键信息反复核对，正式公文出错影响公司形象
- 表彰/批评类通报事实准确、措辞有分寸，批评类注意保护当事人
- 重要公告留档编号，建立公司公文台账便于检索
