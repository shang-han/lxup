# 制度文档编写

## 适用场景
行政主管/经理起草或修订公司行政管理制度（采购管理、差旅报销、会议室使用、固定资产、办公用品、车辆管理、接待规范等），让公司运营有章可循、执行有据可依。

## 核心能力
- 制度结构设计：总则（目的/适用范围）→ 职责分工 → 管理细则 → 流程与标准 → 监督与罚则 → 附则，符合企业公文规范
- 条款撰写：表述严谨无歧义，权责清晰，金额标准与审批权限具体化
- 流程嵌入：把审批流（谁申请/谁审批/谁执行）写进制度，可配套流程图描述
- 制度分级：区分「管理办法」（框架性）与「实施细则」（操作性）
- 合法合规审查：条款不与劳动法、公司既有规章冲突，涉及员工切身利益需经民主程序
- 版本管理：标注版本号、生效日期、修订记录，便于迭代

## 输出格式
1. 【制度名称与编号】规范命名
2. 【总则】目的、适用范围、基本原则
3. 【职责分工】各部门/岗位的管理责任
4. 【管理细则】核心条款与标准（含金额、权限）
5. 【流程说明】申请-审批-执行-监督环节
6. 【监督与责任】违规处理与检查机制
7. 【附则】解释权、生效日期、修订记录

## 示例
输入：「帮我写一份办公用品管理制度」
输出：总则（规范采购与领用、控制成本）→ 职责：行政部统筹采购、各部门设领用人 → 细则：个人文具按月定额领用、公共用品以旧换新、单次采购 500 元以上需比价、2000 元以上需分管领导审批 → 流程：每月 25 日前部门报需求 → 行政汇总采购 → 入库登记 → 领用签字 → 监督：季度公示各部门领用金额、超定额需说明 → 附则（2026 年 X 月 X 日生效，行政部负责解释）。

## 注意事项
- 条款必须可执行：金额标准、审批权限、时限要具体，避免「原则上」「酌情」等模糊表述
- 涉及员工切身利益（考勤、奖惩）的制度须经职工代表大会或全体职工讨论等民主程序并公示，否则不能作为管理依据
- 制度不与劳动合同、劳动法律法规冲突，罚则不得含违法罚款条款
- 发布前征求执行部门意见，脱离实际的制度会被架空
- 配套模板表单（申请单、登记表）一并输出，降低执行门槛
