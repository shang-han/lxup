# 行业基础

## 行业概况
行政管理涵盖办公环境、固定资产、会议接待、差旅管理、采购后勤、制度建设、员工活动等，是公司运营的支持中枢。行政主管的核心能力是统筹协调 + 执行落地 + 成本控制 + 服务意识的综合体。岗位价值正从「事务打杂」向「组织运营效率专家」升级：数字化办公（OA、智能会议室）、行政预算精细化管理、员工体验运营成为新的能力要求。做得好的行政是老板的左膀右臂与团队的稳定器。

## 常用术语
- OA：办公自动化系统，承载审批、通知、流程线上化
- 固定资产：使用期限超过 1 年、达到价值标准的办公设备与家具，需登记盘点
- 低值易耗品：笔纸耗材等消耗性办公用品，通常一次性摊销
- 预算执行率：实际支出 / 预算金额，衡量预算编制合理性与执行控制
- 员工满意度：通过问卷调研的员工对行政服务的满意程度
- 三方比价：同类采购向至少三家供应商询价比较，采购合规的基本要求
- 会议纪要：记录会议决议与行动项的正式文件，区别于流水式会议记录
- 盘点：对固定资产与库存物资的定期清点核对，保证账实相符

## 核心工作流程
每日：办公环境巡检 → 物资领用与采购跟进 → 费用报销处理 → 文件与接待管理
定期：会议组织与纪要输出 → 员工活动策划执行 → 资产盘点 → 供应商评估与合同续签
年度：行政预算编制 → 制度修订与发布 → 供应商年度评估 → 年度总结与规划

## 常见问题
Q：办公用品浪费严重怎么办？
A：实行定额管理（部门/个人月度限额）+ 领用登记台账 + 季度公示各部门消耗 + 预算超支需说明，用透明化与制度化约束浪费。

Q：员工不遵守行政制度怎么办？
A：先检查制度是否合理并充分公示宣导，执行上做到一视同仁、违规必纠，配合正向激励；涉及切身利益的制度须经民主程序制定，执行才有底气。

Q：采购如何既省钱又合规？
A：达到门槛的采购坚持三方比价并留痕，建立合格供应商名录定期评估汰换，采购与验收分离，优先有账期可开票的正规供应商，兼顾成本与审计合规。
