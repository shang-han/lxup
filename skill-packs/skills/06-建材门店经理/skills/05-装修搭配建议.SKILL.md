# 装修搭配建议

## 适用场景
家装客户对装修风格、色彩搭配、材料选择拿不定主意时，门店经理/导购需要根据客户的户型、喜好和预算，提供瓷砖、地板、卫浴、橱柜的整体搭配建议，帮客户做决策、提升整体客单。

## 核心能力
- 风格方案：现代简约、轻奢、奶油风、原木、新中式等主流风格
- 配色逻辑：墙地、家具、软装的整体色彩搭配
- 选材组合：瓷砖/地板/卫浴/橱柜的成套推荐
- 按空间搭配：客厅、厨卫、卧室的选材要点
- 按户型调整：小户型显大、采光差提亮等技巧
- 预算适配：不同预算下的搭配取舍

## 输出格式
1. 【需求分析】户型/风格偏好/预算
2. 【风格定位】整体风格与氛围
3. 【配色方案】主色/辅色/点缀色
4. 【选材组合】各空间的材料推荐
5. 【搭配要点】空间化建议
6. 【备选方案】不同风格/预算的替换

## 示例
输入：「90 平小户型，想做奶油风，帮我搭配下选材」
输出：分析=小户型 + 奶油风 + 预算 → 风格定位：温柔治愈、显大 → 配色：奶白为主 + 原木色点缀 → 选材：客厅奶白柔光砖（显亮显大）、卧室原木色地板、厨卫浅色墙地砖、橱柜奶白门板 → 搭配要点：小户型用浅色系 + 通铺延伸感、避免深色压抑 → 备选：预算更高可换微水泥质感砖。

## 注意事项
- 搭配以客户实际户型和喜好为准，不强推不适合的方案
- 推荐结合店内现有产品与库存，确保能落地
- 客观说明不同材料的优缺点与预算差异
- 涉及承重、水电等专业装修问题提示咨询专业施工方
