# 竞品价格对比

## 适用场景
客户拿别家报价来比价，或门店经理需要定期了解同行竞品价格时，需要客观对比同类产品与价格，找到自身优势，制定应对话术，既不盲目打价格战也不丢单。

## 核心能力
- 竞品调研：同商圈/同品类竞品的产品与价格摸底
- 可比性校正：区分品牌、规格、等级、工艺的差异再比价
- 价值对比：从材质、工艺、环保、售后多维度对比
- 应对话术：客户说「别家更便宜」时的专业回应
- 差异化策略：避价格战、突出性价比与服务
- 定价参考：结合竞品调整自身报价策略

## 输出格式
1. 【竞品清单】调研对象与产品
2. 【价格对比表】同规格产品横向对比
3. 【差异分析】材质/工艺/服务差异
4. 【应对话术】客户比价时的回应
5. 【竞争策略】差异化定位建议

## 示例
输入：「客户说隔壁同规格瓷砖便宜 20 块一平，怎么应对」
输出：对比校正（确认是否同品牌同等级，警惕贴牌/次级品）→ 价值对比：我方优等品 + 更低吸水率 + 免费补货 + 铺贴指导 → 应对话术：「您说的价格我们也能做，但那是不同等级的砖。我们这款是优等品，平整度和防污更好，还包补货和铺贴指导，算下来更省心，差价其实买的是质量和售后」→ 策略：主推差异化与整体服务，不单纯拼单价。

## 注意事项
- 比价基于公开信息，客观真实，不恶意贬低同行
- 对比须同口径（同规格同等级），避免误导客户
- 应对靠价值塑造，不诋毁对手也不虚假承诺
- 遵守《反不正当竞争法》，不搞商业诋毁与虚假宣传
