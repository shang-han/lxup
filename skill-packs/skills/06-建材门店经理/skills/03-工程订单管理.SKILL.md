# 工程订单管理

## 适用场景
建材门店承接装修公司、工地、楼盘精装、酒店等工程/团购订单时，经理需要管理从签约、供货、配送、验收到回款的全流程，保证按期交付、质量达标、款项回收。

## 核心能力
- 订单台账：工程客户、合同金额、供货明细、进度记录
- 供货计划：按工地进度排供货批次与配送时间
- 进度跟踪：下单、生产、配送、到场、验收节点
- 变更管理：增减项、规格变更的记录与确认
- 回款管理：预付款、进度款、尾款的节点与催收
- 风险预警：供货延迟、质量异议、回款逾期提示

## 输出格式
1. 【订单信息】客户/合同/金额/工期
2. 【供货计划】分批次供货与配送安排
3. 【进度台账】各节点状态记录表
4. 【变更记录】增减项与确认
5. 【回款计划】各期款项节点与状态
6. 【风险提示】需关注的预警项

## 示例
输入：「接了个酒店工程，200 间房的地砖，分三批供货，帮我建个管理台账」
输出：订单信息（XX 酒店、合同额 X、工期 X）→ 供货计划：按楼层分三批（一批 80 间、二批 70 间、三批 50 间）+ 配送时间 → 进度台账表（下单/生产/配送/验收节点）→ 回款计划（预付 30%、货到 40%、验收 30%）→ 风险提示：关注首批到场时间、留足备货应对破损补货。

## 注意事项
- 工程合同条款（规格、工期、验收、违约）须明确，建议法务把关
- 供货与验收留书面/影像记录，变更须双方确认
- 严格按合同节点催收进度款，逾期及时预警
- 质保与售后条款清晰，避免交付后纠纷
