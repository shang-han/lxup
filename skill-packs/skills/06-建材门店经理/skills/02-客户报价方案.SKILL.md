# 客户报价方案

## 适用场景
建材/家居门店经理面对家装客户或工程客户，需要根据户型面积、选材档次和预算，生成清晰透明的报价方案，既让客户看得明白、感到专业，又为议价留空间、促成签单。

## 核心能力
- 按空间/品类拆解报价：客厅、厨卫、卧室分项列明
- 用量测算：根据面积估算瓷砖、地板、涂料等用量（含损耗）
- 多档方案：经济/舒适/高端三档供客户选择
- 明细化呈现：单价、数量、合计、工艺费分开列
- 让利设计：总价优惠、套餐价的合理让利空间
- 对比说明：不同档位配置的差异与性价比

## 输出格式
1. 【需求确认】户型/面积/选材/预算
2. 【用量测算】各品类用量（含损耗）
3. 【报价明细】分项价格清单
4. 【多档方案】不同档次的总价对比
5. 【优惠说明】让利方式与签单政策
6. 【备注】工艺、配送、安装与售后条款

## 示例
输入：「客户 90 平三居室，要中档瓷砖全屋，预算 2 万，出个报价」
输出：确认需求（90 平、中档、预算 2 万）→ 用量测算（铺贴面积约 75 平 + 5% 损耗）→ 报价明细（瓷砖单价 × 用量 + 加工费 + 配送）→ 提供两档：经济款 1.7 万 / 推荐款 2.05 万 → 优惠：当日签单送腰线 + 免配送 → 备注：含铺贴指导、破损补货、质保条款。

## 注意事项
- 报价真实透明，明示单价与各项费用，不玩文字游戏
- 用量测算留合理损耗，避免后期补货纠纷
- 口头承诺的赠送、安装、质保须写入报价单/合同
- 遵守《价格法》，明码标价，不搞价格欺诈
