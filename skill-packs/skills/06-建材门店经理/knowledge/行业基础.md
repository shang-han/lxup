# 行业基础

## 行业概况
建材家居行业涵盖瓷砖、地板、卫浴、橱柜、板材、涂料、五金等品类，与房地产行业高度相关，受新房交付和存量房翻新双重驱动。行业特点是单店投资大、客单价高、决策周期长、专业门槛高，客户购买低频但单笔金额大，往往需要多次到店和比较。核心竞争力是「专业知识 + 服务能力 + 渠道资源」：导购要懂产品懂装修，门店要做好设计师渠道、装修公司渠道和小区推广。利润来自产品差价与工程订单，回款周期和库存占用是主要经营压力。当前存量房翻新、整装一体化、线上线下融合是主要趋势。

## 专业技能体系

### 硬技能
- 产品知识覆盖（紧固件/水暖/电工/工具/涂料/管材）
- 规格型号速查与替代方案推荐
- 批发渠道与厂家直采
- 库存ABC分类管理
- 工程项目报价与送货

### 软技能
- 师傅/包工头关系维护
- 老客户赊账管理

## 常用术语
- 进店率：宣传曝光后客户实际进店的转化比例
- 成交率：进店客户中最终成交的比例
- 客单价：平均每单成交金额
- 复购率：老客户再次购买或增项的比例
- 转介绍率：老客户推荐新客户的比例（建材行业重要获客来源）
- 优等品/一级品：瓷砖等建材的质量等级划分
- 环保等级：板材甲醛释放等级（如 E0/E1/ENF）

## 核心工作流程
每日：客户接待 → 产品介绍 → 量房报价 → 跟进促单
每周：小区推广 → 设计师/装企渠道维护 → 竞品调研
每月：销售数据分析 → 库存盘点 → 供应商对账 → 工程订单复盘

## 常见问题
Q：客户说贵怎么办？
A：对比材质、工艺、环保、售后的差异，强调性价比和整体服务，而非单纯拼最低价。

Q：设计师渠道怎么维护？
A：定期拜访建立关系，提供产品培训与样品支持，设置合理的返点激励和带客奖励。

Q：工程订单怎么防回款风险？
A：合同明确付款节点（预付 + 进度款 + 尾款），严格按节点催收，逾期及时预警，必要时控制供货。

Q：客户决策周期长怎么跟进？
A：建立客户档案记录需求与顾虑，分阶段提供方案、案例和优惠信息，保持适度联系不骚扰，关键节点主动促成。
