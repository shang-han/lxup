# 拍摄分镜方案

## 适用场景
脚本确定后、开拍前：把文字脚本拆解为可执行的分镜表，指导摄像/编导现场的景别、运镜、构图、布光与时长安排。

## 核心能力
- 把脚本拆成逐镜头 shot list，标注景别（远/全/中/近/特写）
- 运镜设计（推/拉/摇/移/跟/手持/固定），并说明每个运镜的情绪目的
- 构图建议（三分法、中心、对称、前景遮挡、引导线）
- 器材与参数建议（手机/稳定器/麦克风、帧率、分辨率、竖屏 9:16）
- 布光与场景布置要点（三点布光、自然光朝向、反光补光）
- 时长预算管理，保证总时长符合平台推荐区间

## 输出格式
1. 【拍摄概览】场景、人员、器材清单、总时长
2. 【分镜表】镜号 / 时长 / 景别 / 运镜 / 构图 / 画面内容 / 口播
3. 【运镜要点】难点运镜的动作要领与拍法
4. 【构图与布光】各场景的构图与灯光建议
5. 【拍摄清单】现场 checklist（道具/服装/备电/备用镜头）

## 示例
输入：「把这个早餐剧情脚本拆成分镜表」
输出：12 个镜头的分镜表 → 镜 1：0-3s 榨汁杯特写固定机位三分构图… → 运镜要点：镜 5 手持跟拍保持稳定器水平、镜 8 推镜速度匀速… → 布光：窗边逆光+白板补亮人物面部 → 清单：道具清单、补拍 2 条同事反应镜头备用。

## 注意事项
- 每个运镜要有情绪/叙事目的，避免无意义晃动
- 开头镜头优先特写或动态镜头，帮助拉完播
- 预留备用镜头（反应镜头、空镜）供剪辑选择
- 手机拍摄建议 1080P/60fps 以上并锁定曝光与对焦
- 竖屏构图主体适当靠上，避开底部点赞评论区遮挡
