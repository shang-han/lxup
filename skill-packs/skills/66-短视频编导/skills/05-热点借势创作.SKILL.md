# 热点借势创作

## 适用场景
出现热搜话题、节日节点或行业热点时，快速判断能不能追、怎么追，生成与产品卖点结合的借势内容创意和脚本大纲。

## 核心能力
- 热点与账号定位的匹配度评估（相关性/参与门槛/时效窗口）
- 借势角度挖掘：热点事件 × 产品卖点 × 用户情绪
- 生成借势脚本大纲（热点钩子 → 自然过渡产品 → 情绪/价值收尾）
- 标题、话题标签与评论区运营话术建议
- 风险提示（敏感热点、负面事件、肖像与版权风险）
- 把握热点 24-72 小时生命周期窗口

## 输出格式
1. 【热点评估】匹配度判断、追不追建议、时效窗口
2. 【借势角度】2-3 个切入角度及对应用户情绪
3. 【脚本大纲】热点开头 → 产品过渡 → 结尾引导
4. 【发布建议】标题、话题标签、发布时间、评论区引导话术
5. 【风险提示】需规避的红线与事项

## 示例
输入：「city walk 话题爆了，我们户外水杯账号能蹭吗？」
输出：评估：相关性高、窗口约 3 天、建议追 → 角度：「city walk 装备清单」「通勤包里有什么」 → 大纲：开头「city walk 的人都在带这个」→ 过渡水杯便携卖点 → 结尾引导晒装备 → 发布：标签 #citywalk #通勤好物、周五中午发布 → 风险：不关联高温中暑等事故话题。

## 注意事项
- 灾难、事故、社会负面等敏感热点坚决不蹭
- 涉及明星/IP 注意肖像权与版权风险
- 借势过渡要自然，避免硬植入引发反感掉粉
- 时效类热点 24 小时内出内容，来不及就放弃
- 借势选题仍要服务账号定位，避免只涨粉不转化
