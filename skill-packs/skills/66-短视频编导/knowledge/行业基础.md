# 行业基础

## 行业概况
短视频行业进入存量竞争阶段，抖音/快手/小红书/视频号等平台以推荐算法分发，内容质量与数据表现决定流量。编导的核心工作是把客户产品卖点转化为用户愿意看完的内容，通过内容实现曝光与转化。行业从流量红利转向内容驱动，数据化迭代能力是关键分水岭。

## 专业技能体系
### 硬技能
- 选题策划与脚本写作（开头3秒钩子/反转/共鸣）
- 拍摄运镜与构图
- 剪辑软件操作（剪映/PR）
- BGM与音效选择
- 数据分析（完播率/点赞比/评论率）
### 软技能
- 客户产品理解与创意转化
- 热点敏感度

## 常用术语
- 完播率：看完视频的用户占比，影响推荐的最关键指标
- 黄金3秒：开头 3 秒决定用户是否划走
- 流量池：平台分级推荐机制，数据越好被推入越大的流量池
- 钩子：开头设置的悬念/冲突/结果，吸引用户继续看
- 卡点：剪辑切换与音乐节拍对齐
- 种草：通过内容推荐激发用户购买欲
- 带货：通过短视频/直播实现商品销售
- DOU+：抖音的付费流量投放工具

## 核心工作流程
前期策划：接客户/产品需求 → 选题策划 → 脚本写作 → 脚本内审
拍摄准备：分镜拆解 → 场地道具准备 → 器材检查
现场拍摄：现场调度 → 按分镜拍摄 → 备用镜头补拍
后期制作：粗剪 → 精剪 → 配乐音效 → 字幕花字 → 封面标题
发布复盘：择时发布 → 数据监测 → 48 小时复盘 → 迭代下期

## 常见问题
Q：完播率低怎么办？
A：先看开头 3 秒跳出率：开头跳出高就换更强的钩子（结果前置/悬念/冲突）；中段流失就加快节奏，删气口、加反转或信息密度。

Q：有播放没点赞怎么办？
A：点赞反映情绪共鸣与价值认同：加强价值输出与情绪点，结尾加引导话术；选题阶段优先选有情绪或争议的话题。

Q：客户产品怎么转化才不生硬？
A：不做广告片：把产品放进用户场景（痛点场景 → 产品解决 → 效果对比），说用户语言，一条视频只打一个卖点。

Q：热点要不要条条追？
A：看匹配度与风险：与账号定位相关且不敏感的热点尽快追（24 小时内）；敏感、负面及与定位无关的热点不追。
