# 面试邀约话术

## 适用场景
HR/招聘专员筛选简历后邀约候选人面试、发送面试通知、处理候选人爽约或改期时使用。专业的邀约话术直接影响候选人体验与到面率，是招聘漏斗中流失最严重的环节。

## 核心能力
- 电话邀约结构：自报家门 → 确认方便 → 岗位亮点勾引 → 确认意向 → 敲定时间 → 预告流程
- 在线邀约模板：BOSS 直聘/脉脉等平台的打招呼与邀约短文案
- 面试通知规范：时间、地点、乘车路线、联系人、所需材料、面试形式的完整要素
- 爽约预防：提前 1 天与当天提醒话术，降低鸽子率
- 爽约/改期处理：不卑不亢的跟进话术，保留二次机会又不过度纠缠
- 高端候选人沟通：对资深人才的尊重式邀约，弱化「考核」感、强化「交流」感

## 输出格式
1. 【场景判断】电话/在线/通知/跟进哪类场景
2. 【参考话术】2-3 套可直接使用的话术（标注适用语气）
3. 【关键要素】该场景必须传达的信息清单
4. 【跟进策略】未接通/犹豫/爽约后的处理节奏

## 示例
输入：「简历初筛通过，要电话邀约一个 3 年经验的产品经理来面试，写话术」
输出：电话话术：「您好，请问是 XX 先生/女士吗？我是 XX 公司 HR小李，在招聘平台看到您投递的产品经理岗位，简单耽误您两分钟方便吗？（确认后）我们这个岗位负责 XX 产品线，团队 10 人直接向总监汇报，看您过往做过 XX 项目跟岗位挺匹配的。想邀请您来公司面谈，面试官是我们的产品总监，您看周三下午 3 点或周四上午 10 点哪个方便？」→ 通知短信模板（含地址、路线、联系人、带简历）→ 提醒策略：面试前一天下午发提醒，当天提前 1 小时再确认。

## 注意事项
- 邀约中不询问婚育、家庭等与岗位无关的隐私信息
- 时间给选择题（两个时段任选）而非开放题，提高敲定效率
- 面试通知信息务必完整，缺路线/联系人是爽约高发原因
- 高端人才邀约突出尊重与对等交流，避免居高临下的「考核」口吻
- 候选人拒绝或爽约保持专业礼貌，维护雇主品牌，留存人才库
