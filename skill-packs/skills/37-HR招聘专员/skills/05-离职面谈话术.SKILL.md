# 离职面谈话术

## 适用场景
HR/招聘专员（及 HRBP）对提出离职的员工进行面谈、对核心员工做挽留沟通、通过离职访谈诊断组织问题时使用。高质量离职面谈既能挽留关键人才，又能挖出真实的组织管理问题。

## 核心能力
- 面谈提纲设计：按离职类型（主动/被动、核心/普通）定制问题清单
- 真实原因挖掘：穿透「个人发展」等官方说辞，用追问定位薪酬/上级/氛围/压力等真因
- 挽留话术：针对可挽留的核心员工，按真因给出对策（调薪/调岗/换组/承诺改进），并评估挽留成本与可行性
- 情绪管理：对委屈型、冲突型离职员工的倾听与共情，好聚好散维护雇主品牌
- 交接引导：工作交接、保密与竞业义务提醒、离职手续的规范话术
- 离职数据分析：将访谈结论结构化汇总，输出组织改进建议

## 输出格式
1. 【面谈准备】对象背景、面谈目标、环境安排
2. 【提问提纲】开场 → 原因探究 → 组织评价 → 挽留探询 → 交接提醒
3. 【挽留话术】针对不同真因的挽留方案与表达
4. 【注意事项】本次面谈的敏感点与禁忌
5. 【记录模板】结构化记录与分析框架

## 示例
输入：「一个 3 年的核心后端工程师提离职，说是个人原因，怎么谈」
输出：提纲：开场（肯定贡献、说明保密）→ 真因探究（「除了个人原因，工作上有没有让你不太舒服的地方？比如团队、上级、成长、薪资，都可以直说，今天聊的只用于改进」）→ 组织评价（对直属上级、团队氛围、晋升机制打分）→ 挽留探询（若真因是薪资：对标市场调薪 + 项目奖金；若是上级：探讨换组可能）→ 交接（核心项目交接清单、竞业协议提醒）→ 提醒：不施压不画饼，挽留方案需当场能给授权范围内的答复，谈不拢也保持尊重。

## 注意事项
- 面谈内容严格保密，员工敢说真话的前提是信任，泄露会让后续访谈失效
- 遵守《个人信息保护法》：离职访谈记录属个人信息，限 HR 权限内使用，不外传
- 不贬低员工、不追问去向细节用于竞争目的，好聚好散维护口碑
- 挽留要实事求是，无法兑现的承诺（空头升职加薪）只会二次伤害信任
- 离职数据定期汇总分析，高频真因（如某部门流失集中）要升级到组织改进
