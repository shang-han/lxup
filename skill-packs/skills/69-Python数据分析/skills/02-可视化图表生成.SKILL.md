# 可视化图表生成

## 适用场景
分析结果需要可视化呈现：根据数据特点和表达目标推荐合适的图表类型，生成绘图代码（Matplotlib/Plotly）或 Tableau 制作建议，并附标题、标注与解读要点。

## 核心能力
- 图表类型选择（趋势折线、构成堆叠柱/饼、对比条形、分布直方、相关散点、漏斗图）
- Matplotlib/Plotly 绘图代码（中文字体、样式、双轴、标注）
- 图表信息设计（标题结论化、关键数据标注、减少图表垃圾）
- 看板布局建议（核心指标、维度拆分、下钻层级）
- Tableau 制作建议（维度与度量、筛选器与参数操作）
- 为图表生成一句话解读

## 输出格式
1. 【图表选择】图表类型与选择理由
2. 【绘图代码】完整可运行代码
3. 【样式要点】中文字体、配色、标注位置
4. 【解读要点】标题结论与 2-3 个阅读重点
5. 【替代方案】其他可选图表类型及适用情形

## 示例
输入：「展示近 3 个月各渠道转化率趋势，数据是 date/channel/cvr 三列的 DataFrame」
输出：选择：多系列折线图（趋势对比）→ 代码：Plotly px.line，color=channel，hover 显示数值，大促节点加竖线标注 → 样式：统一色板、Y 轴百分比格式 → 解读：标题「渠道 A 策略调整后 CVR 由 2.1% 稳步升至 3.4%」→ 替代：渠道少时可用小多图分面。

## 注意事项
- 标题写结论，不只写指标名
- 饼图只适合类别少（≤5）的构成，否则用条形图
- Y 轴截断慎用，避免夸大差异
- 颜色少而有意义，突出关键系列
- 图表配一句话解读，降低阅读门槛
