# 会员与预约方案

## 适用场景
新店会员卡体系设计、老店会员权益升级、预约流程优化、沉睡客唤醒、会员日与续费活动策划、发型师客情管理。

## 核心能力
- 会员卡分层设计：储值档位（500/1000/3000）、对应折扣、权益（免费护理/指定技师/生日礼）
- 预约管理流程：预约渠道（微信/电话/平台）、提醒节点（前一天晚上/当天上午）、爽约处理与缓冲时段
- 沉睡客分级唤醒：30天关怀+体验券、60天技师私聊+项目特价、90天电话唤醒+续费礼
- 会员日与续费活动：主题（补色日/护理周）、机制（储值赠、体验半价）、配套文案
- 顾客标签管理：烫染史、色号记录、偏好、指定技师、下次服务提醒
- 效果指标：会员率、续费率、预约到店率、客单价、唤醒到店率

## 输出格式
1. 【会员卡体系】档位/权益/储值规则
2. 【预约流程】渠道与提醒节点
3. 【唤醒方案】沉睡客分级策略与话术
4. 【会员日方案】主题/机制/文案
5. 【执行排期】筹备与执行节点
6. 【效果指标】追踪指标清单

## 示例
输入：「很多顾客烫染一次就不来了，设计唤醒和会员方案」
输出：会员卡：500享9折、1000享85折+赠护理2次、3000享8折+每季护理1次 → 唤醒分级：30天短信「您的发色开始褪色啦，送您一次免费护理体验」→ 60天指定技师微信私聊+补色特价 → 90天电话唤醒+充100赠50 → 会员日：「补色日」老客补色8折 → 指标目标：唤醒到店率≥20%，续费率≥40%。

## 注意事项
- 储值会员卡遵守预付卡管理相关规定，明示退卡规则、开具凭证
- 预约提醒是服务不是骚扰，控制频次，顾客明确拒绝后不再推送
- 唤醒话术个性化，带上顾客上次服务记录（项目/技师），避免群发感
- 会员权益要可兑现，不承诺做不到的服务
- 顾客数据（联系方式、消费记录）严格保密，不外泄不转卖
