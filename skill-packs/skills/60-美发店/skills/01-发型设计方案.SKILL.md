# 发型设计方案

## 适用场景
顾客说「帮我设计个发型」「剪短点你看着来」、发型师需要脸型/发质分析与烫染提案、顾客拿参考图需要预期对齐。

## 核心能力
- 脸型与发型匹配：圆/方/长/椭圆/心形脸与长度、刘海、层次、蓬松位置的关系
- 发质发量考量：细软（增蓬松）、粗硬（去量打薄）、稀疏（增量与发际线处理）、自然卷（顺直或利用卷度）
- 生活场景匹配：打理时间、烫染频率、职业形象需求
- 潮流融合：法式烫、狼尾、气垫烫、法式刘海、公主切等流行元素的应用场景
- 维护成本说明：日常打理方法、修剪/补色周期、后续护理项目
- 沟通提案：参考图对齐方法、效果预期管理、风险前置告知（发质限制、漂染损伤）

## 输出格式
1. 【特征分析】脸型/发质/发量/偏好
2. 【推荐发型】2-3个方案，各附理由
3. 【烫染建议】工艺方向与要点
4. 【日常打理】打理方法与时间成本
5. 【维护周期】修剪与补色频率
6. 【沟通提案】预期确认与风险告知话术

## 示例
输入：「圆脸，细软发，想显瘦好打理」
输出：分析：圆脸需纵向拉伸，细软发缺蓬松 → 方案A：锁骨发层次剪+法式刘海（两侧收紧拉长脸型）；方案B：公主切+发根气垫烫 → 工艺：发根烫增蓬松，可选冷棕色提亮肤色 → 打理：低头倒吹发根，5分钟 → 周期：6-8周修剪一次 → 沟通：拿2-3张同脸型参考图与顾客确认，提前说明细软发蓬松度的维持需要打理配合。

## 注意事项
- 建议基于顾客描述与通用规律，须结合到店面诊实际情况调整
- 发质限制要提前说明，不承诺「和图片一模一样」
- 考虑顾客打理时间与预算，不只推高难度高客单项目
- 有漂染史的头发先评估损伤再设计二次烫染方案
- 服务后交代打理方法与周期，减少「回家就不好看」的纠纷
