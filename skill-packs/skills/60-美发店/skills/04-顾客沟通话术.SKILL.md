# 顾客沟通话术

## 适用场景
新客接待需求挖掘、发型方案沟通、顾客嫌贵、项目升级（剪→烫染护）、办卡推荐时机、效果不满意投诉处理。

## 核心能力
- 服务前需求挖掘：问询清单（场合、打理习惯、过往烫染史、预算、参考图）
- 预期对齐：参考图确认、发质可行性说明、预期差前置告知
- 项目自然升级：从剪发自然带出烫染护（给理由与场景，不强推）
- 价格异议：报价透明化、价值拆解（技师等级/产品档次/效果维持时长）
- 办卡时机与话术：服务满意后、算账法（单次价 vs 会员价）、权益一次讲清
- 投诉处理：效果不满意先共情再补救（免费修复/改色/护理），不推责给顾客发质

## 输出格式
1. 【场景定义】顾客类型与具体问题
2. 【推荐话术】口语化，2-3个版本
3. 【配套动作】检测、参考图确认、算账演示
4. 【禁忌话术】容易招反感的表达
5. 【合规提醒】不过度承诺、不强推、不虚假宣传

## 示例
输入：「顾客只想剪38元的头发，怎么自然推荐烫发」
输出：场景：价格敏感新客 → 话术：剪发时聊「您早上打理头发费劲吗？您这个发质发根扁塌，做个发根烫起床就能出门，店里可以先免费做个头皮检测看看」（给理由+低门槛体验）→ 配套动作：剪发时演示蓬松效果对比，赠护理小样 → 禁忌：「光剪没用」「保证满意」「今天办卡最便宜」→ 合规：顾客明确拒绝即止，留好印象下次转化，不承诺效果。

## 注意事项
- 办卡与项目升级基于顾客真实需求，不为每个顾客设强制推销指标
- 服务前报价明确（含可能的加项），避免结账争议
- 不过度承诺效果（「保证不伤发」「永不褪色」不可说）
- 投诉先解决问题，合理范围免费补救，口碑比一单利润重要
- 顾客隐私（联系方式、照片）须保护，发型照片做宣传前须征得同意
