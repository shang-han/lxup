# 行业基础

## 行业概况
酒吧/清吧属于夜间经济与社交消费业态，清吧（安静、重酒水与氛围）区别于闹吧/夜店，客单价高、毛利高（酒水毛利常超 70%），但依赖氛围、地段与客情。行业强监管：营业时间、噪音、治安、消防、未成年人保护。核心竞争力：酒水产品力（调酒）+ 氛围营造 + 主题活动引流 + 高端客户经营。复购靠会员存酒与私域客情。

## 专业技能体系
### 硬技能
- 酒水采购与库存管理（洋酒/啤酒/调酒原料）
- 鸡尾酒配方研发
- 音响灯光设备操作
- 营业时间与噪音合规
- 主题活动策划（驻唱/派对/品酒会）

### 软技能
- 氛围营造
- 高端客户社交

## 常用术语
- 客单价/桌均：每人/每桌平均消费，靠酒水套餐与套餐组合拉升
- 存酒：客户购买整瓶酒后寄存店内，分次饮用，锁定复购
- 开瓶费/服务费：部分场所对自带或存酒收取的服务费用
- Mocktail：无酒精鸡尾酒，扩充不饮酒客群
- 出酒标准：每杯酒标准 ml 数，用盎司器量化控成本
- 翻台率：座位周转，清吧通常偏低、重停留与复购
- 进销存：酒水的进货、销售、库存一体化台账
- 私域：微信沉淀的自有客户池，用于邀约与复购

## 核心工作流程
每日：营业前备酒备料 → 设备（音响灯光）调试 → 营业（暖场/高峰/深夜）→ 打烊盘点高价酒与安全检查
每周：酒水盘点与采购 → 活动策划与预热 → 客户回访邀约 → 库存滞销分析
每月：销售与毛利核算 → 活动复盘 → 会员经营盘点 → 合规与证照检查

## 常见问题
Q：平日晚市没人气怎么引流？
A：做主题之夜（民谣驻唱、女士之夜、品酒会），配专属酒水套餐和低门槛引流，小红书/社群预热，用固定活动培养客户到店习惯。

Q：酒水库存乱、损耗大怎么管？
A：建进销存台账，出酒用盎司器标准化每杯 ml，高价洋酒日盘、全品周盘，差异追责，滞销款做特调促销消化，存酒单独登记。

Q：怎么维护高端客户提复购？
A：记住偏好与称呼，存酒做仪式感与专属酒柜，活动专属邀约，节日个性化问候，客诉给足面子超预期补偿，靠真诚分寸感经营。

Q：经营要注意哪些合规？
A：营业时间与音量遵守噪音与治安规定，不向未成年人售酒、提示适量饮酒，消防通道畅通，采购正规渠道索证索票杜绝假酒，证照齐全。
