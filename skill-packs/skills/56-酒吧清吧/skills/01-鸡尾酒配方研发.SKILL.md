# 鸡尾酒配方研发

## 适用场景
研发招牌特调与季节限定；设计酒单结构（经典款+招牌款+入门款）；标准化配方保证多调酒师出品一致；核算每杯成本与毛利；为不喝酒的客人设计无酒精款。

## 核心能力
- 给出鸡尾酒配方（基酒、辅料、配比、调酒技法、杯型、装饰）
- 按风味谱（酸甜/烈度/果香）设计产品矩阵
- 标准化配方到 ml 与操作 SOP（摇和/调和/搅和）
- 核算单杯成本与毛利，优化高价基酒使用
- 设计无酒精特调（Mocktail）扩充客群
- 标注出品标准（色泽、分层、装饰）与成本红线

## 输出格式
1. 【酒单结构】经典/招牌/入门/无酒精分类
2. 【配方表】基酒/辅料/配比/技法/杯型
3. 【调制SOP】步骤与出品标准
4. 【成本与定价】单杯成本、毛利
5. 【风味矩阵】覆盖不同口味偏好
6. 【新品建议】季节限定方向

## 示例
输入：「清吧想推几款招牌特调，帮我设计配方和定价」
输出：酒单结构（经典 6 款打底 + 招牌特调 3 款 + 无酒精 2 款）→ 招牌配方「荔枝玫瑰金汤力」（金酒 45ml、荔枝利口酒 15ml、汤力水补满、玫瑰花瓣点缀）→ SOP（摇和后滤入柯林杯、分层摆花瓣）→ 成本 8 元 / 定价 48 元 / 毛利 83% → 风味矩阵（偏甜/偏烈/果香各一款）→ 夏季推杨梅/荔枝限定。

## 注意事项
- 严守食品安全，调酒原料与辅料在保质期内、规范存储
- 不向未成年人售酒，菜单提示适量饮酒、提供无酒精选择
- 配方中的装饰食材须可食用且卫生
- 门店须按许可范围经营，相关证照齐全
