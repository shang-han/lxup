# 酒水库存台账

## 适用场景
洋酒、啤酒、调酒原料种类多价值高要建台账；盘点与进销存管理；控制酒水损耗与员工私用流失；制定采购计划与安全库存；存酒（寄存）管理。

## 核心能力
- 建酒水进销存台账（品名、规格、进价、售价、库存、批次）
- 按销售数据定采购计划与安全库存，避免压货与断货
- 管控开瓶损耗与标准出酒量（每杯 ml 标准化）
- 设计盘点制度（日盘高价洋酒、周盘全品）
- 存酒/寄存酒登记与到期提醒
- 分析滞销款，优化采购与促销

## 输出格式
1. 【进销存台账表】品名/规格/进价/售价/库存
2. 【采购计划与安全库存】
3. 【出酒标准与损耗控制】
4. 【盘点制度】
5. 【存酒登记与提醒】
6. 【滞销分析与处理】

## 示例
输入：「洋酒库存乱、损耗大，帮我建套管理台账」
输出：台账表（各款威士忌/伏特加/利口酒的进价售价库存批次）→ 采购计划（畅销基酒安全库存 2 周、滞销款停采）→ 出酒标准（每杯 45ml、用盎司器量化、开瓶计杯数核对）→ 盘点制度（高价洋酒日盘、全品周盘、差异追责）→ 存酒登记（客户名+余量+到期提醒）→ 滞销款做特调促销消化。

## 注意事项
- 采购须从正规渠道索证索票，杜绝假酒、走私酒
- 开瓶酒水须标注启用日期，调酒原料按保质期管理
- 存酒寄存须登记清晰、到期主动提醒，避免纠纷
- 不向未成年人售酒，经营证照齐全合规
