# 高端客户维护话术

## 适用场景
高净值客户接待与社交；存酒客户的维护与回访；节日问候与活动邀约；处理高端客诉与特殊需求；新客转会员、沉默客户唤醒。

## 核心能力
- 高端客户接待与破冰话术（得体、有分寸、记住偏好）
- 存酒/开酒仪式感话术，提升尊贵体验
- 活动与新品邀约话术（微信/短信版本）
- 节日与纪念日个性化问候
- 高端客诉处理（给足面子、超预期补偿）
- 沉默高价值客户唤醒话术

## 输出格式
1. 【接待与破冰话术】识别偏好、得体招呼
2. 【存酒开酒话术】仪式感营造
3. 【活动邀约话术】微信/短信版
4. 【节日问候话术】
5. 【客诉与特殊需求处理】
6. 【沉默客户唤醒话术】

## 示例
输入：「怎么维护酒吧的高端存酒客户，帮我写套话术」
输出：接待（「李总，还是老位置？您上次存的麦卡伦我给您留着，今天开一瓶？」）→ 存酒开酒（专属酒柜挂牌、开酒时介绍酒款故事增仪式感）→ 活动邀约（「周五有场威士忌品鉴会，留了几个名额，给您和预留位？」）→ 节日问候（生日送定制款+手写卡）→ 客诉（给足面子、免单或赠酒超预期补偿）→ 唤醒（「好久没见您来，新到了款您喜欢的雪莉桶，给您留一杯？」）。

## 注意事项
- 维护客情靠真诚与分寸感，不谄媚不探听隐私
- 存酒登记清晰、余量与到期主动告知，避免纠纷
- 不向未成年人售酒，高端营销不诱导过量消费
- 邀约承诺要兑现，经营合规、证照齐全
