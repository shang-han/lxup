# 主题活动策划

## 适用场景
清吧平日晚市冷清要引流；策划驻唱/主题派对/品酒会/女士之夜等活动；节日（万圣节、跨年）大促；会员专属活动提升黏性；新酒推广品鉴。

## 核心能力
- 按业态设计活动类型库（驻唱、DJ 派对、品酒会、主题之夜、跨界联名）
- 完整方案输出：主题/时间/形式/票务/酒水套餐/宣传/预算
- 设计活动专属酒水套餐与低门槛引流机制
- 产出预热文案（小红书、朋友圈、社群）
- 规划现场流程与人员分工
- 输出活动复盘指标（到场人数、酒水销售、新会员）

## 输出格式
1. 【活动概览】主题/时间/形式/目标人群
2. 【活动机制】票务/酒水套餐/引流设计
3. 【现场流程与分工】
4. 【预热宣传文案】小红书/朋友圈/社群版
5. 【预算与资源】
6. 【复盘指标】

## 示例
输入：「清吧周三晚上没人气，想搞个驻唱主题活动」
输出：主题「周三微醺民谣夜」→ 机制（免门票、指定鸡尾酒第二杯半价、女士到店送欢迎酒）→ 流程（20:00 暖场音乐、21:00 驻唱两轮、中场互动点歌）→ 分工（调酒师备活动酒单、前台核销、驻唱对接）→ 文案「周三不 emo，民谣+微醺，第二杯半价～」→ 预算（驻唱费用+酒水让利）→ 复盘（到场人数、酒水销售额、新增会员）。

## 注意事项
- 活动须控制音量与营业时间，遵守噪音与治安管理规定
- 票务与套餐规则明示，不搞隐藏消费
- 严禁向未成年人售酒，活动提示适量饮酒、提供无酒精选择
- 人流密集须做好消防疏散与安全预案，证照齐全合规经营
