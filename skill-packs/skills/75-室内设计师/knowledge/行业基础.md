# 行业基础

## 行业概况
室内设计师为住宅与商业空间提供整体设计服务，覆盖空间规划、硬装设计、施工图深化、主材选型与软装搭配。行业从「画图 + 施工分包」走向「全案设计 + 整体交付」，客户需求从「装得上」升级为「好看、好用、预算可控」。整装、全屋定制、精装房微改与老房翻新是当前三大增量市场。设计师的核心价值 = 空间问题解决力 + 审美表达 + 预算与落地控制力，而非单纯出图。

## 专业技能体系
### 硬技能
- 空间规划与功能布局（动线/采光/收纳）
- 设计软件（CAD/3dsMax/SketchUp/Enscape）
- 材料选型与工艺了解（地板/墙面/吊顶/柜体）
- 预算编制与施工图深化
- 软装搭配与色彩方案
### 软技能
- 客户生活方式挖掘
- 设计方案讲标能力

## 常用术语
- 动线：人在室内的行动路线（居住动线/家务动线/访客动线），好布局要求短而少交叉
- 硬装/软装：硬装指地墙顶、水电等固定装修；软装指家具、窗帘、灯具、挂画等可移动陈设
- 全案设计：硬装 + 主材 + 软装一体化设计，强调整体效果落地还原
- 情绪板（Mood Board）：将材质、色彩、单品图像拼合呈现设计调性的提案工具
- ENF/E0：人造板甲醛释放等级，ENF 为现行国标最高等级
- 隐蔽工程：水电、防水等完工后被遮蔽的工程，返修代价高，须重点验收留档
- 价值工程：在保功能与效果前提下优化成本的方法，预算控制的核心手段
- 施工交底：设计师向施工方讲解设计意图、节点做法与验收标准的环节

## 核心工作流程
需求阶段：客户沟通 → 生活方式挖掘 → 现场量房 → 户型诊断
方案阶段：平面布局 → 效果方案（3D/全景）→ 材料选型与预算 → 方案讲标与签单
实施阶段：施工图深化 → 图纸会审 → 施工交底 → 工地跟踪与节点验收 → 变更处理
交付阶段：硬装验收 → 软装摆场 → 全案拍摄与复盘

## 常见问题
Q：客户说「我也不知道喜欢什么风格」，怎么沟通？
A：别问风格问生活：几口人、日常作息、兴趣爱好、 disliked 的元素，让客户发「别人家哪个角落喜欢」的参考图。再用情绪板把模糊感觉翻译成具体的色彩、材质与造型供确认。

Q：效果与预算冲突怎么办？
A：做价值工程：保住效果焦点（客厅主视觉、主灯、关键家具），简化次要界面与装饰性项目，给出 2-3 档减配方案让客户做选择题，绝不为了签单口头承诺再偷偷降配。

Q：为什么施工图到现场总出问题？
A：常见三类原因：没现场复尺、节点没考虑材料厚度与工艺、专业间没碰撞检查。对策：出图前现场复核关键尺寸，组织图纸会审，隐蔽工程拍照留档，变更一律走书面变更单。

Q：如何平衡客户的个性化要求与落地性？
A：先肯定需求再谈落地：把个性化想法的施工难度、成本与后期维护代价讲清楚，同时给「效果接近但更易落地」的替代方案，让客户在充分知情后选择，而不是直接否定。
