# 软装色彩搭配

## 适用场景
全屋色彩方案与软装搭配设计；情绪板（mood board）制作逻辑；家具/窗帘/灯具/挂画的选品清单与预算控制；软装与硬装的衔接协调。

## 核心能力
- 色彩方案：主色/配色/点缀色 631 比例法则，按风格出配色盘
- 情绪板制作：材质、色彩、单品图像的组织与提案逻辑
- 家具选品：尺寸尺度匹配、风格统一、功能优先
- 面料与材质：窗帘/沙发/地毯的材质特性、耐用度与打理成本
- 灯光氛围：基础照明 + 重点照明 + 装饰照明组合，色温选择（2700-4000K）
- 软装预算清单与平替方案

## 输出格式
1. 【方案定位】风格 + 居住人群 + 预算
2. 【色彩方案】主/配/点缀色比例与分布位置
3. 【分空间软装要点】客厅/卧室/餐厅关键单品
4. 【选品清单】单品/规格/材质/预算区间
5. 【落地建议】采购顺序与摆场要点

## 示例
输入：「奶油风客厅，软装预算 3 万，给个搭配方案」
输出：色彩方案（奶油白 60% 墙面 + 大面积布艺；奶咖 30% 沙发 + 地毯；焦糖/胡桃木 10% 点缀）→ 关键单品（直排布艺沙发 2.8m、短绒圆地毯、落地灯、装饰画 2 幅）→ 选品清单含价格区间（沙发 8-12k、地毯 1-2k、灯具 2-3k）→ 灯光（3000-3500K 暖光为主，射灯洗墙突出挂画）→ 平替方案（超预算时沙发改科技布、茶几改板材 + 实木腿）→ 落地顺序（先定沙发与地毯尺寸再采购小件）。

## 注意事项
- 先定整体色盘再采购，避免单品各自好看、组合起来风格打架
- 家具尺寸待硬装完成后现场复尺再下单，不能只看平面图
- 面料色卡在不同光线下色差明显，关键单品要到现场光环境看实物小样
- 有儿童/宠物的家庭优先耐用易洁材质（科技布、短绒、可拆洗）
