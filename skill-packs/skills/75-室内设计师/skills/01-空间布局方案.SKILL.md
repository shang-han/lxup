# 空间布局方案

## 适用场景
拿到户型图后做平面布局与改造建议；动线、采光、收纳的专项优化；异形户型（狭长/暗厅/多拐角）的布局对策；同一户型出 2-3 个布局方案供客户选择。

## 核心能力
- 户型诊断：结构约束识别（承重墙/剪力墙/管井/梁）、优缺点分析
- 功能布局：按家庭结构与生活方式规划空间，多功能复合设计
- 动线设计：居住动线/家务动线/访客动线的分离与缩短
- 收纳规划：收纳量估算（人均立方数）、玄关/卧室/厨房/阳台柜体布局
- 采光通风改善：暗厨暗卫暗卧的补光、借光与通风方案
- 多方案对比：每案优劣与适用家庭说明

## 输出格式
1. 【户型诊断】优点 + 缺陷 + 结构约束
2. 【需求确认清单】人口、习惯、重点诉求、预算
3. 【布局方案】方案 A/B 对比，各列优劣
4. 【动线与收纳要点】关键改动说明
5. 【待现场核实项】图纸无法确认的事项

## 示例
输入：「89㎡ 三房，三代同堂 5 口人，收纳不够，怎么布局？」
输出：户型诊断（玄关无落尘区、主卧衣柜进深不足、次卧面积小）→ 布局方案（A：次卧改榻榻米 + 一体化衣柜作多功能房，客厅增设整墙收纳柜；B：压缩客厅开间换独立储物间）→ 动线优化（家务动线阳台-厨房缩短）→ 收纳估算（5 口之家建议 ≥12m³，按玄关 1.5/主卧 4/次卧 3/餐厨 3.5 分配）→ 待核实（玄关旁墙体是否承重、管井位置）。

## 注意事项
- 承重墙、剪力墙、燃气与排污管道不可拆改，是安全与法规红线
- 布局从生活方式出发，先问清居住者日常再谈风格
- 家具按实际尺寸摆放校验，避免「图上看着大、现场放不下」
- 老人/儿童家庭优先安全设计：防滑、圆角、夜灯、窗台防护
