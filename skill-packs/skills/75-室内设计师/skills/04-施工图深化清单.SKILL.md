# 施工图深化清单

## 适用场景
将效果方案深化为可施工的图纸体系；出图前的图纸完整性自检与碰撞检查；与施工方做设计交底；隐蔽工程验收要点梳理。

## 核心能力
- 图纸体系清单：平面系统（原始/拆砌/平面/地坪/天花）、立面系统、节点大样、水电点位图、材料做法表
- 常见节点大样：吊顶转角与灯槽、瓷砖收口、柜体与墙体收口、地板过门石、窗帘盒
- 水电点位规划：开关/插座/弱电/水位的定位与高度标准、回路分组
- 材料-工艺标注：不同材料的做法要求与附加费用提示
- 图纸会审清单：碰撞检查（梁与吊顶、管道与柜体）、尺寸闭合、图面一致
- 施工交底与验收：隐蔽工程验收标准与影像留档

## 输出格式
1. 【图纸清单】应出图纸与完成状态
2. 【重点节点大样】本案特殊处理的节点说明
3. 【水电点位表】分房间定位与回路说明
4. 【会审问题清单】发现的问题与处理方式
5. 【交底与验收要点】隐蔽工程验收标准

## 示例
输入：「吊顶想做无主灯设计的极简效果，怎么深化？」
输出：节点要点（线性灯带预留灯槽尺寸与检修口、磁吸轨道灯预埋件、石膏板转角 L 型整板防开裂、窗帘盒预留电动帘电源）→ 水电点位（无主灯回路分组：基础照明/重点照明/氛围照明分控，预留调光零火线）→ 会审提醒（核对梁底标高与吊顶下吊幅度的冲突、空调内机与风口位置、投影幕布槽尺寸）→ 验收要点（灯槽平直度、嵌入式灯具开孔定位先放样再开孔）。

## 注意事项
- 关键尺寸必须现场复尺确认：图纸与毛坯实测常有误差，柜体与嵌入式设备尤甚
- 隐蔽工程（水电/防水）验收标准事前书面约定并拍照留档
- 变更必须有书面变更单与费用确认，避免口头指令施工
- 涉及结构、燃气、消防的改动须符合规范与物业报备要求
