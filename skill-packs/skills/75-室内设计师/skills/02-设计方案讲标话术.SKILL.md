# 设计方案讲标话术

## 适用场景
向客户汇报平面与效果方案前的讲标逻辑与话术准备；设计理念的表达；预算异议、风格分歧、「再考虑考虑」等常见异议的应对；讲标后的跟进节奏。

## 核心能力
- 生活方式挖掘提问清单：家庭结构、作息习惯、兴趣爱好、居住痛点、预算区间
- 讲标结构：痛点呼应 → 设计理念 → 逐空间方案 → 材料与预算 → 计划
- 场景化表达：把设计点连到用户日常（「早上送孩子上学前，玄关这里可以……」）
- 风格词汇翻译：现代简约/轻奢/奶油/侘寂/新中式与客户口语的双向转译
- 异议应对：太贵了/要再考虑/家人意见不一/想改方案
- 信任构建：案例佐证、流程透明化、承诺边界的表达

## 输出格式
1. 【客户需求回顾】关键需求点呼应
2. 【讲标主线】3 分钟开场 + 分段脚本
3. 【方案讲解要点】逐空间亮点与场景化文案
4. 【异议应对】常见问题与应答话术
5. 【收尾与下一步】推进动作与时间约定

## 示例
输入：「客户是年轻夫妻，喜欢轻奢风但预算紧张，方案怎么讲？」
输出：开场呼应（两人通勤忙、需要充足收纳、想要质感但不想超支）→ 理念「轻硬装重软装」：用色彩与材质体现轻奢，砍掉复杂造型与背景墙 → 逐空间脚本（客厅：奶咖主色 + 金属线条灯具点睛；主卧：衣柜内部按两人衣物量分区）→ 预算分配建议（主材分级配置，焦点面用好的）→ 异议应对（「太贵了」给三档减配方案让客户选，不裸降价）→ 收尾（约定复尺时间，3 天内出软装清单）。

## 注意事项
- 讲标是回答客户的问题，不是展示设计师的审美：先呼应需求再谈方案
- 不过度承诺效果与预算，「保证效果」「绝不超支」类表述有法律风险
- 口头承诺要落纸面（合同或附件），避免交付争议
- 谈价格讲分项依据（哪里值这个钱），而不是靠打折促单
