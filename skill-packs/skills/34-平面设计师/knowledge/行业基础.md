# 行业基础

## 行业概况
设计行业竞争激烈，初级设计师供给过剩、资深设计师稀缺，AI 设计工具正在重塑基础执行类工作的价值分布。服务类型涵盖品牌设计、电商设计、UI 设计、印刷/包装设计等。设计师的核心竞争力从「软件熟练」转向「审美判断 + 沟通表达 + 商业理解 + 交付效率」的综合能力，能讲清楚设计价值的设计师议价能力显著更强。

## 常用术语
- VI：视觉识别系统（Visual Identity），含 LOGO、标准色、标准字及应用规范
- UI：用户界面设计（User Interface），App/网页等数字产品的界面设计
- DPI：打印分辨率，印刷通常要求 300DPI，屏幕显示 72-150PPI
- RGB/CMYK：屏幕显示色域 / 印刷色域，印刷文件须转 CMYK 以防色差
- 栅格系统：版式设计的分栏分格框架，保证排版秩序感
- 出血：印刷品裁切预留的延伸区域，通常四边各留 3mm
- 样机（Mockup）：把设计稿套入真实场景的效果图，用于提案展示
- 源文件：可编辑的工程文件（AI/PSD/CDR），交付与版权归属需单独约定

## 核心工作流程
接单：需求沟通 → 报价方案 → 签订合同 → 收取预付款
创作：素材搜集与参考 → 方案构思（情绪板/草图）→ 设计执行 → 内部自审
交付：提案讲解 → 客户反馈 → 修改调整（约定轮次）→ 最终确认 → 源文件交付 → 收尾款

## 常见问题
Q：客户频繁要求修改怎么办？
A：合同中约定修改次数（一般 3 轮），超出按次收费；每轮修改前请客户一次性汇总意见，避免零散返工；用设计说明引导客户给出具体方向而非模糊感受。

Q：设计费怎么报价？
A：按项目复杂度 + 工时成本 + 市场行情综合定价，拆解成明细项（如 LOGO + 规范 + 应用物料）让报价有据可依，报价单写清包含范围、修改次数与付款方式。

Q：如何避免被「白嫖」方案？
A：坚持预付款后启动，提案阶段提供带水印的预览图，源文件与高清图在尾款结清后交付，重要约定全部落实到文字或合同。
