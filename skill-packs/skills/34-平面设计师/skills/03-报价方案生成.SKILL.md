# 报价方案生成

## 适用场景
平面设计师接到客户询价后，快速生成结构清晰、依据充分的报价方案。覆盖 LOGO/VI、海报、包装、画册、电商设计等常见项目类型，让报价「有据可依」，支撑议价谈判。

## 核心能力
- 报价方式选择：按项目定价 / 按工时定价 / 按物料数量定价，说明适用场景
- 项目拆解估价：把「设计一套 VI」拆成 LOGO、标准色、字体规范、应用物料等可计价项
- 工作量评估：按复杂度、修改轮次、交付周期估算工时，倒推合理报价
- 市场行情参照：给出初级/资深/工作室三档价格带，定位自己的报价区间
- 付款方案设计：预付款比例（通常 30%-50%）、节点付款、尾款条件
- 报价单输出：含项目明细、单价、数量、总价、有效期、不含税/含税说明

## 输出格式
1. 【项目理解】客户需求的复述与范围界定
2. 【报价明细】分项列出的工作项、工作量、单价、小计
3. 【总价与优惠】合计金额、打包优惠说明
4. 【付款方式】预付款 / 阶段款 / 尾款比例与节点
5. 【报价说明】含修改次数、交付周期、版权归属、报价有效期
6. 【议价预案】客户压价时的让步空间与置换策略

## 示例
输入：「客户要做一套茶饮品牌 VI，包括 LOGO 和包装，帮我报价」
输出：项目拆解 → LOGO 设计（3 套方案 + 精修 1 套）8000 元、标准色与字体规范 2000 元、包装延展（杯身 + 外带袋 + 封口贴）6000 元 → 合计 16000 元，打包价 14800 元 → 付款：预付 50% 启动，提案确认后付 30%，源文件交付前付清 20% → 说明：含 3 轮修改，超出按 500 元/轮计，源文件交付后版权归客户、署名权保留 → 议价预案：预算不足可先做 LOGO + 核心规范（10000 元），包装二期再做。

## 注意事项
- 报价先界定范围：「VI 设计」边界模糊，必须列明包含哪些物料，防止无限加项
- 预付款不可省，未收款不启动、源文件付清尾款再交付
- 修改次数与超改收费标准写进报价单，避免后期扯皮
- 版权条款提前约定：源文件、商用授权、署名权分别说明
- 报价留足议价空间，但让步用「减项」而非「降价」，守住单价体系
