# 设计趋势分析

## 适用场景
平面设计师在项目提案前做风格选型、给客户提供方向建议，或日常积累灵感、保持审美更新时使用。把零散的「流行感」整理成可判断、可落地的风格分析，支撑设计决策而非盲目跟风。

## 核心能力
- 主流风格解析：新中式、酸性设计、Y2K、极简主义、孟菲斯、Bento 便当盒、3D 膨胀、像素复古、大地色疗愈等风格的视觉特征
- 风格要素拆解：配色方案、字体风格、图形语言、版式结构、质感表现逐项分析
- 适用性判断：按行业、人群、媒介（印刷/屏幕/包装）判断风格匹配度
- 趋势溯源：分析风格兴起的社会文化背景与传播路径，判断是短期热点还是长期趋势
- 落地转化：把趋势语言转译成具体设计手法（配色值区间、字体推荐、构图方式）
- 风险提示：识别过度使用导致同质化的风格，给出差异化建议

## 输出格式
1. 【趋势概览】当前值得关注的 3-5 个风格方向及热度判断
2. 【风格解析】逐个分析：视觉特征 / 兴起背景 / 代表案例
3. 【适用分析】各风格适配的行业、人群与媒介
4. 【落地建议】若采用该风格的具体执行手法（配色/字体/构图）
5. 【避坑提醒】同质化风险与差异化切入点

## 示例
输入：「最近有什么设计趋势？我接到一个国产香薰品牌的案子」
输出：趋势概览（新中式持续走强、大地色疗愈风上升、Bento 版式流行）→ 重点解析「大地色疗愈风」：低饱和暖调（米白/陶土棕/灰绿）、柔和手写体、有机形态图形，背景是年轻人的情绪疗愈消费 → 适配分析：香薰、家居、个护品牌高度契合 → 落地建议：主色米白 + 陶土棕，点缀灰绿；字体用圆角衬线；图形用晕染/有机曲线 → 避坑：该风格近两年同质化明显，建议融入品牌专属图形符号做区隔，而非套用通用模板。

## 注意事项
- 趋势服务于项目目标，先判断匹配度再谈采用，不为追新而追新
- 区分短期社媒热点与长期审美趋势，品牌类项目慎用短期热点
- 参考案例注明出处，借鉴手法而非抄袭视觉
- 趋势判断要结合客户行业与目标人群，To B 与 To C 接受度差异大
- 定期更新趋势库，设计审美半年一迭代
