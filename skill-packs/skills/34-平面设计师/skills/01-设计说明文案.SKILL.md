# 设计说明文案

## 适用场景
平面设计师在方案提案、交付说明、作品参赛、作品集撰写时使用。把视觉选择翻译成客户/评委能理解的语言，讲清楚「为什么这样设计」，提升方案通过率与作品专业感。

## 核心能力
- 按项目类型适配写法：品牌 LOGO/VI、海报、包装、画册、电商详情页
- 设计理念提炼：从品牌定位、目标人群、行业属性推导创意出发点
- 视觉语言阐释：图形、色彩、字体、版式的选择理由与象征含义
- 结构化表达：背景 → 挑战 → 策略 → 方案 → 效果，逻辑清晰不堆砌辞藻
- 双版本输出：提案版（说服客户，强调商业价值）与作品版（参赛/展示，强调创意方法）
- 控制篇幅：短版 200 字内（配图文案）、长版 500-800 字（完整提案）

## 输出格式
1. 【项目背景】客户、行业、设计任务
2. 【设计理念】一句话核心概念 + 创意来源
3. 【设计解析】图形 / 色彩 / 字体 / 版式逐项说明选择理由
4. 【应用延展】主视觉在各物料上的呈现逻辑
5. 【价值总结】设计如何解决客户的商业/传播问题

## 示例
输入：「给一个新中式茶饮品牌 LOGO 写设计说明」
输出：理念「一叶见山水」——以茶叶舒展形态结合远山剪影，传递东方自然美学 → 图形解析（叶片与山形同构，负空间藏「茶」字意象）→ 色彩（黛绿 + 米白，取自宋画山水色调，区别于常见大红大绿的茶饮配色）→ 字体（定制衬线体，笔画带书法起收笔韵味）→ 应用延展（杯身、外带袋、门店招招牌的层级规范）→ 价值总结：帮助品牌在货架与门头快速建立「新中式」识别度。

## 注意事项
- 说明服务于沟通目标：对客户讲商业价值，对评委讲方法论，不要自说自话
- 每个视觉选择都要有理由，避免「好看」「大气」等空泛表述
- 术语适度：专业词（负空间、栅格）要配通俗解释
- 篇幅匹配场景，提案现场用的说明要口语化、可朗读
- 涉及客户未公开信息时注意保密表述
