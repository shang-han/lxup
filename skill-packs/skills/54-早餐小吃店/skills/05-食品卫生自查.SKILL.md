# 食品卫生自查

## 适用场景
迎接市场监管检查要自查；日常卫生管理需要清单；排查食品安全隐患；建立健康证、进货、消毒等台账；发生卫生问题要整改。

## 核心能力
- 生成每日/每周卫生自查清单（人员、环境、原料、操作、设备）
- 核对证照要求（食品经营许可证、健康证、索证索票）
- 排查高风险点（生熟交叉、烧熟煮透、危险温度带、添加剂）
- 输出 HACCP 关键控制点与记录表
- 制定问题整改清单与责任人
- 建立消毒、留样、废弃物处理记录模板

## 输出格式
1. 【证照核查清单】许可证/健康证/索证索票
2. 【卫生自查表】人员/环境/原料/操作分项
3. 【高风险点排查】
4. 【HACCP控制点与记录】
5. 【整改清单】问题/措施/责任人/时限
6. 【日常记录模板】消毒/留样/废弃物

## 示例
输入：「帮我做一份小吃店迎接检查的卫生自查清单」
输出：证照核查（许可证在有效期、全员健康证、进货票据齐全）→ 自查表（穿戴工作衣帽、生熟砧板分开、原料离地离墙、冷藏温度达标）→ 高风险点（油条用油每日检查、豆制品防变质、添加剂专柜专人）→ HACCP 控制点（油炸温度、复热中心温度）→ 整改清单 → 消毒与废弃物记录模板。

## 注意事项
- 食品经营许可证须在有效期内并按许可范围经营，不得超范围
- 从业人员持有效健康证，每日晨检，有症状立即离岗
- 食品添加剂专柜存放、专人管理、按标准使用，禁止超范围超量添加
- 原料索证索票、进货查验齐全，发现问题食品立即停用并上报
