# 采购比价台账

## 适用场景
面粉、食用油、调料等用量大想降采购成本；要在多家供应商间比价选优；判断批量采购/囤货是否划算；建进货台账应对检查与核算成本。

## 核心能力
- 建供应商比价表（品名、规格、单价、起订量、账期、配送）
- 折算统一计量单位做可比价格对比
- 测算批量采购的资金占用与仓储成本，判断囤货临界点
- 评估供应商资质与稳定性（证照、供货及时率）
- 生成分周期采购计划与安全库存
- 输出进货台账（索证索票留痕）

## 输出格式
1. 【供应商比价表】统一单位价格对比
2. 【推荐供应商与理由】
3. 【批量采购测算】资金/仓储/临界点
4. 【采购计划与安全库存】
5. 【资质审核要点】
6. 【进货台账模板】

## 示例
输入：「面粉和油用量大，怎么比价采购省钱？」
输出：比价表（面粉 A 供应商 25kg 装 95 元 = 3.8 元/kg、B 50kg 装 180 元 = 3.6 元/kg 但起订 5 袋）→ 推荐（B 单价低但需评估仓储与资金占用）→ 批量测算（食用油耐存可月度囤货、面粉防潮按需半月进）→ 采购计划（面粉安全库存 3 天、油 7 天）→ 资质审核（营业执照、食品经营许可、检验报告）→ 进货台账模板。

## 注意事项
- 采购必须索证索票（供应商资质、产品检验报告），台账留存备查
- 不采购来源不明、无标识、过期劣质原料
- 批量囤货须考虑仓储条件（防潮、防鼠、离地离墙）
- 使用合规食用油与添加剂，禁止超范围超量添加
