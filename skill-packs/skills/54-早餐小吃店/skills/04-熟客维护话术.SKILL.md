# 熟客维护话术

## 适用场景
早餐小吃靠社区熟客复购，要记住并维护老顾客；想提升复购与客单；处理熟客抱怨；建微信群/会员做私域；节假日问候维系客情。

## 核心能力
- 熟客识别与个性化招呼话术（记口味、记称呼）
- 复购引导：固定搭配推荐、套餐、集点卡
- 私域社群运营话术（微信群报餐、预留、接龙）
- 熟客抱怨与客诉处理（给面子、快补偿）
- 节假日与特殊时点问候话术
- 新客转熟客的破冰与留存话术

## 输出格式
1. 【熟客招呼话术】识别与个性化
2. 【复购引导】套餐/集点/预留
3. 【社群运营话术】报餐/接龙/通知
4. 【客诉处理话术】
5. 【节日问候话术】
6. 【新客转熟客话术】

## 示例
输入：「早餐店怎么做熟客生意，帮我写套话术」
输出：招呼话术（「王哥还是老样子俩包子一碗豆浆？今天给您多舀勺卤」）→ 复购（集点卡买十送一、固定套餐预留）→ 社群（「明天有现磨豆浆，要的群里接龙，给您留好不用排队」）→ 客诉（「这碗咸了马上给您换，下回给您做淡口」）→ 节日问候 → 新客转熟客（记住口味、加微信进群享预留）。

## 注意事项
- 预留、集点等承诺要兑现，失信一次流失一个熟客
- 社群通知控制频次，避免骚扰
- 优惠须真实，不搞先涨后降
- 维护客情同时守住食品卫生底线，不因熟客简化操作规范
