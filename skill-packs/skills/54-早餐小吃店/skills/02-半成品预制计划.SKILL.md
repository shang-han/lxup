# 半成品预制计划

## 适用场景
早餐起早现做来不及，要提前预制；要规划面团、馅料、卤味的预制量与冷藏冷冻方案；平衡预制与现做口感；建冷链管理记录。

## 核心能力
- 梳理可预制半成品（和面、调馅、包制生坯、卤制、预煮）及预制时点
- 制定冷藏/冷冻存储条件与保存时限
- 按预估销量定每日预制量，避免浪费与断货
- 设计预制-解冻-复热的标准工序
- 标注冷链关键控制点（中心温度、解冻方式、复热温度）
- 输出一周预制备料排期表

## 输出格式
1. 【可预制半成品清单】项目与预制时点
2. 【存储条件与时限】冷藏/冷冻参数
3. 【每日预制量】按销量预估
4. 【预制-复热SOP】解冻、复热工序
5. 【冷链控制点】温度与时限
6. 【一周备料排期表】

## 示例
输入：「我卖包子和粥，怎么提前预制又能保证口感？」
输出：预制清单（头天晚上和面冷藏发酵、调馅冷藏、包好生坯冷冻；粥底预煮）→ 存储（生坯冷冻 -18℃ 存 3 天、馅料冷藏 2 天）→ 预制量（按日均 300 个备 1.2 倍）→ SOP（生坯无需解冻直接蒸、蒸透中心温度 75℃ 以上）→ 冷链控制点（冷藏 0-4℃、复热中心 70℃+）→ 一周排期表。

## 注意事项
- 预制半成品须贴制作日期与保质期标签，先进先出
- 冷链不得断链，解冻应在冷藏或流水中进行，禁止室温久放
- 复热须烧熟煮透，中心温度达标
- 原料索证索票，预制操作符合 HACCP 卫生规范
