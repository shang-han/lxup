# 行业基础

## 行业概况
早餐小吃是刚需、高频、低客单的民生餐饮，抗周期能力强，但辛苦（早起、长时间站立）、利薄、靠走量。业态包括包子、粥、油条豆浆、粉面、煎饼、地方小吃等。核心矛盾：高峰产能有限与出餐速度的矛盾、低客单与成本上涨的矛盾。生存关键：极致出餐效率 + 半成品预制 + 供应链成本控制 + 社区熟客复购。摊位与小店还要兼顾城管合规与食品卫生。

## 专业技能体系
### 硬技能
- 高效出餐流程（高峰期3分钟/单）
- 半成品预制与冷链存储
- 摊位选址与城管合规
- 供应链比价（面粉/食用油/调料批量采购）
- 食品卫生操作规范

### 软技能
- 早起自律管理
- 熟客关系维护

## 常用术语
- 出餐速度：高峰每分钟/每单出品时间，决定坪效与流失率
- 危险温度带：5-60℃ 细菌快速繁殖区间，熟食不宜久放
- 中心温度：食物内部温度，烧熟煮透一般需达 70℃ 以上
- 冷链：从预制到复热全程低温不断链
- 出成率：成品量/原料量，衡量预制损耗
- 安全库存：保证不断货的最低备货量
- 客单价：平均每单金额，靠套餐、加料拉升
- 复购率：老客重复购买占比，熟客生意核心

## 核心工作流程
每日：凌晨备料预制 → 开档出餐 → 高峰流水线作业 → 午间收档清洁 → 次日备货
每周：原料比价采购 → 库存盘点 → 设备清洁保养 → 卫生自查
每月：成本核算 → 熟客/社群维护 → 品种调整 → 证照与台账检查

## 常见问题
Q：早高峰出餐太慢老流失顾客怎么办？
A：把能提前做的都预制前置（包子预蒸保温、豆浆预装），高峰三人流水线分工，固定套餐扫码预点单，目标 90 秒/单。

Q：怎么提前预制又不影响口感和安全？
A：生坯冷冻、粥底预煮，贴好日期先进先出，冷链不断链，复热烧熟煮透中心温度达标，按销量定预制量别囤太多。

Q：面粉油调料怎么采购省钱？
A：统一单位比价，耐存的油可月度批量囤、面粉防潮半月进，算清资金与仓储成本，务必索证索票选有资质供应商。

Q：早餐店怎么做熟客复购？
A：记住口味称呼个性化招呼，建微信群报餐预留免排队，集点卡买十送一，节日问候维系客情，承诺一定兑现。
