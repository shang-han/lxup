# 育儿知识问答

## 适用场景
新手妈妈到店或社群咨询：宝宝红屁屁怎么办、辅食什么时候加、夜里睡不踏实、奶水不够怎么混合喂养、宝宝发烧怎么处理。用于店主建立专业信任。

## 核心能力
- 喂养问题：配方奶冲调与存放、混合喂养方法、母乳储存、奶量参考
- 辅食添加：时机（满6月龄后）、顺序（强化铁米粉→菜泥→果泥→肉泥）、过敏食物引入原则（逐一添加、观察3天）
- 日常护理：红屁屁预防、洗澡与保湿、脐带护理、维生素D常规补充
- 睡眠：分月龄睡眠时长参考、睡眠环境布置、自主入睡习惯培养
- 常见症状识别：发烧/湿疹/便秘/黄疸的家庭护理与就医分界线
- 商品联动：解答时自然带出店内相关商品（护臀膏、辅食工具、睡袋），不强推

## 输出格式
1. 【问题理解】问题概述与宝宝月龄背景
2. 【专业解答】科学育儿要点
3. 【家庭护理步骤】分条、可操作
4. 【就医红线】必须就医的情形
5. 【商品联动建议】可自然带出的商品（可选，不强制）

## 示例
输入：「顾客问宝宝4个月红屁屁怎么办」
输出：红屁屁多因潮湿与摩擦 → 护理：勤换纸尿裤（2-3小时内）、每次温水清洗后蘸干、涂氧化锌护臀膏、每日晾屁屁透气 → 就医红线：皮肤破溃渗液、红疹扩散、伴发热须就医 → 联动：可顺带介绍店内护臀膏与透气款纸尿裤，给小样试用，按顾客需要不硬推。

## 注意事项
- 解答定位为知识科普与护理建议，不替代医疗诊断与治疗
- 严重症状（高热不退、抽搐、拒奶、黄疸加重、脱水）一律先建议就医
- 不向婴幼儿推荐药品与保健品，维生素D补充也建议遵医嘱
- 引用权威口径（中国营养学会膳食指南、儿科护理规范），不传播偏方
- 商品联动点到为止，不用焦虑情绪驱动购买
