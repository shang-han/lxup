# 会员运营方案

## 适用场景
新店会员体系设计、老店会员权益升级、分阶段会员运营（孕期→婴儿→幼儿）、会员日活动策划、沉睡会员激活。

## 核心能力
- 全周期分层设计：孕妈卡→宝宝卡（0-1岁）→成长卡（1-3岁），层级迁移与权益继承
- 积分与成长值规则：消费积分、建档礼、产检陪伴打卡、积分兑品与抵现规则
- 分阶段触达内容：预产期前2周待产包提醒、6月龄转奶提醒、按体重纸尿裤升码提醒、1岁辅食进阶方案
- 会员日活动方案：主题（宝宝爬行赛/育儿讲座/转奶节）、机制、选品、预算
- 沉睡激活：60/90天未消费分层策略，激活权益与触达话术
- 效果指标：会员率、复购率、客单价、阶段流转率、激活到店率

## 输出格式
1. 【会员体系框架】层级、权益、积分规则
2. 【全周期阶段运营】各阶段触达时机与内容
3. 【活动方案】主题/机制/选品/时间/预算
4. 【配套文案】社群版 + 朋友圈版 + 短信版
5. 【执行排期】筹备与执行节点
6. 【效果指标】活动后追踪指标清单

## 示例
输入：「设计一套会员方案，想从孕期就锁客」
输出：三级会员：孕妈卡（建档送待产包9折券+免费孕妇课堂）→ 宝宝卡（每月积分双倍、免费婴儿洗澡1次）→ 成长卡（生日礼+玩具租玩权益）→ 阶段触达：预产期前2周推送待产包清单，5.5月龄推送2段转奶指南 → 会员日「转奶节」：2段奶粉买6赠1 → 配套文案与排期 → 指标：孕妈建档率、阶段流转率。

## 注意事项
- 阶段触达内容以服务和知识为主，不是硬广轰炸，频次每月2-4次为宜
- 会员权益要简单易懂，店员一句话能讲清楚，避免复杂计算
- 孕期与婴童数据涉及个人隐私，不得泄露或转卖，触达用企业微信统一管理
- 活动机制先算清毛利，避免叠加亏损，赠品注意效期
- 沉睡唤醒话术温和，顾客明确拒绝后不再重复打扰
