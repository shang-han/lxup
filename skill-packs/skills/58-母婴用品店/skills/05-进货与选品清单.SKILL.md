# 进货与选品清单

## 适用场景
新店开业选品、日常补货渠道对比、母婴展会（CBME等）采购准备、供应商资质核验、首单数量测算。

## 核心能力
- 渠道对比：品牌区域代理、省批市场、展会直采、品牌总部直供、B2B电商平台
- 资质核验清单：营业执照、食品经营许可证、品牌授权书（有效期与授权链）、奶粉注册证号、发票与可追溯凭证
- 奶粉渠道专项：配方注册证号核验、区域保护与窜货风险、价保与退换货政策
- 选品结构：引流品（主流品牌透明价）/利润品（二三线品牌、功能性单品）/形象品（高端礼盒）配比建议
- 展会采购计划：行程规划、目标品牌与展位清单、谈判要点（首单折扣、退换货比例、陈列物料、导购培训）
- 首单测算：周边0-3岁家庭数 × 预估进店率 × 客单价 → 各品类采购量

## 输出格式
1. 【进货需求】品类/数量/预算
2. 【渠道推荐】各品类推荐渠道与对比
3. 【资质核验表】各供应商核验项与状态
4. 【选品结构】引流/利润/形象品比例与示例
5. 【采购计划】行程/谈判要点/首单数量
6. 【风险提醒】渠道与合规常见坑

## 示例
输入：「新开一家店，奶粉和纸尿裤走什么渠道？首单怎么配」
输出：奶粉对接品牌区域代理（核验授权书+注册证号，谈价保与退换货）→ 纸尿裤省批+品牌直供组合（主流款走批发、高端款走直供）→ 选品：引流品40% + 利润品40% + 形象品20% → 首单按周边3000户家庭 × 15%进店率 × 200元客单价测算 → 风险：拒绝无注册证号货源，警惕超低价临期尾货，退换货条款写进合同。

## 注意事项
- 奶粉必查配方注册证号与批次检验报告，拒绝三无与窜货商品
- 新供应商先小批量试单，验货验资质后再放量，票据留存可追溯
- 展会下单明确交货期与退换货政策，口头承诺要落进合同
- 首单宁少勿压，预留现金流与货架给后续补货
- 价格不是唯一标准，综合评估售后、培训、动销支持
