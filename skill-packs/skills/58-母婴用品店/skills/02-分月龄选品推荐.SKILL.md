# 分月龄选品推荐

## 适用场景
顾客问「X个月宝宝该备什么」「待产包要准备哪些」「1岁宝宝适合什么玩具」，需要按月龄/年龄和预算给出选品清单。

## 核心能力
- 按月龄/年龄选品清单：新生儿包、0-6月、6-12月（辅食期）、1-3岁
- 待产包清单：妈妈用品（产褥垫/哺乳衣/一次性内裤）、住院用品、月子护理
- 服装尺码对应：月龄+身高体重→尺码（52-90），按季节建议买大一码
- 玩具发育适配：0-3月听觉视觉、3-6月抓握口欲、6-12月爬站学步、1-3岁精细动作与认知
- 预算分档：经济/标准/高端三档组合与价格带
- 季节调整：夏季清凉透气、冬季保暖、换季衣物数量控制

## 输出格式
1. 【年龄画像】月龄/年龄，体重身高参考区间
2. 【必备清单】必需品类与数量
3. 【可选清单】按预算的加购项
4. 【尺码与规格建议】服装尺码、纸尿裤码数、奶粉段数
5. 【预算参考】各档位组合价格带
6. 【安全要点】认证与适龄性检查项

## 示例
输入：「3个月宝宝，该准备什么玩具和衣服」
输出：3个月处于抓握与视觉发育期、开始翻身 → 玩具：摇铃、黑白/彩色卡、布书、健身架（均需3C认证、无脱落小零件）→ 服装：66码（适合5-7kg），开襟连体衣2-3件、薄袜、A类纯棉 → 预算：经济档约300元/标准档约600元 → 安全：玩具可入口啃咬、无长绳，衣服按季节买大一码。

## 注意事项
- 月龄发育有个体差异，清单为参考，结合顾客宝宝实际情况调整
- 玩具必须适龄并有3C认证，0-3岁严格规避小零件、长绳、尖锐边角
- 不制造焦虑式推销，必备与可选清晰分开，让顾客按预算自主选择
- 服装推荐A类、浅色、开襟款，避免可拆卸装饰物
- 奶粉数量建议结合喂养方式（母乳/混合/配方），不鼓励囤货
