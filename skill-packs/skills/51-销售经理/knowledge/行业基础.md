# 行业基础

## 行业概况
销售经理/BD 是直接承担营收增长的岗位，覆盖 B2B 直销、渠道拓展、大客户（KA）管理、政企采购等场景。销售模式正从关系型销售向顾问式销售、价值销售转型，CRM 数字化工具成为标配。优秀销售的核心能力：客户洞察 + 方案呈现 + 谈判推进 + 长期经营。收入结构通常为底薪 + 提成/奖金，业绩波动大，淘汰率高，但天花板也高。

## 常用术语
- BD（Business Development）：商务拓展，侧重渠道与新业务合作
- KA（Key Account）：大客户/重点客户
- 销售漏斗：线索→商机→谈判→成交各阶段的数量与金额分布
- 商机转化率（Win Rate）：成交商机数/总商机数
- 客单价：平均单笔合同金额
- 回款率：实际回款金额/合同金额（提成多按回款而非合同计）
- BANT：Budget 预算 / Authority 权限 / Need 需求 / Timeline 周期，商机资格审查框架
- TCO（Total Cost of Ownership）：总拥有成本，含采购+实施+运维+替换成本
- LTV：客户生命周期总价值
- 返点/返利：按采购额返还的折扣，须明示入账，账外暗中即商业贿赂

## 核心工作流程
获客：名单整理 → 背景调研 → 首次触达（电话/微信/拜访/展会/转介绍）
推进：需求调研 → 方案提报 → 报价 → 谈判 → 签约
交付：订单跟进 → 实施交付 → 验收 → 回款
经营：客户盘点 → 复购增购 → 转介绍 → 沉默激活

## 常见问题
Q：客户只盯着价格怎么办？
A：从单价切换到总成本视角（TCO），做价值拆解与分层报价（减配可更低价），用条件式让步代替裸降价。

Q：成交周期长，客户总「再考虑考虑」怎么办？
A：用 BANT 判断是真商机还是陪标；每个阶段给客户增量价值保持存在感；锁定决策链上的关键人，必要时发展内部教练。

Q：回款难怎么预防？
A：签约时明确付款节点与验收标准；交付留痕（签收单/验收单）；到期前 7 天提醒；逾期分级升级（函件 → 高层沟通 → 法律手段）。

Q：怎么和老客户要转介绍？
A：在交付满意/续约成功的正向时刻提出，给明确的介绍对象画像（「您圈子里有没有同样在做 XX 的朋友」），并对介绍结果给予反馈与感谢。
