# 跟进计划

## 适用场景
新客推进节奏设计、长成交周期商机的阶段管理、成交客户的复购/增购维护、沉默客户激活、重点客户周报。

## 核心能力
- 阶段化跟进节奏表：初识 → 需求确认 → 方案提报 → 谈判 → 签约 → 交付 → 回款，每阶段的触达间隔、触点方式、推进动作
- 每次触达提供「价值型理由」（行业资料、诊断小结、案例、政策变化），而非空泛问候
- 跟进记录模板：时间 / 联系人 / 沟通要点 / 客户反馈 / 下一步 / 下次触点日期
- 沉默客户分层激活：30 天（轻触达）/ 60 天（价值内容 + 邀约）/ 90 天（换切入角度或换联系人）
- 复购周期测算与提醒：按合同周期、消耗周期推算续约/补货节点
- 重点客户周报模板：商机进展、风险、所需支持

## 输出格式
1. 【商机阶段判断】当前所处阶段与卡点
2. 【跟进时间线】表格（节点 / 目的 / 方式 / 内容要点 / 话术）
3. 【触达内容建议】每次联系带的「价值钩子」
4. 【风险与预案】可能的拖延/流失信号及应对
5. 【复盘指标】阶段转化率、平均成交周期等

## 示例
输入：「这个客户方案报上去三周了没动静，采购说领导还没看，怎么跟？」
输出：判断卡在决策层未介入 → 时间线：本周给采购发一页纸摘要（降低领导阅读成本）+ 询问上会时间 → 下周约 15 分钟线上演示直击决策层关注点 → 同步准备同行业案例作为推动材料 → 话术：「张经理，知道领导忙，我把方案浓缩成一页纸，重点标了投入产出，您方便时转一下；另外我们下周可以安排 15 分钟线上演示，只讲领导最关心的成本和落地周期，您看周二还是周四合适？」→ 风险预案：若两周内仍无进展，考虑发展内部教练或换路径接触。

## 注意事项
- 跟进频次设上限（同一联系人每周不超过 2 次主动触达），客户明确拒绝即停止
- 承诺的回访/资料交付时间必须兑现，失信一次推进成本翻倍
- 客户沟通信息仅用于本项目推进，不向无关第三方透露
- 关键承诺与口头达成一致后补书面确认（邮件/会议纪要）
