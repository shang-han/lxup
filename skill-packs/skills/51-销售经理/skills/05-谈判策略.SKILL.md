# 谈判策略

## 适用场景
客户压价、付款方式与账期谈判、合同条款磋商、续约谈判、框架协议年度议价、被竞品低价狙击时的应对。

## 核心能力
- 局势评估：双方 BATNA（最佳替代方案）、时间压力、决策权限分析
- 价格异议拆解：总拥有成本（TCO）对比、价值拆分、单价 vs 总量视角转换
- 让步策略设计：条件式让步（降价换账期/换量/换案例授权）、递减式让步节奏、守住底线的表达
- 条款博弈：付款节点、验收标准、违约责任、SLA、续约条件的交换组合
- 僵局破解：换议题、拆包、引入新变量（高层互访、试点合作、分期落地）
- 谈判前清单：目标 / 底线 / 筹码 / 让步阶梯 / 话术预演

## 输出格式
1. 【局势评估】双方筹码与压力点
2. 【谈判目标】理想目标 / 可接受目标 / 底线（含量化）
3. 【策略组合】3-4 套打法及适用条件
4. 【让步阶梯】每一档让什么、换什么、怎么说
5. 【应对话术】高频施压场景的标准回应
6. 【风险提示】法律与履约风险点

## 示例
输入：「客户咬定要打 7 折，我们最多能让到 85 折，怎么谈？」
输出：评估（客户未必真有 7 折预期，多在试探底线）→ 策略：不直接回应折扣，先做价值锚定（TCO 测算 + 同行业采购价区间）→ 让步阶梯：85 折需绑定三年框架或预付半年；若只要价格，可做「减配版 75 折」（去掉两项增值服务）→ 话术：「7 折确实做不到，这个价格连实施成本都覆盖不了，给您做下去服务必然缩水，反而坑了您。这样，如果您能签三年框架，我向公司申请 85 折特批；或者我们把 X 和 Y 两项服务拿掉，给您一个更低的配置价，您看哪种更合适？」→ 风险：任何折扣以书面合同为准，口头承诺不落地。

## 合规红线
- 不以回扣、私人好处换取成交（《反不正当竞争法》第七条，商业贿赂可入刑）；返利、折扣必须明示并如实入账
- 报价不得构成价格欺诈（虚构原价、虚假优惠折价），遵守《价格法》
- 不与竞品或渠道商约定固定转售价格（《反垄断法》纵向垄断协议风险）
- 投标场景严禁围标串标、低于成本价竞标
- 谈判中获取的对方商业秘密负有保密义务，不用于其他项目
