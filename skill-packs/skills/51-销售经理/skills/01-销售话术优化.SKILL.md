# 销售话术优化

## 适用场景
陌拜电话开场、微信首次添加自介、客户拜访破冰、客户异议应对（「不需要」「太贵了」「发资料看看」「要考虑一下」）、临门促单、转介绍请求等场景的话术生成与优化。

## 核心能力
- 按客户类型（决策者/使用者/采购）生成不同侧重的话术版本
- 电话/微信开场白：15 秒说清「我是谁 + 为什么找你 + 对你有什么用」
- 高频异议应对库：价格异议、需求异议、拖延异议、竞品对比异议
- FABE 价值陈述（特性-优势-利益-证据）结构化表达
- 促单话术：假设成交法、二择一法、限时权益法
- 话术诊断：输入现有话术，指出问题并给出改写版本

## 输出格式
1. 【场景判断】客户类型、所处阶段、核心顾虑
2. 【优化话术】2-3 个版本（电话版 / 微信文字版 / 面谈话版）
3. 【要点拆解】每句话的目的说明
4. 【避坑提醒】该场景常见错误说法

## 示例
输入：「客户总说『你把资料发我，我看完联系你』，然后就没下文了，怎么接？」
输出：判断为拖延型异议（软拒绝）→ 话术：「好的王总，资料我马上发您。不过我们方案有 30 多页，怕耽误您时间——您主要关心成本这块还是效果这块？我发您的时候把重点标出来，另外周三上午我给您去个 5 分钟电话，就您关心的那点说清楚就行？」→ 要点：给资料≠放弃推进，锁定具体关注点 + 约定下次触点 → 避坑：不要只回「好的」然后干等。

## 注意事项
- 话术不得含虚假宣传、夸大承诺（《广告法》《反不正当竞争法》）
- 不教暗示回扣、私人好处等商业贿赂话术（返利须明示入账）
- 电话营销遵守时段与频次规范，客户明确拒绝即停止
- 话术是工具不是套路：以真实价值为基础，不诱导客户做出不符合利益的决策
