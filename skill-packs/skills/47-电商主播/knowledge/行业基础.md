# 行业基础

## 行业概况
直播电商市场规模超 4 万亿，主播从业者超千万，是增长最快的电商形态。头部主播单场带货上亿，但大部分中小主播月收入数千元，行业两极分化明显。平台以抖音、快手、淘宝直播、视频号为主。主播核心能力包括口才、产品表现力、粉丝互动与选品眼光，收入来自坑位费 + 销售佣金。直播带货受《广告法》《网络直播营销管理办法》严格约束，合规是长期经营的底线。

## 常用术语
- GMV：直播间的成交总金额
- 转化率：观看用户中下单购买的比例
- 客单价：平均每单金额
- 退货率：购买后退货的比例，直接影响真实收益
- 在线人数：直播间当前同时观看的人数
- 憋单：先攒需求再集中放单的促单手法
- 坑位费：商家为商品上链接支付的固定费用
- 佣金：按销售额比例抽取的分成

## 核心工作流程
开播前：选品 → 排品 → 脚本准备 → 设备调试 → 预热引流
直播中：产品介绍 → 互动抽奖 → 憋单逼单 → 解答问题
下播后：订单整理 → 售后跟进 → 数据复盘 → 下场规划

## 常见问题
Q：直播间没人、人气上不去怎么办？
A：投流引流 + 短视频预热 + 开场福利留人 + 优化话术节奏，靠停留时长和互动率撬动平台自然流量。

Q：怎么提高转化率？
A：限时优惠 + 限量抢购 + 痛点讲解 + 信任背书 + 打消售后顾虑，配合憋单营造抢购氛围。

Q：退货率高怎么降？
A：讲解如实不夸大、尺寸/效果说明清楚、避免冲动下单后的落差，做好售后与发货时效。

Q：新手怎么起步？
A：先确定垂直品类人设、坚持稳定开播时长、打磨话术与排品、靠数据复盘持续迭代，别指望一夜爆火。
