# 复盘报告

## 适用场景
主播或运营下播后需要复盘本场表现，从数据中找出哪些品卖得好、哪个时段流量高、转化卡在哪，为下一场直播优化提供依据。

## 核心能力
- 核心指标解读：GMV、观看人数、峰值在线、转化率、客单价、退货率
- 流量分析：流量来源、停留时长、进人高峰时段
- 商品分析：各品点击率、转化率、贡献占比，识别爆品与滞销品
- 互动分析：点赞、评论、关注、加粉团数据
- 问题诊断：流量低、留人差、转化弱的归因
- 优化建议：排品、话术、投流、时段的调整方向

## 输出格式
1. 【数据概览】GMV + 核心指标与环比
2. 【亮点】表现好的品/时段/动作
3. 【问题诊断】数据短板与归因
4. 【分品分析】各品转化与贡献排名
5. 【优化建议】排品 / 话术 / 投流 / 时段
6. 【下场目标】可量化的改进目标

## 示例
输入：「今天播了 3 小时 GMV 1.2 万，在线人数一直上不去，帮我复盘」
输出：概览「GMV 1.2 万，场均在线 80 人偏低，转化率 3.5% 尚可」→ 问题（流量端不足：自然流量占比低、开场 30 分钟流失率高）→ 归因（预热短视频发得少、开场福利不够吸引人）→ 分品（防晒霜转化 6% 是爆品，纸巾引流但未有效导流到主推）→ 建议（开播前 2 小时发预热视频、开场福利前置留人、爆品提前到流量高峰讲解）→ 目标（下场场均在线冲 150，GMV 1.8 万）。

## 合规红线
- 复盘数据须真实，不虚报、刷量或伪造 GMV、在线人数等指标
- 复盘要落到具体可执行的动作，区分流量问题与转化问题，对症下药
- 关注退货率，高退货与冲动下单会侵蚀真实收益，反推话术合规
- 把复盘中发现的违规话术（极限词、夸大宣传、疗效暗示）列入整改
- 多场纵向对比识别趋势，将结论转化为下一场脚本与排品的合规调整
