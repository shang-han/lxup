# 产品介绍话术

## 适用场景
主播拿到一款商品后，需要快速组织出一段有感染力、突出卖点、贴合使用场景的讲解话术，让观众从「看看」变成「想买」。

## 核心能力
- FABE 结构：特性（Feature）→ 优势（Advantage）→ 利益（Benefit）→ 证据（Evidence）
- 场景化种草：把产品放进用户真实使用场景
- 痛点切入：先戳痛点再给解决方案
- 信任背书：销量、资质、测评、主播亲测
- 价格锚点：与市场/原价对比凸显划算
- 适配品类风格：美妆、食品、日用、服饰各有话术套路

## 输出格式
1. 【痛点开场】一句话戳中需求
2. 【卖点讲解】FABE 逐条展开
3. 【场景代入】谁、什么时候、怎么用
4. 【信任背书】销量/资质/亲测
5. 【价格锚点】对比凸显划算
6. 【行动指令】引导下单话术

## 示例
输入：「帮我写一段无线吸尘器的讲解话术」
输出：痛点「家里有猫有娃，沙发床底全是毛，扫帚越扫越飞？」→ 卖点（25kPa 大吸力 → 比传统强 3 倍 → 一遍吸净不反复 → 附检测报告）→ 场景「床底、沙发缝、车里都能用，单手就能拎」→ 背书「我自己用了俩月，销量 10 万+」→ 锚点「专柜 1299，今天直播间 699 还送替换滤芯」→ 指令「想要的扣 1，上链接 3、2、1」。

## 合规红线
- 禁用「最」「第一」「国家级」「顶级」等《广告法》极限词
- 不虚假宣传：功效描述、检测数据、销量必须有真实依据，不虚构
- 食品、保健品不得宣称治疗、预防疾病的疗效，须提示「不能代替药物」
- 价格对比、原价、亲测背书必须真实，不虚构原价与虚假体验
- 商业合作、广告推广须标注利益相关，不伪装成中立测评
