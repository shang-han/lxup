# 憋单话术

## 适用场景
主播在讲解主推款时需要营造抢购氛围，通过憋单、逼单、催单话术推动犹豫的观众下单，提升当场转化率与 GMV。

## 核心能力
- 憋单铺垫：先种草攒需求，再集中释放库存
- 稀缺感营造：限量、限时、限赠
- 紧迫感话术：倒计时、库存告急、即将恢复原价
- 从众引导：已拍人数、加购数、销量播报
- 临门一脚：打消下单顾虑（运费险、七天无理由）
- 返场话术：错过后的补救与引导关注下一场

## 输出格式
1. 【憋单铺垫】攒需求、抬期待的口播
2. 【开价放单】揭晓价格与福利的高潮话术
3. 【稀缺紧迫】限量/限时/库存播报
4. 【从众引导】已拍人数与抢购氛围
5. 【打消顾虑】售后保障承诺
6. 【倒计时催单】最后冲刺口播

## 示例
输入：「这款面膜我要憋一波，帮我写憋单话术」
输出：铺垫「想要这款的把'想要'刷起来，刷到 500 个我去申请加赠」→ 开价「今天不要 199，直播间拍一发三，到手 69！」→ 稀缺「只备了 300 单，拍完真没了」→ 从众「已经拍了 120 单，手慢的抓紧」→ 打消顾虑「运费险+七天无理由，不好用直接退」→ 倒计时「最后 50 单，3、2、1 上链接，没抢到的扣'没抢到'」。

## 合规红线
- 库存与限量必须真实，不虚构「仅剩 X 单」制造虚假紧迫感
- 不用「全网最低」「史上最强」等《广告法》极限词，不虚假宣传价格
- 承诺的赠品、价格、福利必须真实兑现，售后保障如实说明
- 食品、保健品憋单话术不得宣称疗效，不以健康焦虑诱导下单
- 商业推广须标注利益相关，憋单节奏适度，不透支观众信任
