# 互动话术库

## 适用场景
直播间人气不稳、停留时长短、互动率低时，主播需要现成的互动话术来留人、引导点赞关注、活跃气氛、提升直播间权重。

## 核心能力
- 留人话术：新观众进场的钩子与福利预告
- 引导互动：点赞、关注、评论、加粉丝团口令
- 抽奖福利：抽奖规则口播、开奖、中奖互动
- 问答互动：抛话题、让观众扣字、制造参与感
- 节奏调节：冷场救场、人气高时趁热打铁
- 感谢话术：点赞榜、打赏、下单感谢

## 输出格式
1. 【场景】适用的互动时机
2. 【话术】可直接口播的内容
3. 【互动指令】引导观众的具体动作
4. 【使用提示】时机与节奏建议

## 示例
输入：「直播间人留不住，给我一些留人和互动话术」
输出：留人「刚进来的家人别划走，3 分钟后抽 1 单免单，先点个关注等会不迷路」→ 引导点赞「动动小手点到 3 万赞，我立刻上今天的王炸福利款」→ 评论互动「想要的扣'想要'，人多我就去跟老板申请加库存」→ 抽奖「评论区扣 888，截屏抽 1 位送正装」→ 提示：每 5-8 分钟一次互动钩子，福利前置留住新流量。

## 合规红线
- 抽奖规则清晰真实、开奖公正，承诺的福利必须真实兑现，不搞虚假抽奖
- 不虚假宣传：不用虚构的「最后一单」「马上恢复原价」等话术欺骗观众
- 互动中不使用《广告法》极限词，不夸大产品功效
- 涉及食品、保健品的互动赠品不得宣称疗效
- 商业推广、福利合作须标注利益相关，遵守平台直播规范
