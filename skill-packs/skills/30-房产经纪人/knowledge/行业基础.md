# 行业基础

## 行业概况
全国房产经纪人约 200 万人，行业受政策影响大（限购、限贷、利率调整、「认房不认贷」等），佣金收入为主、业绩波动大。核心能力：房源开发 + 客户匹配 + 谈判技巧 + 交易流程把控。二手房交易「线上挂牌 + 线下带看」结合，平台化（贝壳/链家/安居客等）主导客源入口。合规要求：房源信息真实、不吃差价、交易资金走监管账户。行业向专业化转型，深耕社区、积累口碑是长期生存之道。

## 专业技能体系

### 硬技能
- 房源开发与业主维护（端口/扫街/老客户转介）
- 需求匹配与带看技巧
- 交易流程与税费计算（契税/个税/增值税）
- 贷款政策了解（商贷/公积金/组合贷）
- 房屋基本问题识别（漏水/裂缝/朝向/采光）

### 软技能
- 信任建立与长期跟进
- 买卖双方价格撮合

## 常用术语
- 房源：挂牌出售/出租的房屋，分一般委托与独家委托
- 客源：有购房/租房需求的客户，按意向度分 A（急购）/B（考虑中）/C（观望）级
- 带看：陪同客户实地看房，是意向转化为成交的核心环节
- 意向金（定金）：客户表达购买意向的预付金，须书面约定退还规则
- 网签：在房管局系统签订买卖合同备案，防止一房多卖的关键步骤
- 资金监管：交易房款存入银行专用监管账户，过户完成后划转，保障买卖双方资金安全
- 满二/满五唯一：房产证或契税票满 2 年免增值税、满 5 年且为家庭唯一住房免个税的税费优惠政策

## 核心工作流程
开发：房源开发（业主维护/社区深耕）→ 实地勘查 → 拍照上架 → 价格沟通与业主维护
匹配：客户需求分析 → 精准推荐房源 → 约看安排 → 带看反馈跟进
交易：价格谈判 → 意向金/定金 → 签约 → 贷款审批 → 缴税过户 → 资金监管划转 → 物业交割

## 常见问题
Q：中介费能不能少？
A：服务包含找房、带看、谈判、签约、贷款、过户全程协助，按公示标准与合同约定收费。用专业服务体现价值，不靠无条件让价竞争。

Q：房子有瑕疵怎么办？
A：签约前如实告知（漏水史、噪音、产权纠纷、非正常死亡事件等）并在合同中注明。隐瞒构成欺诈，可能导致合同撤销与赔偿，得不偿失。

Q：怎么判断客户的真实意向度？
A：看行为信号：是否反复看房、是否带家人同行、是否主动询问贷款细节与入住时间。A 级客户（需求明确 + 时间紧 + 预算匹配）优先重点跟进。
