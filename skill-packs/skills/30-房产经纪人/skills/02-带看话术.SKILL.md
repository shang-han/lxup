# 带看话术

## 适用场景
经纪人陪同客户实地看房：新客户首次带看、多套房源集中对比看、意向客户复看。需要标准化的讲解流程与临场应变话术，提升看房体验与转化。

## 核心能力
- 看房前准备清单：钥匙/业主配合确认、带看路线规划、周边配套踩点、客户关注点回顾
- 现场讲解顺序：小区环境与物业 → 楼栋与电梯 → 户型格局 → 采光通风 → 视野景观 → 装修细节
- 场景化卖点讲解：把参数翻译成生活（「早上阳光直接洒到餐桌」「这个书房刚好放书桌加书柜」）
- 瑕疵应对：主动如实说明 + 给出化解方案或价格支撑，不遮掩不回避
- 互动引导：让客户动手体验（开窗听噪音、试水压、看收纳），用确认式提问锁定满意点
- 看房后跟进：确认满意度排序、明确顾虑、安排复看或谈判

## 输出格式
1. 【看房前准备】确认事项与话术预演
2. 【开场话术】到场破冰与路线说明
3. 【讲解要点】按空间顺序的卖点话术
4. 【瑕疵应对】如实告知 + 缓冲话术
5. 【互动引导】体验与确认式提问
6. 【收尾跟进】看房后总结与下一步话术

## 示例
输入：「带刚需夫妻第一次看一套 89 平南北通透三房，房子装修一般但保养干净，业主要价偏硬」
输出：准备：确认钥匙与业主可配合时间、查小区物业费与车位比、了解夫妻通勤地点 → 开场：小区门口介绍楼龄、物业口碑与住户构成 → 讲解：玄关（南北通透，站在客厅感受穿堂风）→ 客厅（下午采光实测）→ 厨房（U 型台面动线）→ 主卧（飘窗利用场景）→ 瑕疵应对：「装修是基础了些，但正好按你们喜好设计，这部分价格已经让出来了」→ 互动：请客户开窗感受隔音、试卫生间水压 → 收尾：问「三套里最在意这套房哪一点」，锁定满意点，约复看或安排与业主谈价。

## 合规红线
- 现场讲解须与房源实际一致，不得承诺未写入合同的改造/赠送（「业主会换家电」须书面确认）
- 已知房屋瑕疵（漏水、噪音、邻里纠纷、非正常死亡事件）如实告知并留痕
- 产权状态（抵押、查封、共有权人）如实说明，不隐瞒促成签约
- 周边规划（地铁、学校）以官方公示为准，不传播未经证实的规划利好
- 不制造虚假紧迫感逼定（谎称「明天就有人签」）
