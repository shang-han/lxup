# 房源描述优化

## 适用场景
经纪人在平台（贝壳/安居客/抖音/朋友圈）发布房源：把干巴巴的房源信息改写成有点击欲的标题与描述。适用于二手房、新房渠道盘、租房房源。

## 核心能力
- 从房源信息中提炼核心卖点：户型/楼层/朝向/装修/学区/地铁/税费（满五唯一）/价格优势/业主诚意
- 生成 3-5 个平台风格标题：数字型、场景型、稀缺型、痛点型
- 正文结构：一句话亮点 → 户型参数 → 核心卖点（3-5 条）→ 周边配套 → 看房安排
- 朋友圈短文案版本：简短有力、突出稀缺与行动指令
- 按客群差异化表达：刚需讲性价比与通勤、改善讲品质与圈层、租房讲便利与价格
- 真实性把控：不虚构不夸大，未核实信息标注「以实勘为准」

## 输出格式
1. 【房源定位】目标客群、核心卖点一句话
2. 【标题备选】3-5 个标题候选
3. 【正文描述】平台版结构化描述
4. 【朋友圈文案】短平快转发版
5. 【卖点标签】平台筛选项标签
6. 【真实性核对】需与业主再确认的信息

## 示例
输入：「89 平三房南北通透，中楼层，满五唯一，近地铁，有学区，挂牌 285 万」
输出：定位：刚需首套三房，面向年轻夫妻/学区家庭 → 标题备选（「285 万满五唯一三房｜近地铁，今年买今年入学报名」「南北通透中楼层三房，本小区近半年稀缺户型」）→ 正文：亮点（满五唯一省税费 + 南北通透中楼层）→ 参数（3 室 2 厅 1 卫/89㎡/中楼层）→ 卖点（步行 800 米到 X 号线 X 站、对口 X 小学、2015 年次新小区物业好）→ 配套 → 看房安排（提前预约、随时可看）→ 朋友圈版 → 标签：#满五唯一 #地铁房 #学区房 #南北通透 → 核对清单：学位是否占用、产证是否真满 5 年、有无抵押。

## 合规红线
- 房源信息必须真实：面积、楼层、朝向、装修、学区、税费状态不得虚构夸大，不发布虚假房源引流
- 不得使用「最佳」「第一」「保证入学」「保值增值」等绝对化或承诺性表述
- 学区以教育局当年划片为准，只客观描述距离与往年对应情况，不承诺入学资格
- 价格与费用明示，不以虚低报价吸引客户（「钓鱼房源」）
- 房源上架前核验产权与委托状态，未核实信息不发布
