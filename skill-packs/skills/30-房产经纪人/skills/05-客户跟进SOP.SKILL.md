# 客户跟进SOP

## 适用场景
经纪人批量管理客源：新客首咨跟进、带看后回访、A/B/C 级客户周期维护、长期沉默客户激活、已成交老客户转介绍。用标准化 SOP 保证不漏客、跟进不骚扰。

## 核心能力
- 客户分级跟进频率：A 级（每周 2-3 次，以新房源/调价为由）、B 级（每周 1 次，价值内容 + 新盘推荐）、C 级（半月-1 月，轻触达养客）
- 跟进话题库：新上房源、同小区调价、成交数据播报、贷款政策变化、带看反馈复盘——而非每次问「考虑得怎样」
- 带看后跟进流程：当天发反馈总结 → 次日针对性再推荐 → 3 日内约复看
- 沉默客户激活理由：新政策（利率下调/限购放松）、本小区新上低价房、年度市场报告
- 老客户维护与转介绍：成交周年/交房周年问候，附「邻居想卖房可免费估价」的自然转介绍钩子
- 跟进日志规范：每次记录跟进内容与客户反馈，周复盘转化漏斗

## 输出格式
1. 【客户分级】分级标准与对应频率
2. 【跟进话题库】不同场景的触达理由与话术
3. 【带看后SOP】按时间节点的回访动作
4. 【激活方案】沉默客户触达理由与话术
5. 【转介绍机制】老客户转介绍时机与话术
6. 【日志规范】跟进记录字段与复盘要求

## 示例
输入：「手上 20 组客户，最近一个月只有 3 组看过房，帮我定跟进计划」
输出：分级：重新评级——3 组带看客户列 A/B 重点跟，17 组未带看按最新沟通重新分 A/B/C → 话题库：A 级用「本小区新上一套中楼层同户型，业主诚心卖」约复看；B 级发本月小区成交单价表 + 2 套新上房源；C 级发贷款利率下调政策解读轻触达 → 带看后 SOP：当天发带看总结（三套房优劣对比表），次日电话问顾虑，第 3 天约复看或调整匹配方向 → 激活：筛出 3 个月未联系客户，用「您关注的 X 小区出了套低于均价的房」破冰 → 转介绍：已成交客户发交房周年祝福，附「邻居想卖房免费上门估价」→ 日志：每次记录话题 + 客户反馈 + 下一步动作，每周一复盘 A 级客户进展。

## 合规红线
- 跟进频率适度，避免早晚休息时间高频打扰
- 跟进内容须真实，不编造虚假房源、虚假调价诱客
- 客户电话与信息严格保密，不泄露不转卖
- 不做无法兑现的承诺（「保证 X 天内买到」）
- 转介绍激励须合法合规，不返佣、不赠送不正当利益
