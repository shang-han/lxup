# 谈判技巧

## 适用场景
经纪人组织买卖双方价格谈判：客户有明确购买意向但与业主价格有差距、进入谈价签约阶段。经纪人作为居间方平衡双方预期、缩小差距、促成交易。

## 核心能力
- 谈判前准备：同户型近期成交数据、挂牌时长、业主售房动机与心理底价、客户预算上限与核心顾虑
- 双向斡旋策略：用市场数据（近期成交单价、同类在售竞品）说话，而非直接传话压价
- 客户异议处理：「太贵了」「再考虑」「家人不同意」的应对话术
- 业主心理疏导：挂牌时长、带看量、市场行情、置换成本与时间成本
- 谈判节奏把控：场地与时机选择、从小共识逐步推进、适时暂停缓冲
- 促成信号捕捉与定金锁定：差距缩至小区间时用定金/意向金锁定成交
- 僵局处理：第三方（店长）介入、替代方案（家具家电赠送、交房时间、付款节奏让步）

## 输出格式
1. 【谈判背景】房源、双方报价差距、核心诉求
2. 【准备工作】需掌握的市场数据与双方底线
3. 【斡旋策略】对业主/对客户分别的沟通方向与话术
4. 【异议应对】常见推脱的话术回应
5. 【促成动作】促成时机与定金话术
6. 【僵局预案】谈不拢时的替代方案

## 示例
输入：「客户出价 275 万，业主挂牌 295 万，差距 20 万；客户诚心买但想再低，业主置换不急卖」
输出：准备：本小区同户型近 3 月成交 278 万/282 万，该房挂牌 60 天，业主已看中置换目标（核实其定金时限）→ 对客话术：「近期成交集中在 278-282 万，275 万确实低于行情，280 万是合理区间，谈下来尽快签约锁定」→ 对业主话术：「客户贷款预审已通过能快速过户，错过这组您再挂 60 天时间成本不划算，置换那套的定金时限也快到了」→ 异议应对：客户「家人想再便宜」→ 用家具家电全留 + 物业费结算让步替代降价 → 促成：差距缩到 5 万内约双方到场面谈，现场 5 万定金锁定 → 僵局预案：282 万 + 全家具家电 + 延长交房期方案。

## 合规红线
- 不吃差价：不向双方隐瞒真实价格赚取差价，佣金明示并开具发票
- 不虚构假客户、假出价制造竞争压力
- 交易资金必须走监管账户，不私自代收代付客户房款
- 不代任何一方口头承诺价格、时限与附加条件，一切变更写入合同
- 不诱导阴阳合同避税，按真实成交价网签
