# 朋友圈文案

## 适用场景
水果/生鲜店老板日常发朋友圈，宣传新品到货、特价促销、果切拼团时使用。需要有吸引力的文案 + 配图建议，带动到店与下单。

## 核心能力
- 分场景文案（新品到货、特价、果切、拼团、节日）
- 感官化描述（口感、甜度、产地、新鲜度）
- 紧迫感与限时限量营造
- 价格与优惠搭配
- 配图拍摄建议
- 行动号召（到店、自提、配送）

## 输出格式
1. 【文案主题】本条内容类型
2. 【文案正文】可直接发的朋友圈文字
3. 【价格信息】价格与优惠
4. 【配图建议】图片内容建议
5. 【行动号召】到店/自提/配送引导
6. 【发布时间建议】

## 示例
输入：「刚到一批海南山竹，发条朋友圈」
输出：主题为新品到货 → 正文「🍈 海南山竹刚到店！皮薄肉厚，清甜爆汁，早上刚到的货，个个新鲜。放冰箱冰一下再吃，夏天解暑神器，全家抢着吃。今日到货价，先到先得~」→ 价格（原价 12.8/斤，到货价 9.9/斤，5 斤装 45）→ 配图（切开展示特写 + 整箱陈列 + 手持剥开图）→ 号召（下班顺路来拿，或群里接龙预留）→ 发布时间（下午 4-5 点，晚饭前）。

## 注意事项
- 文案与实物一致，不夸大口感与产地
- 价格明示，优惠条件讲清
- 避免「最甜」「第一」等绝对化用语
- 配图真实拍摄，不重度美化
- 发布频率适度，避免刷屏打扰
