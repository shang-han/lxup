# 季节性营销方案

## 适用场景
水果/生鲜店在季节更替（夏季西瓜节、冬季柑橘季）与节日（端午、中秋、春节礼盒）做营销时使用，带动销售与客流。

## 核心能力
- 按季节与节日策划主题
- 活动机制（限时特价、满减、拼团、礼盒预订）
- 应季选品建议
- 全渠道文案（朋友圈、社群、店内 POP）
- 执行分工与物料
- 复盘指标

## 输出格式
1. 【活动概览】主题、时间、目标、面向人群
2. 【活动机制】具体玩法与优惠规则
3. 【应季选品】主推品与搭配
4. 【宣传文案】朋友圈版 + 社群版 + POP 版
5. 【执行要点】分工、物料、时间节点
6. 【复盘指标】统计项清单

## 示例
输入：「夏天到了，策划一个西瓜促销活动」
输出：概览（主题「清凉西瓜节」，时间 2 周，目标引流走量）→ 机制（整瓜特价、切开尝鲜价 9.9/斤、买瓜送冰袋、群接龙整瓜第二件半价）→ 选品（8424、麒麟瓜引流，小凤瓜适合单人）→ 文案（朋友圈「夏天的快乐是西瓜给的🍉 8424 到店，脆甜多汁，今日尝鲜价 9.9/斤，下班带一个回家！」）→ 执行（门口堆头陈列、社群提前预告）→ 复盘（西瓜销量、客流、会员新增）。

## 注意事项
- 优惠真实，条件明示
- 应季品保证新鲜，次品剔除
- 避免「第一」「最甜」等绝对化用语
- 大促备货充足，避免缺货
- 活动后复盘，沉淀经验
