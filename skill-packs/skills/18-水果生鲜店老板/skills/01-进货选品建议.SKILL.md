# 进货选品建议

## 适用场景
水果/生鲜店老板进货决策时使用。需要判断进什么货、进多少、什么等级，结合季节、地域、客流与历史销量优化选品与进货量。

## 核心能力
- 按季节与地域推荐应季主打品
- 根据门店定位、客流、历史销量判断进货量
- 等级搭配（引流级 / 利润级 / 形象级）
- 品质挑选要点（成熟度、瑕疵、新鲜度）
- 供应商与渠道建议（批发市场、产地直供、社区团购供货）
- 新品与反季品的风险提示

## 输出格式
1. 【当季推荐】当前可进的应季品
2. 【品类结构】引流品 / 利润品 / 形象品搭配
3. 【进货量建议】各品建议进货量与依据
4. 【挑选要点】进货时的品质标准
5. 【渠道建议】进货渠道与成本参考
6. 【风险提示】易损品与注意事项

## 示例
输入：「现在夏天，我社区店日客流 300 人，怎么进货？」
输出：当季推荐（西瓜、甜瓜、葡萄、桃、荔枝、山竹）→ 品类结构（西瓜引流走量、葡萄桃做利润、车厘子榴莲做形象）→ 进货量（西瓜日进可大些周转快、荔枝易损少进勤进、山竹中量）→ 挑选要点（西瓜看瓜蒂与声音、葡萄看果梗青与果粉）→ 渠道（凌晨批发市场 + 产地直供西瓜）→ 风险（荔枝杨梅易褐变，少量多次）。

## 注意事项
- 易损品少量多次进货，宁可周转快不压货
- 进货严把品质关，残次品当场剔除
- 应季品性价比高，反季与外来品谨慎试量
- 进货量以实际销售数据调整，不凭感觉
- 关注食品安全，不进来源不明产品
