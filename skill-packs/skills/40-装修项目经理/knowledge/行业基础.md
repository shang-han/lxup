# 行业基础

## 行业概况
装修行业市场规模超 5 万亿，家装约占 60%，涵盖新房整装、旧房翻新、局部改造与工装。项目经理（工长）的核心职责是工期控制、质量管控、成本管理、客户沟通四条线并行。行业痛点集中在增项纠纷、工期延误、质量不达标与售后扯皮。竞争格局上，连锁装企拼供应链与标准化，中小装企和施工队靠口碑转介绍与性价比生存，「透明报价 + 过程留痕 + 准时交付」是复购与转介绍的根本。

## 专业技能体系

### 硬技能
- 施工图纸识读与放线定位
- 水电/泥瓦/木工/油漆各工种协调
- 材料预算与采购（瓷砖/板材/涂料/管线）
- 工艺标准与验收要点（空鼓/平整度/水电检测）
- 工期管理与人效计算

### 软技能
- 业主需求沟通与预期管理
- 工人团队管理

## 常用术语
- 隐蔽工程：水电、防水等覆盖后不可见的工程，必须先验收再封闭
- 竣工验收：按标准全面检查工程质量的最后环节，通过后交付
- 增项：合同约定范围之外的额外工程项目，须书面确认
- 工期：从开工到竣工的日历天数，一般按合同约定含顺延情形
- 质保期：装修公司对工程质量的保修期限（通常基础工程 2 年、防水 5 年）
- 半包/全包：半包为人工+辅材、主材业主自购；全包为人工+辅材+主材
- 闭水试验：防水层完成后蓄水 24-48 小时检验是否渗漏
- 成品保护：施工中对门窗、电梯、已完工项目的保护措施

## 核心工作流程
开工前：现场勘查 → 设计交底 → 报价签约 → 材料进场 → 施工准备
施工中：拆改 → 水电 → 防水（闭水试验）→ 泥瓦 → 木工 → 油漆 → 安装，各工种完工即验收
竣工后：整体验收整改 → 保洁 → 软装摆场 → 交付交底 → 质保售后

## 常见问题
Q：工期延迟了怎么办？
A：先分清责任方（业主变更、材料延误还是施工安排问题），协调资源抢关键路径工序，必要时签订书面延期协议顺延工期，主动通报胜过被动解释。

Q：客户想增项但不想加钱怎么办？
A：签约时就讲清增项规则并写入合同，增项出现时出示明细报价与必要性说明，坚持「先书面确认价格再施工」，可以协商优惠但不开免费先例。

Q：客户对比别家报价低很多怎么谈？
A：逐项对比工艺与材料口径，指出低价常见的漏项（水电按实、拆旧、找平），强调按实测计价与增项上限承诺，用透明换信任。

Q：隐蔽工程怎么避免后期纠纷？
A：水电防水覆盖前必须验收、试压试水、拍照留档并发客户确认，管路走向图存档备查。
