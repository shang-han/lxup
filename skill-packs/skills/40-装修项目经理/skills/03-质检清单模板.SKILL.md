# 质检清单模板

## 适用场景
工种完工后项目经理自检、陪客户或第三方监理验收，以及日常工地巡检时使用标准化检查表，确保质量问题在覆盖前被发现和整改。

## 核心能力
- 分工种清单模板：水电（打压试验、线路分路、强弱电间距）、防水（涂刷高度、闭水试验）、泥瓦（空鼓率、平整度、阴阳角、排水坡度）、木工（水平度、接缝、防火防腐）、油漆（无流坠透底、阴阳角顺直）
- 每个检查点给出检查方法、工具与允许偏差值（如墙砖空鼓率≤5%、墙面平整度≤3mm）
- 区分必检红线项（用电安全、防水、排水坡度）与一般项目
- 问题整改记录模板：问题部位 + 照片 + 整改要求 + 责任人 + 限期
- 输出可打印的一页式验收单，含签字确认栏

## 输出格式
1. 【验收信息】房屋、工种、验收日期、参加人员
2. 【检查清单】序号、检查点、标准、检查方法、结果（合格/不合格）
3. 【红线项】必须通过方可进入下道工序的项目
4. 【问题清单】问题部位、描述、整改要求与限期
5. 【结论】合格 / 整改后复验，各方签字栏

## 示例
输入：「明天陪客户验收瓷砖，给我一份检查清单」
输出：验收信息表 → 12 项清单（空鼓率用空鼓锤逐块敲击≤5%、平整度 2m 靠尺≤3mm、接缝直线度、阴阳角方正、厨卫地漏坡度泼水试验……）→ 红线项（湿区无连续空鼓、地漏排水顺畅无积水）→ 问题清单模板（部位/照片/整改限期）→ 结论与客户签字栏。

## 注意事项
- 隐蔽工程（水电、防水）必须在覆盖前验收并拍照存档，覆盖后难以追责
- 空鼓率、平整度等用工具实测实量，不凭目测口头说「差不多」
- 不合格项拍照留证、书面明确整改责任与期限，整改后复验闭环
- 陪客户验收时现场讲解标准与结果，整改承诺落到文字，避免竣工时翻旧账
