# 报价方案生成

## 适用场景
装修公司或施工队向客户出具透明、可对比的报价单（新房整装、旧房翻新、局部改造），或老板审核施工队报价是否存在漏项、模糊项，从源头减少结算纠纷。

## 核心能力
- 按装修类型与面积生成报价骨架：拆改 → 水电 → 防水 → 泥瓦 → 木工 → 油漆 → 安装 → 管理费/税金
- 逐项细化：材料品牌规格、单位、工程量、人工单价、综合单价与合价
- 说明易争议项的计价口径：水电按米实测还是包干、防水涂刷面积、拆墙垃圾清运、异形造型加价
- 附增项规则：增项范围界定、单价口径、必须客户书面确认（微信/纸质签证单）后方可施工
- 输出明细版（合同附件）与汇总版（客户快速浏览）两个版本
- 可对比竞品报价，提示常见「低价陷阱」项（漏报水电、拆旧、找平、成品保护）

## 输出格式
1. 【项目信息】房屋地址、面积、户型、装修类型、报价日期与有效期
2. 【报价汇总】各工种小计、管理费、合计总价
3. 【分项明细】逐项材料品牌规格、单位、数量、单价、合价
4. 【计价说明】水电等弹性项的计算方式与上限约定
5. 【增项约定】可能增项的范围、定价原则与书面确认流程

## 示例
输入：「89 平新房整装，中档配置，帮我做个报价」
输出：项目信息（89㎡三室一厅、整装中档、有效期 15 天）→ 汇总（拆改 0.8 万 / 水电 2.2 万 / 防水泥瓦 3.5 万 / 木工 1.8 万 / 油漆 1.5 万 / 安装 1.2 万 / 管理费 8%，合计约 12.1 万）→ 分项明细（如防水：东方雨虹、湿区 1.8m 高、45 元/㎡……）→ 计价说明（水电现场实测、按米计价、预估上浮不超 10%）→ 增项约定（增项须微信或纸质签证单确认后施工，累计增项不超合同总价 5%）。

## 注意事项
- 报价明细到材料品牌、型号、规格与单价，杜绝「一项、按实结算」式模糊条目
- 水电、拆旧等易超支项提前约定计算方式与价格上限，并写入合同
- 所有增项必须先书面告知价格、经客户确认后再施工，口头约定一律不认
- 不以明显低于成本的报价签单再靠增项回补，这是投诉与差评的最大来源
- 报价单注明有效期与「最终以实测工程量结算」口径，避免被无限期追溯
