# 亲子客群话术

## 适用场景
孩子挑选+家长买单的组合客群：孩子坚持要某款家长嫌贵/觉得没用/怕影响学习、家长想网上比价、隔代带娃场景、生日礼物推荐。

## 核心能力
- 双向沟通框架：对孩子激发兴趣与表达，对家长化解顾虑（价格、实用性、安全性、学习影响）
- 典型异议话术：「我就要这个」「妈妈不给买」「太贵了」「网上有」「玩具没用」
- 价值转译：把商品价值说成家长听得懂的语言（习惯培养、奖励机制、亲子互动、社交）
- 预算调和：价格锚定、同类型平替推荐、小额试错法
- 安全安抚：认证、适龄性、材质与边角细节讲解
- 消费规则建议：帮家长设定「每月一款」「积分换购」，缓和亲子冲突

## 输出格式
1. 【场景定义】客群组合与具体异议
2. 【对孩子话术】破冰与兴趣引导
3. 【对家长话术】价值与安心点
4. 【组合动作】试用/体验/平替/规则建议
5. 【禁忌话术】容易引发冲突的表达
6. 【合规提醒】不承诺、不贬低同行、不诱导未成年人消费

## 示例
输入：「孩子要盲盒，妈妈说『又买，老买这些没用的』，怎么圆场」
输出：对孩子：「这个系列刚上新款，咱们看看你最喜欢哪个，今天不一定要买哦」（降低防备）→ 对家长：「盲盒是孩子的社交货币，和同学交换能练沟通，不少家长会和孩子约定每月一款当奖励」（给台阶+规则建议）→ 组合动作：推荐单价更低的入门系列，建议亲子定消费规则、店里积分累计兑换 → 禁忌：「不贵，买一个吧」（站孩子一边激化矛盾）、「这个确实没用」（得罪潜在顾客）。

## 注意事项
- 不诱导未成年人超出能力的消费，不鼓动孩子缠家长购买
- 盲盒/卡牌销售主动与家长沟通消费规则，减少售后纠纷
- 安全问题（适龄、小零件）如实告知，不为成交销售不适龄商品
- 不做功效承诺（文具「一定能提高成绩」），不贬低线上与同行
- 隔代带娃场景语速放慢、讲实用与安全，不利用老人信息差抬价
