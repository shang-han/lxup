# 库存动销分析

## 适用场景
定期盘点、滞销品处理、补货决策、开学季后库存去化、IP产品止损、断货与压货平衡。

## 核心能力
- 库存健康分类：按品类（书写/绘画/手工/办公/玩具）评估动销率，ABC分类法
- 补货建议：安全库存与补货点（日均销量×补货周期+安全库存）、热销款应急补货渠道
- 滞销分级处理：满减促销→捆绑销售→折扣清仓→退换货→报损
- IP产品止损：热度周期短，30天不动销停止补货，退热信号出现立即去化
- 季节节奏：开学季备货、节假日爆发、考试季、寒暑假库存安排
- 盘点模板：字段设计（商品/规格/库存量/成本/动销天数/处理建议）与差异处理

## 输出格式
1. 【库存概况】各品类库存与动销评估
2. 【补货清单】商品/建议数量/紧急程度
3. 【滞销处理】逐项分级方案
4. 【动销建议】采购节奏与止损规则
5. 【盘点模板】表格字段与使用说明
6. 【重点提醒】IP品与季节品风险

## 示例
输入：「开学季过完还压了200套IP文具礼盒，怎么处理」
输出：评估：IP礼盒季节性与潮流性强，滞销风险高 → 分级处理：第一周搭热门盲盒满减去化 → 第二周会员日5折清仓 → 第三周谈供应商退换或赠老客 → 动销建议：明年开学季IP礼盒备货减20%，留30%预算应急补货 → 止损规则：IP类30天不动销即停补 → 附盘点模板字段。

## 注意事项
- 盘点至少每月一次，热销款与IP品每周快速核对
- 滞销处理要果断，折价比压货占资金划算，先算资金成本
- 退换货以采购合同约定为准，处理时按条款执行并留痕
- IP品遵循「少量试、快速补、及时止」，不主观赌潮流
- 开学季刚需余货可留常规期销售，季节性强品类及时去化
