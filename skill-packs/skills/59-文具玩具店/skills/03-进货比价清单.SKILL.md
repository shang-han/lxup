# 进货比价清单

## 适用场景
日常补货比价、新品寻源（社交平台爆款、展会新品）、供应商筛选与谈判、采购成本优化。

## 核心能力
- 渠道对比：批发市场、1688/线上批发、品牌代理、IP授权商、展会订货会
- 比价表模板：商品/规格/单价/起订量/运费/退换货条款/账期字段
- 新品寻源方法：社交平台爆款反查货源、展会新品首发、品牌订货会、同行调货
- 起订量与现金流平衡：小批量多频次与大单折扣的拆分采购策略
- 质量核验：3C认证、厂名厂址、检验报告、IP授权文件
- 谈判要点：残次品退换比例、账期（现结/月结）、价保、新品试样提供

## 输出格式
1. 【采购清单】商品/规格/数量需求
2. 【渠道对比】各渠道优劣势对比
3. 【比价表】各渠道单价/起订量/运费/条款
4. 【推荐方案】采购拆分与下单建议
5. 【质量核验】证照与授权检查清单
6. 【谈判要点】退换、账期、价保条款

## 示例
输入：「想进一批中性笔和笔记本，帮我对一下批发市场和网上的价」
输出：比价表：中性笔0.5黑——批发市场0.8元/支起订100支当日可提 vs 1688 0.55元/支起订500支另加运费 → 笔记本B5田字格——批发3.2元/本 vs 线上2.6元/本起订200本 → 建议：主流规格走批发市场（补货快、可验货），大流通单品走线上省成本 → 核验：查笔类厂名厂址与生产日期（笔芯有保质期），本册查执行标准 → 谈判：残次退换3-5%、新品供试样。

## 注意事项
- 不只看单价，综合计算运费、起订量、资金占用与断货风险
- 新供应商先小批量试单，验货验资质后再放量
- IP产品必查授权文件，不因低价进仿品盗版
- 比价表与票据留存，每月跑一次市场校价，建立供应商台账
- 爆款寻源先验真伪，警惕「原单」「工厂直供」话术
