# 开学季备货方案

## 适用场景
春秋两季开学前的集中备货期：备多少、备什么、卖不完怎么办、开学活动怎么引流。

## 核心能力
- 备货节奏：T-14天首批到货、T-7天陈列试销、开学周爆发、T+14天长尾补货
- 分学段需求清单：幼儿园（美术/手工）、小学（HB铅笔/田字本/书包/修正带）、中学（中性笔/笔记本/文件夹/便利贴）
- 库存预估方法：周边学生数 × 预估进店率 × 客单价 × 备货系数，按品类拆分
- 选品结构：引流品（本册/笔类透明价）+ 利润品（IP套装/礼盒文具）+ 形象品（高端书包）
- 促销设计：开学大礼包、满减、第二件半价、新会员礼
- 余货处理预案：阶梯降价、捆绑销售、供应商退换、常规期消化

## 输出格式
1. 【备货时间线】各阶段目标与动作
2. 【需求清单】按学段的品类/规格/数量
3. 【库存预估表】各品类预估量与测算依据
4. 【选品与定价】引流/利润/形象品组合
5. 【促销方案】大礼包与活动机制
6. 【余货处理】开学后去化预案

## 示例
输入：「我店在小学旁边，30平，开学季要备多少货」
输出：节奏：开学前2周首批到货（刚需70%），留30%预算应急补货 → 小学清单：HB铅笔×500支、田字本×800本、书包×50个、修正带/橡皮足量 → 预估：周边1000名学生 × 40%进店率 × 25元客单价 ≈ 1万元备货 → 选品：本册笔类引流、IP文具套装做利润 → 促销：39元/59元大礼包 + 第二件半价 → 余货：IP套装开学后搭赠清货，刚需本册常规期消化。

## 注意事项
- 预估仅为参考，结合往年同期销售与周边学校招生变化调整
- 刚需大流通品备足，IP与新奇品类保守备货，避免积压
- 备货前核对产品厂名厂址、3C认证与适用年龄，不因低价进三无货
- 预留货架与现金流，不一次备满，热销款提前锁定应急补货渠道
- 促销先算毛利，大礼包定价不低于组合成本
