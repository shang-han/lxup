# 促销文案生成

## 适用场景
便利店/超市做特价、满减、会员日、节日促销、新品推广时，店长需要配套的促销文案，包括货架价签、店内 POP 海报、社群/朋友圈推送，做到吸睛、清楚、合规。

## 核心能力
- 多渠道文案：价签短语、POP 标题、社群推文、朋友圈、广播稿
- 突出利益点：直降、满减、买赠、会员专享的醒目表达
- 营造紧迫感：限时、限量、仅此 X 天
- 分品类调性：生鲜新鲜感、零食解馋、日用实惠
- 不同字数版本：一句话标语到完整推文
- 合规措辞：规避夸大与绝对化用语

## 输出格式
1. 【活动主题】一句话主题
2. 【价签短语】货架用的简短促销语
3. 【POP 文案】海报标题 + 副标 + 利益点
4. 【社群/朋友圈】完整推送文案
5. 【备选版本】2-3 个不同风格

## 示例
输入：「周末会员日，鸡蛋和牛奶做特价，写文案」
输出：主题「会员狂欢日·鲜品直降」→ 价签：「会员专享 鲜鸡蛋 直降 3 元」→ POP：「本周六日 会员日｜鸡蛋牛奶 限时特惠 错过等下周」→ 朋友圈：「🥚会员日来啦！本周六日，鲜鸡蛋、纯牛奶会员专享价，新鲜到家，记得带会员卡哦~仅限两天！」→ 备选「省钱攻略｜会员日必囤的 3 样」。

## 合规红线
- 促销价格真实，不得先提价再打折、虚构原价
- 规避《广告法》禁用词（「最」「第一」「全网最低」等）
- 限时限量须真实，明示活动起止时间与规则
- 会员价、满减规则表述清晰，避免误导消费者
