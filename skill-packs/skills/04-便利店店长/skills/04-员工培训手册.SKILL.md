# 员工培训手册

## 适用场景
便利店/超市店长需要为新员工和兼职人员编写标准化培训手册，让新人快速掌握收银、理货、商品知识、服务规范和防损要点，缩短上岗周期、保证服务一致性。

## 核心能力
- 岗位 SOP：收银、理货、上架、清洁的分步操作
- 商品知识：品类分区、价签规则、保质期识别
- 服务规范：迎宾、待客、收银话术、投诉应对
- 防损与安全：防盗、消防、应急处理
- 系统操作：收银机、会员、促销的操作要点
- 上岗考核：应知应会的检查清单

## 输出格式
1. 【手册目录】分模块的培训框架
2. 【岗位 SOP】各岗位操作步骤与标准
3. 【服务规范】待客话术与行为准则
4. 【商品与系统】商品知识与设备操作
5. 【安全防损】防盗消防与应急
6. 【考核清单】上岗应知应会检查表

## 示例
输入：「做一份新员工三天上岗培训手册」
输出：三天计划 → Day1 门店制度 + 商品分区 + 收银系统操作 → Day2 理货上架 SOP（先进先出、价签核对）+ 服务话术 → Day3 防损消防 + 投诉应对 + 实操考核 → 每模块含步骤与标准 → 附考核清单：收银零差错、会处理退换货、熟悉保质期检查，由店长签字上岗。

## 注意事项
- 食品安全（效期检查、生熟分开、冷链）列为必修重点
- 收银与钱财岗位强调操作规范与防损纪律
- 内容贴合门店实际，多用图示和步骤少用空话
- 保留培训与考核记录，兼职人员同样需培训上岗
