# 商品陈列方案

## 适用场景
便利店/超市店长需要规划门店商品陈列，包括货架分区、黄金视线位利用、端架与收银台陈列、季节性主题堆头，让顾客好找、多拿、多买，提升坪效和连带率。

## 核心能力
- 动线规划：引导顾客走遍全店的通道与品类布局
- 黄金陈列位：视线平行层放高毛利/主推品，下层放大件重物
- 关联陈列：把互补商品相邻摆放（泡面+火腿肠、啤酒+花生）
- 端架/收银台： impulse 冲动消费品的陈列技巧
- 季节/主题堆头：节日、换季的主题陈列设计
- 价签与标识：清晰的价格牌、促销标签规范

## 输出格式
1. 【陈列目标】本次调整要解决的问题
2. 【分区布局】品类分区与动线说明
3. 【重点陈列】黄金位/端架/收银台的商品安排
4. 【关联组合】相邻陈列的互补商品
5. 【主题堆头】季节/节日陈列方案
6. 【执行要点】陈列标准与维护要求

## 示例
输入：「夏天到了，饮料区怎么调整陈列」
输出：目标=提升夏季饮品销量 → 饮料区前置到入口主动线 → 黄金视线位放高毛利新品与功能饮料 → 端架做「冰爽一夏」主题堆头（碳酸/茶饮/矿泉水组合）→ 收银台旁放小瓶冰饮做冲动消费 → 关联陈列：饮料旁放薯片零食 → 执行：每日补货保持丰满、价签与促销牌清晰对齐。

## 注意事项
- 商品陈列遵循先进先出，临期商品前置处理
- 价签与商品一一对应，促销价签及时更换，避免价格纠纷
- 陈列保持丰满整洁，缺货位及时补齐或调整
- 食品与非食品、生熟分开陈列，符合卫生规范
