# 行业基础

## 行业概况
便利店行业以 7-11、全家、罗森等外资品牌为标杆，国内本土品牌（美宜佳、便利蜂等）快速崛起，全国门店数量超 30 万家。单店日均销售额约 5000-8000 元，核心经营指标为毛利率、库存周转率和来客数。行业特点是高频刚需、客单价低、SKU 多、鲜食占比提升，盈利依赖精细化运营：选址决定客流、商品结构决定毛利、周转决定损耗。当前趋势是「便利店 + 鲜食 + 即时零售（线上到家）」融合，社区型门店靠便利性和熟客经营立足。

## 专业技能体系

### 硬技能
- 商品SKU管理与周转率优化
- 收银系统操作与日结对账
- 供应商谈判与进货周期管理
- 货架陈列与动线设计（黄金视线层）
- 临期商品处理策略

### 软技能
- 社区熟客经营
- 防损防盗意识

## 常用术语
- 周转率：商品销售速度，通常用周转天数衡量库存健康度
- 毛利率：销售毛利占营业额的比例
- 来客数：每日进店消费的顾客人次
- 客单价：平均每单消费金额
- 缺货率：顾客需要时商品断货的比例，影响销售与体验
- 坪效：每平米营业面积创造的销售额
- 动销率：有销售的 SKU 占全部 SKU 的比例
- 鲜食占比：盒饭、关东煮、面包等鲜食销售占比（高毛利品类）

## 核心工作流程
每日：收货验货 → 商品上架陈列 → 价签核对 → 鲜食效期检查 → 卫生清洁 → 日结对账
每周：库存盘点 → 过期商品下架 → 滞销分析 → 补货订货计划
每月：销售数据分析 → 供应商对账 → 促销规划 → 商品汰换

## 常见问题
Q：过期商品怎么避免？
A：严格执行 FIFO 先进先出原则，每日检查保质期，临期商品提前 1 个月预警并做特价处理。

Q：偷盗怎么防范？
A：优化监控布局消除死角，贵重商品加锁或贴防盗扣，提高员工巡店密度，收银规范核对。

Q：怎么提高客单价？
A：做好关联陈列和收银台冲动消费区，推会员满额优惠，培训员工关联推荐话术。

Q：鲜食损耗大怎么办？
A：按历史销量精准订货、少量多次，临期时段打折促销，分析报废数据持续优化订货量。
