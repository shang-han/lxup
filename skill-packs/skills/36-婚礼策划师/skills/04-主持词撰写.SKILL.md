# 主持词撰写

## 适用场景
婚礼策划师/司仪为新人定制主持词，或新人准备父母致辞、新人答谢词时使用。根据新人的故事与婚礼调性撰写既有仪式感又真挚不尴尬的台词，避免千篇一律的套话。

## 核心能力
- 全流程台词：开场白、新人入场、证婚、誓言引导、交换戒指、拥吻、父母环节、感恩环节、祝酒、敬酒串词
- 风格适配：庄重典雅、温馨感人、轻松幽默、新中式古风，匹配婚礼调性与宾客构成
- 个性化定制：把新人的相识故事、职业特征、共同爱好写进台词，制造专属记忆点
- 节奏控制：台词配合音乐 cue 点与环节时长，标注停顿、互动与情绪起伏
- 互动设计：引导宾客鼓掌、举杯、见证的参与感话术
- 父母/新人致辞模板：双方父母致辞、新人答谢词的结构与范文

## 输出格式
1. 【风格定位】主持词基调与适配理由
2. 【完整台词】按仪式环节顺序撰写，标注【音乐】【停顿】【互动】
3. 【个性化段落】融入新人故事的专属内容
4. 【致辞模板】父母致辞/新人答谢词（如需要）
5. 【司仪提示】临场应变与情绪把控要点

## 示例
输入：「新人恋爱 7 年，男方是程序员，写温馨带点幽默的主持词」
输出：开场（温馨）：「有人说陪伴是最长情的告白，今天这对新人用七年证明了这一点……」→ 新郎入场词 → 新娘入场与交接（庄重，父亲托付）→ 誓言引导 → 个性化段落：「作为程序员，他写过无数行代码，但他说，遇到她，是人生唯一一次不需要 debug 的幸运」→ 交换戒指 → 父母环节（感恩，备纸巾提示）→ 祝酒 → 司仪提示：父亲交接环节控制情绪节奏，给足停顿与掌声时间。

## 注意事项
- 台词真挚优于华丽，堆砌辞藻不如讲一个真实细节动人
- 幽默要有分寸，不开低俗玩笑、不拿新人短板调侃、不让任何一方父母难堪
- 提前与新人核对故事细节与禁忌（如离异家庭措辞、不想提的过往）
- 台词留出音乐与掌声的气口，标注 cue 点，避免干讲
- 准备应急串词：设备故障、环节延误时的圆场话术
