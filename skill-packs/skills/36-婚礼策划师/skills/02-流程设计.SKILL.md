# 流程设计

## 适用场景
婚礼策划师为新人编排婚礼当天的执行流程、统筹彩排，或新人自行核对流程完整性时使用。把一场婚礼拆成分钟级的执行表，让四大金刚、伴郎伴娘、酒店各方各司其职，确保当天零纰漏。

## 核心能力
- 全天流程编排：化妆 → 迎亲 → 外景 → 迎宾 → 仪式 → 宴席 → 敬酒 → 送客，按吉时与习俗排布
- 分钟级时间表：每个环节的起止时间、负责人、配合人员、所需物料
- 仪式环节设计：入场、证婚、交换戒指、誓言、拥抱、父母环节的时长与衔接
- 人员分工表：伴郎伴娘、督导、花童、戒童、双方父母的站位与任务
- 应急预案：流程延误、设备故障、天气突变、人员缺席的 Plan B
- 彩排清单：彩排走位、音乐 cue 点、灯光配合的核对项

## 输出格式
1. 【全天时间表】按时间顺序的环节清单（时间 | 环节 | 负责人 | 物料）
2. 【仪式流程】仪式区逐环节细化（含音乐 cue 点）
3. 【人员分工】各角色任务与站位
4. 【物料清单】当天必备物品核对表
5. 【应急预案】高频风险与应对方案

## 示例
输入：「10 月 2 日午宴，帮我排婚礼当天流程」
输出：时间表：06:00 新娘化妆 / 08:08 新郎出发迎亲 / 08:58 到新娘家（堵门游戏控制在 20 分钟内）/ 10:08 接亲回新房 / 11:00 酒店迎宾 / 12:08 仪式开始 / 12:40 宴席开席 / 13:20 新人换装敬酒 → 仪式细化：开场灯光秀 2 分钟 → 新郎入场 → 新娘与父亲入场（音乐 cue：Canon）→ 交接 → 誓言 → 交换戒指 → 拥吻 → 父母上台 → 全家举杯，全程控制在 32 分钟 → 分工表 + 物料清单（戒指、誓言卡、红包、备用丝袜……）→ 预案：仪式超 40 分钟则砍掉父母致辞改为敬酒时说。

## 注意事项
- 所有时间按「提前 10 分钟」预留缓冲，婚礼当天必然有延误
- 迎亲堵门游戏提前与伴娘约定时长上限，防止误吉时
- 仪式时长控制在 30-40 分钟，过长宾客注意力下降
- 关键物料（戒指、誓言卡、身份证）指定专人保管并双份核对
- 彩排必须走全流程含音乐灯光 cue 点，不能只口头过一遍
