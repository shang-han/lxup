# 供应商沟通

## 适用场景
婚礼策划师统筹婚礼供应商（四大金刚、酒店、花艺、车队、甜品、灯光音响），或新人自行对接供应商时使用。确保各方需求对齐、档期锁定、合同无坑、当天配合顺畅，是婚礼统筹能力的核心体现。

## 核心能力
- 供应商清单管理：按类别建立对接台账（联系人、档期、报价、合同状态、付款进度）
- 四大金刚对接：司仪/摄影/摄像/化妆的选标准（看客片、问档期、定人员）与沟通要点
- 场地对接：酒店档期、菜单试菜、桌数浮动条款、进场时间、电力与层高、服务费与进场费
- 合同审核要点：人员锁定条款（防止临时换人）、违约责任、超时费、退款条件
- 排期协同：把各供应商的到场时间、搭建时间、撤场时间编入总时间表
- 沟通话术：询价、压价、确认细节、催进度、处理临时变更的标准表达

## 输出格式
1. 【供应商清单】类别、候选、档期、报价、状态台账
2. 【对接要点】每类供应商必须确认的关键问题清单
3. 【合同审核项】重点条款与风险提示
4. 【排期协同表】各供应商当天到场与作业时间
5. 【沟通话术】询价/确认/变更场景的参考表达

## 示例
输入：「下个月婚礼，四大金刚和酒店还没完全敲定，帮我列对接清单」
输出：台账模板（含摄影/摄像/化妆/司仪/酒店/花艺/车队 7 类）→ 对接要点：摄影摄像务必确认「客片而非样片 + 指定摄影师本人 + 双机位 + 交片张数与精修数量 + 快剪是否包含」；酒店确认「菜单可试菜 + 桌数上下浮动 10% 按实结算 + 免收开瓶费写进合同 + 婚庆进场时间提前 4 小时」→ 合同审核：所有指定人员写进附件，换人需经新人书面同意否则退一赔一 → 排期协同：化妆师 6:00 到新娘家，花艺布置前晚进场，车队 8:00 集合 → 询价话术示例。

## 注意事项
- 指定人员必须写进合同附件，防止「样片是总监、当天来新手」的调包
- 口头承诺全部落到书面，微信确认也算证据，重要条款进合同
- 档期确认后尽快付定金锁定，旺季（五一/十一）手慢无
- 付款按服务节点分期，留尾款在当天服务完成后结清
- 建立供应商群统一同步时间表与变更，避免多方信息差
