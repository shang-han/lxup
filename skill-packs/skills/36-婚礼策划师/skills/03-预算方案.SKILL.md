# 预算方案

## 适用场景
婚礼策划师帮新人做预算规划与费用把控，或新人婚前盘点开支时使用。在总预算约束下合理分配各项花费，既保证婚礼效果又不超支，并识别隐藏费用与砍价空间。

## 核心能力
- 预算结构模型：婚宴餐饮、场地布置、四大金刚（司仪/摄影/摄像/化妆）、珠宝首饰、婚纱礼服、喜糖请柬、其他机动，按经验比例分配
- 档位方案输出：同一总额给出「性价比版 / 均衡版 / 品质版」的结构差异
- 优先级排序：识别新人最在意的环节倾斜预算，次要环节压缩
- 隐藏费用提示：开瓶费、服务费、超时费、进场费、异地交通住宿等易漏项
- 砍价与省钱策略：淡季/工作日折扣、套餐打包、DIY 替代项、熟人渠道
- 预算跟踪表：项目、预算额、实际支出、差额的动态管理模板

## 输出格式
1. 【总预算与原则】总额、分配原则、优先级
2. 【预算分配表】各大项与细分项的金额与占比
3. 【档位对比】不同预算等级的效果差异说明
4. 【隐藏费用清单】容易漏算的费用项
5. 【省钱建议】不影响效果的降本方案
6. 【预算跟踪表】可填写执行的管理模板

## 示例
输入：「总预算 15 万，在二线城市办 20 桌，帮我分配」
输出：分配原则：婚宴与布置占大头、四大金刚不省（影像一辈子看一次）→ 分配表：婚宴餐饮 7.5 万（20 桌 ×3500 + 酒水）/ 场地布置 3 万 / 四大金刚 1.8 万（司仪 5000 + 双机摄影 6000 + 双机摄像 5000 + 化妆 2000）/ 婚纱礼服 1 万 / 珠宝三金 1 万 / 喜糖请柬伴手礼 0.5 万 / 机动 0.2 万 → 隐藏费用：酒店服务费 10%、婚庆进场费、外景用车 → 省钱建议：避开五一/十一选 9 月吉日可省 15% 婚宴费，请柬电子化省 2000 元 → 附预算跟踪表模板。

## 注意事项
- 必须预留 5%-10% 机动金，婚礼几乎必然超预算
- 签合同前逐项确认费用包含范围（如摄像是否含快剪、布置是否含撤场）
- 四大金刚看客片而非样片，低价团队翻车成本远高于差价
- 口头优惠一律写进合同，定金与尾款节点明确
- 尊重双方家庭的预算参与度，大额支出提前与父母沟通避免矛盾
