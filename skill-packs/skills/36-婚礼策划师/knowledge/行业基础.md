# 行业基础

## 行业概况
婚礼市场规模超万亿元，平均婚礼花费 10-30 万元，一站式婚礼会馆与独立策划工作室并存。策划师核心能力是创意设计 + 统筹执行 + 预算控制 + 应急处理的综合体。行业季节性强，五一、十一、春节前后为旺季，档期资源紧张。近年趋势：年轻人更重仪式感与个性化、轻婚礼/目的地婚礼兴起、预算分配从「讲排场」转向「重体验与影像」。

## 常用术语
- 四大金刚：司仪、摄影、摄像、化妆师，婚礼当天的核心服务团队
- 场地布置：婚礼现场的装饰设计，含花艺、灯光、背景、道具
- 婚礼流程：迎亲、仪式、宴席、敬酒等环节的时间编排
- 预算分配：常见比例约为婚宴餐饮 40% / 布置 20% / 四大金刚与人员 20% / 其他 20%
- 婚庆套餐：策划公司打包服务的标准化产品，按档位定价
- First Look：新人仪式前第一次见对方穿婚纱/礼服的惊喜环节
- 督导：婚礼当天统筹各环节执行、提醒新人与工作人员的现场执行人员
- 进场费：酒店向婚庆公司收取的施工进场费用，易被新人漏算

## 核心工作流程
接单：需求沟通 → 创意提案 → 预算规划 → 签约收定金
筹备：场地选定与试菜 → 供应商对接与档期锁定 → 流程与布置设计 → 物料准备 → 彩排
执行：当天统筹（按分钟级时间表推进）→ 应急预案启动 → 送客收尾 → 尾款结算与复盘

## 常见问题
Q：预算有限怎么保证效果？
A：抓重点——预算向仪式区和影像（四大金刚）倾斜，简化次要环节；善用 DIY（电子请柬、手作伴手礼）；避开五一/十一旺季选吉日，成本可省 15%-30%。

Q：下雨天户外婚礼怎么办？
A：筹备期即制定 B 方案（室内备用区或帐篷），写进合同；当天提前 4-6 小时决策是否切换，并安排专人通知宾客与供应商调整布置。

Q：供应商当天临时换人怎么办？
A：合同中写明指定服务人员及违约条款（换人需书面同意，否则退一赔一）；保留尾款在当天服务完成后结算，作为约束筹码。
