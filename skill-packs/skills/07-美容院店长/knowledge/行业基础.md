# 行业基础

## 行业概况
美容行业市场规模超万亿，涵盖面部护理、身体 SPA、抗衰养护等项目。女性消费者占 90% 以上，注重体验和服务，消费决策受口碑和效果影响大。生活美容（非侵入性皮肤护理、身体养护）与医疗美容（注射、光电、手术类）有严格界限，生活美容机构不得开展医美项目。行业以预付费会员卡模式为主，核心竞争力：技术 + 服务 + 客情维护。门店盈利靠老客复购与升单，拓客成本高，私域运营（朋友圈、社群）是关键抓手。

## 常用术语
- 客单价：顾客平均每次消费金额
- 耗卡率：会员卡内金额/次数的消耗速度，反映到店活跃度
- 升单率：从低价项目向高价项目、从单项向疗程转化的比例
- 续卡率：会员卡到期后续费的比例
- 到店率：会员月均到店次数
- A 客/B 客/C 客：按消费能力和贡献度的客户分级
- 平均客龄：顾客从首次到店到流失的平均时长

## 核心工作流程
每日：客户预约确认 → 到店接待 → 皮肤检测与需求沟通 → 项目服务 → 效果确认与家居建议 → 预约下次 → 客情维护（回访/朋友圈互动）
每周：产品耗材盘点 → 员工手法与话术培训 → 营销活动策划与执行 → 沉睡客户唤醒
每月：经营数据分析（耗卡、续卡、升单）→ 业绩复盘 → 下月活动与目标规划

## 常见问题
Q：客户不续卡怎么办？
A：先分析原因（效果不达预期/服务体验/价格/搬迁），再针对性给专属续费优惠 + 增值服务 + 效果回访，切忌临期才想起联系。

Q：员工推销太硬引起反感怎么办？
A：优化话术培训，以皮肤检测数据和效果案例引导，替代强行推销；设定服务中禁推时段，先做好服务再自然过渡。

Q：新客体验完不办卡？
A：体验前做好需求诊断建立专业感，体验中让效果可感知，体验后给限时体验价转化，并留好跟进话术节点。

Q：客流不稳定怎么办？
A：老客转介绍奖励 + 朋友圈/社群内容运营 + 异业合作（美甲、瑜伽、女装）+ 节日主题沙龙引流。
