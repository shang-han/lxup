# 会员营销方案

## 适用场景
美容院策划储值卡销售、老客续卡、新客体验拓客、节日大促（三八节、店庆、双 11）、老带新裂变等会员营销活动时的完整方案与文案。

## 核心能力
- 活动类型库：储值有礼、续卡升级、新客 9.9 体验卡、老带新双向奖励、节日主题沙龙、沉睡客唤醒专场
- 完整方案输出：目标 / 主题 / 时间 / 机制 / 卡项设计 / 预算 / 人员分工与激励 / 物料清单 / 复盘指标
- 机制设计：储值梯度赠送、疗程卡打包价、体验卡转正价抵扣、转介绍双方得项目
- 客群分层触达：A 客私聊专属权益、B 客社群通知、C 客与沉睡客短信/朋友圈唤醒
- 配套文案：朋友圈预告版、社群接龙版、一对一邀约版、店内海报版
- 复盘模板：储值金额、新增会员、续卡率、到店率、转介绍数对比

## 输出格式
1. 【活动概览】目标、主题、时间、面向人群
2. 【活动机制】卡项与优惠规则明细
3. 【客群触达计划】分层名单与触达方式、时间节点
4. 【执行分工】按岗位（店长/顾问/美容师）拆解任务与提成激励
5. 【宣传文案】朋友圈版 + 社群版 + 一对一邀约版
6. 【复盘指标】活动后统计项清单

## 示例
输入：「店庆想做储值活动冲一下业绩」
输出：主题「五周年·宠粉节」→ 机制（充 3000 送 600 + 手部护理 2 次，充 5000 送 1500 + 高端抗衰体验 1 次，老客转介绍双方各得肩颈 1 次）→ 触达：A 客店长亲邀、B 客社群秒杀接龙、沉睡客发「老客专属回归礼」→ 分工与美容师阶梯提成 → 朋友圈文案：「五年了，最好的广告是你们还在。店庆储值一年仅一次，充 5000 送 1500……」→ 复盘表（储值额、新增、转介绍数）。

## 注意事项
- 预付费储值须遵守单用途商业预付卡管理规定，明示退卡规则，不诱导超出消费能力的过度储值
- 活动宣传不得宣称医疗、治疗效果，不使用「最、第一、顶级」等《广告法》禁用词
- 优惠规则写清适用范围与有效期，避免「最终解释权」式霸王条款
- 使用顾客案例或前后对比图做宣传须取得本人书面同意
