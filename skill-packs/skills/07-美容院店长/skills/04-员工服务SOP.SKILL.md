# 员工服务SOP

## 适用场景
店长规范美容师服务流程、培训新员工、统一服务标准、制定质检评分表时使用，确保每位顾客体验一致、减少服务纠纷。

## 核心能力
- 全流程 SOP 拆解：预约确认 → 迎宾接待 → 换鞋奉茶 → 皮肤检测 → 项目讲解确认 → 更衣准备 → 服务操作 → 效果确认 → 家居建议 → 结账与下次预约 → 离店送别 → 次日回访
- 每个环节的标准动作、标准话术、时长要求与禁忌
- 卫生消毒规范：一客一换一消毒、用品分类摆放、仪器使用后清洁登记
- 服务禁语与禁行清单：不议论顾客隐私、不私自加推项目、服务中不闲聊推销
- 质检评分表：按环节设检查项与分值，可直接用于月度考核
- 投诉高发点预防：力度确认、温度确认、时间不缩水、效果预期前置管理

## 输出格式
1. 【流程总览】环节清单与标准时长
2. 【分环节 SOP】每环节：标准动作 / 参考话术 / 禁忌
3. 【卫生规范】消毒与用品管理要点
4. 【质检检查表】检查项 + 分值 + 检查方式
5. 【培训建议】演练重点与考核方式

## 示例
输入：「新来两个美容师，帮我做一份服务流程标准」
输出：12 步全流程表（每步含动作与话术，如服务前「姐，力度重了轻了您随时说」）→ 卫生规范（毛巾一客一消毒、精油开封标注日期）→ 质检表（迎宾 10 分 / 检测讲解 15 分 / 操作规范 30 分 / 效果确认 15 分……总分 100）→ 培训建议：第 1 周跟岗观摩，第 2 周互练考核，店长抽查评分。

## 注意事项
- SOP 中明确：生活美容服务不得涉及任何医疗、治疗性质的操作与宣传，员工话术统一使用养护类表述
- 涉及顾客身体接触的服务须事先告知流程并征得同意，尊重顾客隐私与拒绝权
- 推销环节设边界（如服务操作中段禁推），避免硬性销售损害体验
- 卫生规范须符合公共场所卫生管理要求，消毒记录留存备查
