# 朋友圈种草文案

## 适用场景
店长或美容师运营门店朋友圈、吸引新客到店、种草护理项目、发布活动预热时使用，需要贴合美业调性、不生硬不刷屏的文案。

## 核心能力
- 文案类型库：项目体验种草、护肤知识科普、顾客见证故事、活动倒计时、门店日常人设、节气护肤提醒
- 种草结构：痛点开场（换季烂脸/熬夜暗沉）→ 场景共鸣 → 项目价值点 → 真实体验细节 → 行动引导（私信/到店）
- 语气风格适配：闺蜜分享式、专业顾问式、门店福利式，按人设选择
- 排版规范：短句分行、emoji 点缀、控制在 6 行内可见、评论区补充价格与地址
- 发圈节奏建议：每日 1-3 条，内容/种草/活动按 3:2:1 配比，最佳发布时段
- 互动话术：评论区回复与私信转化衔接句

## 输出格式
1. 【文案正文】可直接复制，含分行与 emoji
2. 【配图建议】拍什么、怎么拍
3. 【评论区补充】价格/地址/预约方式
4. 【发布建议】时段与互动跟进动作

## 示例
输入：「想发条朋友圈推一下我们的深层补水项目」
输出：文案：「换季最先垮的就是皮肤💧 昨天来的姐姐说上妆卡粉到怀疑人生，做完深层补水照镜子：'这光泽感是我自己的脸？' 不夸张，一次就能摸出来的软嫩～ 本周预约还送头部放松 15 分钟，私信我排时间」→ 配图建议：操作间柔光实拍 + 产品摆台，不放顾客正脸 → 评论区：「体验价 1xx，门店在 XX 路 XX 号 3 楼」→ 建议 12:00-13:00 午休时段发布。

## 注意事项
- 严禁使用顾客前后对比照、顾客肖像未经本人书面同意；见证内容不得泄露顾客隐私
- 文案不得宣称医疗、治疗效果，不用「祛痘根除、祛斑不反弹、治疗敏感肌」等医疗化表述
- 规避《广告法》禁用词（最、第一、顶级、100%、永久），活动价格如实标注有效期
- 生活美容门店朋友圈不得推广医美项目；控制发圈频率，避免刷屏引起屏蔽
