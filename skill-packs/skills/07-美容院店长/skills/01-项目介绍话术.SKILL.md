# 项目介绍话术

## 适用场景
美容师或店长向顾客介绍面部护理、身体 SPA、抗衰养护等项目时使用：新客首次到店的项目推荐、服务过程中的自然升单、顾客提出「太贵了/再看看/怕没效果」等异议时的应对。

## 核心能力
- 需求诊断提问清单：皮肤状态、生活习惯、护理目标、预算区间，先问后推
- 按 FABE 结构讲项目：成分/手法/仪器 → 作用原理 → 顾客能感受到的好处 → 同类顾客案例
- 场景化价值塑造：把「补水」说成「上妆不卡粉、素颜也有光泽」的可感知描述
- 异议应对库：太贵了 / 担心没效果 / 回去考虑一下 / 别家更便宜，逐条给应答话术
- 升单过渡设计：单项 → 疗程卡、面部 → 身体项目的自然衔接句与时机判断
- 区分新客首单话术（低门槛体验 + 建立信任）与老客续项话术（效果回顾 + 阶段升级）

## 输出格式
1. 【顾客情况分析】已知信息梳理与待补充提问
2. 【需求诊断问题】3-5 个现场要问的问题
3. 【项目推荐话术】开场切入 → 效果描述 → 价格锚定 → 促成话术
4. 【异议应对】针对最可能出现的 2-3 种异议
5. 【升单过渡】服务中/服务后的衔接句

## 示例
输入：「顾客 30 多岁，皮肤干、有细纹，做了基础补水，想推抗衰项目但她觉得贵」
输出：先问睡眠、防晒、护肤习惯定位干纹成因 → 推荐话术：「姐，基础补水解决的是渴，您这个干纹是胶原流失的信号，就像气球放了气光擦表面没用……」→ 用单次体验价做锚点，对比同龄顾客 3 次后的状态照片 → 异议应对：把年卡折算成每次不到一杯咖啡钱 → 升单：「今天先做一次感受，手法力度您随时喊我」。

## 注意事项
- 生活美容项目不得宣称医疗、治疗效果，禁用「治疗、治愈、祛除、根治、修复细胞」等医疗用语，效果描述用「改善、养护、舒缓」
- 不得向顾客推荐或暗示医美项目（注射、激光、手术类），涉及医美需求应建议其去正规医疗机构
- 不使用「100% 有效、永久、根除」等绝对化用语，案例展示须经顾客同意
- 报价如实，不虚构原价再打折；顾客明确拒绝时停止推销，尊重意愿
