# 产品介绍话术

## 适用场景
导购在接待顾客时需要介绍商品，但常陷入「只会说参数」或「不知从哪讲起」的困境，需要一套有感染力、突出卖点、贴合顾客需求的话术。

## 核心能力
- FABA 结构：特性（Feature）→ 优势（Advantage）→ 利益（Benefit）→ 证据（Evidence）
- 卖点提炼：从一堆参数中抓 2-3 个打动人的核心卖点
- 场景化表达：把产品放进顾客真实使用场景
- 按需调整：根据顾客关注点（价格/品质/送礼）调整侧重
- 体验引导：引导顾客触摸、试用、对比
- 品类适配：服饰、数码、家居、化妆品各有讲法

## 输出格式
1. 【开场衔接】自然引入产品的破冰句
2. 【核心卖点】FABA 逐条展开
3. 【场景代入】谁、什么场合、怎么用
4. 【体验引导】邀请试用/对比的话术
5. 【信任背书】销量/口碑/品质保证
6. 【关联推荐】连带搭配建议

## 示例
输入：「帮我写一段介绍这款羊绒大衣的话术，顾客是位中年女士」
输出：衔接「姐，您眼光真好，这件是我们今年的经典款」→ 卖点（100% 山羊绒 → 比普通羊毛轻一半 → 上身又暖又不压身 → 摸给您感受下这手感）→ 场景「过年聚会、平时出门都能穿，百搭不过时」→ 体验「您上身试试，照镜子看下版型」→ 背书「这款回头客特别多，去年卖断货」→ 关联「配条我们的羊绒围巾更出彩，一套带走给您优惠」。

## 注意事项
- 讲利益而非堆参数，顾客关心「对我有什么用」
- 卖点不超过 3 个，多了记不住
- 引导体验比口头讲解更有说服力
- 如实介绍，不夸大材质与功效
- 关联推荐自然不生硬，基于真实搭配需求
