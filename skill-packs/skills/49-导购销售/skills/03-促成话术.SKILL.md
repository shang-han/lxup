# 促成话术

## 适用场景
顾客已经心动却迟迟不下单、在几款之间犹豫、或表示「再考虑考虑」时，导购需要恰当的促成话术推一把，把意向转化为成交。

## 核心能力
- 成交信号识别：反复触摸、询问售后、讨价还价、征求同伴意见
- 假设成交法：默认顾客已决定，直接推进到付款/开票
- 二选一法：给两个都导向成交的选项
- 稀缺促单：库存、活动、赠品的限时限量
- 从众引导：销量、口碑、同款热卖
- 风险逆转：试用、退换、质保打消顾虑

## 输出格式
1. 【成交信号】识别顾客准备购买的信号
2. 【促成话术】具体可用的促单口播
3. 【促单技巧】所用的方法与切入点
4. 【顾虑消除】对应的保障承诺
5. 【动作衔接】促成后的开单/连带引导

## 示例
输入：「顾客试了衣服很喜欢但说再看看，怎么促成」
输出：信号（反复照镜子、问能不能洗=高意向）→ 话术「姐这件真的写了您的名字，上身效果特别好，今天就剩这一件了，我帮您包起来？」（假设成交+稀缺）→ 或「您是喜欢黑色还是这件焦糖色？两个色都给您留？」（二选一）→ 顾虑「七天无理由退换，不喜欢随时拿来」→ 衔接「再配条裤子就是完整一套，一起带走给您打个折」。

## 注意事项
- 促成要自然，过度施压会引起反感
- 识别真实信号再促成，别在顾客明确拒绝时硬推
- 稀缺与优惠必须真实，不虚构「最后一件」
- 给顾客台阶和选择，保留体面
- 促成后立即衔接开单与连带，避免反悔
