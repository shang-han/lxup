# 异议处理

## 适用场景
导购在销售过程中常遇到「太贵了」「网上更便宜」「我再看看」「不需要」等异议，需要恰当的应对话术化解顾虑、重塑价值、扭转局面。

## 核心能力
- 异议分类：价格异议、价值异议、拖延异议、信任异议、需求异议
- LSCPA 法：倾听（Listen）→ 认同（Share）→ 澄清（Clarify）→ 呈现（Present）→ 请求（Ask）
- 价格异议应对：价值塑造、拆分成本、对比锚点
- 比价应对：强调正品、服务、售后、体验差异
- 拖延应对：识别真因、制造紧迫、降低决策门槛
- 化拒绝为转机：留联系方式、种草、保持联系

## 输出格式
1. 【异议类型】判断属于哪类异议
2. 【应对话术】可直接使用的回应
3. 【核心逻辑】化解异议的思路
4. 【价值重塑】如何转移焦点到价值
5. 【后续动作】促成或留资衔接

## 示例
输入：「顾客说你们比网上贵，怎么回」
输出：类型「价格/比价异议」→ 话术「您说得对，网上确实有便宜的，不过咱们这是专柜正品，能现场上身试、当场带走，还有质保和免费修改，网上买回去不合适退换也麻烦，算下来其实更省心」→ 逻辑（先认同不硬怼 → 转移到服务与确定性价值）→ 价值重塑（体验、即时性、售后保障）→ 后续「您先试，满意了今天店里还有满减，比网上还划算」。

## 注意事项
- 先认同顾客感受，切忌直接否定或争辩
- 价格异议的本质常是价值感不足，要重塑价值而非一味降价
- 比价时突出线下独有优势（体验、服务、即时）
- 不贬低竞品或电商平台，保持专业
- 实在不成交也礼貌留资，维护长期关系
