# 客户需求分析

## 适用场景
顾客进店后，导购需要通过观察与提问快速判断其需求、预算、购买意向，避免盲目推荐导致顾客反感或错失成交机会。

## 核心能力
- 观察技巧：从穿着、同伴、目光停留、触摸动作判断兴趣
- 提问设计：开放式与封闭式提问结合，挖出真实需求
- 需求分层：自用/送礼、刚需/逛逛、价格敏感/品质优先
- 购买意向判断：随便看看 vs 明确目标的信号识别
- 隐性需求挖掘：表面要便宜，实际要性价比或面子
- 输出推荐方向：匹配产品与话术策略

## 输出格式
1. 【观察要点】看什么、怎么判断
2. 【提问清单】了解需求的关键问题
3. 【需求判断】自用/送礼、预算、关注点
4. 【意向分级】购买意向强弱判断
5. 【推荐策略】匹配的产品方向与话术侧重

## 示例
输入：「进来一对年轻情侣看戒指，我怎么判断他们的需求」
输出：观察（看女方目光停留的款式、男方是否主动问价、是否试戴）→ 提问（「是选婚戒还是日常戴？」了解用途；「平时喜欢简约还是带钻的？」了解风格；「大概的预算范围方便说下吗，我帮您精准推荐」）→ 判断（婚戒、女方主导审美、男方关注预算）→ 意向（试戴+问售后=高意向）→ 策略（主推女方喜欢的款式，强调寓意与保值，价格给中档区间+分期付款引导）。

## 注意事项
- 先观察再开口，别一进门就紧跟推销
- 提问自然不审问，给顾客舒适空间
- 送礼场景要兼顾购买者与使用者的双重需求
- 预算问题委婉询问，避免让顾客尴尬
- 判断错了及时调整，别一条道走到黑
