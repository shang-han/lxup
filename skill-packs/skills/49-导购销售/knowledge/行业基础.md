# 行业基础

## 行业概况
导购是零售门店直接面向顾客的销售人员，直接影响门店业绩与品牌形象，各行各业（服饰、数码、家居、化妆品、珠宝等）都需要导购。核心能力是产品知识、沟通技巧与服务意识。在电商冲击下，线下导购的价值更体现在体验、专业建议与客情维护上。业绩考核以成交率、客单价、连带率、会员转化为主，优秀的导购本质上是「被顾客信赖的选购顾问」。

## 常用术语
- 成交率：进店顾客中完成购买的比例
- 客单价：平均每单消费金额
- 连带率：每单平均购买件数
- 会员转化率：新客注册会员的比例
- 回头率：老客再次到店消费的比例
- FABA：特性-优势-利益-证据的销售介绍法则
- 客情：与顾客建立的情感联系与信任关系

## 核心工作流程
迎宾：主动问候 → 观察需求 → 破冰沟通 → 引导入店
接待：需求了解 → 产品展示 → 试用体验 → 专业建议
成交：推荐搭配 → 议价处理 → 开单结算 → 会员注册 → 送宾

## 常见问题
Q：顾客只看不买怎么办？
A：了解其顾虑（价格/需求/对比），强化产品价值、提供试用体验、不强行推销，留下联系方式后续跟进。

Q：顾客砍价太狠怎么办？
A：强调品质、服务与售后价值，用赠品、礼品、会员权益替代直接降价，守住价格底线。

Q：怎么提高连带率？
A：基于真实搭配需求做成套推荐、关联陈列、买赠活动，自然引导而非硬塞。

Q：如何沉淀老客提升复购？
A：建立客户档案打标签、定期回访关怀、上新与活动私信通知、会员权益与转介绍激励，用心经营客情。
