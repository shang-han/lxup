# 行业基础

## 行业概况
养生产业规模持续增长，涵盖中医理疗、艾灸、推拿、足疗、刮痧拔罐等细分领域。消费群体从老年人群向中青年扩展，上班族肩颈腰背调理、女性气血养护需求旺盛。核心卖点：健康效果 + 专业手法 + 服务体验。养生保健服务属于非医疗性健康服务，不得宣称诊断、治疗效果，与医疗机构的推拿科、针灸科有明确界限。门店以会员卡、疗程卡为主要营收模式，技师手法与客情是留客关键。

## 专业技能体系

### 硬技能
- 中医推拿手法（经络/穴位/推拿流派）
- 常见疼痛问题诊断（颈椎/腰椎/肩周）
- 精油选配与使用
- 足疗/刮痧/拔罐等辅疗技术
- 门店卫生与合规经营

### 软技能
- 客户身体状况沟通
- 疗程续费引导

## 常用术语
- 上钟率：技师实际服务时长占可用工时的比例，反映工作量饱和度
- 点钟率：客户指定某位技师服务的比例，体现技师客情与口碑
- 耗卡率：会员卡内金额/次数的消耗速度
- 返店率：客户再次到店消费的比例
- 疗程完成率：客户完成整套疗程计划的比例
- 排钟：按顺序轮流安排技师接单的方式
- 加钟：服务结束后客户主动要求延长时间或加做项目

## 核心工作流程
每日：预约确认 → 客户接待与需求询问 → 技师排钟安排 → 服务跟进 → 收银结算 → 回访记录
每周：产品耗材库存检查 → 仪器设备维护 → 员工排班与手法培训
每月：客户消费分析 → 项目结构优化 → 会员营销活动策划

## 常见问题
Q：客户做了几次觉得没效果怎么办？
A：重新评估体质与预期，制定个性化疗程方案，记录每次调理反应（如酸胀、出痧），用阶段性变化建立信心，而非承诺「包好」。

Q：技师流失怎么办？
A：底薪 + 阶梯提成 + 社保保障 + 手法培训与晋升通道（技师 → 组长 → 店长），核心技师参与点钟分成。

Q：淡旺季客流差距大怎么办？
A：旺季（秋冬进补、三伏三九）做疗程卡锁定，淡季推小项目体验卡与节气主题活动引流。

Q：客户担心养生馆不专业？
A：展示技师资质证书、服务前做体质问询建档、调理反应讲解透明化，用专业流程建立信任。
