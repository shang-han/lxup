# 会员卡营销话术

## 适用场景
客户做完体验或单次服务后邀请办卡、疗程卡销售、会员卡到期续费、客户犹豫时的异议处理等场景的话术支持。

## 核心能力
- 卡项卖点提炼：储值卡（折扣 + 赠送）、疗程卡（锁定效果 + 折算单价）、季卡/年卡（频次自由）分别突出什么
- 办卡时机判断：服务后体感最好时、客户主动问价格时、多次到店未办卡时
- 促成话术结构：效果回顾 → 算账对比（单次 vs 卡项）→ 限时权益 → 低压力收尾
- 异议应对库：太贵了 / 怕坚持不来 / 回家商量 / 别家更便宜 / 怕没效果
- 续卡话术：到期前 2 周分节点触达（感谢回顾 → 专属续费礼 → 临期提醒）
- 不同客群语气适配：老年客户重实惠、上班族重效率、养生爱好者重专业

## 输出格式
1. 【时机判断】该客户现在适合推哪种卡
2. 【核心话术】开场 → 算账 → 促成，可直接使用
3. 【异议应对】针对最可能的 2-3 种异议
4. 【续卡/跟进节点】时间表与对应话术

## 示例
输入：「客户做了 3 次肩颈推拿感觉不错，但说疗程卡太贵了」
输出：算账话术：「姐，单次 168，10 次疗程卡 1180，等于每次 118，还送 2 次艾灸。您肩颈这个情况断断续续做最浪费钱，密集调理 10 次刚好一个周期……」→ 异议应对：怕坚持不来→「卡有效期一年，咱先约好前 4 次时间，我每周提醒您」→ 退一步方案：「要不先办 5 次小疗程试试节奏？」→ 续卡节点：第 8 次时预告续费礼。

## 注意事项
- 办卡话术中不得以疗效承诺诱导办卡（「办卡保证治好」），效果表述用「多数客户反馈、因人而异」
- 如实告知卡项有效期、使用范围、退卡规则，不隐瞒限制条件
- 预付费销售遵守单用途预付卡管理规定，不诱导超出客户承受能力的超大额储值
- 客户明确拒绝时礼貌收住，不围追推销，保护客户体验
