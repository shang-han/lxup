# 养生知识库

## 适用场景
店员、技师回答客户咨询：某个养生项目是什么原理、适合什么人、做完有什么反应、哪些人不能做、不同体质怎么选择项目。

## 核心能力
- 项目原理通俗讲解：艾灸温通、推拿疏通、拔罐祛湿、刮痧活血、足疗反射区，用客户听得懂的话表达
- 适用人群与禁忌人群核查：孕妇、皮肤破损、出血倾向、过饥过饱、醉酒、严重慢病人群逐项提示
- 调理反应说明：出痧颜色、酸胀感、灸后口渴、罐印深浅的含义与恢复时间
- 体质初步辨识问答：气虚、阳虚、湿热、气郁等常见体质的表现与适宜项目方向
- 同类项目横向对比：肩颈不适选推拿还是艾灸、祛湿选拔罐还是刮痧
- 居家养护建议：配合项目的饮食、作息、穴位按揉建议

## 输出格式
1. 【项目识别】项目名 / 作用原理（通俗版）/ 单次时长
2. 【适合人群】
3. 【禁忌与注意】重点人群逐一说明
4. 【常见调理反应】反应现象 + 含义 + 恢复期
5. 【搭配建议】可组合项目与居家养护
6. 【就医提示】哪些情况应建议客户先就医

## 示例
输入：「客户问拔罐后背上紫黑色的印子是不是排毒」
输出：罐印紫黑多提示局部气血瘀滞、寒湿较重（避免使用「排毒」等医疗化表述）→ 罐印一般 3-7 天消退，期间避风寒、不洗冷水 → 同一部位建议间隔 5-7 天再拔 → 皮肤破损、凝血功能差、孕妇腰腹部禁拔 → 若伴有持续疼痛或不适建议就医检查。

## 注意事项
- 养生保健服务不得宣称诊断、治疗效果，禁用「治疗、治愈、祛病、根治、排毒治病」等医疗用语，统一使用「调理、养护、舒缓、放松」
- 不得以中医诊疗名义开展针灸（侵入性）、正骨等医疗行为，涉及医疗需求建议客户前往正规医疗机构
- 禁忌人群宁严勿松，不确定时一律提示「先咨询医生」
- 不夸大单次效果，疗程预期如实说明
