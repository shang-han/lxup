# 季节性养生方案

## 适用场景
养生馆按季节、节气策划主题调理活动（三伏灸、冬季进补、春季疏肝），或给客户出季节性居家调养建议，带动淡季客流与疗程卡销售。

## 核心能力
- 四季调养要点库：春养肝疏泄、夏养心祛湿、秋润肺防燥、冬补肾温阳，对应推荐门店项目
- 节气活动方案：三伏贴/三九灸、惊蛰排寒、冬至进补灸等主题活动的目标、机制、选品、文案
- 客户调养建议：按季节给客户的饮食、起居、穴位按揉、到店项目组合建议
- 节气朋友圈/社群内容：节气知识 + 门店项目软植入的文案模板
- 淡旺季衔接：用节气节点把单次客户转化为季节疗程卡
- 活动话术包：邀约话术、到店讲解话术、朋友圈预热文案一套齐

## 输出格式
1. 【节气/季节要点】养生原理通俗讲解
2. 【推荐项目组合】门店现有项目的搭配建议
3. 【活动方案】（如适用）主题 / 机制 / 时间 / 文案
4. 【客户调养建议】饮食 + 起居 + 穴位，可转发客户
5. 【宣传文案】朋友圈或社群版本

## 示例
输入：「快到三伏天了，想做个三伏灸活动」
输出：要点：三伏天阳气最盛，借天时温灸可助阳散寒，适合手脚冰凉、易感冒、关节怕冷人群 → 项目组合：三伏灸 3 次（初伏/中伏/末伏各 1 次）+ 督脉推拿 → 活动机制：3 次套餐价 7 折，老客带新各加赠 1 次拔罐 → 邀约话术：「张姐，您一到冬天就手脚凉，三伏灸一年就这一回借天时的机会……」→ 朋友圈文案：「头伏将至｜一年中『晒背补阳』的黄金 40 天来了……」→ 提醒：发热、实热体质、孕妇不宜，到店先问询再安排。

## 注意事项
- 「冬病夏治」等传统说法引用须谨慎，不得表述为治疗、治愈疾病，统一用「传统养生理念认为、辅助调养」
- 禁忌人群（实热体质、发热期、孕妇、皮肤破损处）必须在方案与话术中明示筛查
- 不借节气制造健康焦虑诱导消费，建议内容科学平实
- 客户出现明确疾病症状时，调养建议后固定附加「不适请及时就医」
