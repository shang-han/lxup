# 服务标准SOP

## 适用场景
门店经理制定与培训保洁、深度清洁、洗衣洗护服务的标准化流程，统一服务动作与验收标准，用于员工培训、上门作业指导与质量检查。

## 核心能力
- 保洁 SOP：进门规范（穿鞋套、工具摆放）→ 分区作业顺序（由上到下、由里到外）→ 厨卫重点 → 收尾自检 → 客户验收
- 深度清洁 SOP：高温蒸汽、玻璃、家电拆洗等专项流程与时长标准
- 洗衣收洗流程：收衣检查（瑕疵拍照、洗涤标签核对）→ 分类洗涤 → 去渍处理 → 熨烫质检 → 包装交付
- 验收标准清单：分区域可量化标准（无水渍、无毛发、无灰尘残留），可直接打印做验收单
- 工具与药剂规范：分区毛巾分色管理、药剂稀释比例、不同材质（实木、真皮、大理石）禁忌
- 培训与考核设计：新员工带教周期、实操考核评分表

## 输出格式
1. 【流程总览】步骤清单与标准时长
2. 【分步骤规范】每步：动作标准 / 禁忌 / 常见错误
3. 【验收标准表】分区域检查项
4. 【工具药剂清单】配置与使用规范
5. 【考核评分表】分值与扣分项

## 示例
输入：「做一份 2 小时日常保洁的服务标准给新阿姨培训用」
输出：流程：进门（鞋套、垫布、与客户确认重点区域，5 分钟）→ 卧室整理除尘（25 分钟）→ 客厅（25 分钟）→ 厨房（30 分钟，油烟机表面 + 灶台 + 台面）→ 卫生间（30 分钟，马桶/淋浴区/镜面）→ 收尾自检 + 邀客户验收（5 分钟）→ 验收表：台面无水渍、地面无毛发、镜面无痕……→ 禁忌：不翻动客户私人物品、不坐客户沙发休息、湿毛巾不擦电器。

## 注意事项
- 上门服务规范中写入安全条款：身份工牌出示、不独处收现金、客户财物不触碰不挪用
- 药剂使用须标注安全事项（戴手套、通风、禁止混用如 84 与洁厕灵），防止安全事故
- 洗衣流程坚持收衣当面检查拍照，瑕疵事先确认，从源头减少赔付纠纷
- SOP 须随客户反馈与事故复盘定期修订，不做一成不变的摆设
