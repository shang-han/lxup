# 会员套餐方案

## 适用场景
门店设计会员套餐与营销活动：保洁次卡/年卡、洗衣储值卡、深度清洁套餐、节日促销，提升锁客与复购。

## 核心能力
- 套餐类型库：保洁 10 次/20 次卡、包月包年定期保洁、洗衣储值卡（充赠）、洗护季卡、深度清洁组合包
- 定价逻辑：按单次价 × 次数给梯度折扣，折算单次成本展示，锁定高频刚需
- 权益设计：会员价、优先预约、专属客服、生日礼、转介绍奖励
- 客群分层方案：家庭定期保洁客（年卡）、单身白领（包月）、奢侈品护理客（储值 + 保值洗）
- 营销活动设计：开年保洁节、换季洗护周、老带新双向奖励、沉睡会员唤醒券
- 销售话术配套：不同套餐的卖点一句话 + 算账促成话术

## 输出格式
1. 【套餐清单】名称 / 内容 / 定价 / 折算单次价
2. 【目标客群】每款套餐对应人群
3. 【会员权益表】
4. 【营销活动】（如适用）主题 / 机制 / 时间 / 文案
5. 【销售话术】主推套餐的促成话术

## 示例
输入：「设计几款保洁会员套餐，主打家庭定期客户」
输出：套餐：①包月卡（每月 2 次 2 小时保洁）xxx 元/月，比单次省 25%；②10 次卡（90 天内用完）xxx 元；③年卡（24 次 + 送 2 次深度清洁）xxx 元 → 主推年卡话术：「您家本来一个月就要做两次，年卡算下来一次才 xx 元，还白送两次年底深度清洁，档期会员优先约，不怕年底约不上」→ 权益：会员专属客服 + 优先预约 + 转介绍各得 1 次 → 唤醒活动：90 天未消费会员发「回归 5 折体验券」。

## 注意事项
- 预付式套餐须明示有效期、使用规则与退卡退费方式，遵守单用途预付卡管理规定
- 套餐包含的服务范围写清楚（面积上限、时长、加项计费），避免使用中扯皮
- 促销价格真实，不虚构原价制造折扣假象
- 年卡客户的服务质量是口碑命脉，不因是套餐客户降低服务标准
