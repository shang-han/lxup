# 投诉处理流程

## 适用场景
门店处理客户投诉：服务质量不达标、洗衣损坏染色、人员爽约迟到、物品损失纠纷、平台差评，需要标准流程、赔偿尺度与沟通话术。

## 核心能力
- 投诉分级：轻度（服务瑕疵）/ 中度（返工 + 补偿）/ 重度（财物损坏、人身言语冲突），分级响应时限与处理权限
- 标准处理流程：倾听记录 → 致歉共情 → 核实情况 → 给出方案 → 执行补偿 → 回访闭环
- 赔偿标准表：保洁返工免费、洗衣按衣物折旧与洗涤协议赔付、爽约赠券、贵重物品按事先约定
- 分场景安抚话术：情绪激动客户、要求高额赔偿客户、平台差评回复
- 证据管理：服务前后照片、收衣检查记录、沟通记录留存要点
- 复盘模板：客诉原因归类（人/流程/沟通）→ 整改措施 → 制度修订

## 输出格式
1. 【投诉定级】级别 / 响应时限 / 处理人
2. 【处理步骤】编号流程，每步动作与话术
3. 【补偿方案】可选方案与尺度依据
4. 【沟通话术】致歉 + 方案 + 跟进，可直接使用
5. 【复盘记录模板】原因 / 措施 / 预防

## 示例
输入：「客户投诉洗坏的羽绒服袖子染色了，很生气，怎么处理」
输出：定级：中度，店长 2 小时内响应 → 步骤：①致歉共情：「实在抱歉，您先别急，这件事我们肯定负责到底」；②核实：调收衣检查照片确认染色是否收衣时已有；③方案：重洗修复，无法修复按折旧赔付（参照洗涤协议，上限为洗涤费的 X 倍或保值金额）；④补偿：本次洗费免单 + 洗护券 → 话术：「您这件衣服我们按协议赔付 xxx 元，这次洗护免单，再送您两张券表示歉意……」→ 复盘：查染色订单是否混洗违规，修订色牢度预检流程。

## 注意事项
- 先处理情绪再处理问题，第一时间不推责不争辩，态度决定事态
- 赔偿以事先书面约定（服务协议、收衣单据）为依据，无约定时按公平原则协商，重大赔付店长审批
- 平台差评 24 小时内公开回复（致歉 + 已处理 + 联系方式），同时私信沟通解决，不与客户互怼
- 每起客诉必须复盘归档，同类问题重复发生须修订制度，而非只赔钱了事
