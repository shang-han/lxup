# 客户沟通话术

## 适用场景
门店客服或经理与客户沟通的全流程：电话咨询报价、首次预约确认、服务前需求核对、服务中异常沟通、服务后回访与续费引导。

## 核心能力
- 咨询报价话术：先问清面积/件数/需求再报价，报「区间 + 计费方式」，不报死价也不含糊
- 预约确认话术：时间、地址、服务内容、注意事项（家中留人、贵重物品收纳）逐项确认
- 需求预期管理：明确「包含什么、不包含什么」，外墙、高空、特殊污渍等加项提前说清
- 服务中沟通：增项请示话术、发现问题（如家具已有损坏）告知话术
- 回访话术：满意度询问 + 问题收集 + 会员/复购自然引导
- 异议应对：嫌贵、比别家贵、质疑专业性等常见异议应答

## 输出格式
1. 【场景判断】沟通所处环节与目标
2. 【话术正文】可直接使用，口语化
3. 【关键确认项】本环节必须问清/说清的事项清单
4. 【异常应对】客户提出异议或意外情况时的应答

## 示例
输入：「客户打电话问保洁多少钱，帮我写报价话术」
输出：话术：「您好，我们日常保洁是按面积算的，90 平以内两小时 xxx 元，包含卧室、客厅、厨卫的表面清洁。先问下您家大概多大面积？这次是想做日常保洁还是深度清洁？有没有需要重点处理的，比如油烟机或者玻璃？」→ 关键确认项：面积、服务类型、重点需求、是否首次（首单体验价）→ 嫌贵应对：「我们是两人上门、自带全套工具和环保药剂，做完您验收不满意免费返工，算下来比找散工省心也不贵。」

## 注意事项
- 报价与服务范围以书面确认为准（微信文字或订单备注），口头承诺要同步留痕，避免结算争议
- 不夸大服务能力（如承诺「什么污渍都能去掉」），不确定项如实告知需现场评估
- 客户家庭住址、联系方式、家中情况属隐私，仅限服务必需范围使用
- 回访引导续费点到为止，客户表示不满意时先解决问题再谈营销
