# 行业基础

## 行业概况
家政行业市场规模超万亿，涵盖保洁、保姆、月嫂、育儿嫂、养老护理等细分，从业人员超 3000 万，市场需求旺盛但服务质量参差不齐。洗衣门店（干洗、水洗、奢侈品护理）是社区生活服务的重要组成，与家政共享会员客群。行业核心痛点：人员管理与服务标准化、客户信任建立。预付费会员卡是主流模式，口碑与复购决定门店生死，旺季集中在年末大扫除、换季洗护与节假日。

## 专业技能体系

### 硬技能
- 保洁服务标准化（开荒/日常/深度/消杀）
- 清洁剂与设备选型（分类/配比/安全）
- 员工培训与考核体系
- 客户派单与调度系统
- 保险与纠纷处理
- 面料识别与水洗/干洗工艺选择
- 去渍技术（油渍/血渍/果汁/墨水分类处理）
- 干洗设备操作与维护（四氯乙烯/石油溶剂/湿洗）
- 衣物修复（织补/改色/皮革护理）
- 收发流程标准化（标签/拍照/登记）

### 软技能
- 信任建立与售后跟进
- 灵活用工管理
- 客户贵重衣物沟通
- 投诉与赔偿处理

## 常用术语
- 接单率：订单被成功承接的比例
- 客诉率：产生投诉的订单占比
- 复购率：客户再次下单的比例
- 人单匹配率：服务人员技能与订单需求的匹配程度
- 服务完成率：按时按标准完成的订单比例
- 返工率：需要重新返工处理的订单比例
- 洗护事故率：洗衣过程中出现损坏、染色等问题的比例

## 核心工作流程
每日：接单派单 → 客户需求确认 → 服务人员调度 → 服务过程跟进 → 验收回访 → 收银结算
每周：员工技能培训 → 服务质量抽查 → 下周排班 → 物料耗材补充
每月：经营数据分析（接单率、客诉率、复购率）→ 客户回访汇总 → 供应商结算

## 常见问题
Q：阿姨放鸽子怎么办？
A：建立备选人员池，订单超时未到自动触发补派；接单前双重确认，旺季订单预留机动人员；爽约纳入服务人员考核。

Q：客户不满意服务质量？
A：第一时间道歉 + 免费返工 + 必要时更换人员 + 补偿券，先解决情绪再复盘原因，重大客诉店长亲自跟进。

Q：洗衣损坏怎么赔？
A：收衣时当面检查并拍照留档、贵重衣物建议保值清洗并书面约定，按洗涤标签与事先约定的赔付标准处理，不推诿。

Q：报价总被嫌贵？
A：按面积/件数/难度的标准化报价表公开透明，拆解服务内容（几遍清洁、含哪些区域），对比游击队的风险成本。
