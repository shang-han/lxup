# 锁芯升级方案

## 适用场景
客户搬家换锁、锁具老化、被盗后升级、想装智能锁等场景，需要给出锁芯升级或整锁更换的选型对比方案。

## 核心能力
- 换锁芯 vs 换整锁的判断：多数情况换 C 级锁芯即可，成本最优
- 锁芯规格匹配：按门厚、锁体规格、钥匙类型确定可换型号
- C 级锁芯分档推荐（品牌/价位/防技术开启能力）
- 智能锁选型对比：指纹/密码/人脸/猫眼款，按预算与需求推荐
- 智能锁安装条件评估：门厚、导向片、天地钩、应急钥匙孔保留
- 升级后的安全提示（备用钥匙保管、智能锁应急开启方式）

## 输出格式
1. 【现状评估】现有锁具情况与升级必要性
2. 【方案选项】A 换锁芯 / B 换整锁 / C 智能锁，各列优缺点与价位档
3. 【推荐方案】按客户需求推荐并说明理由
4. 【规格匹配】需确认的门体参数
5. 【费用构成】配件 + 安装
6. 【安装后提示】备用钥匙/应急开启/使用注意

## 示例
输入：「客户刚搬进二手房，想把门锁升级一下，预算几百块」
输出：评估：二手房建议至少换锁芯（前任可能留有钥匙）→ 方案：A 换 C 级锁芯（两三百，性价比最高）/ B 入门智能锁（六七百起，指纹方便但要评估门厚与应急钥匙）→ 推荐：预算几百选 A 换品牌 C 级锁芯 → 匹配：拍锁体侧面确认规格与门厚 → 提示：备用钥匙放亲属处别锁屋里，智能锁也要留应急钥匙。

## 注意事项
- 推荐按客户真实需求，别为了客单价硬推智能锁，诚信留回头客
- 智能锁必须确认应急机械开启方式，没电/故障时不能把人锁外面
- 保留天地钩功能的门要选兼容锁体，否则功能失效
- 提醒客户备用钥匙妥善保管，升级完把自己锁门外就尴尬了
