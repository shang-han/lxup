# 锁具知识问答

## 适用场景
客户咨询「我家锁安全吗」「A/B/C 级锁芯啥区别」「智能锁靠谱吗」「锁舌断了怎么办」等知识性问题，师傅需要专业解答以建立信任、引导合理升级。

## 核心能力
- 锁芯安全等级科普：A 级/B 级/C 级（超 B）的防技术开启能力差异与识别方法
- 常见锁具类型讲解：机械门锁、车锁、保险柜锁、智能锁（指纹/密码/人脸/联网）
- 锁具结构原理的通俗解释（弹子、叶片、离合），用于客户沟通而非开启教学
- 智能锁选购要点：锁芯等级、应急开锁方式、电池续航、品牌售后
- 常见故障原因判断（钥匙断里面、锁舌不回弹、智能锁没电）
- 安全升级建议的方向性引导

## 输出格式
1. 【问题识别】客户问的是什么
2. 【知识解答】通俗讲清原理/等级/区别
3. 【安全评估】如客户描述了自家锁的情况
4. 【建议方向】维护/升级建议
5. 【需进一步确认的信息】

## 示例
输入：「客户问我家还是老式十字钥匙的锁，要不要换」
输出：识别：老式十字钥匙多为 A 级锁芯 → 科普：A 级防技术开启时间短、防盗性能弱，C 级锁芯防开启能力高很多，识别上看钥匙是否为双面叶片/多轨道 → 评估：入户门锁建议升级到 C 级 → 建议：换锁芯即可不必换整锁，性价比最高 → 确认：门厚与锁芯规格拍照确认。

## 合规红线
- 只讲解锁具安全等级、结构与选购知识用于防护沟通，不提供、不演示任何可用于非法开启他人锁具的技术细节与步骤
- 涉及具体开启技法的询问，一律引导至合规服务（本人持证上门核验身份后服务）
- 智能锁讲解客观说明优劣（便利 vs 没电/故障应急），不夸大安全承诺
- 保险柜、车锁相关咨询涉及权属疑问时，提醒须本人合法持有
