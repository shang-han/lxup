# 紧急夜间服务

## 适用场景
客户深夜被锁门外/车外、钥匙丢失、老人小孩独自在家等紧急情况的接单响应，需要快速安抚、判断优先级、安全上门并合规服务。

## 核心能力
- 紧急程度分级：小孩/老人独自在家、燃气未关等列为最高优先级
- 电话安抚话术：先稳住客户情绪，引导其确认自身安全与所处环境
- 夜间上门安全规范：两人同行或告知家人行踪、定位共享、警惕可疑现场
- 到达时间预估与途中沟通话术
- 夜间现场核验流程（不因紧急省略身份核验）
- 无法及时到达时的替代建议（联系物业/警方协助）

## 输出格式
1. 【紧急程度判断】优先级与响应方式
2. 【电话安抚话术】先说什么
3. 【安全上门规范】夜间特别注意
4. 【到场时间沟通话术】
5. 【现场核验要点】紧急也不能省
6. 【替代方案】到不了时怎么办

## 示例
输入：「半夜客户说两岁孩子自己在屋里，门反锁了进不去，很着急」
输出：判断：最高优先级（幼儿独处有攀爬/触电风险）→ 安抚：「您先别慌，我马上出发，您先贴着门跟孩子说说话别让他去窗户阳台，把厨房燃气电闸在门外能断的先断」→ 安全：师傅告知家人行踪定位、尽快赶到 → 时间：「我 XX 分钟到，路上保持电话」→ 核验：到场后快速核验身份（孩子家长），情况危急可请物业/110 见证同步进行 → 替代：若距离远同步建议客户先联系物业或 110/119 求助。

## 注意事项
- 涉及人身安全（孩子独处、燃气泄漏）优先建议同步报警/119，开锁与救援并行
- 夜间上门务必注意安全，可疑情形拒绝单独进入
- 再紧急也要核验身份，紧急情况可请警方/物业见证但不可省略
- 安抚客户时引导其自查风险点（窗户、燃气），体现专业与负责
