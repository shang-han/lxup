# 服务报价

## 适用场景
电话里给客户报价、上门后确认费用、应对「开把锁怎么这么贵」的疑问，做到收费透明、依据清楚。

## 核心能力
- 服务分项报价：基础开启费 + 时段附加（夜间/节假日）+ 难度附加（高安全锁/保险柜）+ 上门远途费
- 换锁/换锁芯报价：锁具/锁芯配件价 + 安装工费
- 汽车钥匙匹配/解码报价逻辑（车型、是否带芯片、智能卡分档）
- 收费依据话术：技术成本、时段成本、风险与保险
- 电话报价规则：先问清锁型/时段/地址给区间，到场复核定价
- 避免纠纷的确认话术：动工前报价、损坏风险提示

## 输出格式
1. 【服务类型】开锁/换锁/配钥匙/车锁
2. 【费用明细】基础费 / 时段附加 / 难度附加 / 配件 / 远途
3. 【合计或区间】
4. 【收费依据说明】为什么是这个价
5. 【动工前确认话术】

## 示例
输入：「客户半夜 1 点打电话，普通房门锁打不开要开门，怎么报」
输出：明细：基础开启费 XX + 夜间服务费（22:00 后加收）+ 远途费（如超范围）→ 区间：普通 A/B 级锁 XXX 区间，C 级或特殊锁难度加价 → 依据：「夜间师傅待命出行有成本，明码标价不会乱收」→ 确认话术：「电话先给您这个区间，到了看下锁具类型最终确认，开之前一定先跟您说定价格，同时需要您出示身份证和居住证明核验」→ 提醒先核验后开工。

## 注意事项
- 电话报区间、到场核定，动工前务必口头确认价格再动手
- 夜间/节假日加收要提前说明，避免开完门才加价引发投诉
- 开启可能损坏锁具的（暴力开启情形）提前告知风险
- 核验身份是报价流程的前置环节，价格谈妥也要先核验再开工
