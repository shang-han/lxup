# 开锁身份核验流程

## 适用场景
接到开锁请求（家门、车门、保险柜）后、动手开启之前，完成法定的身份核验与登记备案，既保护客户财产安全，也保护师傅自己不卷入纠纷或违法风险。

## 核心能力
- 电话初筛话术：先问清是谁的门/车、为什么打不开、能否提供证明，过滤可疑请求
- 到场核验清单：身份证 + 与开锁对象的关联证明（房产证/租房合同/行驶证/物业证明/邻居见证）
- 无法现场证明时的处理：请物业/社区民警到场见证，或引导报警处理
- 开锁服务登记台账模板：时间、地址、客户姓名证件号、开锁对象、见证人、师傅签字
- 向客户解释核验必要性的话术（「这是公安要求，也是保护您家安全」）
- 识别高风险/异常情形并拒绝服务

## 输出格式
1. 【电话初筛话术】接单时先问什么
2. 【到场核验清单】逐项核对的证明
3. 【无法证明的处理流程】
4. 【登记台账模板】可复制的登记表字段
5. 【对客解释话术】让客户理解配合
6. 【拒绝服务情形】哪些情况不开

## 示例
输入：「有人打电话说把自己锁门外了，让我去开门」
输出：初筛：「好的师傅先问下，是您自己住的房子吗？带身份证了吗？能出示房产证或租房合同吗？」→ 到场清单：身份证本人核对 + 产权/居住证明任一（房产证/租约/近月水电费单/物业证明）→ 若只有身份证无居住证明：联系物业或社区民警到场见证，无法见证的建议报警 → 登记：填台账（地址、证件号、见证人、时间）双方签字 → 话术：「核验是公安备案要求，也防止有人冒充业主，耽误您两分钟配合下」。

## 合规红线
- 开锁前必须核验客户身份证 + 产权/居住证明（或物业/民警见证）并登记台账，手续不全一律不开，这是法律底线
- 服务须向公安机关备案，从业人员持备案资质上岗，台账留存备查
- 不承接身份可疑、证明不全、客户含糊回避的请求；涉及纠纷（如夫妻/房东租客争执开门）要求双方到场或报警处理
- 本技能及所有技能均不提供可用于非法入侵的具体技术细节，只做合规服务流程指导
- 发现客户意图可疑或涉及赃物（保险柜无法说明来源等），拒绝服务并视情报警
