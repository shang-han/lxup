# 行业基础

## 行业概况
开锁配钥是受公安机关严格监管的特种服务行业，从业人员须备案、持证上岗，开锁服务须核验客户身份并登记。需求来自忘带钥匙、锁具故障、搬家换锁、汽车钥匙匹配、智能锁安装等，即时性强、夜间与节假日单占比高、客单价可观。核心竞争力是响应速度、合规规范与社区信任。行业正向智能锁安装调试等高附加值服务延伸，纯技术开锁占比下降。

## 专业技能体系
### 硬技能
- 各类锁具结构熟悉（门锁/车锁/保险柜/智能锁）
- 无损开锁技法（单勾/锡纸/撞匙/技术开）
- 锁芯更换与升级
- 汽车钥匙匹配与解码
- 智能锁安装与调试
### 软技能
- 客户身份核验与安全意识
- 紧急响应与夜间服务

## 常用术语
- 锁芯等级：A 级/B 级/C 级（超 B），按防技术开启能力划分，C 级最高
- 技术开启：不破坏锁具的开启方式，合规服务须核验身份后进行
- 无损开锁：开启后锁具仍可正常使用的技法，优先采用
- 弹子/叶片锁：机械锁芯的两种主流结构
- 芯片钥匙：汽车钥匙内置防盗芯片，需匹配解码才能启动
- 智能锁应急开启：没电/故障时用机械钥匙或外接电源的兜底方式
- 公安备案：开锁从业者与每次开锁服务的登记留存要求
- 天地钩：防盗门上下联动锁点，换锁时需确认兼容性

## 核心工作流程
接单：电话问清对象/情况 → 判断紧急程度 → 约定或急赴
核验：到场核验身份证 + 产权/居住证明（或物业/民警见证）→ 登记台账
服务：技术开启/更换/匹配 → 试锁确认 → 收费 → 安全建议
收尾：台账签字留存 → 智能锁使用指导 → 老客户维护

## 常见问题
Q：客户说「就是自己家门，还要什么证明这么麻烦」怎么办？
A：耐心解释「这是公安备案的硬性要求，也是保护您家不被别人乱开，耽误两分钟配合下」，核验是底线不能省，实在无证明就请物业或社区民警见证。

Q：哪些情况必须拒绝服务？
A：身份可疑、无法提供任何权属或居住证明且无见证、涉及权属纠纷双方未到场、保险柜说不清来源等，宁可少做一单也不能担违法风险。

Q：夜间急单怎么兼顾速度与安全？
A：电话先安抚并引导客户自查风险，涉及人身安全建议同步报警；师傅夜间上门告知行踪、可疑现场不单独进入；到场后核验流程不因紧急省略。

Q：怎么积累口碑又不踩线？
A：靠响应快、收费透明、合规规范做社区口碑，和物业、房产中介建立正规合作；宣传只讲安全升级服务，不展示开启技术细节。
