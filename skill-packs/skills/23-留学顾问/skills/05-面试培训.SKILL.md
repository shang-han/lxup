# 面试培训

## 适用场景
学生收到海外院校（尤其商科、港新、部分本科项目）的面试邀请后，留学顾问为其提供系统的面试培训：高频问题、应答逻辑、视频面试礼仪与模拟演练，提升录取概率。

## 核心能力
- 院校面试形式解析：真人面（招生官/校友/Kira）vs 录制视频面试
- 高频问题库：自我介绍、为何选校/专业、职业规划、行为面试题（Behavioral）
- 应答框架：STAR 法则、动机—能力—匹配结构
- 行为面试（Leadership/Teamwork/Challenge）的故事准备
- 视频面试技巧：环境、设备、眼神、语速、时间控制
- 反问环节设计：向招生官提问的好问题
- 一对一模拟面试 + 逐点点评反馈

## 输出格式
1. 【面试概况】院校 / 形式 / 时长 / 侧重
2. 【高频问题与应答框架】分类整理
3. 【行为面试故事库】STAR 结构化素材
4. 【自我介绍与反问】模板
5. 【视频面试技巧】环境与表现
6. 【模拟演练】问答示例 + 点评

## 示例
输入：「我收到了港中文商科的视频面试邀请，第一次面，怎么准备？」
输出：面试概况（Kira/真人面、英文为主、约15-20分钟）→ 高频问题（自我介绍、why this program、职业规划、行为题）→ 应答框架（动机-能力-匹配）→ 行为面试故事库（领导力/团队合作各准备1-2个STAR故事）→ 自我介绍与反问模板 → 视频面试技巧（背景整洁、眼神看镜头、控制每题1-2分钟）→ 一段模拟问答+点评。

## 注意事项
- 应答真实自然，避免背诵痕迹，招生官反感套话
- 突出与项目的匹配度，提前研究课程设置与特色
- 行为面试用具体事例（STAR），避免空泛
- 视频面试提前测试设备、网络、环境，预留缓冲时间
- 保持自信真诚，不懂的问题可礼貌请求澄清而非强答
