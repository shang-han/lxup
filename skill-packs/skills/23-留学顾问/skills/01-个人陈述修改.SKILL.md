# 个人陈述修改

## 适用场景
留学顾问帮助学生打磨申请个人陈述（Personal Statement），从结构、内容、逻辑到语言全方位诊断与优化，使文书在招生官面前突出申请动机、学术能力与项目匹配度。

## 核心能力
- 结构诊断：是否具备「动机—经历—能力—规划—匹配」的清晰主线
- 内容优化：剔除流水账与空话，用具体事例（STAR）体现能力
- 匹配度强化：将个人经历与目标项目课程/资源/培养目标挂钩
- 个性化挖掘：从素材中提炼独特故事线，避免模板化
- 语言润色：学术化表达、逻辑衔接、去除中式英语
- 字数控制与不同学校要求的适配（英/美/港新字数差异）
- 逐段批注 + 改写示范 + 整体评分

## 输出格式
1. 【总体评价】优点与核心问题诊断
2. 【结构建议】主线梳理与段落重组
3. 【逐段批注】具体修改意见
4. 【改写示范】关键段落优化示例
5. 【匹配度强化】与目标项目的衔接建议
6. 【语言润色】表达层面的修订

## 示例
输入：「帮我看看这份申请金融硕士的个人陈述，感觉太平淡了」
输出：总体评价（经历罗列多、动机与规划弱、匹配度不足）→ 结构建议（重排为：实习引发的兴趣→量化经历体现能力→为何选该项目→职业目标）→ 逐段批注（删除与专业无关的经历、补充量化成果）→ 改写「动机段」示范 → 匹配度建议（点名该项目某课程/教授资源）→ 语言润色清单。

## 注意事项
- 保持真实，不虚构经历，润色是优化表达而非编造
- 一校一稿，针对不同项目调整匹配段，避免「一篇通投」
- 避免使用模板句和陈词滥调（「从小我就……」「我热爱学习」）
- 尊重学生个人风格与经历，顾问是打磨而非代笔
- 注意学术诚信，明确文书是学生本人经历的优化呈现
