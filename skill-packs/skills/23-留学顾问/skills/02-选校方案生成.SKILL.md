# 选校方案生成

## 适用场景
留学顾问为学生制定申请院校清单时，需要综合背景、目标、预算、地域偏好，生成「冲刺—匹配—保底」梯度合理、可执行的选校方案。

## 核心能力
- 背景评估：GPA、标化、院校背景、实习科研、语言成绩综合定位
- 梯度设计：冲刺/匹配/保底按比例配置，控制申请风险
- 院校信息整合：录取要求、申请轮次、截止日期、学费、就业数据
- 多国/多地区联申方案，分散风险并把握时间线
- 结合预算、地域、就业/移民意向筛选
- 专业细分方向匹配（同校不同项目难度差异）
- 输出含时间表的申请执行计划

## 输出格式
1. 【背景定位】学生综合条件概述
2. 【选校梯度表】冲刺/匹配/保底院校清单（含专业、要求、截止日期）
3. 【录取概率分析】各档院校的竞争解读
4. 【联申策略】多地区组合与时间协调
5. 【申请时间表】各校各轮次关键节点
6. 【建议与提醒】选校逻辑与注意事项

## 示例
输入：「双非本科，GPA3.5，雅思7.0，想申英国商科硕士，预算40万，帮我选校」
输出：背景定位（双非、均分尚可、雅思达标）→ 选校梯度（冲刺：王爱曼华部分项目；匹配：布里斯托/格拉斯哥/伯明翰；保底：南安/利兹/卡迪夫）→ 录取概率分析（双非需注意名单限制）→ 联申建议（可加港新/澳洲分散）→ 申请时间表（各校轮次截止）→ 提醒（关注认可院校名单、部分项目需 GMAT、尽早递交）。

## 注意事项
- 数据以各校当年官网要求为准，标注查询时效
- 不打包票承诺录取，客观呈现概率与风险
- 关注院校「认可名单/list」对双非/均分的硬性限制
- 梯度要真保底，避免全冲刺导致失学
- 结合就业/移民/预算等真实诉求，而非唯排名论
