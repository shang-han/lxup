# 行业基础

## 行业概况
中国每年出国留学人数超 80 万，主要目的地为英、美、澳、加、日及港新地区。留学服务链条涵盖选校 → 申请 → 签证 → 行前，核心交付物是文书质量与录取结果。行业竞争核心在于成功案例积累、文书专业度与信息及时性（招生政策、截止日期、签证新规）。服务周期长（半年到两年），客户决策谨慎、客单价高，口碑与转介绍是获客关键。近年呈现低龄化、多国联申、背景提升前置等趋势。

## 常用术语
- GPA：平均学分绩点，申请的核心硬指标
- 标化成绩：托福/雅思（语言）、GRE/GMAT（研究生）、SAT/ACT（本科）等标准化考试分数
- 早申（EA/ED）：早期申请，通常录取率更高，ED 有绑定约束
- RD：常规轮申请
- 留位费：录取后确认入学需缴纳的押金，抵充学费
- PS/CV/RL：个人陈述 / 个人简历 / 推荐信，申请三大文书
- 联申：同时申请多个国家/地区以分散风险
- 背景提升：科研、实习、竞赛、夏校等软实力的规划积累

## 核心工作流程
规划期：背景评估 → 目标定位 → 考试规划 → 活动/科研规划
申请期：选校清单 → 文书创作修改 → 网申提交 → 面试辅导
录取后：定校确认 → 缴纳留位费 → 签证办理 → 住宿安排 → 行前指导

## 常见问题
Q：我这个背景能申到什么学校？
A：综合 GPA、标化成绩、院校背景、实习科研、文书表现五项评估，给出冲刺/匹配/保底梯度，不打包票。

Q：文书怎么写才出彩？
A：围绕「个人故事线 + 专业认知 + 职业规划 + 与学校的匹配度」展开，避免流水账和空泛口号，用具体事例体现能力与动机。

Q：要不要找中介/顾问？
A：信息不对称、时间紧张、需要文书专业打磨的学生适合找顾问；核心价值在选校策略、文书质量与流程把控。

Q：申请截止日期怎么把控？
A：建立各校各轮次截止时间表，提前 2-4 周完成文书，预留推荐信、成绩单、送分等环节时间，避免压线。
