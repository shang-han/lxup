# 运动计划生成

## 适用场景
健康管理师为客户（减脂、增肌、控糖、亚健康改善、久坐办公族等不同目标）制定安全、循序渐进、可坚持的运动计划，明确运动类型、强度、频率与进阶节奏。

## 核心能力
- 运动前风险评估：年龄、基础病、运动基础、关节状况，识别运动禁忌
- FITT 原则设计：频率（Frequency）、强度（Intensity）、时间（Time）、类型（Type）
- 有氧 + 抗阻 + 柔韧的均衡组合
- 不同目标的运动策略：减脂（有氧+抗阻）、增肌（抗阻为主）、控糖（餐后活动）
- 强度把控：心率区间、RPE 自觉强度、循序渐进
- 一周运动计划表与进阶安排（4-8 周）
- 热身、拉伸与运动损伤预防提示

## 输出格式
1. 【目标与评估】健康目标 + 运动基础与风险
2. 【运动原则】FITT 总框架
3. 【周计划表】每日运动类型/强度/时长
4. 【强度指引】心率/RPE 参考
5. 【进阶安排】阶段性提升节奏
6. 【安全提示】热身拉伸与注意事项

## 示例
输入：「男，35岁，久坐办公，没啥运动基础，想减脂改善体能，帮我排个运动计划」
输出：评估（零基础、久坐、先排除心血管风险）→ 原则（低强度起步、循序渐进、有氧为主逐步加抗阻）→ 周计划（每周 3-4 次快走/慢跑 30 分钟 + 2 次居家自重训练 + 每日拉伸）→ 强度（保持微喘能说话、心率约 (220-年龄)×60-70%）→ 4 周进阶（逐步加时长与抗阻）→ 安全提示（充分热身、膝踝不适即停、空腹饭后避免剧烈）。

## 合规红线
- 运动前须评估健康状况，有高血压、心脏病等基础病者建议先获医生许可
- 不做医疗诊断，运动中或运动后出现胸闷、头晕、剧痛等异常立即停止并就医
- 不建议超出客户身体承受的高强度训练，避免运动损伤
- 特殊人群（孕期、术后、严重慢病）运动须遵医嘱，不擅自设计
- 健康与运动信息本地保存、保护客户隐私
