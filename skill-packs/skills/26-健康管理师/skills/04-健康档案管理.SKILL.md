# 健康档案管理

## 适用场景
健康管理师为客户建立并维护长期健康档案，规范采集与记录健康信息、随访情况与干预效果，便于纵向追踪指标变化、评估管理成效，同时确保信息安全合规。

## 核心能力
- 档案结构设计：基本信息、既往史、家族史、体测数据、生活方式、干预方案、随访记录
- 标准化信息采集表单与字段定义
- 随访记录模板：时间、指标、依从性、调整、下次计划
- 指标纵向追踪表：体重、血压、血糖等关键指标的时间序列
- 干预效果评估与阶段小结
- 档案分类、检索与到期随访提醒机制
- 隐私保护与数据安全管理规范

## 输出格式
1. 【档案结构】字段框架与分类
2. 【信息采集表】建档需收集的项目
3. 【随访记录模板】单次随访填写格式
4. 【指标追踪表】关键指标时间序列
5. 【阶段小结模板】效果评估与调整
6. 【管理规范】存储、检索、保密要求

## 示例
输入：「我要给一批企业客户建健康档案，帮我设计档案结构和随访记录模板」
输出：档案结构（基本信息/健康史/体测/生活方式/干预方案/随访六大模块）→ 建档采集表（含家族史、慢病史、过敏史、运动饮食习惯）→ 随访记录模板（日期、本次指标、依从性、方案调整、下次随访时间）→ 指标追踪表（体重/BMI/血压/腰围按月记录）→ 阶段小结模板（季度效果评估）→ 管理规范（专人管理、加密存储、本地保存、未经授权不外泄）。

## 合规红线
- 健康信息本地保存、加密管理，严格保护客户隐私
- 未经客户书面授权，不向任何第三方提供健康档案信息
- 如实记录，不篡改、不伪造健康数据与随访记录
- 档案仅用于健康管理目的，不作其他商业用途
- 档案中发现异常指标应及时提示客户就医，不隐瞒不延误
