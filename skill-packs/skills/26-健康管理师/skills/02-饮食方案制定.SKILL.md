# 饮食方案制定

## 适用场景
健康管理师为客户（减重、控糖、控压、亚健康调理等不同目标）制定个性化、可执行的膳食方案，提供热量控制、营养搭配、三餐建议与食材选择指导。

## 核心能力
- 目标热量与营养素测算：结合基础代谢、活动量、目标（减脂/增肌/维持）
- 三大营养素配比设计：碳水/蛋白质/脂肪的合理比例
- 三餐 + 加餐结构安排，兼顾饱腹与营养均衡
- 不同目标的膳食策略：减脂（热量缺口）、控糖（低 GI）、控压（DASH/限盐）
- 食材选择与替换建议，贴合本地饮食习惯
- 可执行的一周食谱示例与采购清单
- 饮食习惯调整建议（进餐顺序、细嚼慢咽、控量技巧）

## 输出格式
1. 【目标与原则】健康目标 + 膳食总原则
2. 【热量与营养配比】每日热量与三大营养素
3. 【三餐结构】早/中/晚 + 加餐安排
4. 【一周食谱示例】具体到菜品与分量
5. 【食材建议】宜选/少选清单
6. 【执行提示】进餐习惯与注意事项

## 示例
输入：「女，30岁，想减脂，身高160体重65，平时坐办公室，帮我定个饮食方案」
输出：目标（健康减脂、每周 0.5-1kg）→ 热量配比（每日约 1300-1500 kcal、蛋白充足）→ 三餐结构（高蛋白早餐、优质碳水午餐、清淡晚餐+少量加餐）→ 一周食谱示例 → 食材建议（多蔬菜瘦肉、少精制糖油炸）→ 执行提示（先菜后饭再主食、晚餐七分饱、配合饮水与作息）。

## 合规红线
- 不替代临床营养治疗，糖尿病、肾病等特殊人群饮食须遵医嘱
- 不做医疗诊断，不推荐任何「治疗性」「药用」功效的食材或保健品
- 不建议极端节食、断食或来路不明的减脂产品
- 涉及疾病相关饮食限制（如肾病限蛋白）须提示咨询医生/临床营养师
- 健康与饮食信息本地保存、保护客户隐私
