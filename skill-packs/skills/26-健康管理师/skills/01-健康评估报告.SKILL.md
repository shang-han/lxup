# 健康评估报告

## 适用场景
健康管理师在客户建档或随访时，根据健康问卷、体测数据（身高体重、血压、腰围、体脂等）与体检结果，生成结构化的健康评估报告，识别健康风险并指明生活方式改善方向。

## 核心能力
- 体测指标计算与解读：BMI、体脂率、腰臀比、基础代谢率、血压分级
- 健康问卷信息整合：家族史、生活习惯、既往史、饮食运动情况
- 健康风险分级：按指标划分低/中/高风险并说明依据
- 生活方式问题识别：饮食、运动、作息、压力、烟酒等可干预因素
- 改善方向建议：指向饮食/运动/作息调整，不涉及诊疗
- 异常指标识别与就医提示
- 报告语言通俗、客观，兼顾专业性与可读性

## 输出格式
1. 【基本信息】年龄/性别/体测概览
2. 【指标解读】各项数据与参考范围对照
3. 【风险分级】整体与分项风险评级
4. 【生活方式评估】饮食/运动/作息现状
5. 【改善方向】生活方式干预重点
6. 【就医提示】需就医/复查的异常项

## 示例
输入：「男，42岁，身高172体重85，血压140/92，帮我出个健康评估」
输出：基本信息 → 指标解读（BMI 28.7 属超重、血压 140/92 达高血压一级范围、估算体脂偏高）→ 风险分级（心血管代谢风险中-高）→ 生活方式评估（需补充饮食运动作息信息）→ 改善方向（控制总热量、增加有氧、规律作息、限盐限酒）→ 就医提示（血压偏高建议心内科就诊确认是否需药物干预，勿自行判断）。

## 合规红线
- 不做医疗诊断、不下疾病结论，评估仅限健康风险与生活方式层面
- 不替代医生，异常指标（高血压、高血糖、肝肾功能异常等）一律引导就医
- 不建议用药、不调整客户现有医嘱用药方案
- 健康信息本地保存、严格保密，不向第三方泄露
- 表述客观，不夸大风险制造焦虑，也不淡化真实问题
