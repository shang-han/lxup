# 行业基础

## 行业概况
健康管理行业受「健康中国 2030」政策利好，市场需求旺盛、专业人才缺口大。服务涵盖健康体检评估、慢病管理、亚健康调理、体重管理、生活方式干预等，服务对象包括企业团体客户与个人客户。健康管理师的核心是「治未病」——通过饮食、运动、作息、心理等生活方式干预降低疾病风险、改善健康指标，而非医疗诊治。服务以「评估—干预—跟踪」为闭环，强调长期陪伴与数据化随访。

## 常用术语
- BMI：身体质量指数 = 体重(kg) / 身高(m)²，衡量胖瘦的常用指标
- 体脂率：脂肪重量占体重的比例，比 BMI 更能反映身体成分
- 基础代谢率（BMR）：静息状态下维持生命消耗的最低热量
- 慢病：高血压、糖尿病、高血脂等慢性非传染性疾病
- 健康管理方案：饮食 + 运动 + 作息 + 心理的综合干预方案
- 腰臀比：腰围与臀围之比，反映中心性肥胖与代谢风险
- 随访：对客户健康指标的定期跟踪回访
- 健康危险因素：吸烟、缺乏运动、不合理膳食等可干预的风险因素

## 核心工作流程
客户接诊：健康问卷 → 体测评估 → 报告解读 → 方案制定
干预阶段：饮食指导 → 运动监督 → 作息调整 → 心理支持
跟踪阶段：定期回访 → 指标监测 → 方案调整 → 效果评估

## 常见问题
Q：健康管理多久能看到效果？
A：生活方式干预一般 2-4 周开始有体感变化，3 个月左右指标（体重、血压、血糖等）可有明显改善，需坚持。

Q：健康管理需要吃药吗？
A：以生活方式干预为主，不替代医疗。涉及用药须遵医嘱，发现异常指标应及时引导就医，不与临床治疗冲突。

Q：客户依从性差、坚持不下来怎么办？
A：设定可达成的小目标、建立随访与打卡机制、给予正向反馈、找到客户内在动机，必要时降低方案难度循序渐进。

Q：健康管理和看病有什么区别？
A：健康管理聚焦「未病先防、慢病防控」的生活方式干预，不做诊断和治疗；已患病者以临床治疗为主，健康管理为辅并须配合医生。
