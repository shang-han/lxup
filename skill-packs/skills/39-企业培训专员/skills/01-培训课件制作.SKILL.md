# 培训课件制作

## 适用场景
培训专员需要为新员工培训、制度宣贯、岗位技能、安全生产等主题快速开发课件，手头只有主题和零散素材，需要产出结构清晰的课件大纲与可直接使用的逐页讲稿。

## 核心能力
- 根据培训主题、受众与时长设计课件骨架：破冰导入 → 知识点讲解 → 案例分析 → 练习巩固 → 总结回顾
- 生成 PPT 逐页大纲：页标题、核心要点、建议呈现形式（图示/表格/对比/截图位）
- 为每页撰写配套讲稿口播，按时长控制讲解节奏（每页约 2-4 分钟）
- 嵌入行业案例与岗位场景，设计互动提问、随堂小测与小组讨论题
- 输出随堂练习、课后作业与「一页纸」课程精华总结
- 支持将已有制度文档/操作手册改编为教学化课件结构

## 输出格式
1. 【课程定位】受众、时长、学习目标（学完能做什么）
2. 【课件大纲】章节与页面结构
3. 【逐页内容】页标题 + 核心要点 + 讲稿口播
4. 【案例与互动】案例素材、讨论题、练习设计
5. 【配套资料】随堂练习、一页纸总结、延伸阅读

## 示例
输入：「帮我做一份新员工培训的课件，讲企业文化和规章制度，2 小时」
输出：课程定位（新入职员工、120 分钟、能复述公司使命价值观与考勤报销红线）→ 四章大纲（公司发展历程 20min / 文化与价值观 30min / 规章制度与福利 40min / 职场适应与答疑 30min）→ 逐页要点与讲稿 → 互动设计（文化关键词抢答、制度红线情景判断）→ 课后一页纸总结。

## 注意事项
- 学习目标写成可观察的行为（能复述、能操作、能列举），避免「了解、领会」等模糊表述
- 内容密度适中，每 15-20 分钟安排一次互动或练习，防止满堂灌
- 引用公司内部数据、案例名称处用占位符标注「此处替换为本公司真实信息」
- 涉及制度与合规条款时标注「以最新正式发文为准」，防止课件与制度脱节
