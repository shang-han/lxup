# 培训效果评估

## 适用场景
培训结束后需要向管理层汇报培训效果，或培训价值受到质疑需要建立系统化评估机制，用数据证明培训投入的回报并提出改进方向。

## 核心能力
- 按培训类型选择评估层级：通用课做反应层+学习层，重点项目延伸到行为层与结果层
- 生成评估工具文本：满意度问卷、随堂考试题、训后行为观察表、上级访谈提纲
- 设计数据采集时点与责任人：训后即采、训后 1 个月、训后 3 个月
- 定义评估指标口径：满意度、通过率、行为改善率、业绩前后对比
- 输出评估报告：结论先行 → 数据详表 → 问题归因 → 课程改进建议
- 估算培训 ROI：收益量化口径与成本归集方法

## 输出格式
1. 【评估目标】评估层级、要回答的业务问题
2. 【评估设计】评估维度、工具、时点与样本
3. 【评估工具文本】问卷/考卷/观察表/访谈提纲
4. 【数据分析框架】指标定义与统计口径
5. 【报告结构】结论 → 数据详表 → 改进建议

## 示例
输入：「上个月做了销售技巧培训，老板问效果怎么样，帮我设计评估方案」
输出：评估目标（学习层+行为层+结果层，回答培训是否带动成交）→ 设计（训后即考 10 题 + 1 个月上级行为观察 + 3 个月业绩对比）→ 工具文本（试卷、8 项行为观察表含「是否使用 SPIN 提问」等、业绩对比口径）→ 分析框架（通过率、行为达标率、训前训后成交率对比）→ 报告结构与归因建议模板。

## 注意事项
- 并非所有培训都要做到结果层评估，低成本通用课止于反应层+学习层即可，避免过度投入
- 行为层评估依赖上级配合，提前说明评估目的（改进课程而非考核个人），降低抵触
- 数据对比要有可比基线：同组训前训后对比，或设对照组
- 评估结论如实呈现，包含数据局限与干扰因素，避免为讨好上级夸大成果
