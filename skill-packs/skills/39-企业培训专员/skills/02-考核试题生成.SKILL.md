# 考核试题生成

## 适用场景
培训后考核、新员工转正考试、岗位技能定期测试、合规年度考试等场景，培训专员需要依据讲义或制度文档快速出一份规范的试卷与答案。

## 核心能力
- 从培训材料/制度文档中提取考点，生成考点分布表，保证覆盖核心内容
- 支持多题型组卷：单选、多选、判断、填空、简答、案例分析
- 按难度配比（基础 60% / 中等 30% / 应用 10%）与分值结构控制题量
- 每题附标准答案与解析，解析指向对应知识点或文档出处
- 输出试卷版（供打印/线上考试导入）与答案版分离的两份文档
- 可对同一材料生成 A/B 卷或补考卷，避免原题重复

## 输出格式
1. 【试卷信息】考核主题、时长、总分、题型分值结构
2. 【试卷正文】按题型分组，每题标注分值
3. 【答案与解析】逐题对应，解析简明点出考点
4. 【难度与覆盖】难度配比与考点覆盖统计

## 示例
输入：「根据消防安全培训讲义出一份考核题，满分 100 分」
输出：试卷信息（100 分 / 60 分钟）→ 试题（单选 20×2 分、多选 10×3 分、判断 20×1 分、简答 1×10 分）→ 每题附答案与解析（如「灭火器使用四步：提拔握压」对应讲义第 3 节）→ 难度分布（基础 60% / 中等 30% / 应用 10%）→ 单独附答案版。

## 注意事项
- 考点均匀覆盖核心章节，避免集中在某一小节或只考细枝末节
- 选项长度与句式相近、干扰项有迷惑性，避免格式上泄露答案
- 涉及数字、条款、流程顺序的题目严格依据源文档，拿不准处标注「以原文为准」
- 高利害考试（转正、持证上岗）建议备 A/B 卷并设及格线与补考规则
