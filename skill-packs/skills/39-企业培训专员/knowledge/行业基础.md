# 行业基础

## 行业概况
企业培训涵盖新员工培训、岗位技能培训、管理能力培训、合规培训等类型，是企业人才梯队建设与组织能力沉淀的关键环节。培训形式包括线下集训、线上课程、在岗实践与混合式教学。核心考核指标：培训覆盖率、满意度、知识转化率。随着数字化学习平台与企业知识库普及，培训专员的工作重心正从「组织办班」转向「设计学习体验、推动行为改变、沉淀组织知识」。

## 常用术语
- 培训覆盖率：受训员工占总员工的比例
- 培训满意度：培训后问卷对课程与讲师的评分
- 知识转化率：所学内容实际应用到工作中的比例
- 柯氏四级评估：反应（听得怎样）、学习（学会什么）、行为（回去用没用）、结果（对业绩的影响）四级评估模型
- 培训 ROI：培训收益与培训成本的比值
- 培训需求调研：通过问卷、访谈、绩效数据识别能力差距与培训重点
- 课程体系：按岗位序列与职级设计的结构化课程地图
- 内训师：经认证承担内部授课任务的业务骨干或管理者

## 核心工作流程
需求期：培训需求调研 → 能力差距分析 → 课程规划 → 讲师筛选 → 预算制定
执行期：课程开发 → 培训通知与组织 → 培训实施 → 现场管理 → 考核评估
跟进期：效果评估 → 行为跟踪 → 数据复盘 → 课程迭代优化

## 常见问题
Q：培训效果怎么评估？
A：按柯氏四级落地：反应层（满意度问卷）+ 学习层（考试/实操）+ 行为层（训后 1-3 个月上级评价与行为观察）+ 结果层（业绩指标前后对比）。

Q：员工不愿意参加培训怎么办？
A：内容紧贴实际工作痛点，邀请部门负责人做开班动员，培训与考核晋升挂钩，并设置优秀学员奖励。

Q：内部讲师怎么管？
A：建立讲师选拔、认证、授课费与评优制度，提供课程开发辅导，把授课贡献纳入绩效或晋升评价。

Q：知识库建了没人用怎么办？
A：把知识库嵌入实际工作流（新人入职必修、答疑先查库），设贡献激励与定期评审，指定每篇文档负责人。
