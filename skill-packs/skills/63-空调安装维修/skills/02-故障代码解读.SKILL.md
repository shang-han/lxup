# 故障代码解读

## 适用场景
客户报修说「空调显示 E1/ flashing 灯」「开了半天不凉」，师傅远程初步判断或上门前准备工具配件，快速锁定故障范围。

## 核心能力
- 按品牌系列（格力/美的/海尔/大金/变频通用）解读常见故障代码含义
- 现象归类：不制冷/不制热、不开机、漏水、异响、跳闸、通讯故障
- 给出从简到繁的排查路径（遥控器设置→滤网→电源→传感器→冷媒→主板）
- 判断用户可自行处理项（滤网脏堵、模式设错）与必须上门项
- 结合代码预估所需配件与工时，辅助接单报价
- 安全类故障（漏电跳闸、R32 泄漏）的应急处置提示

## 输出格式
1. 【代码/现象含义】对应的故障部件或系统
2. 【常见原因】按概率排序
3. 【排查步骤】逐步操作与仪表
4. 【用户可自查项】让客户先试的简单操作
5. 【上门准备】需带的配件/工具
6. 【安全提示】涉及电/冷媒的风险点

## 示例
输入：「客户说格力空调显示 E1，室内机不制冷」
输出：E1 多为高压保护/系统高压异常 → 原因：①外机冷凝器脏堵散热不良 ②外机风扇不转 ③冷媒过多或系统堵塞 ④高压开关故障 → 自查：让客户先看外机有没有转、散热片是不是很脏（脏了清洗可缓解）→ 上门准备：压力表、万用表、清洗设备 → 安全：检测前断电，R32 机型检漏禁用明火。

## 注意事项
- 同一代码不同品牌/年份含义不同，务必让客户拍铭牌确认具体型号
- 远程解读给方向不给定论，话术留余地（「大概率是…，具体上门测了才知道」）
- 高压/低压保护往往是系统问题的表象，别只复位了事，要查根因
- 涉及制冷剂操作提示持证规范作业，R32 泄漏先通风断电
