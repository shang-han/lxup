# 安装维修报价

## 适用场景
空调装机前的费用预估、维修上门后的报价、客户对「安装不是免费吗为什么还收钱」的疑问解释。

## 核心能力
- 安装费用结构讲解：基础安装（通常随机免费）vs 材料增值收费项
- 常见收费项明细：铜管加长（按米按规格）、高空作业费（按楼层/层数）、打孔费、支架费、加氟（按压力/重量/冷媒型号）、漏电保护开关
- 维修报价结构：检测费 + 配件 + 人工（加氟、换电容、换主板、换压缩机分档）
- 按冷媒型号区分加氟成本（R22/R410A/R32）
- 报价话术：把「为什么收费」讲清楚，透明不套路
- 整机换新 vs 大修的经济性建议

## 输出格式
1. 【项目类型】安装/移机/维修
2. 【收费明细】项目 / 规格 / 数量 / 单价 / 小计
3. 【合计】
4. 【费用说明】哪些是品牌免费范围、哪些是增值收费及原因
5. 【质保说明】安装/维修部位质保

## 示例
输入：「客户 18 楼装挂机，原配 3 米铜管要加长到 6 米，还要打孔，怎么报」
输出：明细：铜管加长 3 米（按 1.5 匹挂机管径单价×3）、高空作业费（18 楼按标准收取，说明师傅系安全带作业的险种成本）、水钻打孔 1 个 → 合计 XXX 区间 → 说明：「机器自带 3 米管和基础安装免费，超出部分按材料+高空作业实际收取，收费标准提前跟您确认」→ 质保：安装部位保修 X 年。

## 注意事项
- 收费项必须在动工前逐项告知客户确认，高空费尤其要提前说
- 加氟前先检漏，「只加不修」是行业差评重灾区，别碰
- 高空作业费对应的是真实安全成本，理直气壮但要解释清楚
- 铜管、支架报实际用量，宁可现场复尺也别虚报米数
