# 旺季调度管理

## 适用场景
夏季高温空调单爆满，需要合理安排每日安装/维修路线与时间窗、应对客户催单与插队请求、管理多师傅团队的调度。

## 核心能力
- 每日排单建议：按区域集中派单减少路途、按作业类型（安装/维修）批量安排
- 预约时间窗话术：给区间不给准点、预留缓冲、提前半天确认
- 单量超载时的接单策略：分级排优先级（完全不制冷+老人小孩家庭优先）、引导非急单错峰
- 客户催单安抚话术：如实告知排队情况、给明确预计时间、不空头承诺
- 插队请求处理话术：加价急单规则或婉拒
- 旺季前准备提醒：常用配件备货、车辆工具检查、防暑安排

## 输出格式
1. 【排单方案】当日/次日单子的区域化路线与时间窗建议
2. 【预约话术】新客户预约怎么回
3. 【催单安抚话术】排队客户怎么回
4. 【优先级建议】哪些单优先、哪些可协商错峰
5. 【备货/准备提醒】

## 示例
输入：「今天有 6 个安装单分散在三个区，还有客户一直打电话催，帮我排一下和回话」
输出：排单：按区分三组，A 区 2 单上午连做、B 区 3 单下午、C 区 1 单收尾，各给 2 小时时间窗 → 预约话术：「现在高温季单子排到 X 天后了，给您约周三上午可以吗，前一天晚上跟您确认」→ 催单安抚：「您家单子我记着呢，排在周四，师傅到了提前半小时给您打电话，实在抱歉让您久等」→ 优先级：有老人婴儿的全不制冷单优先协商插空。

## 注意事项
- 再忙也不承诺做不到的时间，失信比晚到更伤客源
- 高温季师傅防暑与作息要排进去，疲劳作业出安全事故全白干
- 催单客户给「确定性」（明确日期+提前通知）比反复道歉有用
- 旺季宁可少接单保质量，安装返工和差评的代价远大于多赚一单
