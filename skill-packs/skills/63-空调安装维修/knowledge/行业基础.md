# 行业基础

## 行业概况
空调行业「三分产品、七分安装」，安装与维修质量直接决定使用效果和寿命。需求高度季节性：夏季制冷（5-8 月）、冬季制热（部分地区）为绝对旺季，旺季单量可达淡季数倍，对调度与体力都是考验。服务形态：品牌售后、平台派单、个体师傅（小区+熟人渠道）。高空作业是行业特有风险点，持证上岗与保险是硬性要求。冷媒正从 R410A 向可燃的 R32 切换，对规范操作要求更高。

## 专业技能体系
### 硬技能
- 空调选型与安装位置评估
- 铜管焊接与制冷系统操作
- 高空作业安全规范
- 多联机/中央空调系统调试
- 故障代码解读与快速定位
### 软技能
- 客户安装方案解释
- 旺季调度与预约管理

## 常用术语
- 匹数：制冷量通俗单位，1 匹≈2500W 制冷量，对应约 10-15㎡
- 冷媒/制冷剂：R22（淘汰中）、R410A、R32（可燃，需抽真空）
- 抽真空：安装时用真空泵抽空管路空气水分，R32/R410A 必做
- 铜管加长：原厂配管不够时按米收费加接，影响制冷效果
- 高空作业费：4 楼及以上外墙作业收取的安全成本费
- 多联机/中央空调：一拖多系统，分歧管分配冷媒到多个内机
- 高压/低压保护：系统压力异常的自我保护停机（故障代码常见来源）
- 冷凝水：制冷产生的排水，需坡度排向地漏，否则室内漏水

## 核心工作流程
接单：问型号/现象/楼层 → 判断安装或维修 → 预约时间
安装：现场勘查 → 方案沟通确认 → 打孔装支架 → 挂内外机 → 接管抽真空 → 试机
维修：复现故障 → 读码/检测 → 报价确认 → 维修（检漏/加氟/换件）→ 试机
收尾：讲解使用 → 收款 → 质保告知 → 旺季复盘与备货

## 常见问题
Q：客户问「安装不是免费的吗，怎么还收这么多」？
A：解释「免费的是随机基础安装（标准配管+人工），超出部分如加管、高空、打孔是实际材料和作业成本」，动工前逐项报价确认，透明就没争议。

Q：旺季单子接不完怎么取舍？
A：按区域集中排单降路途、完全不制冷+特殊家庭优先、非急单引导错峰，宁可少接保质量，安装返工代价远大于多赚一单。

Q：加氟要不要先检漏？
A：必须检漏。制冷剂是密封循环不会无故减少，缺氟必有漏点，只加不修用不了多久又漏，还砸自己招牌。

Q：高空作业最要注意什么？
A：持证上岗、安全带双钩挂牢、地面设警戒、工具系防坠绳，6 级以上大风和雷雨高温红色预警一律停工，命比单重要。
