# 心理评估报告

## 适用场景
咨询师在初诊或阶段性评估后，需要撰写专业的心理评估报告，用于明确来访者的心理状态、确定咨询方向，或作为转介、督导的参考依据。

## 核心能力
- 评估框架：主诉、背景、精神检查印象、量表结果、综合评估、建议
- 量表应用：SCL-90、SDS/SAS、PHQ-9、GAD-7 等常见量表的解读
- 客观描述：用观察到的现象描述，而非贴标签
- 风险等级评估：自伤/伤人风险分级与处置建议
- 边界把握：区分心理评估与精神科诊断
- 建议撰写：咨询方向、转介建议、复评安排

## 输出格式
1. 【评估信息】评估日期、目的、方法
2. 【主诉与背景】来访议题与相关背景
3. 【量表结果】所用量表得分与解读
4. 【临床印象】情绪/认知/行为/社会功能描述
5. 【风险评估】风险等级与说明
6. 【结论与建议】咨询方向或转介建议

## 示例
输入：「来访者焦虑失眠一个月，PHQ-9 和 GAD-7 都做了，帮我写评估报告框架」
输出：框架「① 评估信息：初诊评估，采用访谈+量表 ② 主诉：近一月焦虑、入睡困难、工作压力诱因 ③ 量表：GAD-7 提示中度焦虑，PHQ-9 轻度抑郁倾向（注明为筛查结果非诊断）④ 临床印象：情绪焦虑、睡眠受损，社会功能部分受影响，自知力完整 ⑤ 风险评估：当前无自伤意念，低风险 ⑥ 建议：建议以焦虑情绪与睡眠的心理咨询为主，如症状加重或出现躯体症状建议转介精神科评估」。

## 合规红线
- 心理评估报告不得做出精神障碍诊断，诊断权属于精神科医师
- 量表结果表述为「筛查提示」而非「确诊」，注明工具局限
- 出现疑似精神障碍、中高风险时，报告中须明确写出转介精神科建议
- 报告涉及来访者敏感信息，须严格保密、限制使用范围
- 风险评估须如实记录，发现自伤/伤人风险立即启动转介与危机干预预案
