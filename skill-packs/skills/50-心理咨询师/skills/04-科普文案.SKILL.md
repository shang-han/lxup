# 科普文案

## 适用场景
心理咨询师通过科普内容进行公众教育或个人品牌建设时，需要撰写关于情绪管理、压力调节、心理知识的文章或推文，既要专业准确又要通俗易传播。

## 核心能力
- 选题库：焦虑、抑郁情绪、睡眠、亲子关系、职场压力、自我成长
- 通俗化表达：把专业概念转化为大众能懂的语言与比喻
- 结构组织：现象引入 → 原理科普 → 自助方法 → 求助指引
- 多平台适配：公众号长文、短视频脚本、朋友圈短文
- 去污名化：消除对心理问题的误解与偏见
- 求助引导：区分正常情绪与需专业帮助的信号

## 输出格式
1. 【标题】吸引人且不失专业的标题
2. 【现象引入】从常见困扰切入
3. 【原理解读】心理学机制通俗讲解
4. 【自助方法】可操作的调节建议
5. 【求助指引】何时需要专业帮助
6. 【话题标签】平台话题词

## 示例
输入：「帮我写一篇关于缓解焦虑的科普推文」
输出：标题「总觉得要出事？可能是焦虑在给你'假警报'」→ 引入（明明没事却心慌、睡不踏实的常见体验）→ 原理（焦虑是大脑的预警系统，过度激活会误报，通俗比喻）→ 自助（腹式呼吸 4-7-8、着陆技术、限制过度信息摄入）→ 指引（若持续两周以上、影响生活工作，建议寻求专业心理咨询）→ 标签 #心理健康 #焦虑 #情绪管理。

## 合规红线
- 科普内容不得提供诊断或替代专业咨询/治疗，须注明「不替代专业帮助」
- 涉及抑郁、自伤等敏感话题须谨慎，避免诱导或详细描述危险行为，附求助渠道
- 不做疗效承诺，不用「根治」「一定能好」等绝对化表述
- 引用研究或数据应可靠，不传播伪心理学与伪科学
- 尊重来访者隐私，案例须彻底改编脱敏，不使用可识别素材
