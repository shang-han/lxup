# 跟进计划

## 适用场景
咨询师在咨询进程中或结束阶段，需要为来访者制定跟进随访计划，巩固咨询效果、预防复发、监测风险，并做好必要的转介衔接。

## 核心能力
- 复评节点：按阶段安排量表复评与进展评估
- 频次调整：从规律咨询逐步过渡到巩固性随访
- 复发预防：识别预警信号、巩固已学技能
- 风险监测：对有风险来访者的定期安全随访
- 转介衔接：需医疗介入时的转介与协同跟进
- 结束处理：咨询关系的妥善收尾与回访安排

## 输出格式
1. 【跟进目标】巩固效果 / 预防复发 / 风险监测
2. 【复评节点】时间与评估方式
3. 【随访安排】频次与形式
4. 【复发预防】预警信号与应对技能
5. 【转介衔接】需医疗介入的触发条件与流程
6. 【结束回访】咨询结束后的回访计划

## 示例
输入：「来访者焦虑咨询接近尾声，帮我做个跟进计划」
输出：目标「巩固咨询效果、预防焦虑复发」→ 复评（结束后 1 个月、3 个月各做一次 GAD-7 复评）→ 随访（从每周 1 次过渡到每两周、每月 1 次巩固，共 3 个月）→ 复发预防（识别睡眠变差、回避社交等预警信号，坚持呼吸放松与认知记录练习）→ 转介（如出现持续躯体症状或情绪明显恶化，建议转介精神科）→ 结束回访（结案后 1 个月主动回访一次，评估适应情况）。

## 合规红线
- 随访须在来访者知情同意下进行，尊重其是否继续跟进的意愿
- 风险来访者的随访须包含安全评估，发现自伤/伤人风险立即启动危机转介
- 随访记录同样遵循最小化留存与保密原则
- 不以来访者依赖为目的延长咨询，适时妥善结束
- 涉及精神科协同的跟进，明确心理咨询与医疗的边界与分工
