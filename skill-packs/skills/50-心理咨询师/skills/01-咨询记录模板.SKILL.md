# 咨询记录模板

## 适用场景
咨询师在每次咨询后需要撰写规范的个案记录，既要满足专业留档与督导需要，又要遵循保密与最小化留存原则，保护来访者隐私。

## 核心能力
- 标准记录结构：主诉、现状评估、会谈内容、干预技术、风险标注、下次计划
- 客观记录原则：区分客观观察与主观推断，避免评判性语言
- 风险记录：自伤/伤人风险的识别与记录规范
- 最小化留存：只记录专业必要信息，不记录无关隐私细节
- 保密标注：记录材料的保管与访问权限说明
- 多流派适配：CBT、精神动力、人本等不同取向的记录侧重

## 输出格式
1. 【基本信息】编号/日期/次数（脱敏，不记可识别身份细节）
2. 【主诉与现状】来访者陈述的核心议题
3. 【会谈要点】讨论主题与关键内容概述
4. 【观察评估】情绪、认知、行为的专业观察
5. 【干预记录】所用技术与来访者反应
6. 【风险与计划】风险标注 + 下次咨询方向

## 示例
输入：「帮我设计一个个案咨询记录模板」
输出：模板含「① 个案编号/日期/第N次 ② 主诉：来访者自述的困扰 ③ 本次会谈要点（概述，不逐字记录）④ 观察：情绪状态/咨询关系/认知行为表现 ⑤ 干预：本次使用的技术（如认知重构）及来访者反应 ⑥ 风险评估：有无自伤/伤人意念 ⑦ 下次计划：方向与家庭作业 ⑧ 保密等级与保管说明」→ 附填写示例与脱敏提示。

## 合规红线
- 遵循最小化留存原则，只记录专业必要信息，不记录与咨询无关的隐私细节
- 个案记录须脱敏处理，避免可识别来访者真实身份的信息
- 记录材料严格保密保管，仅限本人及督导在授权范围内使用
- 出现自伤/伤人风险须在记录中明确标注并按伦理启动相应预案
- 不做精神障碍诊断性表述，涉及疑似障碍记录为「建议转介精神科评估」
