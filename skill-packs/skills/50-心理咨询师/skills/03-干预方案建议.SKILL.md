# 干预方案建议

## 适用场景
咨询师在评估后需要为来访者制定结构化的咨询干预方案，明确咨询目标、采用的技术路线、频次与阶段安排，并与来访者商定。

## 核心能力
- 目标设定：将模糊诉求转化为具体、可评估的咨询目标
- 技术路线：按议题匹配 CBT、认知重构、正念、情绪聚焦等技术
- 阶段规划：初期（建立关系）、中期（干预）、后期（巩固）
- 频次与疗程：建议咨询频率与大致次数区间
- 家庭作业：配合咨询的自助练习
- 效果评估：阶段性复盘与方案调整机制

## 输出格式
1. 【议题概述】核心咨询议题
2. 【咨询目标】具体化、可评估的目标
3. 【技术路线】拟采用的方法与依据
4. 【阶段安排】各阶段任务与频次
5. 【自助作业】咨询间隔的练习建议
6. 【评估调整】效果评估与方案迭代节点

## 示例
输入：「来访者社交焦虑，想制定个咨询方案」
输出：议题「社交场景回避与紧张」→ 目标（降低社交焦虑主观评分、逐步恢复必要社交功能）→ 技术（以 CBT 为主：识别自动化负性思维 + 认知重构 + 渐进式暴露，辅以放松训练）→ 阶段（1-3 次建立关系与心理教育、4-10 次认知与暴露练习、11-12 次巩固预防复发，每周 1 次）→ 作业（焦虑日记、放松练习、分级暴露任务）→ 评估（每 4 次用量表复评，据进展调整）。

## 合规红线
- 干预方案须与来访者知情同意、共同商定，尊重其自主决定
- 明确心理咨询的适用范围，疑似精神障碍须先转介精神科、遵医嘱后再配合咨询
- 发现自伤/伤人风险立即启动危机干预与转介，不得仅按常规方案处理
- 不承诺确定疗效与疗程，目标设定保持弹性
- 方案涉及的技术须在咨询师胜任力范围内，超出能力及时转介或寻求督导
