# 行业基础

## 行业概况
心理健康行业快速发展，持证咨询师人数超百万，社会对心理咨询的需求持续增长。服务形式包括个体咨询、团体咨询与线上咨询，主流咨询流派有认知行为（CBT）、精神分析、人本主义、家庭治疗等。心理咨询有严格的伦理边界，核心原则包括保密、知情同意、价值中立与胜任力。行业强调专业与伦理并重，咨询师须在能力范围内执业，涉及精神障碍诊断与危机情况必须转介，不得越界。

## 常用术语
- 共情：理解并感受来访者情绪体验的能力
- 咨访关系：咨询师与来访者之间建立的专业关系
- 案例督导：由资深咨询师对咨询案例进行指导与监督
- 伦理边界：咨询师与来访者关系中的专业限制（如避免双重关系）
- DSM-5：美国精神障碍诊断与统计手册（诊断权属精神科）
- 转介：将超出咨询范围的来访者转给更合适的专业机构或医师
- 知情同意：咨询前向来访者说明设置、费用、保密原则并取得同意

## 核心工作流程
初次：建立关系 → 收集信息 → 问题评估 → 目标商定 → 签署协议
咨询中：探索问题 → 认知重构 → 行为改变 → 情绪调节 → 资源挖掘
结束：目标达成评估 → 分离处理 → 预防复发 → 回访安排

## 常见问题
Q：心理咨询一次就能好吗？
A：通常 8-20 次为一个相对完整的疗程，短期具体问题可能 3-5 次有改善，效果因人而异，需与来访者商定阶段性目标。

Q：心理咨询能解决什么问题？
A：主要适用于情绪困扰、人际关系、自我成长、压力管理、创伤修复等，但不能替代精神科对精神障碍的诊断与治疗。

Q：心理咨询和精神科有什么区别？
A：心理咨询面向心理困扰，通过谈话技术助人自助；精神科负责精神障碍的医学诊断与药物治疗。疑似精神障碍须转介精神科评估。

Q：来访者透露想自伤怎么办？
A：属于保密例外，须立即进行风险评估、启动危机干预、联系相关支持系统并按伦理转介，生命安全高于保密原则。
