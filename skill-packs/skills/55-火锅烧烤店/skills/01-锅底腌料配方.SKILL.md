# 锅底腌料配方

## 适用场景
研发招牌锅底（牛油/清汤/番茄/菌汤）与烧烤腌料；把师傅手感配方固化成标准；调整辣度咸度适配当地口味；核算配方成本；设计自助蘸料台的配比。

## 核心能力
- 给出锅底/腌料的原料配比与炒制/腌制工艺参数（时间、火候、温度）
- 按锅量/批次换算配方，保证多店一致
- 核算单锅/单份腌料成本，优化高价值香料使用
- 设计辣度、咸度梯度与地域口味调整方案
- 输出蘸料台基础酱料配比与推荐搭配
- 标注关键风味控制点与保存条件

## 输出格式
1. 【配方名称】用途 / 适配食材
2. 【原料配比表】各料克重与比例
3. 【制作工艺】炒制/腌制步骤与参数
4. 【批次换算】
5. 【成本核算】单锅/单份成本
6. 【风味控制点与保存】
7. 【蘸料搭配建议】

## 示例
输入：「帮我做个川味牛油火锅锅底配方，标准化到每锅」
输出：牛油红汤锅底 → 配比（牛油 1500g、豆瓣 300g、干辣椒 200g、花椒 80g、姜蒜、香料包、高汤）→ 工艺（炼油 → 下豆瓣姜蒜炒香 → 下辣椒花椒香料炒 20 分钟控温 120℃ 防糊 → 兑高汤）→ 每锅换算（一份锅底料 400g + 高汤 1500ml）→ 成本（单锅料约 18 元）→ 控制点（油温不超 130℃ 防香料发苦）→ 蘸料搭配（香油蒜泥解辣）。

## 注意事项
- 香料、添加剂按食品安全标准使用，禁止超范围超量添加
- 锅底老油须合规处理，严禁回收复用废弃油脂（地沟油）
- 原料须索证索票，辣椒、香料等把控品质与农残
- 配方固化前多批次试做验证，门店须按许可范围经营
