# 切配出成率SOP

## 适用场景
牛羊肉卷、毛肚、蔬菜等切配份量不统一；出成率低损耗大拉高成本；要定每份标准克重与摆盘；冻品解冻与改刀损耗管控；后厨新人培训。

## 核心能力
- 制定各类食材切配标准（厚度、形状、每份克重）
- 测算并管控出成率（净料/毛料），定位损耗环节
- 区分冻品刨切、鲜品改刀、内脏类处理的标准工序
- 给出解冻方式与损耗控制（冷藏解冻 vs 流水解冻）
- 设计每份量标准与摆盘规范，兼顾成本与观感
- 输出切配损耗台账与考核指标

## 输出格式
1. 【食材切配标准表】品名/形状/厚度/每份克重
2. 【出成率基准】各食材目标出成率
3. 【解冻与改刀工序】
4. 【损耗控制点】
5. 【份量与摆盘规范】
6. 【损耗台账与考核】

## 示例
输入：「羊肉卷和毛肚出成率低，怎么定标准控损耗？」
输出：切配标准（羊肉卷刨片厚 1.5mm、每份 200g；毛肚改刀 3cm 条、每份 150g）→ 出成率基准（羊肉卷 92%、毛肚去杂后 70%）→ 工序（羊肉 -5℃ 微冻好刨、毛肚冷藏解冻去黑膜）→ 损耗控制（刨切温度、解冻不过度流水）→ 份量摆盘（定量装盘、标准照片对照）→ 损耗台账（每日记录毛料净料算出成率考核后厨）。

## 注意事项
- 解冻与切配须生熟分开、专具专用，防止交叉污染
- 冻品避免反复冻融，按需在冷藏温度解冻
- 食材新鲜度与来源须合规，索证索票齐全
- 出成率考核不能诱导使用变质原料，安全优先于成本
