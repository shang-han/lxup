# 行业基础

## 行业概况
火锅与烧烤是中餐中标准化程度最高、社交属性最强的聚餐型业态，客单价高、毛利可观，但重资产（排烟通风、消防）、重运营（人力、食材损耗）。行业季节性强（火锅冬旺、烧烤夏旺）。核心竞争力：锅底/腌料口味壁垒 + 食材供应链与出成率控制 + 聚餐场景营销 + 安全合规。头部靠供应链与连锁，单店靠特色口味与本地客情。

## 专业技能体系
### 硬技能
- 锅底/腌料配方研发
- 食材切配标准化（出成率控制）
- 排烟通风系统设计
- 蘸料台自助模式运营
- 淡旺季人力调配

### 软技能
- 聚餐场景营销
- 安全应急处理（烫伤/火灾）

## 常用术语
- 出成率：净料重量/毛料重量，切配损耗控制核心指标
- 翻台率：每张桌日均接待批次，聚餐业态产能指标
- 桌均/客单价：每桌/每人平均消费，靠套餐与酒水拉升
- 锅底：火锅汤底，分牛油红汤、清汤、番茄、菌汤等
- 老油：反复熬制的火锅油，废弃油脂严禁回收复用
- 自助蘸料台：顾客自调蘸料的模式，含酱料配比与成本管控
- 排烟通风：烧烤火锅的除烟换气系统，关系合规与体验
- 人工成本占比：人工费用/营业额，一般控 18-22%

## 核心工作流程
每日：原料验收切配 → 锅底/腌料预制 → 备餐 → 营业（午市/晚市/夜宵）→ 收档清洁与安全检查
每周：食材盘点与采购 → 出成率与损耗分析 → 排烟/消防/燃气检查 → 员工培训
每月：成本与毛利核算 → 营销活动复盘 → 淡旺季人力调整 → 证照台账检查

## 常见问题
Q：食材切配损耗大怎么控？
A：定每份标准克重与切配规格，测算各食材目标出成率，规范解冻与改刀工序，建损耗台账考核后厨，安全前提下抠损耗。

Q：锅底口味怎么稳定又有特色？
A：把招牌锅底配方量化标准化到每锅，严控香料炒制油温与时间，核算成本优化高价香料，形成别家复制不了的口味壁垒。

Q：淡旺季客流差很多，人力怎么配？
A：核心岗全职保底、高峰用小时工，旺季增兼职、淡季调休培训，人工成本控在营业额 18-22%，按分时段客流排班。

Q：烫伤火灾风险高怎么防？
A：制定应急预案（烫伤冲脱泡盖送、油锅起火断氧禁用水、燃气泄漏关阀开窗），酒精炉熄灭后再添液，定期消防演练与排烟管道清洗。
