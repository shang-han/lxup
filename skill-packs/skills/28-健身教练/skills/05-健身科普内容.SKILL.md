# 健身科普内容

## 适用场景
教练做个人 IP 建设与获客：抖音/小红书/公众号发布健身知识、训练误区辟谣、热点话题解读（明星健身法、网红训练法）、会员群每周科普。把专业知识变成通俗、有传播力的内容。

## 核心能力
- 按平台生成不同风格：小红书干货清单体、抖音口播脚本、公众号深度长文
- 辟谣模板：误区 → 真相 → 科学依据 → 正确做法
- 选题方向库：训练类（局部减脂？深蹲伤膝？练后酸痛=有效？）、饮食类（碳水发胖？蛋白粉激素？晚上不能吃？）、常识类（出汗=减脂？女生举铁变金刚芭比？）
- 标题与钩子设计：悬念式、反差式、痛点直击式开头
- 专业概念生活化类比（如「热量赤字像账户透支」）
- 互动与转化设计：评论区话题、引导体验课/关注

## 输出格式
1. 【选题定位】目标受众、发布平台、内容目标
2. 【标题/钩子】3 个备选
3. 【正文】结构化科普内容
4. 【依据来源】科学共识/文献/教材出处
5. 【互动设计】评论引导、关注转化钩子
6. 【敏感自查】夸大宣传与虚假承诺检查

## 示例
输入：「写一条小红书辟谣『出汗多=减脂效果好』」
输出：标题备选（「暴汗服真的能减肥？你减的是水不是肉」「练完大汗淋漓=瘦了？真相扎心」）→ 正文：观点（出汗是体温调节，流失的是水分，喝水就回来）→ 依据（脂肪分解产物是二氧化碳和水，主要经呼吸排出）→ 正确做法（看热量缺口与长期坚持，而非出汗量）→ 判断方法：以周为单位看围度与训练表现变化 → 互动：「你交过哪些健身智商税？评论区聊聊」。

## 注意事项
- 内容须符合运动科学共识，不传播「局部减脂」「快速瘦身」等伪科学
- 不承诺「X 天瘦 X 斤」「包效果」等具体效果
- 规避广告法绝对化用语，未经授权不得转载他人对比照片
- 涉及医疗话题（康复、疾病与运动）提示咨询医生，不越界诊断
- 保持专业边界，不贬损任何体型与训练方式
