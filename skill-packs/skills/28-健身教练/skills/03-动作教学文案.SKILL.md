# 动作教学文案

## 适用场景
教练产出教学内容：社媒动作教学帖、课内动作讲义、短视频动作分解脚本、场馆训练墙贴。把一个动作拆解成初学者能听懂、能照做的教学要点。

## 核心能力
- 动作流程分步拆解（起始姿势→动作过程→顶点控制），逐步说明要领
- 常见错误清单与纠正方法（如深蹲膝内扣、硬拉弓背、卧推耸肩）
- 发力感提示语：「想象用肘部向后拉」「脚掌拧入地面」「臀部夹紧推起」
- 呼吸模式与安全要点（核心收紧、脊柱中立、保护站位）
- 短视频脚本结构：钩子开头（痛点/误区）→ 动作分解 → 错误示范 → 总结互动
- 进阶/退阶变式推荐（新手简化版、高阶进阶版）

## 输出格式
1. 【动作概况】动作名称、目标肌群、难度、所需器械
2. 【动作分解】分步骤动作要领
3. 【发力要领】2-3 条最关键的发力/发力感提示
4. 【常见错误】错误表现 + 纠正方法
5. 【安全要点】呼吸、保护、不适信号
6. 【变式建议】退阶/进阶选择

## 示例
输入：「写一条抖音杠铃深蹲教学脚本」
输出：钩子：「深蹲总是先动膝盖？难怪腿粗臀不翘」→ 动作分解：站位（杠铃置于斜方肌、肩胛收紧）→ 站距（与肩同宽脚尖微外八）→ 下蹲（先屈髋后屈膝、膝随脚尖方向打开、胸口挺起）→ 起身（臀部发力蹬地）→ 发力提示：「坐向身后的椅子」「膝盖向外拧」→ 常见错误：膝内扣/弓背/脚跟离地及纠正 → 呼吸：下蹲吸气憋气撑住、起身呼气 → 变式：新手高脚杯深蹲、进阶暂停深蹲。

## 注意事项
- 教学须基于主流训练科学共识，不传播「局部减脂」「肌肉塑形改变形状」等伪科学
- 高风险动作（大重量硬拉、颈后推举等）强调须有现场指导
- 文案无法替代现场保护，提醒观众量力而行、循序渐进
- 错误示范内容须明确标注「错误动作」避免模仿
- 涉及伤病康复类内容提示咨询医生或物理治疗师
