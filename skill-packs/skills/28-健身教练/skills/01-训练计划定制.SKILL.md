# 训练计划定制

## 适用场景
私教为会员制作训练方案：新手入门计划、增肌分化训练、减脂训练组合、居家无器械训练、备赛期安排。输入会员信息（目标、训练经验、可用器械、每周可训天数）即可生成可直接执行的周计划。

## 核心能力
- 按目标与水平选择训练框架：新手全身 3 练/周、初中级分化 4-5 练/周、减脂力量+有氧组合
- 逐日安排具体动作、组数/次数/组间休息，并给出负荷选择标准（XX RM 或 RPE 自感强度）
- 落实渐进超负荷：设定每周进阶规则（如达到次数上限后加 2.5kg）
- 适配训练条件：健身房全器械版与居家哑铃/自重版双方案
- 包含热身与放松安排、易伤部位预防提示
- 特殊情况调整：久坐人群肩胛稳定、膝关节不适的动作替换、女性臀腿侧重

## 输出格式
1. 【会员概况】目标、水平、训练条件、每周安排
2. 【周框架】每日训练主题与整体逻辑
3. 【每日明细】逐动作列出组次、休息、负荷标准
4. 【进阶策略】每周加负荷的方式与幅度
5. 【热身与放松】训练前后安排
6. 【注意事项】常见动作错误与防伤提示

## 示例
输入：「男，新手，目标增肌，每周能去健身房 3 天，没有训练基础」
输出：全身训练 3 练/周（如周一/三/五）→ 每日以复合动作为主（深蹲/卧推/划船/推举/硬拉变式）共 6-8 个动作，3 组 × 8-12 次 → 负荷标准：选能完成且最后 2 次吃力的重量 → 进阶规则：每周尝试加 1 次，达到 12 次后加 2.5kg → 热身 5 分钟动态拉伸 + 正式组前空杆递增组，训练后静态拉伸 → 提示：新手先学动作模式再追重量，空杆掌握要领后再上片。

## 注意事项
- 制定计划前询问伤病史、慢性病与心血管风险，高血压、心脏病人群提示先获医生许可
- 新手计划以动作学习为主，不盲目上大重量
- 疼痛动作必须替换而非硬练，急性疼痛立即停训就医
- 计划为通用参考，须结合会员现场表现指导调整
- 不承诺「X 天练出 X 效果」，合理设定会员预期
