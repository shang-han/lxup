# 行业基础

## 行业概况
健身行业私教渗透率持续提升，一线城市私教课单价 300-800 元/节。行业模式从「卖卡驱动」转向「服务留存驱动」，续课率是教练收入与场馆经营的核心。核心能力：专业训练技术 + 沟通销售能力 + 客户维护意识。新兴方向包括线上私教、小团课、垂直细分（产后修复、青少年体适能、银发抗阻训练）。行业竞争激烈，持续学习认证与口碑积累是长期发展的关键。

## 专业技能体系

### 硬技能
- 运动解剖学基础与体态评估
- 训练方案设计（增肌/减脂/康复/产后）
- 瑜伽体式编排与正位纠正
- 营养学基础与饮食建议
- 急救处置（CPR/AED）

### 软技能
- 学员激励与进度追踪
- 个人IP打造（短视频/社群）

## 常用术语
- RM：最大重复次数，如 10RM 指尽全力最多做 10 次的重量，用于确定训练负荷
- 组数/次数：每个动作单组重复次数与总组数，训练计划的基本单位
- 分化训练：按身体部位（胸/背/腿/肩/臂）分天训练的安排方式
- HIIT：高强度间歇训练，短时高效燃脂，适合时间紧张人群
- 超量恢复：训练后身体机能恢复到高于训练前水平的现象，是训练效果产生的原理
- 渐进超负荷：逐步增加训练负荷（重量/次数/组数），肌肉持续增长的核心原则
- 体脂率：脂肪重量占体重的比例，衡量身体成分的关键指标

## 核心工作流程
新客阶段：体测评估（体成分、动作筛查）→ 需求沟通 → 目标设定 → 课程规划与销售
训练阶段：动作教学 → 训练指导与负荷调整 → 保护辅助 → 训练记录与跟踪
维护阶段：饮食建议 → 日常训练监督 → 效果反馈与复盘 → 续课与转介绍推进

## 常见问题
Q：怎么快速增肌？
A：三大要素——渐进超负荷训练（每周力量训练 3-5 次）+ 充足蛋白质（1.6-2.2g/kg 体重）+ 充足睡眠（7-8 小时）。肌肉生长需要时间，合理预期为每月 0.5-1kg 瘦体重。

Q：练了一段时间没效果怎么办？
A：从三个维度排查：训练（强度够不够、有没有渐进超负荷）、饮食（蛋白质和总热量吃够没有）、休息（睡眠与恢复）。多数问题出在吃不够或睡不够。

Q：会员续课怎么谈？
A：先让会员看到效果（定期体测对比、训练记录进阶），用数据和阶段成果沟通价值，不做高压推销；在目标达成、课时将尽等自然节点带出下一阶段规划。
