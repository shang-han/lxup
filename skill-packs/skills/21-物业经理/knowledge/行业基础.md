# 行业基础

## 行业概况
物业管理行业规模超万亿，覆盖住宅、商业、写字楼等业态。物业费是主要收入来源，增值服务（家政、团购、广告位、停车管理）是增长点。核心指标：收缴率、业主满意度、投诉率、报修完成率。行业从「看门护院」向「社区生活服务运营商」转型，业主体验与口碑是续签合同、拓展项目的关键。政策层面受《物业管理条例》、业主委员会自治机制约束。

## 常用术语
- 物业费收缴率：实收物业费 / 应收物业费的比例，是项目盈亏的生命线
- 业主满意度：满意度调查综合得分，直接影响合同续签
- 投诉率：投诉户数 / 总户数，反映服务质量
- 报修完成率：按时限完成维修工单的比例
- 公共收益：电梯广告、停车费、场地租赁等公共区域经营收入，按规定应公示
- 业委会：业主委员会，代表全体业主行使自治管理权
- 专项维修资金：用于共用部位、共用设施设备大修更新的资金
- 四保一服：保安、保洁、保绿、保修 + 客服服务，物业基础服务模块

## 核心工作流程
每日：巡楼检查 → 报修处理 → 投诉处理 → 保洁监督 → 安保交接
每周：安保巡逻 → 设备维护（电梯/水泵/消防）→ 绿化养护 → 员工例会
每月：物业费催缴 → 业主沟通回访 → 公共收益公示 → 工作报告
每年：年度预算 → 满意度调查 → 维修计划 → 业委会述职

## 常见问题
Q：业主不交物业费怎么办？
A：先解决业主合理诉求，再上门沟通说明费用用途，持续拖欠可发催缴函，最后通过调解或法律途径（诉讼/支付令），避免断水断电等违规手段。

Q：邻里纠纷（漏水/噪音）怎么处理？
A：分别沟通了解双方诉求 → 现场勘查取证 → 居中调解 → 出具书面处理方案，涉及结构或产权争议的引导走专业鉴定或司法途径。

Q：公共收益怎么管理？
A：单独建账、定期（通常每季度）向全体业主公示收支明细，按规定比例补充专项维修资金或经业委会决议分配。

Q：业主群里出现负面舆情怎么办？
A：第一时间正面回应、不删帖不对抗，线下快速处理并反馈结果，用解决效率化解情绪，必要时邀请业委会共同见证。
