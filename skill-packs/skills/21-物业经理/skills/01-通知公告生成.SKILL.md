# 通知公告生成

## 适用场景
物业客服、管家需要向业主发布停水停电、电梯检修、外墙清洗、消防演练、缴费提醒、节日祝福、装修管理等各类通知时，快速生成措辞得体、要素完整、可直接张贴或发群的规范文本。

## 核心能力
- 识别通知类型（停水停电/维修施工/缴费/安全/节日/管理规约），匹配对应语气与结构
- 自动补全六要素：事由、时间、影响范围、应对措施、联系方式、落款
- 分渠道输出：单元门张贴版（正式）、业主群版（简洁+emoji分段）、短信版（精简）
- 停水停电类自动加入「提前储水/充电、关好电器阀门」等安全提示
- 缴费类通知兼顾催缴力度与情感温度，避免生硬对立
- 标题醒目、落款含物业服务中心名称与日期，符合张贴规范

## 输出格式
1. 【标题】醒目事由标题
2. 【正文】事由 → 时间 → 影响范围 → 应对提示
3. 【联系方式】物业电话/管家微信
4. 【落款】服务中心名称 + 发布日期
5. 【多渠道版本】张贴版 / 业主群版 / 短信版（按需）

## 示例
输入：「明天上午9点到下午3点3号楼停水检修，帮我发个通知」
输出：标题《关于3号楼临时停水的通知》→ 事由（供水管道检修）→ 时间（X月X日 09:00-15:00）→ 影响范围（3号楼全体住户）→ 提示（提前储水、关闭水龙头阀门、恢复后短时水质浑浊请放流）→ 联系电话 → 落款XX物业服务中心 + 日期 → 另附业主群精简版。

## 注意事项
- 时间务必精确到起止时刻，范围写清楼栋/单元，避免「近期」「部分区域」等模糊表述
- 涉及费用、罚则的内容须有合同或管理规约依据，不得随意承诺赔偿
- 安全类提示（用电、用气、高空坠物）不可省略
- 语气尊重业主，多用「敬请谅解」「感谢配合」，少用命令口吻
