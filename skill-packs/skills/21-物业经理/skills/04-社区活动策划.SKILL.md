# 社区活动策划

## 适用场景
物业、社区运营在传统节日、邻里季、便民服务日、业主答谢等节点策划社区活动，目标是增进邻里关系、提升业主满意度与物业费收缴意愿、塑造物业服务口碑。

## 核心能力
- 活动类型库：节日联欢（春节/中秋/重阳）、邻里百家宴、便民服务日（磨刀/理发/义诊）、亲子跳蚤市场、社区运动会、消防/反诈宣传、绿化认养
- 完整方案输出：目标 / 主题 / 时间地点 / 流程 / 物料 / 预算 / 人员分工 / 风险预案
- 低预算高参与的玩法设计，撬动业主志愿者与共建商家资源
- 配套宣传：业主群预告文案、海报文案、活动后回顾稿
- 效果评估指标：参与户数、满意度、互动量、收缴带动
- 针对老人/儿童/年轻家庭不同人群的适配设计

## 输出格式
1. 【活动概览】主题、目标、时间、地点、面向人群
2. 【活动流程】分时段执行脚本
3. 【物料与预算】清单 + 费用估算
4. 【人员分工】岗位与职责
5. 【宣传文案】预告版 + 回顾版
6. 【风险预案】天气/安全/秩序应急
7. 【评估指标】活动后统计项

## 示例
输入：「中秋节想在小区搞个邻里活动，预算不多」
输出：主题「月圆中秋·情暖邻里」→ 流程（18:30 签到领月饼 → 19:00 猜灯谜/做花灯 → 19:30 百家宴分享 → 20:00 赏月合影抽奖）→ 物料预算（月饼、灯谜条、灯笼DIY材料、奖品约 2000 元内）→ 分工（主持/签到/秩序/摄影）→ 业主群预告文案 → 风险预案（雨天移至架空层、用电安全、老人儿童看护）→ 评估指标（参与户数、满意度问卷）。

## 注意事项
- 预算透明，优先争取商家共建/赞助降低成本
- 涉及用火用电、食品、儿童的环节必须配安全预案与责任人
- 活动不扰民，控制音量与时间，提前发通知
- 留好影像资料与签到记录，作为满意度与宣传素材
- 商业植入适度，避免引起业主反感
