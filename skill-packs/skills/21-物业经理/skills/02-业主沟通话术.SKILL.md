# 业主沟通话术

## 适用场景
物业管家、客服在催缴物业费、处理业主抱怨、上门拜访、调解纠纷、回应质疑等面对面或线上沟通场景中，需要既守住立场又不激化矛盾的措辞与应答脚本。

## 核心能力
- 物业费催缴分层话术：首次提醒 / 二次跟进 / 逾期面谈 / 法务函前置沟通，逐级升级
- 抱怨与质疑的「先共情、再解释、后方案」三段式应答
- 上门拜访破冰话术（自我介绍、来意说明、留联系方式）
- 邻里纠纷、装修扰民等矛盾的居中劝解措辞
- 面对「服务不到位凭什么收费」等典型异议的应答口径
- 群聊公开回应话术：正面、克制、转线下处理

## 输出格式
1. 【场景判断】识别沟通类型与业主情绪
2. 【开场白】破冰与来意
3. 【核心话术】分步骤的完整应答脚本
4. 【异议应对】常见反驳的回应口径
5. 【收尾】达成共识、留痕、后续跟进
6. 【注意事项】该场景的沟通禁忌

## 示例
输入：「业主说电梯老坏、卫生差，不肯交物业费，怎么沟通？」
输出：先共情（「您反映的电梯和卫生问题确实影响体验，我们很重视」）→ 说明已采取的整改（列出近期维保记录、保洁加强安排）→ 区分「服务改进」与「缴费义务」两条线 → 邀请业主现场查看整改、留下管家联系方式 → 给出「问题持续可再反映，费用方面也希望您支持」的收尾 → 禁忌提示：不承诺减免、不与业主争辩、不当众承诺无法兑现的时限。

## 注意事项
- 催缴全程留痕（记录时间、对象、内容），为后续法律程序保留证据
- 不得以停水停电、停梯、限制门禁等方式催缴，属违规
- 不随意承诺费用减免或赔偿，减免须经公司审批
- 情绪激动的业主先安抚、不硬碰，必要时转上级或邀请业委会介入
