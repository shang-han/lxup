# 物业报告生成

## 适用场景
物业项目经理、管家需要向上级公司、业委会或业主大会提交月度/季度/年度工作报告、运营分析、收缴情况汇报、述职报告时，快速生成数据清晰、结构规范、重点突出的正式文本。

## 核心能力
- 报告类型库：月度运营报告、年度工作总结、收缴率专项分析、满意度调查汇报、业委会述职、项目交接报告
- 核心指标呈现：收缴率、报修完成率、投诉率、满意度、公共收益、能耗成本，同比环比分析
- 「成绩—问题—改进」三段式结构，突出数据与整改闭环
- 下阶段工作计划按模块（四保一服）拆解，配目标与责任人
- 语言正式、客观，问题部分不回避，改进措施可量化
- 可生成适合 PPT 汇报的提纲要点版

## 输出格式
1. 【报告标题】周期 + 项目 + 类型
2. 【核心数据概览】关键指标一览
3. 【重点工作回顾】分模块完成情况
4. 【存在问题】客观列出
5. 【改进措施与计划】量化、配责任人与时限
6. 【需协调事项】需上级/业委会支持项
7. 【汇报提纲版】PPT 要点（按需）

## 示例
输入：「帮我写个6月份的物业月报，收缴率92%，处理报修186单，投诉3起都解决了」
输出：标题《XX小区物业服务中心6月运营报告》→ 数据概览（收缴率92%、报修186单完成率、投诉3起100%闭环）→ 重点回顾（四保一服分项）→ 问题（未缴费户分析、夏季用电报修上升）→ 改进（针对性催缴、设备巡检加密）→ 7月计划（防汛、消防演练）→ 需协调事项 → 附 PPT 提纲。

## 注意事项
- 数据务必真实、口径一致，注明统计周期与来源
- 问题与改进一一对应，避免「只报喜不报忧」
- 涉及公共收益、专项维修资金须如实披露、合规表述
- 计划可量化、可考核，避免空话套话
- 对外（业委会/业主）版本注意措辞分寸，敏感数据按规定披露
