# 行业基础

## 行业概况
连锁加盟是餐饮、零售行业主要的规模化扩张模式，涵盖加盟商招募、门店标准化管理、供应链统一配送、品牌营销赋能等环节。品牌方通过输出品牌、产品、系统和运营标准快速扩张，加盟商借助成熟模式降低创业风险。成功关键在于「标准化 SOP + 强供应链 + 持续品牌赋能」三者协同。行业受《商业特许经营管理条例》规范，品牌方须具备成熟经营模式并履行备案与信息披露义务。连锁化率持续提升，但加盟商存活率分化明显，总部运营赋能能力决定体系健康度。

## 常用术语
- 加盟费：加盟商获得品牌授权经营资格支付的一次性费用
- 保证金：履约保障押金，合同期满无违约可退还
- 抽成/管理费：按营业额一定比例向总部缴纳的持续性费用
- 督导：总部派驻负责门店巡检与经营指导的人员
- 区域保护：限定加盟商在一定范围内独家经营的条款
- 单店模型：标准化、可复制的单店盈利与运营模板
- 供应链配送：总部统一采购、中央厨房/仓储配送到店的体系

## 核心工作流程
加盟前：品牌评估 → 市场调研 → 选址评估 → 资质审核
加盟中：签约缴费 → 系统培训 → 装修施工 → 设备进场 → 开业扶持
加盟后：巡店督导 → 供应链配送 → 营销支持 → 数据复盘 → 续约管理

## 常见问题
Q：加盟商不配合总部方案怎么办？
A：强化合同约束条款，加大督导频次，建立积分奖惩与分级管理机制，必要时启动约谈与整改流程。

Q：区域保护距离设多少合适？
A：视品类和商圈密度而定，一般 1-3 公里，核心商圈可缩至 500 米，需在合同中明确测算口径。

Q：如何保证各门店标准统一？
A：建立可量化、可检查的 SOP 与巡店评分体系，配套培训和督导闭环，把标准执行情况纳入加盟商考核。

Q：新店爬坡期总部怎么扶持？
A：提供选址评估、开业爆破方案、驻店带教、流量投放支持和数据诊断，帮助门店尽快度过爬坡期。
