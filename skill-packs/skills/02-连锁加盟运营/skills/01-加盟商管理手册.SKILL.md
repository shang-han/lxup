# 加盟商管理手册

## 适用场景
连锁品牌总部运营人员需要一份系统的加盟商管理手册，用于规范招商筛选、合同签订、开业扶持、日常督导、续约与退出全流程，降低加盟纠纷、保证品牌一致性。

## 核心能力
- 加盟商招募标准：资金门槛、经营理念、选址能力、从业经验的筛选维度
- 加盟流程设计：咨询 → 考察 → 评估 → 签约 → 培训 → 开业的标准节点
- 合同关键条款梳理：加盟费、保证金、抽成、区域保护、违约责任
- 总部扶持清单：选址、装修、培训、供应链、营销的支持内容与边界
- 日常管理与督导机制：巡店频次、数据上报、奖惩积分
- 续约与退出机制：续约条件、转让规则、解约清算

## 输出格式
1. 【手册目录】分章节的管理框架
2. 【加盟流程】各节点的标准动作与责任人
3. 【权责划分】总部与加盟商的权利义务对照
4. 【管理制度】督导、上报、奖惩、保密等制度
5. 【续约与退出】续约条件、转让与解约规则
6. 【附件清单】合同模板、评估表、验收表等

## 示例
输入：「帮我搭一个加盟商管理手册的框架，重点是区域保护和督导」
输出：目录（总则/招募/合同/扶持/督导/续约/退出）→ 区域保护条款（同城直线距离不低于 1.5 公里、核心商圈 500 米，违约处理）→ 督导机制（每月至少 1 次现场巡店 + 周报数据上报 + 积分奖惩）→ 附加盟商评估表与巡店检查表模板。

## 注意事项
- 加盟合同须符合《商业特许经营管理条例》，品牌方需具备「两店一年」及备案资质
- 区域保护、抽成、退出等关键条款表述清晰，避免模糊引发纠纷
- 手册作为管理依据应版本化管理，修订及时同步全体加盟商
- 涉及保证金、违约金的条款建议经法务审核
