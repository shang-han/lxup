# 培训材料制作

## 适用场景
连锁总部需要为新加盟商、新店长、一线员工制作标准化培训材料，涵盖品牌文化、产品操作、服务规范、食品安全、系统使用等内容，支持集中授课和门店自学。

## 核心能力
- 课程大纲设计：按岗位（店长/店员）和模块分层
- 课件结构化：学习目标 → 知识点 → 案例 → 实操 → 考核
- 讲师手册：逐页讲解要点、时间分配、互动设计
- 实操带教清单：岗位技能的步骤化训练表
- 考核题库：选择/判断/情景题，检验掌握程度
- 新人上岗路线图：从入职到独立上岗的分阶段计划

## 输出格式
1. 【课程信息】主题、对象、时长、学习目标
2. 【内容大纲】分章节的知识框架
3. 【课件要点】每页/每节的核心内容
4. 【实操清单】岗位技能训练步骤
5. 【考核题目】配套测验题与参考答案
6. 【上岗计划】分阶段的学习与带教安排

## 示例
输入：「做一份新员工入职培训大纲，三天能上岗」
输出：三天课程表 → Day1 品牌文化与门店制度、Day2 产品制作实操（配步骤带教表）、Day3 服务流程与收银 → 每模块含学习目标 + 知识点 + 案例 + 实操 → 附 20 道考核题（含食品安全判断题）→ 上岗标准：实操达标 + 考核 80 分以上由师傅签字。

## 注意事项
- 食品安全、消防、合规内容列为必修并单独考核
- 内容贴合门店实际场景，多用案例和实操少用纯理论
- 培训材料版本化管理，随产品/制度更新同步修订
- 保留培训与考核记录，作为上岗与检查依据
