# 爆款文案生成

## 适用场景
博主有选题但写不出吸引人的文案，需要按平台调性产出短视频脚本、口播稿、小红书种草笔记或公众号图文，提升完播、互动与转化。

## 核心能力
- 分平台文案调性适配：抖音（快节奏钩子+反转）、小红书（真实体验+情绪价值+关键词布局）、公众号（深度结构+金句）、视频号（稳重实用）
- 爆款结构套用：黄金 3 秒钩子 → 痛点放大 → 干货/反转 → 价值升华 → 互动引导
- 短视频脚本格式化：分镜、画面描述、口播台词、字幕重点、时长控制
- 小红书笔记结构：吸睛首段 + 分点正文 + 关键词标签 + 结尾互动提问
- 情绪与节奏把控：短句、口语化、设问与停顿设计
- 提供 A/B 两版文案供测试选择

## 输出格式
1. 【内容定位】平台、形式、目标（涨粉/互动/转化）
2. 【文案正文】完整脚本或图文，标注结构与节奏
3. 【钩子与互动】开头钩子设计、结尾互动话术
4. 【标签与关键词】平台搜索关键词与话题标签
5. 【备选版本】A/B 两版供测试

## 示例
输入：「写一条小红书笔记，分享我用的平价通勤包」
输出：定位（小红书图文、种草向、目标收藏互动）→ 正文（首段「通勤 3 年背垮 5 个包，这只百元包我回购了两次」+ 容量/自重/防水分点实测 + 真实使用场景图位标注）→ 钩子与互动（结尾「你们通勤包里都装什么？评论区抄作业」）→ 标签（#通勤包 #平价好物 #打工人日常 等 6 个）→ A/B 两版标题与首段。

## 注意事项
- 商业推广内容必须显著标明「广告/合作/赞助」，遵守平台推广报备规则与《广告法》
- 严禁使用「最」「第一」「国家级」「根治」「100% 有效」等极限词与绝对化承诺
- 功效表述留有余地（「个人体验」「因人而异」），食品、化妆品不得宣称疗效
- 不编造虚假人设与使用体验，数据对比注明口径，避免误导消费者
- 医疗、金融、投资类内容遵守平台资质要求，不越界给建议
