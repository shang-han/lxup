# 行业基础

## 行业概况
自媒体创作者数量超千万，头部创作者年收入超千万，但绝大多数处于数千到数万元的腰部区间，是典型的「二八分化」行业。主要平台包括抖音、小红书、B 站、视频号、公众号、快手，内容形式涵盖短视频、图文笔记、直播。变现方式主要有广告接单、电商带货、知识付费、平台激励与打赏。核心竞争力是持续输出能力、内容差异化与粉丝信任，算法推荐机制下「完播率+互动率」决定流量分发。

## 常用术语
- 粉丝画像：粉丝的年龄、性别、地域、兴趣分布，决定选题与商业化方向
- 完播率：完整观看视频的用户比例，短视频推荐机制的核心指标
- 互动率：点赞、评论、收藏、转发占播放量的比例
- 转化率：内容引导到关注、加购、成交的比例
- 垂直领域：账号专注的细分内容方向，越垂直标签越精准、商业价值越高
- 人设：创作者在内容中呈现的形象与性格定位
- 黄金 3 秒：视频开头 3 秒决定用户是否继续观看
- 私域：沉淀在微信等自有渠道、可反复触达的粉丝池

## 核心工作流程
定位期：选择垂直领域 → 确定人设 → 拆解竞品 → 找到差异化切入点
创作期：选题策划 → 素材收集 → 脚本撰写 → 拍摄剪辑 → 标题封面优化 → 发布
运营期：评论互动维护 → 数据复盘 → 选题迭代 → 变现尝试 → 账号定期复盘

## 常见问题
Q：新手应该选什么领域？
A：选「自己擅长 + 有市场需求 + 能可持续输出」三者的交集，先垂直做深再考虑扩展，避免大而全。

Q：做了 3 个月不涨粉怎么办？
A：系统复盘：定位是否垂直、开头钩子是否留人、选题是否有需求、标题封面是否合格，逐项优化而不是盲目加更。

Q：怎么接到第一个广告？
A：1 万粉以上可入驻平台官方接单渠道（如星图、蒲公英），同时用优质数据案例主动联系契合品牌，报价从低起步积累合作案例。

Q：接广告需要注意什么？
A：必须走平台报备并显著标注「广告/合作」，文案遵守广告法（禁用极限词、不夸大功效），只推荐自己认可且资质齐全的产品。
