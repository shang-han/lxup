# 行业基础

## 行业概况
家装水电维修与改造是典型的本地生活服务，需求来自存量房维修、二手房翻新和局部改造，订单碎片化、即时性强。个人师傅多靠社区口碑、物业合作与线上平台（58、啄木鸟、美团）获客，收入按单结算，淡旺季明显。核心竞争力是手艺规范、响应快、报价透明，回头客和转介绍是主要订单来源。电工属特种作业，须持电工证上岗。

## 专业技能体系
### 硬技能
- 电路故障排查与布线规范（短路/漏电/跳闸判断）
- 水管安装与维修（PPR焊接/排水/防水）
- 开关/插座/灯具安装标准
- 电表/配电箱配置
- 安全操作规范（断电作业/绝缘工具）
### 软技能
- 客户报价与时间预约
- 服务口碑积累

## 常用术语
- 回路：配电箱中由一个空开/漏保控制的一路线路
- 漏保（漏电保护器）：检测到漏电即跳闸，插座回路必配
- 线径：导线截面积（mm²），决定载流量，如 BV 2.5/4/6
- 强电/弱电：220V 电力线路与网络/电视/电话等信号线路
- PPR 管：家装主流给水管，热熔焊接连接
- 打压测试：水路完工后加压（通常 0.8MPa）保压检漏
- 活线工艺：线管内电线可抽动更换，弯头不超过 2 个
- 等电位：卫生间局部等电位联结，防触电的强制规范

## 核心工作流程
接单：客户咨询 → 问清故障/需求 → 远程初判 → 约定上门时间
上门：现场勘查 → 定位故障 → 报价确认 → 断电验电 → 施工
收尾：通电/打压测试 → 客户验收 → 收款 → 质保告知 → 好评引导
经营：客户记录 → 定期回访 → 朋友圈案例 → 物业/装修公司渠道维护

## 常见问题
Q：客户嫌报价高怎么办？
A：拆开材料品牌和工艺讲（阻燃管、足方线、活线工艺的价值），给「基础版/标准版」两档方案让客户选，不裸降价。

Q：上门后发现是另一个师傅能修的活/判断错了怎么办？
A：如实告知，只收少量上门检测费甚至免收，留个好印象比这一单更重要，客户下次还会找你。

Q：老房没有地线怎么办？
A：明确告知无地线的安全风险，建议增设接地或优先更换带漏保的回路，不得默认将就施工。

Q：怎么积累稳定客源？
A：深耕 2-3 个小区做口碑，和物业、五金店、装修公司建立转单合作，出租房房东是高频复购客户要重点维护。
