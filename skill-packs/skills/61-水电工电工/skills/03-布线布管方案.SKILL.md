# 布线布管方案

## 适用场景
老房电路改造、新房水电定位、增加回路/插座/水路点位时，需要一份回路划分、线径选择、走线方式与材料清单方案，供师傅施工和向客户交底。

## 核心能力
- 回路划分建议：照明、普通插座、厨房、卫生间、空调各独立回路，漏保/空开配置
- 线径与空开匹配（照明 2.5mm²、普通插座 2.5/4mm²、厨卫及柜机 4mm²、中央空调 6mm² 等）
- 走线方式建议（横平竖直、强弱电间距≥30cm、厨卫走顶便于检修）
- 水路方案：PPR 冷热水管区分、走顶/走地利弊、打压验收标准
- 按方案生成材料清单（管材、线材、底盒、空开、漏保数量估算）
- 标注施工规范要点（活线工艺、线管弯头数、接头处理）

## 输出格式
1. 【需求确认】户型/改造范围/大功率电器清单
2. 【回路设计方案】各回路用途、线径、空开规格
3. 【走线方式】强电/弱电/水路走向与规范要求
4. 【材料清单】名称 / 规格 / 估算数量
5. 【施工规范要点】关键工艺与禁止事项
6. 【验收标准】绝缘电阻、打压测试等验收项

## 示例
输入：「90 平两居老房全屋电路改造，有 3 台空调、厨房烤箱和即热热水器」
输出：回路方案（照明 1 路、普通插座 2 路、厨房 1 路 4mm²、卫生间 1 路 4mm²、3 台空调各 1 路）→ 总开 63A+ 漏保配置 → 强弱电分管分距 → 材料清单（BV 线各规格米数、PVC 线管、86 暗盒约 40 个、空开数量）→ 规范：单管穿线不超管径 40%、厨卫插座带防溅盒 → 验收：绝缘电阻≥0.5MΩ。

## 合规红线
- 方案必须符合国家住宅电气规范：插座回路配漏电保护、卫生间局部等电位、空调及大功率电器独立回路
- 严禁铝线铜线直接对接、严禁无地线方案、严禁电线直埋无管敷设
- 涉及电表容量不足、进户线改造，提示客户向供电部门申请，不得私改电表与表前线
- 即热式热水器等大功率设备须核算入户容量与线径，容量不足时明确告知不可强装
