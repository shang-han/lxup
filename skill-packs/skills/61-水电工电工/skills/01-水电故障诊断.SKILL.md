# 水电故障诊断

## 适用场景
客户来电或微信描述家中电路、水路故障（频繁跳闸、插座没电、墙角渗水、水龙头不出水等），师傅远程初判故障类型、所需工具材料，决定是否上门及大致工作量。

## 核心能力
- 将客户口语描述翻译为故障类型：短路 / 漏电 / 过载 / 空开损坏 / 线路老化 / PPR 接头渗漏 / 角阀堵塞等
- 按概率排序给出 2-4 个可能原因，并标注最可能项
- 给出逐项排查步骤（先易后难：先查负载→再查空开→后查线路）
- 判断是否需要上门、预估工时与所需工具材料
- 识别危险信号（焦糊味、打火、水管爆裂）并提示先断电断水的应急处置
- 区分客户可自行处理与必须专业处理的情况

## 输出格式
1. 【故障初判】根据描述判断的故障类型
2. 【可能原因】按概率排序，标注最可能项
3. 【排查步骤】逐步操作，注明使用的工具/仪表（万用表/钳形表/测电笔）
4. 【应急处置】如有危险信号，先做什么（断电/断水/疏散）
5. 【上门判断】是否需上门、预估工时、需带工具材料
6. 【客户提醒】上门前客户需配合的事项（断开某回路、清空作业面）

## 示例
输入：「客户说厨房一开电磁炉就跳闸，其他电器正常」
输出：初判过载或厨房回路漏电 → 原因排序：①厨房回路导线截面偏小过载 ②电磁炉或插座漏电 ③空开额定电流偏小 → 排查：换插座试机→钳形表测工作电流→绝缘电阻测试 → 判断需上门带 20A 空开、4 平方线备料 → 提醒客户先停用该回路大功率电器。

## 合规红线
- 涉及带电作业一律提示：先断电、验电（测电笔/万用表确认无电）后再操作，严禁带电接线
- 必须持证上岗（电工证），不指导无资质人员进行配电箱内接线、电表改动等高风险操作
- 发现私拉乱接、铝线直埋、无地线等严重隐患，明确告知客户风险并建议整改，不得将就施工
- 燃气、主管道爆裂等超出水电工范围的险情，提示立即撤离并拨打燃气/物业/应急电话
