# 施工报价单

## 适用场景
接活前给客户报价、上门检查后出维修报价、装修水电改造整体报价。需要一份让客户看得懂、显得专业、减少扯皮的明细分项报价。

## 核心能力
- 按「材料费 + 人工费 + 其他费用」三分项结构出报价
- 材料逐项列明名称、规格、数量、单价（如 PPR 管 DN25、BV 线 4mm²、86 型暗盒）
- 人工费按计价方式列明（按点位/按米/按项/按工时）
- 标注费用边界：是否含开槽、垃圾清运、高空、夜间、远途费
- 给出合理价格区间参考（师傅可按本地行情调整）
- 附报价说明条款（增项需确认、质保期、付款方式）

## 输出格式
1. 【项目概况】施工地点类型、工作内容概述
2. 【材料明细】表格：名称 / 规格 / 数量 / 单价 / 小计
3. 【人工费明细】表格：项目 / 计价方式 / 数量 / 单价 / 小计
4. 【其他费用】开槽、清运、高空、夜间等（无则写「无」）
5. 【合计与付款】总价、定金比例、尾款节点
6. 【报价说明】有效期、增项确认规则、质保期

## 示例
输入：「客户家卫生间水管漏水，要换一根 PPR 热水管大概 3 米，加两个角阀」
输出：材料明细（PPR 热水管 DN25 ×3 米、直接/弯头若干、全铜角阀 ×2）→ 人工费（管路更换按项计 + 打压测试费）→ 其他（旧管拆除、垃圾清运）→ 合计约 XXX 元（给区间）→ 说明：完工打压 0.8MPa 保压 30 分钟验收，管路质保 X 年。

## 注意事项
- 报价前先问清现场条件（楼层、是否开槽、有无电梯），避免上门后大幅加价伤口碑
- 材料写明品牌或「同等档次可替换」，防止客户拿最低价材料比价
- 增项必须施工前口头+微信确认，报价单注明「未列明项目另计」
- 对老人、回头客可给一口价简化版，减少逐项解释成本
