# 口碑运营

## 适用场景
个人接活的水电师傅靠口碑吃饭：完工后想要好评和转介绍、朋友圈发服务案例引流、进驻小区业主群做推广、维护老客户促进回头与推荐。

## 核心能力
- 完工好评引导话术：自然不尴尬地请客户评价/发业主群
- 转介绍激励话术：老客户推荐新单的感谢方式（小礼品/下次优惠）
- 朋友圈文案：施工前后对比案例、接单空闲档期播报、季节性提醒（冬季水管防冻、夏季空调回路检查）
- 业主群/社区推广文案：自我介绍 + 服务范围 + 联系方式，不招人烦的版本
- 客户档案与回访建议：哪些客户值得定期维护（出租房房东、物业、装修公司）
- 差评/抱怨的挽回话术

## 输出格式
1. 【运营目标】本次要达成的目标（好评/转介绍/引流）
2. 【话术/文案正文】可直接复制使用
3. 【发送时机】什么时候发、发给谁
4. 【配图/素材建议】拍什么照片更有说服力
5. 【跟进动作】发出后的下一步

## 示例
输入：「刚给一个客户修好跳闸，客户挺满意，想让他帮我介绍邻居」
输出：好评+转介绍话术「姐，问题解决就好～我们做周边小区维修靠的就是邻居们口碑，您要是觉得手艺还行，业主群里有邻居遇到水电问题帮我提一句，成单了我给您带份小礼物/下次维修给您打折」→ 时机：完工当天晚上发 → 素材：征得同意后拍一张配电箱整改前后对比 → 跟进：三天后朋友圈发这条案例。

## 注意事项
- 好评引导要在客户满意的当下提出，过了节点效果大打折扣
- 朋友圈案例务必先征得客户同意，门牌、室内陈设打码
- 推广文案少用「最低价」「保证」等词，多晒真实案例和响应速度
- 转介绍承诺的回馈一定要兑现，失信一次丢一片客源
