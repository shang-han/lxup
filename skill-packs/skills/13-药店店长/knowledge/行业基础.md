# 行业基础

## 行业概况
全国零售药店超 60 万家，行业高度监管。GSP 认证、处方药管理、医保定点是关键门槛。受政策影响大（带量采购、门诊统筹、医药分家）。利润来源：药品差价 + 保健品 + 医疗器械 + 健康服务。连锁化率持续提升，单体药店靠专业服务和社区客情生存。

## 常用术语
- 坪效：每平米销售额
- 客单价：平均每单金额
- 关联销售：主药 + 搭配药品的连带比例
- 会员占比：会员消费占总销售比例
- 医保占比：医保支付金额占比
- 近效期：有效期临近（通常 6 个月内）的药品
- 首营企业/首营品种：首次发生供需关系的企业/首次采购的药品，需审核建档
- 双通道：定点医疗机构和定点药店两个医保报销渠道

## 核心工作流程
每日：收货验收 → 药品陈列 → 温湿度记录 → 处方审核 → 销售服务 → 日结对账
每周：近效期药品检查 → 库存盘点 → 补货计划 → 员工培训
每月：GSP 自查 → 供应商对账 → 销售数据分析 → 会员回访

## 常见问题
Q：近效期药品怎么处理？
A：提前 3-6 个月预警，促销催销，与供应商协商退换货，过期药品按不合格品报损销毁。

Q：顾客不信任推荐怎么办？
A：用专业问询建立信任（问症状、问病史），推荐时说清搭配理由，不强推高价药，主动提示便宜替代。

Q：处方药销售要注意什么？
A：凭处方销售，执业药师审核签字，登记购买信息，含麻黄碱复方制剂限购并登记身份证。

Q：客流下降怎么办？
A：深耕社区（免费检测、健康讲座）、做慢病会员管理提复购、接入线上平台（美团买药/饿了么）拓增量。
