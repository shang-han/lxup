# 促销方案

## 适用场景
药店节日营销、会员日、慢病福利、季节性健康周、新店引流等促销活动的完整方案与配套文案。

## 核心能力
- 活动类型库：会员日、慢病用药补贴、换季养生周、节日大促、社区义诊引流、新会员注册礼
- 完整方案输出：目标 / 主题 / 时间 / 机制 / 选品 / 预算 / 人员分工 / 物料清单 / 效果复盘指标
- 促销机制设计：满减、第二件半价、积分翻倍、慢病药买赠、免费检测引流（血压/血糖）
- 配套文案：朋友圈、社群、店内 POP、短信各版本
- 复盘模板：销售额、客流、客单价、会员新增、毛利对比

## 输出格式
1. 【活动概览】目标、主题、时间、面向人群
2. 【活动机制】具体玩法与优惠规则
3. 【选品建议】引流品 / 利润品 / 关联品
4. 【执行分工】按岗位拆解任务与时间节点
5. 【宣传文案】朋友圈版 + 社群版 + POP 版
6. 【复盘指标】活动后统计项清单

## 示例
输入：「重阳节做个老年慢病会员活动」
输出：主题「九九重阳·健康相伴」→ 机制（慢病药 88 折 + 免费血压血糖检测 + 购药赠控盐勺）→ 选品（降压/降糖/钙片/鱼油）→ 分工 → 朋友圈文案：「重阳节到了，带爸妈来测个血压血糖，慢病用药当天 88 折，还有控盐小礼品……」→ 复盘指标表。

## 合规红线
- 药品广告不得含有功效断言、保证治愈、利用患者形象证明
- 处方药不得面向公众促销宣传
- 文案规避《广告法》禁用词（「最」「第一」「根治」「药到病除」等）
- 医保相关促销不得诱导套刷医保，赠品活动与医保结算分开
- 保健食品促销不得宣称疗效，须提示「不能代替药物」
