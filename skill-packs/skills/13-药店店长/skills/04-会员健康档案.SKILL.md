# 会员健康档案

## 适用场景
为慢病会员（高血压、糖尿病、高血脂等）建立健康档案，记录用药情况，生成复购提醒与定期回访计划，提升会员黏性与专业信任。

## 核心能力
- 建档模板：基本信息 / 病史 / 过敏史 / 慢病指标 / 当前用药清单 / 家庭成员
- 用药记录结构化：药名、规格、用法、开药日期、预计用完日期
- 复购提醒：按用量推算药品耗尽节点，生成提醒话术
- 按病种生成健康宣教要点（低盐饮食、血糖监测频率、足部护理等）
- 回访计划表：回访节点（1 周/1 月/3 月）+ 回访问题清单 + 记录模板

## 输出格式
- 档案类：结构化表单（Markdown 表格或键值清单）
- 回访类：时间线表格（节点 / 目的 / 话术 / 记录栏）
- 宣教类：分条要点，口语化，便于店员转述

## 示例
输入：「给一位 65 岁高血压会员建档，吃氨氯地平，每月一盒」
输出：档案表单（含血压记录栏、合并用药栏）→ 用药记录（30 天用量，第 25 天触发提醒）→ 提醒话术：「张叔，您的降压药大概还能吃五天，记得别断药，顺路来量个血压？」→ 宣教要点（清晨服药、定期量血压、低盐饮食）。

## 合规红线
- 健康信息属于个人敏感信息，提示会员知情同意，档案仅本地保存、不上传
- 只做用药记录与提醒，不做诊断、不擅自建议换药/调量
- 指标异常（如血压持续偏高）话术中固定加入「建议复诊」引导
- 档案字段最小化，不收集与服务无关的隐私
