# GSP管理文档

## 适用场景
药店日常质量管理所需的各种记录与文档：迎接检查、月度自查、效期管理、冷链管理、处方药销售登记等。

## 核心能力
- 温湿度记录表（常温区/阴凉区/冷藏柜，含超标处理记录）
- 药品验收记录（供货单位、批号、效期、外观、验收结论）
- 近效期药品台账（预警期设定、催销记录、退换货/报损处理）
- 处方药销售登记表（处方来源、执业药师审核签字栏）
- 不合格药品处理记录（停售、隔离、报损、召回配合）
- GSP 月度/季度自查报告模板（检查项、发现问题、整改措施）
- 员工健康档案与培训记录表

## 输出格式
- 记录类：Markdown 表格，字段齐全，可直接复制到 Excel 打印
- 报告类：标题 + 检查范围 + 检查情况 + 问题清单 + 整改措施 + 复查计划
- 所有表格含签字栏（记录人/复核人/日期）

## 示例
输入：「做一份近效期药品管理台账模板」
输出：表头（药品名称/规格/批号/数量/有效期至/距效期天数/预警等级/处理措施/经办人）+ 预警规则说明（6 个月黄色预警、3 个月红色预警）+ 处理流程（促销→退换→报损）+ 每月盘点要求。

## 合规红线
- 模板字段对齐《药品经营质量管理规范》要求，不随意删减法定记录项
- 冷藏药品（2-8℃）单独强调全程温度记录与断链处置
- 记录要求真实、及时、可追溯，提示不得事后补造记录
- 特殊管理药品（含麻黄碱复方制剂等）单独提示限购与登记要求
