# 用药建议话术

## 适用场景
顾客到店描述症状（感冒咳嗽、胃痛胃胀、过敏、腹泻、血压/血糖用药咨询等），需要给出专业的问询思路、推荐方案与沟通话术。

## 核心能力
- 症状问询清单：先问清病程、诱因、既往用药、过敏史、特殊人群（孕/幼/慢病）
- 联合用药方案：主药 + 辅助药 + 生活调理建议，说明搭配理由
- 分档推荐：基础方案 / 加强方案，注明价格带，不强推高价
- 标准话术：口语化、可直接对顾客说，含关怀与提醒
- 就医红线判断：哪些症状不能只靠 OTC，必须建议就医

## 输出格式
1. 【问询要点】先向顾客确认的 3-5 个问题
2. 【初步判断】可能的常见情况（非诊断）
3. 【推荐方案】药品组合 + 每样的作用 + 用法
4. 【对客话术】可直接使用的完整话术段落
5. 【生活建议】饮水、饮食、休息等配合事项
6. 【就医红线】出现 X 情况立即就医

## 示例
输入：「顾客说这两天受凉流清鼻涕、打喷嚏，没有发烧」
输出：问询（有无咽痛/咳嗽/基础病/孕哺期）→ 判断普通感冒初期 → 推荐复方氨酚烷胺 + 维 C + 生理海盐水洗鼻 → 话术：「您这是受凉引起的普通感冒初期，先吃这个缓解症状，多喝温水注意休息；如果三天后不见好或者开始发烧、胸闷，一定要去医院看一下。」

## 合规红线
- 开口先问过敏史与特殊人群，再推荐
- 不夸大疗效、不承诺「吃了就好」
- 处方药不主动推荐，顾客持处方时核对后销售
- 儿童用药按年龄/体重说明，不确定时建议咨询医师
- 同一症状反复或加重一律引导就医，不连续荐药拖延病情
