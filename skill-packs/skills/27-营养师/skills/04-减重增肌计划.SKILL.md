# 减重增肌计划

## 适用场景
客户有明确的体重或体成分改善目标：减重减脂、增肌、突破体重平台期、产后恢复、婚前塑形。营养师输出饮食与运动结合、可分阶段执行的完整计划。

## 核心能力
- 按目标设定热量策略：减脂缺口 300-500kcal/日，增肌盈余 200-300kcal/日
- 蛋白质定向配置：减脂期 1.6-2.2g/kg 体重、增肌期 1.6-2.4g/kg，并分配到各餐
- 运动配合建议（力量训练为主 + 有氧辅助），协调训练前后营养时机（训前加餐、训后补充）
- 设定阶段目标（4 周/8 周/12 周），明确每阶段重点与合理进度
- 平台期识别与应对：代谢适应评估、饮食再平衡、训练调整
- 设计追踪指标：体重、围度、体脂、力量水平、体态照片对比

## 输出格式
1. 【现状评估】身体数据、目标、难点分析
2. 【总体策略】热量与三大营养素框架、饮食+运动组合
3. 【饮食方案】每日餐食框架与执行要点
4. 【运动建议】训练频率、类型、强度参考
5. 【阶段目标】各阶段里程碑与合理预期
6. 【追踪调整】监测频率、触发调整的条件

## 示例
输入：「男，28 岁，178cm/90kg，体脂 26%，想减到 80kg，每周能去健身房 2 次」
输出：热量目标约 1900-2100kcal/日（缺口约 500）→ 蛋白质 150g/日、碳水集中在训练前后 → 饮食：三餐 + 训后加餐框架（鸡胸/鱼虾/蛋奶 + 糙米薯类 + 大量蔬菜）→ 运动：力量训练增至每周 3 次 + 2 次低强度有氧 → 阶段目标：4 周 -3kg（适应期）、12 周 -7~8kg → 追踪：每周固定晨起称重、每 2 周测围度拍照，连续 2 周体重不动则重新评估热量。

## 注意事项
- 合理减重速度为每周 0.5%-1% 体重，过快提示肌肉流失与反弹风险
- 不推荐减肥药、代餐断食、催吐等极端方式，不推销任何产品
- 有进食障碍倾向（厌食/暴食）的客户应建议心理/医疗干预，不为其制定热量限制方案
- 未成年人、孕产妇、术后人群不适合减重计划，建议先咨询医生
- 效果因人而异，不承诺「X 天瘦 X 斤」等具体结果
