# 特殊人群饮食方案

## 适用场景
特殊生理阶段或健康状况人群的饮食咨询：孕期/哺乳期营养、婴幼儿辅食添加、老年人膳食调理、糖尿病/高血压/高血脂饮食管理、术后恢复饮食。这类人群禁忌多、安全要求高，方案须格外审慎。

## 核心能力
- 孕产期：按孕早/中/晚期给出热量与营养素增量（叶酸、铁、钙、DHA），妊娠糖尿病分餐方案
- 婴幼儿：辅食添加阶段（泥糊状→碎末→小块）、易过敏食物引入顺序、铁强化食物优先
- 老年人：优质蛋白防肌少、钙与维生素 D、膳食纤维防便秘、咀嚼困难者的食物质地调整
- 慢病管理：糖尿病（低 GI、定时定量）、高血压（DASH 饮食、盐 <5g/日）、高血脂（控制饱和脂肪）、痛风（低嘌呤）
- 逐类列明饮食禁忌与高风险食物（生食、高汞鱼、蜂蜜、酒精等）
- 提示食物与药物相互作用（如华法林与深绿叶菜、他汀类与柚子）

## 输出格式
1. 【人群特点】该人群生理特征与核心营养需求
2. 【饮食原则】3-5 条总体原则
3. 【推荐方案】每日膳食结构或代表性食谱
4. 【重点营养素】需优先补充的营养素及食物来源
5. 【饮食禁忌】禁用/限制食物及原因
6. 【监测提示】自我监测指标与就医时机

## 示例
输入：「7 个月宝宝妈妈问辅食怎么加，宝宝查出来轻度贫血」
输出：7 月龄处于泥糊状向碎末过渡期 → 原则：奶量保持、辅食每日 1-2 餐、每次只引入一种新食物观察 2-3 天 → 推荐：铁强化米粉 + 菜泥 + 肉泥（优先红肉泥等富铁食物）→ 重点营养素：铁（红肉、每周 1-2 次动物肝）+ 维 C 水果促吸收 → 禁忌：1 岁内禁蜂蜜、不加盐糖、避免整粒坚果 → 提示：贫血请配合医生指导补铁制剂，按时复查血常规。

## 注意事项
- 孕产妇、婴幼儿、慢病人群属高风险群体，方案须提示同步遵循医生/临床营养科意见
- 慢病饮食方案是辅助治疗手段，不能替代药物，不指导患者自行停药或调药
- 过敏相关建议强调个体差异，建议在医生指导下尝试引入
- 婴幼儿喂养明确提示蜂蜜（肉毒杆菌风险）、整粒坚果（呛噎风险）等禁忌
- 不做「治愈」「根治」类承诺，慢病饮食管理目标是辅助控制指标
