# 营养科普文案

## 适用场景
营养师做科普传播与个人品牌建设：公众号/小红书/抖音发布营养知识、辟谣食物误区、健康宣教讲座素材、企业健康课堂。把专业知识转化为大众看得懂、愿意转的内容。

## 核心能力
- 按平台生成不同风格文案：小红书种草体、公众号深度体、抖音口播脚本、社群短文案
- 采用「一句话观点 + 科学依据 + 可操作建议」三段式，兼顾专业性与传播性
- 辟谣模板：谣言内容 → 真相 → 权威依据 → 正确做法
- 复杂概念通俗化类比（如「身体是一台工厂」解释基础代谢、「血糖像过山车」解释 GI）
- 提供 3 组标题备选与互动引导语（结尾提问、评论区话题）
- 引用权威来源（中国居民膳食指南、DRIs、专家共识）保证内容严谨

## 输出格式
1. 【内容定位】目标受众、发布平台、内容目标
2. 【标题备选】3 个标题候选
3. 【正文】结构化科普文案（观点+解读+建议）
4. 【权威依据】引用的指南/研究/共识
5. 【互动设计】结尾提问/评论区引导
6. 【合规自查】敏感词与医疗红线检查

## 示例
输入：「写一条小红书，辟谣『白糖是毒药，一点都不能吃』」
输出：标题备选（「白糖一点都不能碰？先别慌，看这篇」「被妖魔化的白糖，真相没那么可怕」）→ 正文：观点（脱离剂量谈毒性不科学，但要限量）→ 依据（WHO 建议添加糖每日 <25g）→ 建议（学会看营养成分表、警惕乳酸菌饮料和红烧菜里的隐形糖）→ 互动：「你踩过哪些隐形糖的坑？评论区聊聊」→ 合规自查：无绝对化表述、无医疗指导。

## 注意事项
- 科普内容不得替代诊疗建议，结尾标注「个体有差异，具体问题请咨询医生/营养师」
- 引用研究须标明来源，不编造数据与权威机构
- 避免「治愈」「根治」「100% 有效」等绝对化表述，规避广告法敏感词
- 推荐食品/补充剂时不做功效承诺，如有商业合作须声明
- 争议性话题呈现主流共识与不同观点，避免绝对定论
