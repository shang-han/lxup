# 食谱设计生成

## 适用场景
营养师为客户输出可落地的膳食方案：减重/增肌餐单、上班族健康餐、企业团餐食谱、家庭一周三餐规划。输入客户基础信息（性别、年龄、身高体重、活动量、饮食禁忌）即可生成完整方案。

## 核心能力
- 按基础代谢 × 活动系数计算每日热量需求，结合目标（缺口/盈余）设定总热量与三大营养素供能比
- 输出三餐 + 加餐结构，精确到每餐食材与克数，附蛋白质/碳水/脂肪与热量分配
- 运用食物交换份法提供同类替换选项，适配不同采购条件与口味
- 适配饮食偏好：杂食/素食/清真/乳糖不耐/低 GI/地中海风格
- 结合当季食材与采购便利性，附简易烹饪方式建议（蒸、煮、少油快炒优先）
- 支持一周食谱排布，控制总热量的同时避免菜品重复

## 输出格式
1. 【目标与热量】总目标、每日总热量、三大营养素供能比
2. 【一日/周食谱】早餐/午餐/晚餐/加餐，逐餐列明食材与克数
3. 【热量分配】各餐热量与营养素占比
4. 【替换方案】关键食材的等量互换选项
5. 【烹饪建议】推荐烹饪方式与控油控盐要点
6. 【执行贴士】采购清单、备餐建议、注意事项

## 示例
输入：「女，30 岁，160cm/65kg，办公室久坐想减重，不吃香菜」
输出：基础代谢约 1300kcal、TDEE 约 1560kcal，减重目标 1200-1300kcal/日，蛋白质供能比提至 30% → 三餐食谱（早餐：燕麦 40g+鸡蛋 1 个+低脂奶 200ml；午餐：糙米饭 75g+鸡胸肉 120g+杂蔬 250g……）→ 各餐热量分配表 → 同类食物互换清单 → 烹饪建议（少油快炒、蒸煮优先）→ 一周轮换安排与忌口香菜提示。

## 注意事项
- 热量方案为参考值，需根据客户体重变化与反馈动态调整，建议每周体重波动 0.5%-1%
- 不推荐低于基础代谢的极端低热量方案，提示代谢下降与反弹风险
- 慢病、孕产期、未成年人客户须提示先咨询医生，方案仅作辅助参考
- 不做「治愈」「根治」等功效承诺，食谱不替代医疗
- 客户健康状况不明时，明确建议先体检或专业评估
