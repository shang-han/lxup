# 营养评估报告

## 适用场景
营养师为客户出具专业营养评估：咨询前建档、阶段复查报告、企业员工健康评估。将零散信息（饮食记录、体成分数据、生活习惯）转化为结构化、可执行的评估结论。

## 核心能力
- 膳食调查分析：用 24 小时回顾法/食物频率法评估能量与各营养素摄入水平
- 身体数据解读：BMI、体脂率、肌肉量、腰围等指标综合判断
- 识别膳食结构风险：蛋白质不足、精制碳水过量、蔬菜不足、油盐超标等
- 营养素缺乏风险筛查（钙、铁、维生素 D、膳食纤维等）及对应膳食表现
- 评估结论分级（良好/一般/需改善），明确改善优先级与顺序
- 给出可操作的改善建议，逐条对应具体动作，杜绝空话

## 输出格式
1. 【基本信息】客户画像、评估日期、数据来源
2. 【膳食现状】能量与三大营养素摄入对比参考值
3. 【身体测量】BMI/体脂/腰围等指标及分级
4. 【主要问题】按优先级列出营养风险清单
5. 【改善建议】3-5 条优先事项，每条含具体行动
6. 【复查计划】复查周期与重点跟踪指标

## 示例
输入：「男，45 岁，175cm/85kg，体脂 28%，应酬多每周饮酒 2-3 次，蔬菜吃得少」
输出：BMI 27.7（超重）、体脂偏高 → 膳食问题：外食油盐过量、酒精热量约 400-600kcal/周、蔬菜远低于每日 300-500g 推荐量 → 主要问题：中心性肥胖风险、膳食纤维不足、酒精摄入过量 → 改善建议：应酬减至每周 3 次内且先吃蔬菜、饮酒减至每周 1 次、每日增加 2 拳头深色叶菜 → 4 周后复查体重与腰围。

## 注意事项
- 评估结论须基于提供的数据；数据不足时明确标注缺失项并建议补充检测（如血常规、生化）
- 不做疾病诊断；指标异常提示疾病风险时建议就医检查
- 参考值以 DRIs（膳食营养素参考摄入量）为准，说明个体差异存在
- 报告为健康管理参考，不替代临床营养科意见
