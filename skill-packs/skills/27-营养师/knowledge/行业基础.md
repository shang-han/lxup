# 行业基础

## 行业概况
营养师服务涵盖个人营养咨询、企业团餐设计、食品研发支持、母婴营养、慢病饮食管理等领域。2026 年全国持证营养师超 50 万人，随着健康中国战略推进和居民健康意识提升，行业需求持续增长。核心能力：扎实的营养学知识 + 科学的膳食设计 + 沟通引导能力。服务形式包括一对一咨询、团体膳食指导、线上订阅制健康管理、健康科普内容输出。收入来源：咨询费、方案服务费、企业顾问合作、内容变现。

## 常用术语
- 宏量营养素：碳水化合物、蛋白质、脂肪三大产能营养素，提供能量与身体构建原料
- 微量营养素：维生素与矿物质，需要量小但缺乏会引发健康问题
- GI 值：血糖生成指数，反映食物升高血糖的速度，高 GI 食物升糖快、不利于血糖管理
- DRIs：膳食营养素参考摄入量，设计每日营养素摄入量的权威依据
- 基础代谢率：人体静息状态下消耗的能量，是热量计算的基数
- TDEE：每日总能量消耗 = 基础代谢 × 活动系数，减重增肌热量摄入的参照基准
- 食物交换份法：将食物按类别分组、同类等量互换，保证营养均衡的同时丰富食谱花样

## 核心工作流程
咨询阶段：膳食调查（24 小时回顾法/食物频率问卷）→ 营养评估（体成分、生化指标参考）→ 目标设定 → 方案制定
执行阶段：食谱设计 → 食材采购建议 → 烹饪方式指导 → 每日打卡与进度跟踪
评估阶段：数据前后对比 → 方案调整 → 效果确认 → 长期维持指导

## 常见问题
Q：减肥期间能吃主食吗？
A：能吃。用粗粮杂豆（燕麦、糙米、红豆、红薯）替代部分精米白面，控制总量每餐一拳头左右即可。

Q：需要吃营养补剂吗？
A：优先通过均衡膳食获取营养；经评估确实缺乏时（如孕妇补叶酸、素食者补维生素 B12）再针对性补充。

Q：客户坚持不了食谱怎么办？
A：降低执行门槛——用食物交换法提供替换选项、设置弹性餐、目标拆分为每周阶段，配合打卡陪伴与正向反馈。
