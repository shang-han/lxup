# 行业基础

## 行业概况
手机数码零售市场竞争激烈，新机发布节奏快，利润空间被电商挤压。利润来源：手机差价（薄利）+ 配件（高利）+ 维修 + 运营商业务（转售/宽带）。门店靠现场体验、售后服务与运营商业务与电商形成差异化。以旧换新与分期是降低决策门槛、提升成交的重要手段。

## 专业技能体系

### 硬技能
- 主流品牌产品线知识（苹果/华为/小米各型号差异）
- 二手机检测与估价
- 维修配件供应链
- 运营商套餐代理
- 防盗与保险配置

### 软技能
- 技术参数翻译成用户语言
- 售后纠纷处理

## 常用术语
- 进店转化率：进店 → 成交比例
- 客单价：平均每单金额
- 配件附带率：购机同时购买配件比例
- 运营商业务占比：开户/宽带业务占比
- 以旧换新占比：旧机抵扣购新机比例
- IMEI：手机设备识别码，用于保修与真伪查询
- 保修：厂商质保期内的免费维修服务
- 碎屏险/延保：付费增值保障服务

## 核心工作流程
每日：产品陈列 → 客户接待 → 演示体验 → 成交开单
每周：库存盘点 → 价格更新 → 竞品调研
每月：销售分析 → 运营商结算 → 以旧换新处理

## 常见问题
Q：客户说网上更便宜怎么办？
A：强调正品保障 + 售后服务 + 现场体验 + 赠品优惠，如实对比价格与服务差异。

Q：怎么提高配件销售？
A：开箱即推荐，提供全套解决方案，套餐优惠价打包。

Q：怎么提升进店转化？
A：做好需求挖掘、演示体验、诚实对比，用以旧换新和分期降低决策门槛。

Q：库存管理重点是什么？
A：热门机型保周转防缺货，滞销机型控库存，临退市机型及时清仓。
