# 促销活动策划

## 适用场景
新品上市、节假日、店庆、开学季、清仓等节点策划门店促销活动，以引流和提升销售为目标。

## 核心能力
- 活动类型库（新品首发、节日大促、店庆、学生特惠、以旧换新节）
- 完整方案输出：目标 / 主题 / 时间 / 机制 / 选品 / 预算 / 分工 / 物料
- 促销机制设计（直降、赠品、套餐优惠、分期免息、以旧换新）
- 配套文案（朋友圈、店内 POP、短信、社群）
- 执行分工与时间节点
- 复盘指标（销售额、客流、客单价、配件附带率、会员新增）

## 输出格式
1. 【活动概览】目标、主题、时间、面向人群
2. 【活动机制】具体玩法与优惠规则
3. 【选品建议】引流品 / 利润品 / 配件
4. 【执行分工】按岗位拆解任务与时间节点
5. 【宣传文案】朋友圈版 + POP 版 + 短信版
6. 【复盘指标】活动后统计项清单

## 示例
输入：「新品上市，策划一场门店促销活动」
输出：概览（目标新品首发冲量，主题「新品尝鲜节」，时间本周末）→ 机制（新品购机立减 200 + 送贴膜与保护壳 + 12 期免息 + 以旧换新额外补 300）→ 选品（新品引流、配件做利润）→ 分工（接待/演示/算账/物料各岗）→ 文案（朋友圈「新品到店抢先上手，周末到店购机送好礼 + 分期免息，贴膜免费限量」）→ 复盘（新品销量、配件附带率、会员新增）。

## 注意事项
- 促销信息真实，赠品与优惠条件明示
- 避免「全网最低」「顶级」等广告法禁用词
- 限量赠品标明数量与时限，避免纠纷
- 质保与售后不因促销缩水
- 活动前做好预算与人员排布，活动后及时复盘
