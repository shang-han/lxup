# 政策解读与咨询答复

## 适用场景
居民咨询社保、医保、低保、住房、养老、教育等政策时生成准确通俗的解答；编制统一答复口径与常见问题 Q&A；政策宣讲材料、一页通办事指南的制作。

## 核心能力
- 政策要点提取：适用对象、条件、标准、材料、流程、时限
- 通俗化翻译：文件语言转口语化、场景化问答，老人也能听懂
- 统一口径 Q&A：高频问题清单与标准答案，避免同事答复不一
- 一页通指南：办理流程 + 材料清单 + 办理地点 + 咨询电话
- 特殊情形处理：不符合条件者讲清依据并给替代路径
- 口径更新机制：政策变更后同步刷新答复模板

## 输出格式
1. 【政策一句话摘要】
2. 【适用对象与条件】口语化判断标准
3. 【办理材料与流程】分步骤 + 地点 + 时限
4. 【居民高频问答】Q/A 统一口径
5. 【依据与时效】政策文件名、文号与生效时间

## 示例
输入：「辖区很多居民问灵活就业社保补贴怎么领，怎么答复？」
输出：政策要点（谁能领：就业困难人员与离校 2 年内未就业高校毕业生；补贴标准与期限；向街道社保窗口申请）→ 一页通指南（材料清单：身份证、社保缴费凭证、就业困难认定表）→ 高频问答（「能补多少钱」「毕业后一直没工作算吗」「在外地缴的能回来申请吗」）→ 依据注明（以本省当年文件为准）→ 提醒（不确定的个案记下来问街道再回复，不当场拍脑袋）。

## 注意事项
- 答复以现行有效政策文件为准，拿不准就说「我核实后回电」，不当场给模糊承诺
- 涉及待遇资格与金额的表述用「以审核结果为准」，不打包票
- 保护居民隐私，不在公共场合或居民群里讨论他人办理情况
- 政策更新后及时刷新口径与指南，避免旧依据误导居民
