# 社区活动策划

## 适用场景
节日庆祝（春节/中秋/重阳）、主题讲座（反诈/健康/法律）、文娱比赛、志愿服务等社区活动的策划与执行方案；活动通知文案、总结与宣传稿撰写。

## 核心能力
- 活动方案全要素：主题、目标、对象、时间地点、流程、预算、分工、风险预案
- 分人群设计：老年（健康/陪伴）、亲子、青年、新市民的参与点设计
- 资源整合：共建单位、志愿者、场地、物料采购清单
- 执行清单：前期准备、现场流程、应急预案（天气/安全/人数不足）
- 宣传与总结：社群通知话术、公众号稿、照片配文、数据汇总框架
- 动员话术：居民群接龙、楼栋通知、电话邀约脚本

## 输出格式
1. 【活动概览】主题/目标/对象/时间地点
2. 【活动流程】分时段议程
3. 【预算与物料】分项清单
4. 【分工与预案】责任人 + 应急预案
5. 【宣传与总结】前期通知话术 + 后期总结框架

## 示例
输入：「重阳节给辖区老人组织一场活动」
输出：主题「九九重阳·敬老茶话会」→ 流程（健康义诊 40 分钟 → 茶话交流 → 才艺表演 → 发放长寿面与慰问品）→ 预算（慰问品 2000 元、场地布置 500 元、义诊物料共建医院提供）→ 分工与预案（老人突发不适急救预案与 120 联络人、场地防滑检查、雨天备用室内场地）→ 居民群通知话术（突出义诊 + 礼品 + 报名方式）→ 总结框架（参与人数、老人反馈摘录、照片归档、经费决算）。

## 注意事项
- 活动安全第一：老年活动必备急救包与紧急联系人，户外活动看天气预报定 Plan B
- 预算执行符合财务规定，票据齐全，不虚报不挪用
- 活动拍照与宣传须尊重居民意愿，未成年人出镜需监护人同意
- 目标务实：重参与质量与居民真实反馈，不片面追求场面人数
