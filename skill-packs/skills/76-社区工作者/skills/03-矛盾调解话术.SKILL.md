# 矛盾调解话术

## 适用场景
邻里纠纷（噪音/漏水/宠物/停车）、家庭矛盾、物业矛盾等常见纠纷的调解准备与现场话术；信访接待的应答思路；调解记录与协议书要点撰写。

## 核心能力
- 纠纷事实梳理：双方诉求、争议焦点、证据与时间线整理
- 调解策略：背对背分开沟通 → 面对面协商，先处理情绪再处理事情
- 法律依据引用：民法典相邻关系、噪声污染、宠物管理等常用条款的通俗解释
- 折中方案设计：2-3 个可执行的折中选项供双方选择
- 信访接待：倾听记录 → 依据解释 → 流程引导 → 升级上报判断
- 文书输出：调解记录、调解协议书模板要点

## 输出格式
1. 【纠纷概况】当事人、起因、争议焦点
2. 【双方诉求与底线】
3. 【调解思路】步骤与对应话术（背对背/面对面）
4. 【依据与折中方案】法律政策依据 + 可选折中方案
5. 【协议要点与回访】

## 示例
输入：「楼上漏水泡了楼下天花板，双方为赔偿谈不拢，怎么调解？」
输出：分别了解（楼下主张赔偿 5000 元 + 精神损失，楼上认为要价过高且怀疑是公共管道问题）→ 核查焦点（漏水点责任认定：先约物业共同查勘，必要时请专业检测）→ 背对背话术（对楼下：理解损失感受，引导按实际修复费用主张；对楼上：讲清相邻关系法律责任，拖延可能承担更多）→ 折中方案（A 双方共同委托第三方评估按评估价分担；B 楼上负责修复、楼下自担误工费；C 引入物业与业委会共同见证）→ 协议要点（修复时限、费用分担、后续免责条款）→ 一周后回访复检。

## 注意事项
- 调解坚持自愿、合法原则：不强迫和稀泥，调解不能代替司法裁决
- 涉及人身安全、疑似违法犯罪（家暴、伤害）立即报警并上报，不单独处置
- 保持中立姿态，不偏袒任何一方，避免先入为主定性
- 调解协议书面签署，重大纠纷建议引导司法确认以获强制执行力
- 当事人隐私严格保密，纠纷细节不在居民中传播
