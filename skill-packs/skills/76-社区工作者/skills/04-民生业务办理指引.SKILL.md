# 民生业务办理指引

## 适用场景
为居民生成低保、临时救助、高龄津贴、残疾人补贴、医疗救助等民生业务的办理指南；面向老人、残障人士等特殊群体的上门讲解与帮办代办要点；申请被驳回时的补正指引。

## 核心能力
- 事项流程拆解：申请条件 → 所需材料 → 办理窗口 → 时限 → 结果反馈
- 材料清单规范：必备/可选/告知承诺制材料的分类说明
- 特殊群体帮办：老年人、残障人士的帮办代办流程与陪同要点
- 驳回情形处理：常见驳回原因与补正建议
- 政策联动提示：可叠加享受的政策组合（低保 + 医疗救助 + 临时救助）
- 表达关怀：涉及困难群体的措辞尊重隐私与尊严

## 输出格式
1. 【事项名称与政策依据】
2. 【申请条件】口语化判断标准
3. 【材料清单】必备/可选分项
4. 【办理流程与时限】分步骤
5. 【常见问题与温馨提醒】

## 示例
输入：「有位独居老人子女无赡养能力，家里困难，能申请低保吗？要什么材料？」
输出：条件说明（户籍/居住要求，共同生活家庭成员人均收入与财产低于当地低保标准）→ 材料清单（申请表、身份证与户口簿、收入证明、家庭经济状况核对授权书、子女赡养能力情况说明）→ 流程（社区受理与入户调查 → 街道审核 → 区级民政审批 → 公示 → 按月发放）→ 时限（一般 20-30 个工作日）→ 温馨提醒（可同时咨询医疗救助与临时救助；老人行动不便可预约帮办代办，材料由社工协助整理）。

## 注意事项
- 资格审核权在审批部门，社区是受理与初核角色，不对结果打包票
- 入户调查、民主评议必须如实记录，弄虚作假有纪律与法律风险
- 申请人家庭信息严格保密，公示范围与内容按制度执行不扩大
- 对不符合条件者耐心解释依据并指引其他救助路径，不简单回绝
