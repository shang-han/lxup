# 行业基础

## 行业概况
在线教育市场规模超 4000 亿元，覆盖 K12 素养、成人职业教育、技能培训、兴趣学习等。主流模式为录播课 + 直播课 + 社群运营 + 1 对 1 辅导的组合，知识付费与训练营模式普及。运营核心指标是完课率与转化率（免费→付费、低价→高价、续费）。流量红利见顶后，精细化运营、私域沉淀与口碑转介绍成为增长关键。课程运营贯穿课前（内容策划/制作/推广）、课中（学员管理/社群/答疑）、课后（复盘/回访/迭代）全流程。

## 常用术语
- 完课率：完成全部/主要课程学习的学员比例，反映内容质量与黏性
- 转化率：免费/低价用户 → 付费用户的比例
- LTV：用户生命周期总价值，衡量单个用户的长期贡献
- CAC：获取一个付费用户的成本，需与 LTV 对比看投放效率
- 续费率：课程到期后继续付费/续报的比例
- 训练营：以社群+打卡+辅导驱动的周期性学习产品，转化主力
- SOP：社群/运营的标准化流程
- 私域：可反复触达的用户池（微信群、公众号、个人号）

## 核心工作流程
课前：内容策划 → 课程制作 → 详情页上线 → 推广预热
课中：学员管理 → 社群运营 → 作业批改 → 答疑辅导 → 促转化
课后：课程复盘 → 数据整理 → 学员回访 → 迭代优化

## 常见问题
Q：完课率低怎么办？
A：缩短单课时长、增加互动与练习、设置打卡激励与证书、社群同伴监督、关键节点提醒回访。

Q：怎么提高付费转化？
A：用低价/免费体验课引流，在训练营中展示效果与口碑，配合限时优惠、从众案例、1 对 1 跟进促单。

Q：续费率上不去？
A：做好学习效果外化（成果展示、阶段测评）、提前在课程后期铺垫续报、给予老学员专属权益与服务。

Q：社群死群没人说话？
A：固定栏目（早报/打卡/话题）、设计互动激励、安排「托」带动、及时响应、定期清理与分层运营。
