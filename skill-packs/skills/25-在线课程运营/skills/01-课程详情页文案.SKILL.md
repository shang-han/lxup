# 课程详情页文案

## 适用场景
在线课程运营为新课上架、训练营招生撰写详情页/落地页文案，需要在几屏之内抓住用户痛点、讲清课程价值、建立信任并促成下单。

## 核心能力
- 目标人群画像与痛点挖掘，开篇即戳中需求
- 课程卖点提炼：结果导向（学完能获得什么）而非堆砌内容
- 详情页结构：痛点 → 解决方案 → 课程大纲 → 讲师背书 → 学员成果 → 价格锚点 → 限时行动
- 信任构建：讲师资历、学员案例、数据成果、保障承诺
- 转化钩子：限时优惠、名额、赠品、无忧退款
- 标题与副标题的吸睛写法
- 适配不同平台（自有平台/小程序/第三方）的文案风格

## 输出格式
1. 【主标题+副标题】吸睛且点明价值
2. 【痛点开篇】目标人群与核心困扰
3. 【课程解决方案】学完能得到什么
4. 【课程大纲亮点】分模块价值点
5. 【信任背书】讲师/学员/数据
6. 【转化引导】价格、优惠、行动按钮文案

## 示例
输入：「我们上了一门面向职场新人的Excel高效办公课，帮我写详情页文案」
输出：主标题「从加班到准点下班：职场新人 Excel 高效办公实战」→ 痛点（数据整理慢、报表反复返工、不会函数与透视表）→ 解决方案（21 天掌握高频函数+透视表+自动化，效率提升 3 倍）→ 大纲亮点（分 4 模块、每节配实操作业）→ 信任背书（讲师 10 年数据分析经验、3000+ 学员好评）→ 转化引导（原价 299 限时 99、前 100 名赠模板包、7 天无理由退款）。

## 注意事项
- 卖点要结果导向、可感知，避免「干货满满」这类空话
- 不夸大效果、不虚构师资与学员数据，遵守广告法
- 价格与优惠真实，限时/限量须属实，避免虚假营销
- 文案规避绝对化用语（「最」「第一」「保证学会」）
- 行动引导明确，降低决策门槛（试看、退款保障）
