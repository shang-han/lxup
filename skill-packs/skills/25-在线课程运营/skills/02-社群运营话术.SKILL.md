# 社群运营话术

## 适用场景
在线课程运营/班主任在训练营、学员社群中，需要开营破冰、日常打卡、答疑维护、促活互动、续费转化等全流程话术，保持社群活跃并推动学习完课与转化。

## 核心能力
- 社群生命周期话术：开营 → 破冰 → 日常打卡 → 中期激励 → 结营 → 续费
- 群公告/通知模板：开营须知、打卡规则、课程提醒、活动通知
- 促活互动设计：话题讨论、投票、接龙、红包、晒成果
- 答疑与督学话术：鼓励坚持、提醒进度、一对一关怀
- 续费/转化铺垫话术：自然不生硬，先价值后转化
- 死群激活与负面情绪处理话术
- 分时段运营节奏（早报、午间、晚间打卡）

## 输出格式
1. 【运营节点】当前所处阶段
2. 【话术/公告】可直接使用的文案
3. 【互动设计】配套促活动作
4. 【使用时机】发送时间与频率
5. 【注意事项】该节点的沟通要点

## 示例
输入：「训练营第三天了，群里打卡的人越来越少，帮我想点话术激活一下」
输出：判断为「中期疲劳期」→ 激励话术（「已经坚持到第 3 天，你已经超过 50% 的人啦！今晚 8 点直播答疑，打卡满 5 天解锁加餐资料～」）→ 互动设计（发起「晒今日学习笔记」接龙、连打卡 5 天抽奖）→ 一对一私聊未打卡学员的督学话术 → 使用时机（晚间 7-8 点）→ 注意（多鼓励少施压、给具体小目标）。

## 注意事项
- 以陪伴与价值感驱动，少用命令式与焦虑营销
- 转化要在持续提供价值后自然铺垫，避免硬广刷屏
- 及时响应学员问题，营造被重视的氛围
- 控制发群频率，避免打扰导致屏蔽退群
- 负面情绪私下处理，不在群内公开争执
