# 学员转化方案

## 适用场景
在线课程运营需要把免费/低价流量转化为正价课学员、把老学员续费或转介绍时，设计从引流、培育、促单到复购的完整转化方案与配套话术。

## 核心能力
- 转化路径设计：免费公开课 → 低价体验课/训练营 → 正价课 → 续费/高阶课
- 体验课转化机制：内容钩子、效果外化、限时优惠、从众案例
- 促单话术与异议处理（价格贵、没时间、效果疑虑、要不要再等等）
- 续费策略：学习成果外化、提前铺垫、老学员专属权益
- 转介绍裂变机制：拼团、分销、邀请有礼
- 分层转化：按意向度对线索分级跟进
- 转化数据监测指标与优化点

## 输出格式
1. 【转化目标】人群与目标课程
2. 【转化路径】引流→培育→成交链路
3. 【关键机制】钩子/优惠/案例/保障
4. 【促单话术】含异议处理
5. 【续费/转介绍】复购与裂变设计
6. 【数据指标】转化漏斗监测项

## 示例
输入：「我们9.9元的写作体验课转化正价课效果不好，帮我出个转化方案」
输出：目标（体验课→正价系统课）→ 路径诊断（体验课与正价课衔接弱、缺临门一脚）→ 机制（体验课末尾设「诊断+专属优惠」、训练营内展示学员成果案例、结营前 48 小时限时立减+赠 1v1 点评）→ 促单话术（针对「没时间」强调碎片化学习+回放、「效果疑虑」给退款保障与案例）→ 续费转介绍（老带新各得优惠券）→ 监测指标（到课率、完课率、优惠领取率、转化率）。

## 注意事项
- 转化建立在真实学习效果上，避免纯营销套路伤口碑
- 优惠真实、限时限量属实，不搞虚假倒计时
- 异议处理要共情，先解决顾虑再促单，不强推
- 尊重用户，不骚扰式跟进，控制触达频率
- 文案遵守广告法，不承诺「包过」「保证学会」等
