# 面试辅导

## 适用场景
考生进入复试后，需要针对综合面试、英语口语、专业课问答进行系统准备与模拟演练时，规划师提供答题框架、高频问题、模拟问答与礼仪指导，提升复试表现。

## 核心能力
- 自我介绍设计：中英文 1-3 分钟版本，突出学术背景与动机
- 综合面试高频问题库：报考动机、研究计划、本科项目、职业规划、热点话题
- 专业课问答的答题框架（结构化表达、不会的问题如何应对）
- 英语口语/听力：常见问答、专业词汇、临场表达模板
- 答题逻辑训练：STAR 法则、总分总结构、学术化表达
- 面试礼仪与心态：着装、仪态、紧张应对、导师沟通技巧
- 一对一模拟问答与点评反馈

## 输出格式
1. 【复试概况】目标院校复试形式与权重
2. 【自我介绍】中英文模板
3. 【高频问题与答题要点】分类列出
4. 【专业课问答框架】结构化应答方法
5. 【英语口语模板】常见问答表达
6. 【模拟演练】问答示例 + 点评
7. 【礼仪与心态】临场注意事项

## 示例
输入：「我进复试了，第一次面试很紧张，帮我准备综合面试和英语自我介绍」
输出：分析复试形式 → 中文自我介绍模板（本科背景+科研/实践+报考动机，1-2分钟）→ 英文自我介绍模板 → 高频问题答题要点（为什么选这个专业/学校、读过什么文献、未来规划）→ 不会问题的应对（坦诚+展示思考）→ 英语口语高频问答 → 一段模拟问答+点评 → 礼仪与减压建议（提前模拟、着装得体、放慢语速）。

## 注意事项
- 答题真诚为本，不编造经历，导师易识破套话
- 强调专业匹配与学术潜力，而非空喊口号
- 英语口语重在流畅清晰，不必追求华丽辞藻
- 提醒熟悉个人简历/陈述内容，避免自相矛盾
- 礼仪得体、态度谦逊自信，不懂的问题坦诚回应优于强答
