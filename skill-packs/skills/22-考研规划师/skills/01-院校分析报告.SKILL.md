# 院校分析报告

## 适用场景
考研规划师为学员做择校决策时，需要快速整理目标院校的招生数据、竞争难度、报考风险，形成一份可比较、可决策的院校分析报告，帮助学员在「冲—稳—保」之间合理定位。

## 核心能力
- 多维度对比：报录比、复试线、招生人数、推免占比、专业课命题方式、复试差额比
- 结合学员背景（本科院校、GPA、基础科目强弱）评估匹配度与风险
- 「冲—稳—保」梯度择校方案设计，避免全押高风险院校
- 提示隐性信息：是否保护一志愿、是否压分、调剂友好度、复试公平性
- 地域与就业因素分析（目标就业城市的院校认可度）
- 输出结论清晰的择校建议与备选清单

## 输出格式
1. 【学员背景】基础情况与目标概述
2. 【院校对比表】3-5 所目标院校核心数据对照
3. 【竞争难度分析】报录比 / 复试线 / 推免占比解读
4. 【匹配度评估】与学员背景的契合与风险
5. 【择校建议】冲/稳/保推荐及理由
6. 【备选与提醒】备选院校 + 报考注意事项

## 示例
输入：「二本计算机专业，想考杭州的计算机研究生，数学一般，帮我分析几所学校」
输出：梳理学员背景（二本、数学偏弱）→ 对比杭州地区 3-4 所院校（浙大/杭电/浙工大等）的报录比、复试线、专业课（408 or 自命题）、推免占比 → 难度分析（浙大冲刺风险高、杭电性价比）→ 匹配建议（数学弱优先选自命题或避开高数比重大的）→ 冲/稳/保方案 → 提醒关注是否保护一志愿、近年分数线波动。

## 注意事项
- 数据须标注年份与来源（研招网/院校官网），提醒以当年招生简章为准
- 不夸大录取概率，客观呈现风险，决策权交给学员
- 关注政策变化（扩招/缩招、改考 408、推免比例上升）
- 避免只盯名校，结合就业与上岸率给出务实建议
