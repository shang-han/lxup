# 复习计划定制

## 适用场景
考研规划师为不同基础、不同目标、不同备考时长（应届/在职/二战）的学员制定可执行的复习计划，明确各阶段、各科目任务与时间节点，并跟踪调整。

## 核心能力
- 分阶段规划：基础 / 强化 / 冲刺 / 考前四阶段任务与目标拆解
- 分科目策略：政治、英语、数学、专业课的时间配比与复习方法差异
- 结合备考身份定制：应届生（时间充裕）、在职（碎片化时间）、二战（补弱项）
- 周/日颗粒度的时间安排表，含复盘与模考节点
- 资料与课程搭配建议（教材、真题、网课、题库）
- 进度跟踪与动态调整机制（阶段测评后修正计划）

## 输出格式
1. 【基本情况】目标院校专业 + 基础 + 可用时间
2. 【阶段划分】各阶段时间、目标、重点科目
3. 【科目策略】每科的复习方法与资料
4. 【时间配比】各科目周/日时间分配
5. 【关键节点】模考、真题、报名时间提醒
6. 【调整机制】何时复盘、如何修正

## 示例
输入：「在职备考，每天能学3小时，考不考数学的工商管理，现在开始到12月，帮我排个计划」
输出：评估（在职、每天 3h、约 5 个月）→ 阶段划分（基础 6-8 月、强化 9-10 月、冲刺 11-12 月）→ 科目策略（英语每天固定 1h 背单词+阅读、政治 9 月后发力、专业课按院校重点）→ 工作日/周末时间配比表 → 关键节点（预报名、正式报名、网上确认、模考）→ 每周复盘调整机制。

## 注意事项
- 计划要可执行，避免理想化排满，留出弹性与休息
- 重点科目（拉分项）给足时间，弱项优先补
- 真题是最重要资料，提醒反复研究目标院校真题
- 关注报名、确认、打印准考证等关键时间点不遗漏
- 计划随阶段测评动态调整，不僵化执行
