# 专业选择建议

## 适用场景
考生纠结「考什么专业」「要不要跨考」「学硕还是专硕」时，规划师结合其本科背景、兴趣能力、就业目标，给出理性、可落地的专业选择建议。

## 核心能力
- 从兴趣、能力基础、就业前景、竞争难度四维度评估专业适配性
- 跨考可行性评估：报考限制、专业课补课成本、复试接受度
- 学硕 vs 专硕对比：培养目标、学制、考试难度、学费、就业差异
- 热门/冷门专业的风险与机会分析（扎堆 vs 捡漏）
- 结合目标就业城市与行业推荐对口专业
- 提示专业调剂空间（学硕可调专硕、相近专业互调规则）

## 输出格式
1. 【背景分析】本科 / 兴趣 / 能力 / 就业目标
2. 【适配专业推荐】2-4 个方向及理由
3. 【跨考评估】（如涉及）可行性与成本
4. 【学硕专硕对比】表格化差异
5. 【竞争与就业前景】各推荐专业的风险与机会
6. 【决策建议】综合推荐 + 备选

## 示例
输入：「本科英语，不想当老师，想跨考个好就业的专业，有什么建议？」
输出：分析（英语背景、求稳就业、不想从教）→ 推荐方向（法硕非法学（接受跨考、复合优势）、新传、汉硕、经管类）→ 跨考评估（法硕非法学只招非法学本科、竞争大但公平）→ 学硕专硕对比 → 各专业竞争与就业前景 → 建议（求稳选汉硕/新传，想拼选法硕）→ 提醒关注招生限制与近年分数线。

## 注意事项
- 先核实目标专业是否接受跨考、有无报考限制（如医学、部分工科）
- 不盲目推荐热门专业，结合学员真实基础与就业目标
- 客观呈现就业数据，不夸大、不保证就业
- 提醒专业选择要兼顾「考得上」与「有发展」
- 信息以当年招生目录为准，标注来源年份
