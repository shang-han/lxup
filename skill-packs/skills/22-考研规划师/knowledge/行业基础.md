# 行业基础

## 行业概况
全国考研报名人数超 450 万，整体录取率约 30%，竞争激烈且信息高度不对称。考试科目一般为政治 + 外语 + 专业课（部分专业考数学），初试后还有复试（笔试+面试）。规划师的核心价值在于「消除信息差」：专业选择、院校选择、复习规划、调剂时机四大环节。服务贯穿咨询评估、备考跟踪、考前冲刺、复试调剂全周期，口碑与上岸率是机构核心竞争力。

## 常用术语
- 报录比：报名人数 / 录取人数，衡量竞争激烈程度的核心指标
- 复试线：进入复试的最低分数线（院校/专业各自划定）
- 国家线：教育部划定的进入复试/调剂的最低资格线（分 A/B 区）
- 自划线：34 所自主划线高校自行划定复试线，通常高于国家线
- 调剂：第一志愿未录取或复试被刷，通过调剂系统申请其他院校
- 学硕/专硕：学术型硕士（偏研究）/ 专业型硕士（偏应用）
- 统考/自命题：专业课全国统一命题 vs 招生院校自主命题
- 等额/差额复试：复试人数等于 / 多于招生计划数

## 核心工作流程
学员咨询：背景评估（本科/成绩/基础）→ 目标定位 → 专业建议 → 院校推荐
备考阶段：复习规划 → 资料推荐 → 进度跟踪 → 阶段测评 → 心理疏导
考前冲刺：模拟测试 → 查漏补缺 → 应试技巧 → 心态调整
考后阶段：成绩查询 → 复试/调剂指导 → 录取确认

## 常见问题
Q：考不上怎么办？
A：提前规划「冲—稳—保」梯度，备考同时关注调剂信息，必要时准备就业/二战/出国多路径备选。

Q：跨专业考研难度如何？
A：专业课需额外投入时间，部分专业（医学、法硕非法学等）有报考限制；但复试时复合背景可能是加分项，关键看目标专业是否接受跨考。

Q：择校主要看什么？
A：综合报录比、复试线、招生人数、专业课难度、地域与就业、自身基础六项，避免盲目冲名校或扎堆热门院校。

Q：复习节奏怎么安排？
A：一般分基础（3-6月）、强化（7-9月）、冲刺（10-12月）三阶段，政治后期发力、英语数学长期积累、专业课按院校命题重点推进。
