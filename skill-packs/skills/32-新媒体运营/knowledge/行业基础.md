# 行业基础

## 行业概况
新媒体覆盖公众号、抖音、小红书、微博、B 站、快手、视频号等平台，内容形式包括图文、短视频、直播。核心指标：粉丝量 + 互动率 + 转化率。行业竞争白热化，内容同质化严重，平台算法迭代快（推荐机制、流量扶持方向频繁调整）。运营核心能力：选题策划 + 内容生产 + 数据分析 + 用户运营。变现路径：广告商单（星图/蒲公英）、电商带货、知识付费、私域导流。AIGC 工具大幅降低生产门槛后，运营价值向判断力、创意与用户洞察转移。

## 常用术语
- 粉丝数：关注账号的用户数，基础规模指标
- 阅读量/播放量：内容被浏览/播放次数，衡量曝光
- 互动率：点赞/评论/收藏/转发占阅读量的比例，衡量内容质量与用户粘性
- 收藏率：收藏量占阅读量比例，小红书与干货型内容的核心权重指标
- 涨粉率：新增粉丝/发布内容数（或周期），衡量涨粉效率
- 转化率：内容引导用户完成目标动作（关注/购买/报名）的比例
- 完播率：视频被完整观看的比例，决定能否进入下一级流量池的核心因素
- 私域流量：品牌/个人可自主反复触达的用户资产（微信好友/社群），与平台公域相对

## 核心工作流程
每日：选题策划 → 内容制作（脚本/拍摄/剪辑/排版）→ 发布（时间/标题/标签）→ 互动回复与评论区运营
每周：数据复盘（内容表现排名）→ 竞品分析 → 下周内容计划
每月：粉丝增长与画像分析 → 变现路径复盘 → 下月运营规划与资源申请

## 常见问题
Q：涨粉慢怎么办？
A：先查定位清晰度（用户凭什么关注你）、内容垂直度与爆款率；用追热点换流量、差异化内容做留存、多平台分发扩开口、互推与连麦借势撬动流量。

Q：怎么做爆款？
A：爆款 = 强开头（3 秒钩子）+ 高信息密度或情绪共鸣 + 互动引导（评论/收藏钩子）。爆款不可复制但结构可以：拆解同赛道爆款，提炼可复用的选题角度与叙事结构。

Q：内容不错但数据差？
A：数据由内容与分发共同决定：检查标题封面点击率、发布时间、标签人群是否精准、初始流量池完播率是否达标，再决定是改内容还是改运营动作。
