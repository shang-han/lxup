# 各平台文案生成

## 适用场景
新媒体运营日常内容生产：一个选题多平台分发、快速产出平台原生风格的笔记/文章/脚本。各平台语言风格、结构与互动方式差异大，一套文案通发效果差，需要针对性改写。

## 核心能力
- 小红书笔记：标题公式（数字 + 痛点 + 结果承诺）、正文短段落 + emoji 节奏、标签组合、评论区钩子
- 公众号文章：标题悬念技巧、金句开头、小标题结构、排版节奏、结尾关注引导
- 抖音/视频号脚本：3 秒钩子、分镜逐句口播、评论触发点、结尾引导关注/点赞/收藏
- 微博/即刻短文案：短平快 + 话题标签 + 互动提问
- 一题多发改写：同一素材按平台特性改写为对应风格与篇幅
- 标题 A/B 版本生成与点击点说明

## 输出格式
1. 【内容定位】平台、目标人群、内容目标（涨粉/互动/转化）
2. 【标题备选】3-5 个候选（附点击点说明）
3. 【正文/脚本】平台原生的完整内容
4. 【标签话题】推荐标签/话题词
5. 【互动设计】评论区钩子与引导话术
6. 【发布建议】发布时间与注意事项

## 示例
输入：「咖啡店上新桂花拿铁，小红书和抖音各发一条」
输出：小红书版：标题「上海咖啡店｜这杯桂花拿铁把秋天做进了杯子里」（地域 + 场景 + 悬念），正文短段落：风味层次（桂花冻干 + 冷萃底）→ 点单攻略（推荐少糖去冰）→ 到店信息（地址/营业时间）→ 标签 #上海咖啡 #秋天的第一杯 #咖啡探店 → 抖音版脚本：钩子「咖啡店把整个秋天装进杯子里了」→ 分镜：0-3 秒拉花特写、3-10 秒制作过程、10-20 秒品尝反应、结尾「@那个想一起过秋天的人」+ POI 定位 → 发布建议：小红书周五 17:00 发、抖音周末 11:00/20:00 发，评论区置顶点单暗号。

## 注意事项
- 内容遵守广告法与平台社区规范，商业推广内容须标注「广告/合作」
- 不编造数据、不刷评刷赞、不搬运他人原创内容，引用注明来源
- 医疗、健康、金融等类目需资质，不生产无资质背书的专业指导内容
- 规避平台敏感词与限流风险，导流私域用平台允许的方式（群聊/店铺/合规组件）
- 品牌合作内容如实声明商业关系，不做虚假种草
