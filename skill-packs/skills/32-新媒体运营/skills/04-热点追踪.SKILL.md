# 热点追踪

## 适用场景
运营借热点获取流量：突发热搜、爆梗、节日营销节点、行业大事件。需要快速判断「能不能蹭、怎么蹭」，并在时效窗口内产出与账号定位结合的借势内容。

## 核心能力
- 热点借势价值判断：账号相关性 × 话题热度 × 借势难度 × 时效性，四维快速打分
- 关联角度挖掘：找到热点与账号领域的结合点（硬蹭掉粉，软结合显功力）
- 快速内容框架：热点背景一句话引入 → 与账号领域的结合点 → 本账号的价值输出 → 互动钩子
- 热点标题公式：热词 + 悬念 + 账号价值点
- 时效判断：突发热点窗口通常 24-72 小时，节日/可预期热点提前预制
- 借势风险排查：负面热点、争议事件、明星塌房不蹭；政治敏感、灾难事故严禁营销借势

## 输出格式
1. 【热点概况】话题背景、热度、主要平台
2. 【借势判断】相关性打分与蹭/不蹭结论
3. 【结合角度】2-3 个热点与账号的结合点
4. 【内容方案】标题 + 正文框架/脚本
5. 【发布策略】平台、时间、标签
6. 【风险检查】敏感点与规避建议

## 示例
输入：「秋天第一杯奶茶上热搜了，我们账号做职场干货，能蹭吗？」
输出：热点：#秋天第一杯奶茶# 微博阅读超 10 亿、抖音热榜，社交属性强 → 判断：相关性中等（可结合职场场景），热度高、难度低、窗口内 → 角度：①「职场人的秋天第一杯奶茶点单攻略」（省钱 + 低糖实用向）②「同事都在点奶茶，如何优雅拒绝/拼单」（职场社交向）→ 方案：小红书标题「秋天第一杯奶茶，职场人怎么喝不长胖」，正文：低卡点单公式 + 咖啡因时段建议 + 办公室拼单话术 → 发布：当天 15:00-17:00 点单高峰前，标签 #秋天第一杯奶茶 #职场日常 → 风险：不贬低任何品牌、健康建议标注个体差异、不编造「奶茶成分黑幕」类不实信息。

## 注意事项
- 负面热点（事故、灾难、社会恶性事件）绝不借势营销
- 不传播未经证实的热点信息，引用资讯注明来源
- 涉明星/IP 注意肖像权与版权，不擅自使用他人素材
- 保持账号调性，不为流量硬蹭无关热点损伤定位
- 节日与敏感纪念日营销注意文化习俗与情感分寸，避免消费苦难
