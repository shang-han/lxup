# 选题规划

## 适用场景
运营做周/月度内容计划，或新账号冷启动：解决「发什么、什么时候发」。围绕账号定位建立可持续输出的选题系统，避免灵感式更新与内容漂移。

## 核心能力
- 从账号定位推导选题方向：3-5 个内容支柱（如职场号 = 干货技能 + 行业观察 + 个人成长 + 互动话题）
- 选题挖掘渠道：平台热榜、评论区提问、竞品爆款、搜索下拉词、用户私信、行业日历（节日/大促/行业事件）
- 内容系列化设计：把选题做成可持续更新的系列（编号/主题），提升粉丝追更期待
- 内容日历排布：按平台特性安排更新频率、时段与内容类型配比（干货/互动/带货）
- 选题优先级评估：按搜索需求/共鸣度/生产成本/转化价值四维打分
- 输出选题库结构：主题/角度/标题草稿/素材来源/状态

## 输出格式
1. 【账号定位】目标人群与核心价值主张
2. 【内容支柱】3-5 个选题方向与配比
3. 【选题库】具体选题清单（含角度与标题草稿）
4. 【系列规划】系列主题与更新计划
5. 【内容日历】周/月排期表
6. 【评估标准】选题打分与优先规则

## 示例
输入：「做家居收纳账号，主发小红书，每周 4 更，出一个月选题计划」
输出：定位：帮租房/小户型人群解决收纳痛点 → 支柱：收纳干货技巧（40%）+ 前后对比案例（25%）+ 好物清单（20%）+ 互动答疑（15%）→ 选题库 20 条：「6㎡ 小卧室收纳逆改」「300 元预算出租屋收纳改造」「买了后悔的收纳神器排雷」「换季衣柜断舍离清单」…… → 系列：「小户型逆袭日记」每周 1 更、「本月收纳好物」每月 1 期 → 日历：周一干货、周三对比、周五好物、周日互动 → 评估：对比案例型优先（易爆 + 可引流好物），带货选题绑定店铺商品。

## 注意事项
- 选题须围绕账号定位，不为追无关热点破坏垂直度
- 涉及对比、测评、推荐的内容须有事实依据，不贬低同行与竞品
- 医疗、金融、法律等门槛话题谨慎，无相应资质不生产指导性内容
- 选题库保持 1.5-2 倍更新量的储备，应对突发断更
- 定期用数据淘汰持续低产方向，滚动迭代选题库
