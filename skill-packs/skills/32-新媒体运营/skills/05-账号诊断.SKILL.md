# 账号诊断

## 适用场景
运营给账号做体检：涨粉停滞、互动率低、变现无转化、定位混乱。适用于自查，或接手新账号/代运营客户账号时系统定位问题、排出改进优先级。

## 核心能力
- 定位诊断：账号「为谁提供什么价值」是否清晰，主页信息（昵称/简介/头像/背景图）能否一句话说清关注理由
- 内容垂直度检查：选题分布是否集中、近期有无跑题内容导致粉丝画像混乱
- 数据健康度检查：阅读/互动率对标同量级账号基准线，完播率/收藏率等平台核心指标
- 增长瓶颈拆解：涨粉停滞/互动低定位到具体原因（内容？发布？定位？限流？）
- 变现路径检查：粉丝到转化的链路是否通畅（简介引导、评论区、店铺/私域入口）
- 输出分级改进方案：立即项（当天可改）与中期项（内容策略调整）

## 输出格式
1. 【账号现状】粉丝量、更新频率、近期数据概览
2. 【定位诊断】定位清晰度与主页信息问题
3. 【内容诊断】垂直度、选题类型表现差异
4. 【数据诊断】对标同量级基准，瓶颈指标
5. 【问题清单】按影响优先级排列
6. 【改进方案】立即项 + 中期项

## 示例
输入：「抖音职场号 5 万粉，近 2 个月涨粉停滞，视频平均 3-5 千播放，帮我诊断」
输出：现状：更新频率 3 条/周正常，但均播从 2 万跌至 3-5 千 → 定位：简介写「分享职场日常」过于模糊，无关注理由 → 内容：近期选题混杂（剧情 + 干货 + 蹭热点），垂直度下降；完播率仅 12%（同量级基准 20%+），前 3 秒流失 60% → 数据：播放卡在 500 初始流量池，完播与互动未达标未进下一级流量池 → 问题排序：①开头钩子弱（3 秒流失 60%）②定位不聚焦 ③简介无价值承诺 → 立即项：简介改为「每周 3 个职场硬技能，给 1-3 年新人」，停发跑题蹭热点内容；中期项：选题统一为「1 分钟职场技能」系列，开头改痛点提问式钩子，每条视频埋一个评论争议点。

## 注意事项
- 诊断基于对方提供的数据与内容，不凭空猜测；数据缺失项明确标注
- 不轻易归因「被限流」，先排除内容与定位问题（真限流是小概率事件，先看是否违规）
- 改进建议须可执行可验证，避免「提升内容质量」类空话
- 涉医疗、金融、教育等强监管领域的账号，先检查资质合规问题
- 不承诺「诊断后 X 天涨粉 X 万」，改进效果需要观察周期
