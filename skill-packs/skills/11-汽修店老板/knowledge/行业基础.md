# 行业基础

## 行业概况
汽车后市场规模超 1.5 万亿，涵盖维修保养、美容改装、钣金喷漆、轮胎等业务，全国维修门店超 60 万家，竞争激烈。利润来源：工时费 + 配件差价，保养是高频引流业务，维修是利润业务。行业信任痛点突出，「透明报价、以诚待客」是中小门店对抗 4S 店与连锁品牌的最大武器。新能源车保有量快速增长，三电维修能力成为新门槛。客户留存靠专业 + 诚信 + 档案化服务。

## 常用术语
- 进场台次：每日/每月进店维修的车辆数
- 客单价：平均每车维修消费金额
- 毛利率：配件与工时的综合毛利水平
- 返修率：同一问题短期内再次返修的比例，是质量红线指标
- 客户留存率：再次到店消费的客户比例
- 工时定额：完成某维修项目的标准工时数，报价基础
- 三滤一油：机油、机滤、空滤、空调滤，常规小保养项目

## 核心工作流程
每日：接车登记 → 环车检查 → 故障诊断 → 报价确认 → 维修施工 → 三级质检 → 交车结算 → 三日内回访
每周：配件库存盘点 → 设备工具维护 → 经营数据分析（台次、客单价、毛利）
每月：客户回访与保养提醒 → 技师培训 → 供应商对账与结算

## 常见问题
Q：客户觉得报价贵怎么办？
A：报价单拆成工时 + 配件明细逐项讲解，提供原厂件/品牌件不同方案对比，主动对标 4S 店价格，让客户看到「同样的件省了多少」，而非一味降价。

Q：维修质量怎么保证？
A：三级质检（技师自检 → 组长复检 → 店长抽检），返修免费并追溯责任，建立客户车辆档案跟踪维修历史。

Q：客户不信任怕被坑？
A：旧件展示返还、维修过程可观看、报价先确认后施工、增项必须电话征得同意，用透明流程建立信任。

Q：保养客户流失到 4S 店或线上？
A：做好保养档案与到期提醒，提供比 4S 店性价比更高的正品保养套餐，叠加免费检测、洗车等增值服务提升黏性。
