# 维修报价方案

## 适用场景
汽修店为车主出具维修报价：常规保养报价、故障维修报价、事故维修报价，需要明细清晰、多方案可选、附带沟通话术，解决车主「怕被坑」的顾虑。

## 核心能力
- 报价单结构：车辆信息 / 故障描述 / 维修项目 / 工时费 / 配件明细（名称、品牌、单价、数量）/ 合计 / 质保承诺
- 多档方案设计：同一项目给原厂件 / 品牌件 / 再制造件方案，标注差异与适用场景，由车主选择
- 价格对标：与 4S 店同类项目价格对比呈现，突出性价比
- 增项管理话术：拆检后发现新问题时的电话请示话术，先报价后施工
- 报价沟通话术：逐项讲解逻辑，把专业术语翻译成车主听得懂的话
- 套餐化报价：保养套餐打包价（工时 + 配件 + 增值服务）

## 输出格式
1. 【车辆与故障信息】
2. 【报价明细表】项目 / 工时 / 配件 / 单价 / 小计
3. 【方案对比】（如适用）原厂 vs 品牌件方案差异
4. 【质保说明】质保里程/时长
5. 【沟通话术】讲解与促成话术

## 示例
输入：「卡罗拉 6 万公里大保养，帮我出个报价方案」
输出：明细：机油机滤（全合成机油 + 原厂机滤，工时 1h）、空滤、空调滤、火花塞 4 支、变速箱油重力更换、刹车油、节气门清洗 → 合计拆为配件 xxx + 工时 xxx → 方案对比：机油原厂 vs 品牌全合成两档，差价 xx 元 → 对标 4S 店同项约贵 40% → 话术：「6 万公里这次是中期大保养，火花塞和变速箱油不换的话后面油耗和顿挫都会上来，这两项建议这次一起做，其他按检查结果来……」→ 质保：维修项目质保 3 个月/5000 公里。

## 注意事项
- 先报价、确认后施工，增项必须经车主同意（电话/微信留痕），严禁先斩后奏
- 配件品牌、产地如实标注，不以品牌件冒充原厂件，旧件主动展示或返还
- 安全相关项目（刹车、转向、轮胎）报价时说明风险后果，不夸大也不淡化
- 质保条款写进结算单，口头承诺同步书面化
