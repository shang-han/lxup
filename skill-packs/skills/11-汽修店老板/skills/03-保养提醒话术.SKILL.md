# 保养提醒话术

## 适用场景
汽修店基于车辆档案主动提醒客户：机油保养到期、轮胎/电瓶更换周期、年检到期、保险续保、季节性养护（换季空调、冬季电瓶），并做维修后回访。

## 核心能力
- 按触发源生成提醒：按里程（5000/10000 公里）、按时间（6 个月）、按项目周期（轮胎 3 年、电瓶 3-4 年）
- 多渠道版本：短信版（70 字内）、微信版（口语 + 权益）、电话版（含应对脚本）
- 提醒内容结构：称呼 + 车辆信息 + 提醒事由 + 风险说明（轻量）+ 到店权益 + 预约方式
- 维修后回访话术：3 日回访确认用车感受，顺带告知注意事项
- 季节性批量提醒：换季空调清洗、入冬前电瓶检测等群发文案
- 语气分寸：提醒而非骚扰，突出「帮你记着」的服务感，不制造焦虑

## 输出格式
1. 【提醒类型与对象】
2. 【话术正文】短信 / 微信 / 电话版本
3. 【权益钩子】到店附带的检测或优惠
4. 【跟进策略】未响应的二次触达时间与方式

## 示例
输入：「张先生上次保养是半年前，该提醒了」
输出：微信版：「张哥好，您那台轩逸上次保养是去年 12 月，按半年周期该换机油啦～ 全合成机油超期不换容易积碳。这周来保养送全车 20 项检测 + 免费洗车，周末档期紧张，我给您留个工位？」→ 短信版：「【XX 汽修】张先生，您的轩逸保养已到期，本周到店赠免费检测+洗车，预约回 T」→ 电话应对：没时间→「保养 40 分钟，可以早上送来下班开走」→ 二次触达：一周后发季节性提醒顺带再提。

## 注意事项
- 提醒必须基于真实维修保养记录，不虚构「系统显示需要更换」
- 触达频率适度：同一客户同一事项提醒不超过 2 次，客户明确拒绝后停止
- 客户手机号、车牌、维修记录属个人隐私，仅限本店服务使用，严禁外泄或转卖
- 回访中客户反馈维修问题立即转售后处理，不拖延不推诿
