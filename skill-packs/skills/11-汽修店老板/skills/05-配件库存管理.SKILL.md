# 配件库存管理

## 适用场景
汽修店管理配件库存：确定常备件范围与安全库存、建立盘点与出入库制度、处理呆滞件、评估与开发供应商，平衡「不压货」与「不误工」。

## 核心能力
- 常备件清单设计：按周转频率分级（高频：机油、三滤、常用灯泡；中频：刹车片、电瓶、雨刮；低频：按订单采购）
- 安全库存模型：按历史月消耗 × 供货周期设定补货点与最高库存
- 出入库与盘点制度：领用登记、以旧换新核销、月度盘点差异处理
- 呆滞件处理：车型淘汰件促销、供应商退换、同行调剂
- 供应商管理：多渠道比价（原厂/品牌/拆车件）、账期谈判、质量追溯与退换货机制
- 库存资金分析：库存周转天数、资金占用、缺货误单统计

## 输出格式
- 清单类：配件分级清单表（品名 / 规格 / 安全库存 / 补货点 / 供应商）
- 制度类：出入库与盘点流程编号步骤
- 分析类：库存指标与改进建议

## 示例
输入：「帮我整理一份门店常用配件库存管理方案」
输出：常备件清单：高频件（0W-20/5W-30 机油各规格、各车型机滤空滤、常用型号电瓶、雨刮、灯泡）设最低库存触发补货；中频件（主流车型前后刹车片、刹车油、防冻液、火花塞）按 1-2 套备货；低频件下单即采（约定供应商 2 小时送达）→ 制度：领料凭工单、旧件回收登记、每月 25 日盘点、差异超 2% 查原因 → 供应商：每类件保持 2 家供应商比价，签订质量追溯与退换协议 → 呆滞件：超 180 天未动件季度清理（促销/退换）。

## 注意事项
- 配件来源渠道可追溯，进货留存凭证，坚决不以副厂件/拆车件冒充原厂件销售
- 安全件（刹车、转向、轮胎相关）只采购正规渠道产品，杜绝三无配件
- 库存记录与实物必须一致，账实不符及时核查，防止跑冒滴漏
- 召回批次配件立即停用并通知已安装客户，履行告知义务
