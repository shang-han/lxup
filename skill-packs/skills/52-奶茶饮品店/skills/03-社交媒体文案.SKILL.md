# 社交媒体文案

## 适用场景
新品上市要造势；门店想做小红书/抖音同城种草；朋友圈日常营业与活动预告；节日限定款推广；打造网红打卡点吸引年轻人到店。

## 核心能力
- 按平台调性产出文案：小红书种草体、朋友圈短文案、抖音口播脚本
- 提炼产品卖点与「出片点」（颜值、季节限定、隐藏喝法）
- 设计标题钩子与话题标签，提升同城曝光
- 结合年轻客群语境（网感词、emoji、玩梗）但不低俗
- 生成到店引流钩子（暗号领小料、打卡返现）
- 规划一周/一月的内容选题日历

## 输出格式
1. 【目标与人群】推广目的、面向客群
2. 【核心卖点】3 个记忆点
3. 【文案版本】小红书版 + 朋友圈版 + 抖音口播版
4. 【标题与话题标签】
5. 【引流钩子】到店转化设计
6. 【配图/拍摄建议】

## 示例
输入：「夏天上了一款杨梅冰茶，想发小红书引流到店」
输出：目标（夏季限定引流年轻女性）→ 卖点（手剥鲜杨梅、冰爽解腻、紫红渐变出片）→ 小红书版：「姐妹们！这杯杨梅冰茶我能喝一整个夏天🥹 手剥鲜杨梅+冷萃绿茶，紫红渐变绝绝子，无糖也好喝～」→ 朋友圈版（短）→ 抖音口播脚本 → 标题「杨梅季限定｜XX 路这杯被挤爆」+ #同城探店 #夏日饮品 → 钩子（到店说暗号加杨梅）→ 拍摄建议（逆光拍渐变分层）。

## 注意事项
- 文案不得含「最」「第一」「全网首家」等《广告法》禁用词
- 宣传用料须真实，标注的鲜果、鲜奶不得用粉末/植脂末冒充
- 涉及功效表述须谨慎，不得宣称减肥、美白、养生疗效
- 引流活动规则要写清有效期与限制，避免到店纠纷
