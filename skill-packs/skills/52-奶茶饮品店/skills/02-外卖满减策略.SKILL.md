# 外卖满减策略

## 适用场景
外卖单量上不去、曝光低；想设计满减/折扣/新客券组合；要提升客单价又不亏毛利；新店冷启动冲单量；节假日大促活动配置。

## 核心能力
- 根据客单价与毛利率反推满减门槛与让利幅度（避免「赔本赚吆喝」）
- 设计阶梯满减（如满 25 减 8、满 45 减 15）引导凑单提客单
- 规划引流款 / 利润款 / 凑单款的菜单结构与定价锚点
- 搭配新客立减、折扣商品、会员券形成组合拳
- 测算活动前后单均成本、平台扣点、实际到手毛利
- 给出活动上线节奏与复盘指标（曝光、进店率、下单转化率、复购）

## 输出格式
1. 【现状诊断】客单价 / 毛利率 / 平台扣点
2. 【满减方案】阶梯门槛与让利，含设计逻辑
3. 【菜单结构】引流 / 利润 / 凑单款分工
4. 【组合优惠】新客券 + 折扣 + 满减搭配
5. 【毛利测算】活动前后单均利润对比
6. 【上线节奏与复盘指标】

## 示例
输入：「客单价 22 元，毛利 65%，美团扣点 20%，想做满减冲单量」
输出：诊断（客单偏低需凑单）→ 阶梯满减（满 25 减 6、满 40 减 12，引导第二杯）→ 菜单结构（9.9 元引流柠檬茶 + 16 元利润拿铁 + 3 元小料凑单）→ 组合（新客立减 5 + 第二杯半价）→ 测算（满 25 减 6 后实收 19 元、扣点后到手 15.2 元、原料 7.7 元、净毛利仍正）→ 节奏（先上 3 天测转化再放量）+ 复盘指标。

## 注意事项
- 满减让利须先扣平台佣金再算成本，防止到手价低于原料成本
- 标价与活动价须真实，避免「先涨后减」虚假促销被平台处罚
- 外卖饮品须用防洒封口与保温/保冰措施，减少差评与客诉
- 出品仍须遵守食品安全，外卖与堂食用料标准一致、索证索票齐全
