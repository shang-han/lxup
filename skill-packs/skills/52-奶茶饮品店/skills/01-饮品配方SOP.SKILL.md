# 饮品配方SOP

## 适用场景
老板研发出新品后要把「手感配方」固化成全员可复制的标准；新员工培训需要出杯手册；连锁多店要保证口味一致；调整甜度/冰量/杯型时要有统一基准。

## 核心能力
- 把手工配方拆解为「原料 + 克重 + 顺序 + 时间 + 温度」的标准参数
- 生成不同杯型（中杯/大杯）与甜度冰量（全糖/七分/无糖、少冰/去冰）的换算表
- 核算单杯原料成本与建议售价、毛利率
- 输出关键控制点（茶汤萃取时间、奶精/果糖添加顺序、封口前摇匀次数）
- 给出可量产的预制备料方案（茶汤、小料分时段煮制计划）
- 标注口感校准标准（色泽、挂壁、甜感、回甘）供出品抽检

## 输出格式
1. 【饮品名称】杯型 / 甜度冰量规格
2. 【原料配比表】每杯原料、克重、品牌型号
3. 【出杯步骤】编号步骤，含时间、温度、手法
4. 【杯型甜度换算表】
5. 【成本与定价】单杯成本、建议售价、毛利率
6. 【关键控制点】影响口感的核心参数
7. 【预制备料计划】分时段备料量

## 示例
输入：「我做了一款栀子绿茶拿铁，帮我做成标准 SOP，大杯 700ml」
输出：栀子绿茶拿铁（大杯 700ml）→ 配比表（栀子绿茶汤 250ml、鲜奶 200ml、果糖 30g、冰块补满）→ 步骤（85℃ 萃取 4 分钟 → 加果糖搅匀 → 加冰 → 注入鲜奶 → 摇 8 下封口）→ 甜度冰量换算 → 成本 4.2 元 / 建议售价 16 元 / 毛利 74% → 控制点（萃取超 5 分钟会涩、鲜奶后加防分层）→ 备料计划。

## 注意事项
- 原料须符合食品安全标准，果糖、奶精等添加剂严禁超范围、超量使用
- 茶汤、鲜奶等半成品须标注制备时间与废弃时限，超时即弃
- SOP 中的克重要用电子秤量化，避免「适量」导致多店口味漂移
- 新品上市前建议小批量试卖，根据顾客反馈微调再固化
