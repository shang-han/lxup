# 行业基础

## 行业概况
新式茶饮是餐饮中增速最快、连锁化程度最高的细分赛道之一，头部品牌密集、产品迭代极快（季节限定、联名款）。门店毛利看似高（60-70%），但被租金、人工、外卖平台扣点和原料损耗摊薄。生存关键：爆款产品力 + 标准化出品 + 线上运营（外卖+社交媒体）+ 精细化成本控制。年轻客群对颜值、话题、体验敏感，复购靠产品与会员运营。

## 专业技能体系
### 硬技能
- 饮品配方SOP标准化
- 原材料采购与保存（茶叶/奶精/果糖保质期管理）
- 设备维护（封口机/制冰机/萃茶机）
- 外卖平台运营（美团/饿了么满减策略）
- 选址评估（人流量/竞品密度/租金占比）

### 软技能
- 年轻员工管理
- 社交媒体内容运营

## 常用术语
- 出杯：完成一杯饮品的制作，出杯速度决定高峰期产能
- 客单价：平均每单金额，常靠第二杯、加料、套餐拉升
- 毛利率：（售价-原料成本）/售价，茶饮原料毛利高但净利被费用摊薄
- 先进先出（FIFO）：先入库原料先使用，防止过期
- 效期：原料保质期，开封后另计启用时限
- 满减：外卖平台按订单金额阶梯让利，用于提客单与冲单量
- 曝光-进店-下单转化漏斗：外卖从展示到成交的逐层转化率
- 坪效：每平米营业额，衡量铺位利用效率

## 核心工作流程
每日：备料煮茶 → 设备点检 → 营业出杯 → 外卖接单 → 日结对账 → 效期检查
每周：原料盘点 → 补货采购 → 社交媒体内容发布 → 外卖数据复盘
每月：销售与毛利分析 → 新品研发试卖 → 租金人工成本核算 → 员工培训

## 常见问题
Q：外卖满减怎么设才不亏本？
A：先算清平台扣点和原料成本，满减让利要扣完佣金再比对成本，设阶梯门槛引导凑单提客单，保本线以下的让利坚决不做。

Q：原料老是过期浪费怎么办？
A：建效期台账分级预警，鲜奶勤进少囤，开封贴启用日期，坚持先进先出，临期原料优先用于爆款催用。

Q：怎么吸引年轻人到店？
A：做好小红书/抖音同城种草，打造高颜值出片产品和打卡点，配合暗号领料、限定款制造话题，沉淀会员做复购。

Q：选址最该看什么？
A：一看人流与客群是否匹配饮品消费，二算租金占比（不超预估营业额 15-20%），三查竞品饱和度，四确认能办食品经营许可证。
