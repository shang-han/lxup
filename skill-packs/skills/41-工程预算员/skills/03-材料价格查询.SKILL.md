# 材料价格查询

## 适用场景
预算套价、过程核价与竣工结算时需要确定材料价格；面对甲方/监理/审计对价格的质疑，需要拿出有据可查的取价依据与对比分析。

## 核心能力
- 梳理取价渠道与优先级：造价管理部门发布的信息价（造价信息）、建材市场实地询价、厂家报价、电商平台价、历史项目采购价
- 编制材料询价对比表：材料名称规格、多渠道报价、含税口径、运距与付款方式、建议取价
- 材料调差方法：信息价差额调整、合同约定风险幅度（如±5%）内承包方自担、超出部分调差计算
- 应对甲方核价/审计质询：附报价单、信息价截图与文号、询价记录，形成证据链
- 识别价格影响因素：品牌档次、采购批量、运输距离、付款条件、是否含安装
- 跟踪主材价格波动，对钢材、铜材、商砼等敏感材料提示调差时机与签证办理

## 输出格式
1. 【询价清单】材料名称、规格型号、单位、需求数量、到场时间
2. 【价格对比表】多渠道报价与含税、运费口径对比
3. 【取价建议】建议采用价格及依据
4. 【调差计算】风险幅度、调差公式与金额（如涉及）
5. 【依据附件】报价单来源、信息价文号、联系方式与有效期

## 示例
输入：「C30 商品混凝土甲方核价 480 元/m³，我们套的信息价是 505，怎么回复？」
输出：询价清单 → 对比表（当期信息价 505 元/m³〔含税到场〕、三家搅拌站报价 495-515、历史合同价 498）→ 取价建议（建议取 500，依据为信息价与三家市场报价加权平均）→ 调差计算（合同约定±5% 风险承包方承担，施工期信息价较基准期上涨 7%，超出 2% 部分按实调差）→ 附件清单（信息价文号与页码、三家报价单扫描件）。

## 注意事项
- 取价口径统一标明含税/除税、是否含运至现场、付款条件，口径不同的报价不可直接比较
- 信息价没有的材料用市场询价替代，保留不少于三家报价并留档
- 调差严格按合同约定的风险幅度与调整方法执行，及时办理价格确认单
- 价格波动大的材料分批锁定价格，避免结算时扯皮
- 甲供材料不计入施工方报价，但要核对规格与到场时间
