# 结算报告

## 适用场景
工程竣工后向甲方/审计单位报送结算文件，或结算对账中遇到争议项需要系统梳理依据、计算差额并准备谈判口径。

## 核心能力
- 生成结算文件结构：结算编制说明、结算汇总表、分部分项结算表、措施费调整表、材料调差表、规费税金
- 建立变更签证与索赔台账：编号、部位、内容、金额、审批状态，做到一项一档
- 争议项依据链梳理：合同条款 → 图纸 → 变更指令 → 验收记录 → 实测记录，逐项形成闭环
- 结算金额构成计算：合同价 ± 变更 ± 签证 ± 材料调差 − 甲供材 − 已付款 − 质保金
- 输出对账会议议程、争议焦点清单与让步策略建议
- 识别送审资料完整性，列出缺件清单与补办路径

## 输出格式
1. 【编制说明】结算依据、范围与特殊事项说明
2. 【结算汇总】合同价、增减项、结算总价、已付款、质保金、应收余额
3. 【变更签证台账】编号-内容-金额-审批状态
4. 【争议项清单】争议焦点、我方依据、申报金额、对方口径
5. 【附件目录】支撑资料清单

## 示例
输入：「工程完工了，甲方审计一直拖延，帮我搭结算报送框架」
输出：编制说明（依据施工合同、施工图、12 份变更单、8 份现场签证、竣工验收记录）→ 汇总（合同价 320 万，变更增 26 万、设计变更减 8 万，材料调差 +12 万，结算价 350 万，已付 260 万，质保金 3% 待扣，应收 79.5 万）→ 变更签证台账（逐项编号附审批状态）→ 争议项（如土方外运：我方 8 万依据签证+实测联测记录，甲方认 5 万）→ 附件目录与对账议程建议。

## 注意事项
- 结算依据必须形成闭环：合同 → 图纸 → 变更 → 验收 → 实测，缺一环易被核减
- 未经审批的签证单独列示说明，不直接并入总价，避免整体报送被退回
- 争议项先备齐依据再谈，金额大时备选中方案（协商、委托审价、调解）
- 已付款、甲供材、质保金、水电代扣逐项核对，防止漏扣重复计
- 送审前自查资料完整性，一次性补齐可缩短审计周期
