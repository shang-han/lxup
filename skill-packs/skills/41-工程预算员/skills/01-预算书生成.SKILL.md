# 预算书生成

## 适用场景
预算员编制投标报价、施工图预算、招标控制价，或复核清单列项与定额子目是否完整、计价口径是否合规，需要规范的结构与编制说明。

## 核心能力
- 按专业（土建/安装/装饰）生成预算书结构：封面、编制说明、汇总表、分部分项清单计价表、综合单价分析表、主要材料表
- 协助清单列项：项目编码、项目名称、项目特征描述、计量单位、工程量
- 提供定额套用与换算思路：按施工工艺选子目，说明人工/材料/机械换算依据
- 综合单价组成分析：人工费、材料费、机械费、管理费、利润与风险
- 撰写编制说明：编制依据（图纸版本、定额文号、信息价期数、取费标准）、假定条件、未含事项
- 核对暂列金、暂估价、规费税金按规定单列，不参与竞争性下浮

## 输出格式
1. 【编制依据】图纸、定额、取费标准与信息价期数
2. 【预算汇总】分部分项费、措施费、其他项目费、规费税金、总价
3. 【清单计价表】编码-名称-特征-单位-工程量-综合单价-合价
4. 【综合单价分析】重点清单项的人材机与管理费利润组成
5. 【编制说明】依据、假定、未含项与风险提示

## 示例
输入：「办公楼装修工程的装饰部分投标预算，用本地 2020 定额」
输出：编制依据（施工图 2026.5 版 + 2020 装饰定额 + 当期造价信息 + 营改增后取费）→ 汇总表（拆除、楼地面、墙柱面、天棚、油漆涂料各分部费用）→ 清单计价表（含完整特征描述，如「600×600 抛釉砖、1:3 水泥砂浆粘贴、含勾缝」）→ 重点项综合单价分析 → 编制说明（暂估价材料单列、未含消防改造与业主二次设计部分）。

## 注意事项
- 编制依据写明定额文号、信息价期数与取费文件，保证可追溯可复核
- 清单项目特征描述完整（材质、规格、工艺、部位），避免结算时因特征不清扯皮
- 定额与实际工艺不符时按规则换算并在编制说明中注明依据
- 暂列金、暂估价、安全文明施工费等不可竞争费用按规定单列
- 投标报价核对招标文件实质性条款（工期、付款、风险范围），避免废标
