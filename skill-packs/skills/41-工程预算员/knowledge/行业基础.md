# 行业基础

## 行业概况
工程造价行业贯穿工程项目全过程，涵盖土建、安装、装饰、市政等专业，核心工作是「识图 → 算量 → 套价 → 调价」。预算员的日常包括招标控制价与投标报价编制、施工过程进度款审核、变更签证计价与竣工结算审核。行业以清单计价规范（GB 50500）与地方定额为计价依据，广联达等算量计价软件是主要生产力工具。造价成果直接影响项目盈亏，数据严谨性与依据可追溯性是职业生命线。

## 专业技能体系

### 硬技能
- 工程量清单编制与定额套用
- 造价软件操作（广联达/鲁班/斯维尔）
- 施工工艺与材料市场价格跟踪
- 招标控制价与竣工结算审核
- 合同条款对造价影响分析

### 软技能
- 与施工方/甲方双向沟通
- 数据精度与责任心

## 常用术语
- 定额：国家或地方发布的工程人材机消耗量标准，是套价的基础
- 清单计价：按工程量清单逐项报价，综合单价包干、工程量按实结算
- 综合单价：完成清单项目所需的人工费+材料费+机械费+管理费+利润及一定风险
- 暂列金：招标人预留的不可预见费用，按实发生、余额归发包人
- 暂估价：招标阶段无法确定价格的材料或专业工程的暂定价格
- 措施费：为完成工程施工发生的非实体项目费用（脚手架、模板、安全文明施工等）
- 变更签证：设计变更与现场签证的合称，是结算调整的主要依据
- 竣工结算：工程完工后发承包双方确认的最终工程价款

## 核心工作流程
投标阶段：研读招标文件与图纸 → 工程量计算 → 定额套价 → 综合单价分析 → 投标报价
施工阶段：进度款申报审核 → 变更签证计价 → 材料核价与调差 → 月度成本盘点
竣工阶段：结算书编制 → 报送与对量对价 → 争议处理 → 结算定案与归档

## 常见问题
Q：预算超支了怎么办？
A：先审核超支项，区分量差还是价差：量差优化施工方案与损耗管控，价差核查能否走合同约定的调差机制，同时加快未计价变更的报送。

Q：甲方砍价太狠怎么应对？
A：守住合理利润底线，用完整的计价依据（定额、信息价、报价单）支撑单价，突出材料品牌、工艺标准与售后承诺的价值差异，必要时调整方案而非一味降价。

Q：清单工程量与图纸算出来对不上？
A：以计算规则为准核对扣减边界，清单量按图示净量、定额量含施工余量，两者口径不同属正常，结算时按合同约定调整。

Q：变更签证怎么计价才稳？
A：合同有相同或类似单价的参照执行，没有的按定额与信息价组价，办理及时、附件齐全（指令、图纸、实测、影像），随发生随报送。
