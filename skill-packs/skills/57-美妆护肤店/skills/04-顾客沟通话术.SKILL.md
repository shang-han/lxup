# 顾客沟通话术

## 适用场景
门店接待破冰、需求询问、顾客异议处理（「太贵了」「我上网买」「怕过敏」「用了没效果」）、连带推荐、老客回访与复购提醒。

## 核心能力
- 非推销式接待开场：观察式提问（看肤质状态、看顾客在看的品类），给顾客挑选空间
- 结构化需求询问：肤质/皮肤困扰/过敏史/预算/品牌偏好/现有护肤流程
- 信任建立动作设计：免费肤质测试、试用装耳后试用、主动告知过敏风险
- 异议处理：价格（单次使用成本折算/套装组合）、线上比价（服务与试用价值）、过敏顾虑（成分说明+试用建议）、效果疑虑（合理预期管理）
- 连带建议话术：套装搭配、季节补充、同系列升级（给理由不强推）
- 老客回访：使用打卡（3/7/30天）、补货提醒、会员日通知

## 输出格式
1. 【场景定义】顾客类型与具体异议
2. 【推荐话术】口语化，2-3个版本
3. 【配套动作】肤质测试、试用装体验等动作
4. 【禁忌话术】容易说错、适得其反的表达
5. 【合规提醒】不能说的表述（医疗承诺、功效保证）

## 示例
输入：「顾客说『我回去网上查查再决定』，怎么回」
输出：场景：比价犹豫型 → 话术A：「没关系，网上查是应该的，我先给您免费测个肤质，您知道自己的肤质需求，上网买也不会买错」→ 话术B：「这款可以先在耳后试一下，肤感和过不过敏当场就知道，这是网上比不了的」→ 配套动作：肤质测试 + 赠小样 → 禁忌：「网上都是假货」「我保证有效」→ 合规：不贬低线上渠道，不承诺功效。

## 合规红线
- 不得承诺功效（「保证美白」「7天见效」均不可说）
- 不得使用医疗效果表述（「治疗痘痘」「修复敏感肌」属医疗宣称）
- 不贬低同行或线上渠道拉客，只突出自身服务价值
- 过敏风险须主动告知，不为成交隐瞒（不可说「谁都能用、绝对不过敏」）
- 规避广告法极限词
