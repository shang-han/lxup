# 种草文案

## 适用场景
新品上新种草、节日促销、会员日预告、换季补水专题、门店日常内容运营（社群群发、朋友圈、小红书笔记）。

## 核心能力
- 平台适配：社群群推（简短+互动话尾）、朋友圈（体验感+九宫格）、小红书（标题+正文+话题标签）
- 卖点提炼：成分故事、肤感描述、适用肤质与场景、价格锚点、门店服务（免费肤质测试/试用装体验）
- 体验感结构：痛点开场 → 使用体验 → 变化描述 → 行动号召
- 合规改写：自动识别并替换医疗功效宣称、极限词、禁用表述，输出合规版本
- 节点话题库：新品首发、换季维稳、节日礼盒、会员日、老带新
- 互动设计：试用申领、话题问答、晒单有礼、到店暗号

## 输出格式
1. 【卖点提炼】3个核心卖点（成分/肤感/场景）
2. 【文案正文】所选平台的主版本
3. 【标题/开场备选】2-3个变体
4. 【行动号召】到店试用/进群接龙/预约引导话术
5. 【合规核对】已识别并替换的敏感表述清单

## 示例
输入：「给新到的玻尿酸精华写一条朋友圈种草」
输出：卖点：大小分子玻尿酸双重补水、清爽不黏腻肤感、适合换季干燥肌 → 文案：「换季皮肤干到起皮刺痛？新到的这支玻尿酸精华，上脸清爽秒吸收，化妆水后叠一层，一整天不紧绷，会员到店可以先试再决定~」→ 标题备选2条 → 行动号召「本周到店免费测肤质，试用装先到先得」→ 合规核对：无医疗功效与极限词问题。

## 合规红线
- 种草文案不得宣称医疗作用（禁用「抗敏」「消炎」「祛痘」「治皮炎」），美白类须有特证方可宣称
- 禁用「药妆」「医学护肤」表述，规避「最」「第一」「立刻」「100%」等极限词
- 前后对比用真实分享口吻，不暗示保证功效，不使用患者/病变图片
- 成分功效宣称应有依据，建议以产品备案信息为准，不编造实验数据
- 跨境/代购产品如实说明渠道与售后，不冒充品牌总代
