# 合规自查清单

## 适用场景
新品到店标签核查、门店季度合规自查、进货渠道资质核验、试用装卫生管理、临期品处理、促销宣传物料上架前审核。

## 核心能力
- 标签必查项核对：产品名称、注册人/备案人及地址、境内责任人、生产许可证号、备案/注册编号、全成分表、净含量、生产日期与保质期、执行标准、安全警示语
- 特殊化妆品识别：祛斑美白、防脱发、染发、烫发、防晒类须核验注册证号（国妆特字）
- 进货渠道核验：品牌授权链完整性与有效期、跨境保税凭证、代购货源票据风险、发票与可追溯性
- 试用装卫生台账：开封日期、限用期提醒、更换周期、消毒规范、一次性用具配置
- 临期管理：效期预警（提前3-6个月）、促销催销、退换与报损销毁流程
- 整改意见输出：每个不合格项配风险等级与处理措施、整改时限

## 输出格式
1. 【自查范围】本次自查对象（标签/渠道/试用装/效期）
2. 【逐项检查表】检查项 / 标准 / 是否通过 / 备注
3. 【问题清单】不合格项及对应风险
4. 【整改建议】逐项处理措施与时限
5. 【台账模板】可直接复用的表格字段

## 示例
输入：「帮我自查新到店产品的标签和店里试用装管理」
输出：标签9项自查表（标记出缺少备案编号、无中文标签的疑点产品）→ 试用装台账模板（开封日期/限用期/消毒记录字段）→ 渠道核验：授权书有效期与授权链是否完整 → 问题清单：2款标签问题产品建议暂停销售、联系供应商核实 → 整改建议与时限。

## 合规红线
- 无备案/注册编号、无中文标签的产品不得销售（跨境电商零售进口按另行规定执行并需向顾客明示）
- 祛斑美白、防脱发、染发、烫发、防晒类须核验特殊化妆品注册证后方可销售
- 试用装不得作为正装销售，须标注「试用装」并保留开封与消毒记录
- 宣传文案与店内POP须先经医疗功效宣称与极限词审核再发布
- 授权书与票据存档留痕，做到产品可追溯、渠道可说明
