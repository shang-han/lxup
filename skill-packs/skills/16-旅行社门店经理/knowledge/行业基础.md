# 行业基础

## 行业概况
旅行社行业经历疫情后逐步复苏，但面临自由行和 OTA 平台的双重挤压。转型方向：主题旅游 + 定制服务 + 本地生活。核心竞争力：资源整合 + 专业服务。收入来源为跟团游、定制游、票务、签证代办等，门店靠口碑、专业度与回头客生存。行业受《旅游法》《旅行社条例》规范，合规经营是底线。

## 常用术语
- 成团率：报名人数 / 最低成团人数
- 毛利率：产品利润 / 销售收入
- 客单价：平均每单金额
- 复购率：再次购买比例
- 投诉率：投诉订单占比
- 纯玩团：全程无购物、无强制自费的行程
- 不合理低价游：低于成本报价、靠购物回扣补亏的违规产品
- 旅行社责任险：旅行社依法必须投保、承保自身责任造成损失的险种

## 核心工作流程
每日：客户咨询 → 行程推荐 → 订单处理 → 售后回访
每周：产品上新 → 价格更新 → 数据分析
每月：供应商对账 → 营销策划 → 培训学习

## 常见问题
Q：客户嫌贵怎么办？
A：拆解费用构成，对比自由行的成本与麻烦，强调省心 + 保障 + 纯玩的价值。

Q：行程中出问题怎么办？
A：启动应急预案 + 即时沟通 + 化解方案，依据合同与《旅游法》处理。

Q：怎么减少投诉？
A：口头承诺都写进合同，提前管理客户预期，行中跟进落实，把问题解决在萌芽。

Q：门店转型方向是什么？
A：做主题游、定制服务、本地生活体验，用专业服务与口碑建立差异化。
