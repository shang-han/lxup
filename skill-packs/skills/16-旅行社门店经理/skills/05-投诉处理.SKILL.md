# 投诉处理

## 适用场景
客户在行程中或行程后对行程缩水、服务不达标、强制购物、航班延误等提出投诉时使用。需要化解矛盾、合理化解纠纷、维护门店口碑。

## 核心能力
- 投诉分类（行程缩水、住宿标准不符、导游服务、强制购物、不可抗力）
- 先安抚情绪后处理，倾听并完整记录诉求
- 事实核查与责任判定
- 化解方案设计（退差价、补偿、道歉、体验补偿）
- 不可抗力与突发事件（航班延误、自然灾害）处理
- 投诉复盘与预防复发

## 输出格式
1. 【投诉分类】类型与核心诉求
2. 【应对策略】处理思路与语气
3. 【话术文本】安抚与沟通话术
4. 【化解方案】具体补偿/解决方式
5. 【跟进与预防】回访与改进措施

## 示例
输入：「客户投诉行程中承诺的海景房实际是侧海景，要求赔偿」
输出：分类为住宿标准不符·核心是赔偿 → 策略（先道歉共情，核实事实，给出化解）→ 话术「非常抱歉给您旅途添了堵，您反馈的房型标准问题我们立即核实，如合同约定海景实际未达标，我们一定负责到底……」→ 化解（退还房差价，视严重程度赠补偿券或部分退款）→ 跟进（回访确认满意，追责地接环节，列入服务红线）。

## 合规红线
- 依据《旅游合同》与《旅游法》处理，不推诿、不拖延
- 不可抗力依法处理，合理分担费用、退还未发生费用
- 化解承诺须可兑现并书面留存
- 不侵害客户合法权益，不打击报复投诉人
- 重大投诉与安全事故按要求上报文旅主管部门
