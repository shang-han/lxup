# 产品介绍文案

## 适用场景
旅行社门店经理推广线路产品时使用。撰写朋友圈/公众号/门店 POP/短信等渠道的收客文案，突出产品卖点，激发客户报名欲望。

## 核心能力
- 分渠道文案（朋友圈短版、公众号长版、门店 POP 版、短信版）
- 结构：钩子 + 目的地吸引力 + 行程亮点 + 价格权益 + 行动号召
- 产品卖点突出（纯玩无购物、小团、一价全包、独家体验）
- 按人群做情感诉求（亲子/老年/蜜月/团建）
- 季节与限时紧迫感营造（早鸟、余位、花期）
- 规避广告法禁用表达

## 输出格式
1. 【目标人群与渠道】面向谁、投放哪
2. 【标题/钩子】吸引注意的开头
3. 【正文文案】对应渠道的产品介绍
4. 【行程亮点】3-5 个核心亮点
5. 【价格与权益】价格、早鸟/优惠
6. 【行动号召】引导咨询报名

## 示例
输入：「写个新疆小团纯玩游的朋友圈收客文案」
输出：人群为想玩新疆又怕麻烦的年轻人/情侣，渠道朋友圈 → 钩子「这个夏天去新疆，2-6 人小团纯玩，带个相机和好心情就行」→ 正文（独库公路+伊犁花海 8 日深度，专业司导+包车，全程纯玩无购物）→ 亮点（2-6 人迷你团、当地包车、纯玩、行程可调）→ 价格权益（早鸟立减 500/人，3 人成团再优惠）→ 号召（扫码咨询排期与报价）。

## 合规红线
- 不使用「最低价」「第一」「最好」等绝对化用语，遵守广告法
- 价格与服务内容须真实，不虚假宣传、不诱导
- 「纯玩无购物」等承诺须与实际行程一致并写入合同
- 不宣传不合理低价游，不以回扣/自费为噱头招徕
- 宣传内容标明旅行社名称与经营许可证号
