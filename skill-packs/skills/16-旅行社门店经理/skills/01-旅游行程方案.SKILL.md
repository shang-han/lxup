# 旅游行程方案

## 适用场景
旅行社门店经理/跟团游销售为客户策划行程时使用。包括跟团游线路方案设计、定制游行程编排、按客户预算与人群（亲子/老年/蜜月/团建）调整行程。

## 核心能力
- 按目的地、天数、预算、人群偏好规划线路
- 每日行程细化（景点、交通、住宿、餐饮、自由活动）与节奏把控
- 预算报价拆解（交通、住宿、门票、餐饮、导服、保险）
- 行程卖点提炼与差异化描述（纯玩无购物、小团、深度体验）
- 季节与目的地适配（最佳季节、高原反应、签证要求）
- 多版本方案（经济版 / 品质版 / 定制版）供客户选择

## 输出格式
1. 【方案概览】目的地、天数、适用人群、主题
2. 【每日行程】Day1-DayN 安排与亮点
3. 【预算报价】费用拆解与总价
4. 【方案卖点】核心吸引力与差异化
5. 【服务包含】含与不含项目清单
6. 【注意事项】签证、气候、特殊提示

## 示例
输入：「一家三口去云南 6 天 5 晚，预算人均 5000 左右」
输出：概览（云南亲子游，6 天 5 晚，节奏舒缓适合老人小孩）→ 每日行程（D1 昆明接机 → D2-3 大理洱海慢游 → D4-5 丽江古城与玉龙雪山、提示高原 → D6 返程）→ 预算（往返机票 + 四星住宿 + 包车 + 门票，约 4980 元/人）→ 卖点（纯玩无购物、亲子友好体验、全程包车）→ 含与不含 → 注意事项（玉龙雪山海拔较高，老人小孩备防晒与抗高反）。

## 合规红线
- 不设计「不合理低价游」，不以购物回扣、自费项目填补成本
- 行程须与合同一致，变更须征得客户同意并书面确认
- 依法购买旅行社责任险，提示客户购买旅游意外险
- 持牌经营，不无资质操作，不转包挂靠
- 高风险项目（高原、涉水、探险）须特别提示并告知适宜人群
