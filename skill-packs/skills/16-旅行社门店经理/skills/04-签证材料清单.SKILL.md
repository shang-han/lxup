# 签证材料清单

## 适用场景
旅行社门店接待出境游客户咨询签证办理时使用。根据目的地国家与签证类型（旅游/商务/探亲）为客户生成清晰的材料清单与办理流程。

## 核心能力
- 按目的地与签证类型生成材料清单（基础材料 + 身份分类材料）
- 按申请人身份差异化（在职、自由职业、学生、退休、未成年）
- 材料要求说明（有效期、翻译公证、照片规格、银行流水）
- 办理流程与办理时长说明
- 常见拒签原因与规避提示
- 免签 / 落地签 / 电子签国家的注意事项

## 输出格式
1. 【签证类型】国家与签证类别、办理方式
2. 【基础材料】所有人通用材料
3. 【身份材料】按申请人身份的差异材料
4. 【材料要求】翻译、公证、有效期等具体要求
5. 【流程与时长】步骤与预计办理时间
6. 【注意事项】常见拒签原因与规避

## 示例
输入：「一家三口办日本旅游签，丈夫在职、妻子自由职业、孩子学生」
输出：类型为日本个人旅游签（单次）→ 基础材料（护照有效期 6 个月以上、照片、申请表、身份证、户口本）→ 身份材料（丈夫：在职证明+银行流水；妻子：收入/存款证明；孩子：出生证明+在读证明，可随父母资产）→ 要求（流水余额达标、材料齐全真实）→ 流程（备齐→送签→使馆审核约 7-10 工作日）→ 注意（材料真实不造假，电话回访回答与材料一致）。

## 合规红线
- 不协助客户伪造、变造材料，材料真实性由申请人负责
- 不承诺「包过」「100% 出签」，出签由使馆决定
- 代办收费透明、分项明示，不超标准收费
- 保护客户个人信息与材料安全，不泄露
- 区分代办与使馆责任，提前告知拒签风险与费用承担
