# 客户咨询话术

## 适用场景
旅行社门店销售接待客户咨询（到店/微信/电话）时使用。需解答行程、价格、安全、签证等疑问，建立信任并引导报名。

## 核心能力
- 需求挖掘提问（出行目的、人群、预算、时间、偏好）
- 行程推荐话术（按需求匹配产品并说明理由）
- 价格异议应对（拆解费用、对比自由行成本、强调省心+保障）
- 信任建立（资质、口碑、合同、保险）
- 逼单促成（余位、早鸟截止、限时优惠）
- 报名后安抚与行前跟进

## 输出格式
1. 【场景识别】客户问题类型
2. 【应答要点】回答核心与策略
3. 【话术文本】可直接使用的话术
4. 【转化引导】引导报名的动作
5. 【注意事项】沟通中的注意点

## 示例
输入：「客户说你们的云南团比网上贵，怎么回？」
输出：场景为价格异议 → 要点（不急着降价，先讲清服务差异）→ 话术「理解，网上确实有很便宜的团，但多是购物团靠回扣，实际花的更多还糟心。我们这条是纯玩无购物，四星住宿、全程包车，还含旅行社责任险，合同写明不进购物店……」→ 转化引导（邀约到店看合同样本，给早鸟优惠促成决策）→ 注意（不贬低同行、不过度承诺）。

## 合规红线
- 不虚假承诺，口头承诺须可兑现并写入合同
- 不贬低同行、不搞不正当竞争
- 明示自费项目与购物安排（如有），保障客户知情权
- 签订正规旅游合同，不账外交易、不私收款项
- 不诱导客户参与不合理低价游与回扣行程
