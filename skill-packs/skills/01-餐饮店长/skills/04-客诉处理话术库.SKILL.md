# 客诉处理话术库

## 适用场景
门店遇到顾客现场投诉（菜品异物、口味、上菜慢、服务态度）或线上差评时，店员和店长需要一套标准、得体的应对话术，做到先安抚情绪、再解决问题、最后挽回顾客。

## 核心能力
- 覆盖高频场景：异物/变质、上菜慢、口味不佳、服务态度、排队等位、账单争议
- 「先情绪后问题」四步法：道歉共情 → 了解事实 → 给方案 → 回访跟进
- 分级补偿建议：重做 / 免单 / 赠券 / 退款的适用边界
- 现场话术与线上差评回复两套版本
- 化解升级：情绪激动顾客的降温与上报机制
- 复盘沉淀：把客诉转为后厨/服务改进项

## 输出格式
1. 【场景识别】属于哪类客诉、紧急程度
2. 【处理步骤】分步骤的操作指引
3. 【应对话术】现场口语版（可直接说出口）
4. 【补偿建议】分级方案与授权边界
5. 【线上回复】平台差评回复模板（如适用）
6. 【复盘改进】防止复发的门店动作

## 示例
输入：「顾客说汤里有头发，很生气，在大厅嚷嚷」
输出：场景=异物投诉（高敏感）→ 步骤：先带离座位到安静处、真诚道歉、确认情况、立即处理 → 话术：「非常抱歉让您有这样不愉快的体验，这道菜我马上给您撤下重做，今天这单我帮您免掉，实在对不起」→ 补偿：免该菜品 + 赠果盘 → 复盘：检查后厨工帽佩戴与出餐口卫生。

## 合规红线
- 涉及疑似食物中毒、异物伤害，第一时间保留样品与记录，必要时协助就医，不得隐瞒
- 不推诿、不与顾客争执，更不得言语或肢体冲突
- 补偿授权在门店权限内执行，超出及时上报，不随意承诺
- 线上回复不泄露顾客隐私，不互怼，不删评逃避（平台规则内处理）
