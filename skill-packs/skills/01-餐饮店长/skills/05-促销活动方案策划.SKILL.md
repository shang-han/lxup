# 促销活动方案策划

## 适用场景
门店需要做新店开业引流、节假日大促、会员日锁客、淡季激活、新品推广等活动时，店长需要一份含目标、机制、选品、分工、文案和复盘指标的完整可执行方案。

## 核心能力
- 活动类型库：开业爆破、节日大促、会员储值、储值锁客、新品试吃、社群拼团
- 完整方案：目标 / 主题 / 时间 / 机制 / 选品 / 预算 / 分工 / 物料 / 复盘指标
- 促销机制设计：折扣、满减、买赠、储值送、第二份半价、引流爆品
- 多渠道文案：朋友圈、社群、店内海报、短视频口播
- 预算与毛利测算：确保活动不伤利润
- 复盘模板：客流、营业额、新客、复购对比

## 输出格式
1. 【活动概览】目标、主题、时间、目标人群
2. 【活动机制】具体玩法与优惠规则
3. 【选品建议】引流品 / 利润品 / 关联品
4. 【预算与测算】成本预估与预期收益
5. 【执行分工】按岗位的任务与时间节点
6. 【宣传文案】朋友圈 + 社群 + 海报版
7. 【复盘指标】活动后统计项清单

## 示例
输入：「下个月店庆，想做个三天活动拉一波人气」
输出：主题「X 周年庆·三重好礼」→ 机制（全场 88 折 + 储值 500 送 100 + 到店送小菜）→ 选品（招牌菜做引流、高毛利饮品做利润）→ 三天预算与预期营业额测算 → 前厅后厨分工表 → 朋友圈文案：「我们 X 岁啦！这三天全场 88 折，充 500 送 100，老顾客带新朋友还送招牌小菜……」→ 复盘指标表。

## 注意事项
- 宣传文案规避《广告法》禁用词（「最」「第一」「国家级」等）
- 明码标价，折扣前后价格真实，不得先涨后降搞虚假优惠
- 储值类活动明确使用规则与退款条款，避免预付卡纠纷
- 活动量力而行，先算毛利再定力度，防止赔本赚吆喝
