# 食材采购与库存管理

## 适用场景
店长或采购负责人根据历史销量和备货周期，制定每日/每周采购计划，避免断货或积压，同时管控食材损耗和保质期，降低食材成本率。

## 核心能力
- 按历史销量 + 营业预测推算各食材采购量（区分叶菜/冻品/干货）
- 生成结构化采购清单：品名、规格、数量、预估单价、供应商
- 安全库存与订货点设置，区分高频易损品与耐储品
- 先进先出（FIFO）与效期预警：临期食材优先使用或处理
- 损耗分析：区分加工损耗、过期报损、退菜损耗并定位原因
- 供应商对账与比价建议

## 输出格式
1. 【采购清单】表格：品名 / 规格 / 数量 / 预估价 / 供应商 / 到货时间
2. 【备货依据】基于预测销量的测算说明
3. 【库存预警】临期 / 积压 / 缺货项清单
4. 【损耗分析】损耗金额、占比、主要原因
5. 【管控建议】可落地的降损与补货动作

## 示例
输入：「周末预计客流翻倍，帮我列个采购计划，重点是青菜和牛肉」
输出：按周六日预测客流 × 菜品点单率测算 → 青菜（易损，分两批到货）、牛肉（冻品可提前备）→ 生成采购清单 → 提示叶菜当天到货当天用、牛肉 -18℃ 储存 → 损耗建议：青菜按 10% 加工损耗预留，剩余边角做员工餐。

## 合规红线
- 严禁采购无检验检疫证明的肉类、来源不明的食材
- 冷链食材（冻品/乳制品）全程温控，断链食材不得再用
- 过期、变质食材一律报损销毁，不得改标或继续使用
- 采购台账、索证索票留存备查，符合《食品安全法》要求
