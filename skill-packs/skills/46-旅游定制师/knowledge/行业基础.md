# 行业基础

## 行业概况
定制旅游是旅游行业的高端细分市场，满足个性化、高品质的出行需求。客户群体以高净值人群、家庭出游、企业团建、蜜月旅拍为主，采用服务费制取代传统团费制。相比跟团游，定制游独立成团、行程灵活、体验更深，价格通常高出 20-50%。定制师的核心价值在于需求洞察、资源整合与全程服务，拼的不是低价而是专业与省心。行业受季节与假期影响明显，需提前规划与供应链协调。

## 专业技能体系

### 硬技能
- 行程规划与路线设计（景点/交通/住宿/餐饮串联）
- 讲解词撰写与现场讲解
- 应急事件处理（天气/交通/疾病/走失）
- 票务与供应商谈判（酒店/车队/景区）
- 签证/出入境政策更新跟踪

### 软技能
- 团队气氛调动
- 不同人群讲解方式切换

## 常用术语
- 深度游：深入当地文化与生活的慢节奏旅行方式
- 私家团：独立成团、不与陌生人拼团的出行形式
- 定制行程：根据客户需求个性化设计的专属路线
- 目的地婚礼：在旅游目的地举办婚礼的旅行形式
- 主题旅行：以摄影、美食、徒步等特定主题串联的旅行
- 服务费制：按行程总价一定比例或固定金额收取的定制服务费
- 行前说明：出发前向客户交代行程细节与注意事项的环节

## 核心工作流程
接单：需求沟通 → 客户画像 → 预算评估 → 意向确认
设计：行程草案 → 航班推荐 → 酒店筛选 → 体验设计 → 报价
执行：预订确认 → 行前说明 → 旅行保障 → 实时协助 → 售后回访

## 常见问题
Q：定制比跟团贵多少，贵在哪？
A：通常贵 20-50%，但获得独立成团、个性化行程、灵活调整与全程专属服务，省去跟团的赶场与购物环节。

Q：行程中遇到突发情况怎么办？
A：提供 24 小时应急联系，提前准备备用方案，遇航班取消、天气变化及时调整，并协助保险理赔。

Q：怎么判断客户的真实预算？
A：委婉询问区间，结合出行档次、住宿要求、过往出行经验综合判断，出经济/舒适/高端三档方案供选择。

Q：如何提高方案成交率？
A：需求摸准再设计、方案突出定制差异、报价透明分项、及时响应迭代，用专业和省心打动客户而非拼低价。
