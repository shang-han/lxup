# 行程方案生成

## 适用场景
定制师接到客户出行需求后，需要快速产出一份结构完整、节奏合理、体现个性化的行程方案，用于向客户提案并促成签约。

## 核心能力
- 按天数/预算/人群（亲子/情侣/银发/团建）匹配行程节奏
- 每日安排「上午-下午-晚间」时段，景点动线顺路不绕路
- 合理控制单日强度，避免赶场，预留弹性时间
- 融入差异化体验：小众景点、当地美食、文化互动、主题玩法
- 标注交通方式、预计时长、住宿建议、备选方案
- 支持按客户反馈快速迭代多版本

## 输出格式
1. 【方案概览】目的地 / 天数 / 主题 / 适合人群 / 预算区间
2. 【每日行程】Day1..DayN 的时段安排与动线
3. 【体验亮点】区别于常规团的个性化项目
4. 【交通住宿】大交通 + 当地交通 + 住宿档次建议
5. 【备选/弹性】雨天或临时调整预案
6. 【报价框架】费用构成说明（详见预算规划）

## 示例
输入：「一对情侣想五一去大理玩 4 天，预算人均 4000，喜欢拍照和小众」
输出：概览「大理 4 日·慢生活旅拍之旅，人均约 4000」→ Day1 抵达+古城闲逛+人民路咖啡 → Day2 洱海环湖（小众 S 湾+喜洲古镇拍照）→ Day3 苍山感通索道+寂照庵多肉小院 → Day4 双廊日出+返程 → 亮点（白族扎染体验、海景民宿露台旅拍）→ 交通（租车环湖+古城步行）→ 住宿（海景民宿 2 晚+古城客栈 1 晚）→ 备选（雨天改室内扎染+博物馆）。

## 注意事项
- 行程须匹配客户真实预算与体力，不堆砌景点
- 景点开放时间、季节适宜性、预约要求要核实
- 标注自由活动时间，避免把客户时间排满
- 涉及高风险项目（漂流、徒步）提示安全与保险
- 方案体现「定制」差异，避免与常规跟团雷同
