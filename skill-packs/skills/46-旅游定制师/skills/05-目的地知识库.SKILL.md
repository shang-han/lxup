# 目的地知识库

## 适用场景
定制师在设计方案或回答客户咨询时，需要快速调取某个目的地的景点、美食、交通、最佳季节、当地习俗等知识，保证专业度与信息准确性。

## 核心能力
- 目的地概况：地理、气候、最佳旅游季节、淡旺季
- 景点知识：核心景点、小众景点、门票预约、游览时长
- 美食与特产：必吃美食、老字号、伴手礼
- 交通住宿：到达方式、当地交通、住宿区域特点
- 文化习俗：民族风情、禁忌、礼仪、节庆
- 实用信息：证件、货币、通讯、医疗、安全提示

## 输出格式
1. 【目的地概况】位置 / 气候 / 最佳季节
2. 【必玩景点】核心 + 小众，含游览建议
3. 【必吃美食】特色与推荐去处
4. 【交通住宿】到达与内部交通、住宿区域
5. 【文化习俗】风情与禁忌
6. 【实用提示】证件、安全、医疗等

## 示例
输入：「客户想去甘肃敦煌，给我一份目的地知识」
输出：概况（甘肃西部、温带大陆性气候、最佳 5-6 月与 9-10 月，避开酷暑与风沙）→ 必玩（莫高窟需提前预约、鸣沙山月牙泉看日落、雅丹魔鬼城、阳关玉门关）→ 美食（驴肉黄面、杏皮水、敦煌酿皮）→ 交通（敦煌机场/火车站，景点分散建议包车）→ 文化（丝路文化、佛教石窟艺术，参观莫高窟禁拍照）→ 提示（昼夜温差大带外套、防晒补水、莫高窟门票务必提前订）。

## 注意事项
- 景点开放时间、预约规则、票价以官方最新信息为准
- 季节与气候提示要准确，直接影响体验与安全
- 尊重当地民族习俗与宗教禁忌，避免冒犯
- 高原、沙漠、边境等特殊目的地要强化安全与证件提示
- 信息不确定时明确标注，建议客户出行前再核实
