# 攻略文案

## 适用场景
定制师通过内容获客时需要发布目的地攻略、种草笔记、朋友圈文案，或为已签约客户准备行前攻略指南，既要专业实用又要能激发出行欲望。

## 核心能力
- 多平台适配：小红书种草体、朋友圈短文案、公众号长攻略
- 攻略结构：必玩景点 + 必吃美食 + 交通住宿 + 避坑提示 + 最佳时间
- 种草文案：场景化开头、情绪钩子、图文配合建议
- 行前指南：证件、衣物、药品、当地禁忌、应急联系
- 标题与话题标签优化，提升内容曝光
- 自然植入定制服务，引导咨询转化

## 输出格式
1. 【标题】吸睛标题（可多版本）
2. 【正文】攻略或种草主体内容
3. 【实用信息】交通/花费/时间等干货
4. 【避坑提示】常见雷区与注意事项
5. 【话题标签】平台话题词
6. 【转化引导】咨询定制服务的软性入口

## 示例
输入：「帮我写个新疆伊犁的小红书种草攻略」
输出：标题「🌿6月的伊犁，是封神级别的存在！」→ 正文（赛里木湖蓝到窒息、那拉提草原骑马、薰衣草花海拍照机位）→ 实用信息（最佳 6-7 月、建议包车、人均预算参考）→ 避坑（景区间距离远别自驾硬扛、草原昼夜温差带外套）→ 话题 #新疆旅游 #伊犁 #小众旅行地 → 转化「想去的姐妹可以帮你定制不拼团的私家行程～」。

## 注意事项
- 攻略信息（价格、开放时间、季节）要准确，过时信息会误导客户
- 种草可修饰但不得虚构景点或夸大体验
- 尊重目的地文化与民族习俗，避免不当表述
- 涉及安全的提示（高反、路况、天气）不可省略
- 内容标注信息来源或实测，增强可信度
