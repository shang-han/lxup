# 预算规划

## 适用场景
客户咨询「这趟大概花多少钱」「能不能便宜点」时，定制师需要给出透明、分项、可调整的预算方案，既体现专业又能灵活匹配不同预算。

## 核心能力
- 费用分项拆解：大交通 / 住宿 / 当地交通 / 门票 / 餐饮 / 服务费 / 保险
- 按档次出多版本：经济 / 舒适 / 高端三档报价对比
- 人均与总价双口径，注明人数与是否含儿童
- 识别可省与不可省项，给出控费建议
- 标注淡旺季价差、提前预订优惠
- 说明定制服务费收取方式，体现价值

## 输出格式
1. 【预算总览】人均 + 总价，含档次定位
2. 【费用明细】逐项拆分与占比
3. 【档次对比】经济/舒适/高端差异
4. 【控费建议】可调整项与省钱技巧
5. 【费用说明】含/不含项、自费项目、服务费规则
6. 【浮动提示】淡旺季、人数、预订时点的影响

## 示例
输入：「一家四口暑假去青岛 5 天，帮我算下预算，孩子一个 8 岁一个 4 岁」
输出：总览「舒适档人均约 3200，4 人合计约 1.1 万（4 岁儿童多数免票占床减半）」→ 明细（往返高铁约 3000、海景酒店 4 晚约 3200、当地包车约 1500、门票餐饮约 2000、服务费+保险约 1300）→ 控费（避开 7 月底最高峰、酒店选近地铁非一线海景可省 800）→ 说明（含定制服务费 8%，不含个人消费与自费项目）→ 提示（暑期价格上浮，建议提前 1 个月锁定酒店）。

## 注意事项
- 报价必须透明分项，避免「一口价」引发后期纠纷
- 明确写出含与不含，自费项目提前告知
- 儿童、老人票价规则说清楚
- 不低价引流后加项，定制服务价值要体现在明处
- 淡旺季、汇率、预订时点导致的浮动需标注
