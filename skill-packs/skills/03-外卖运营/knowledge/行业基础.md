# 行业基础

## 行业概况
外卖运营是餐饮行业的核心增长引擎，美团和饿了么两大平台占据 90% 以上市场份额，外卖渗透率持续提升。运营核心是围绕转化漏斗做精细化优化：曝光量 → 进店率 → 下单率 → 复购率，每个环节都有对应的优化手段。行业竞争激烈，同质化严重，需在产品、定价、活动、评价、配送时效上全面打磨。平台规则（扣点、活动、流量分配）影响大，运营人员须熟悉规则并在其内优化。利润受平台扣点（通常 15%-25%）和满减补贴双重挤压，精细化核算毛利是关键。

## 常用术语
- 曝光量：用户在列表/推荐位看到店铺的次数
- 进店率：曝光后点击进入店铺的转化率
- 下单率：进店后实际下单的转化率
- 客单价：平均每单实付金额
- 满减率：参与满减活动的订单占比
- ROI：推广投入产出比（推广带来的营业额 ÷ 推广花费）
- 双平台运营：同时在美团、饿了么经营并分别优化

## 核心工作流程
每日：查看数据（曝光/进店/下单）→ 调整菜品与活动 → 回复评价 → 处理异常订单
每周：竞品分析 → 活动策划 → 菜品与套餐优化 → 推广复盘
每月：经营复盘 → 营销规划 → 成本与毛利分析 → 菜单迭代

## 常见问题
Q：新店怎么快速起量？
A：抓住 7 天新店流量扶持期，满减 + 折扣引流品 + 推广三管齐下冲销量和评分，为后续自然流量打基础。

Q：差评怎么处理？
A：24 小时内回复，真诚致歉 + 说明原因 + 补偿方案 + 改进承诺，同时私信沟通争取顾客谅解或修改评价。

Q：曝光高但下单率低怎么办？
A：检查进店后的转化要素——菜品图片、定价、套餐、评价、起送门槛，逐一优化短板环节。

Q：活动做了没利润怎么办？
A：核算平台扣点 + 满减叠加后的真实毛利，用引流品带利润品，控制让利幅度，靠走量而非单纯降价。
