# 套餐搭配方案

## 适用场景
外卖运营专员需要通过设计单人套餐、双人餐、家庭餐和凑单小食组合，提升客单价和连带率，同时用引流品带利润品，优化整体毛利结构。

## 核心能力
- 套餐分层：一人食工作餐、双人分享、家庭多人餐
- 引流品 + 利润品 + 关联品的组合逻辑
- 凑单设计：搭配小食/饮品帮顾客凑满减门槛
- 定价策略：套餐价低于单点总和，制造划算感
- 毛利测算：核算平台扣点与活动后的净毛利
- 命名与呈现：套餐名称突出场景与优惠

## 输出格式
1. 【套餐清单】名称 + 组成 + 定价
2. 【搭配逻辑】引流/利润/关联品说明
3. 【定价测算】单点总价 vs 套餐价的让利
4. 【毛利核算】扣点与活动后净毛利
5. 【凑单建议】小食/饮品凑满减组合
6. 【上线建议】主推套餐与展示位

## 示例
输入：「帮我设计几个套餐，客单价想提到 35」
输出：单人工作餐（招牌菜+米饭+饮品，29 元）→ 双人分享餐（2 荤 1 素+2 米饭，58 元）→ 凑单小食（卤蛋/小酥肉 3-8 元帮凑满减）→ 定价让利说明（套餐比单点省 6-10 元）→ 毛利核算（扣点 20% 后双人餐净利约 30%）→ 建议双人餐设为主推占首屏。

## 注意事项
- 套餐让利要算清平台扣点与满减叠加后的真实毛利，避免赔本
- 组合内菜品口味与分量搭配合理，避免「凑数」影响体验
- 套餐命名清晰标明份数与适用人数，减少误解投诉
- 定期根据销量数据淘汰滞销套餐、迭代爆款组合
