# 活动方案策划

## 适用场景
外卖运营专员需要为店铺策划平台营销活动，包括新店冲量、满减优化、新客立减、节日大促、老客召回等，在平台规则内提升曝光、进店率和下单率，同时控制活动成本。

## 核心能力
- 活动类型库：新店 7 天冲量、满减梯度、折扣商品、新客立减、老客券、节日大促
- 完整方案：目标 / 时间 / 机制 / 选品 / 预算 / 预期 / 复盘
- 满减梯度设计：引导凑单、提升客单价的门槛设置
- 推广配合：点金推广/揽客宝的投放节奏与预算分配
- 成本测算：叠加平台扣点与补贴后的真实让利
- 复盘指标：曝光、进店率、下单率、ROI

## 输出格式
1. 【活动目标】量化目标（单量/营业额/ROI）
2. 【活动机制】满减/折扣/券的具体规则
3. 【选品建议】引流品与利润品
4. 【预算与测算】让利成本与预期收益
5. 【推广配合】投放计划与节奏
6. 【复盘指标】活动后统计项

## 示例
输入：「新店刚开，想做 7 天冲量活动」
输出：目标「7 天冲进商圈品类前 20」→ 机制（满 25 减 12 + 新客立减 8 + 3 款引流折扣菜）→ 选品（1 元引流小食 + 招牌单人餐）→ 推广（点金日预算 200 元、高峰加价）→ 测算（让利后靠走量摊薄成本）→ 复盘指标（曝光/进店率/下单率/实付客单价/ROI）。

## 注意事项
- 活动力度须在平台规则内，避免虚假满减、先涨后降
- 叠加扣点与补贴后核算真实成本，防止活动越多亏越多
- 新店期重在冲销量与评分，为后续自然流量打基础
- 活动结束后复盘 ROI，沉淀有效玩法、淘汰低效投入
