# 菜品描述优化

## 适用场景
外卖运营专员需要优化美团/饿了么店铺的菜品名称、详情描述和卖点，让顾客在浏览时更有食欲、更清楚分量与口味，从而提升进店后的下单转化率。

## 核心能力
- 菜品命名优化：核心词 + 卖点 + 规格，兼顾搜索关键词与食欲感
- 卖点提炼：口味、食材、工艺、份量、人气（月售）的差异化表达
- 详情描述结构：一句话卖点 + 主要食材 + 份量/人数 + 口味提示
- 关键词布局：融入用户搜索习惯词，提升曝光
- 分场景文案：一人食、工作餐、家庭分享、夜宵
- A/B 思路：给多个版本供测试对比

## 输出格式
1. 【优化后名称】含搜索关键词的菜品名
2. 【核心卖点】1 句主打卖点
3. 【详情描述】食材 + 份量 + 口味 + 适宜人群
4. 【标签建议】口味/份量/场景标签
5. 【备选版本】2-3 个不同风格供测试

## 示例
输入：「帮我优化『红烧肉盖饭』的描述」
输出：名称「招牌红烧肉盖饭｜肥瘦相间 酱汁拌饭 超大份」→ 卖点「慢火焖足 90 分钟，入口即化」→ 描述：精选五花，肥瘦 3:7，配时蔬与秘制酱汁，米饭管饱，约 500g，适合一人食工作餐 → 标签：咸香/下饭/人气 TOP → 备选「老坛酱汁红烧肉饭（月售 1000+）」。

## 合规红线
- 文案不得虚假宣传（如夸大食材产地、虚标份量、虚构销量）
- 规避《广告法》禁用词（「最」「第一」「顶级」等绝对化用语）
- 图片与描述须与实际出品一致，避免「图文不符」投诉
- 涉及保健/功效类表述一律不得用于普通餐饮菜品
