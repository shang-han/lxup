# 竞品分析报告

## 适用场景
外卖运营专员需要定期分析同商圈、同品类的竞品店铺，了解对手的产品结构、定价、销量、活动力度和评价，找到自身差距与机会点，制定应对策略。

## 核心能力
- 竞品清单筛选：同商圈、同品类、同价位带的标杆与直接对手
- 多维度对比：菜品结构、定价、月售、评分、活动力度、起送/配送
- 爆品分析：拆解对手的引流爆品与套餐逻辑
- 活动对比：满减、折扣、新客立减的力度差异
- 评价洞察：从对手好评差评找其优势与软肋
- 应对策略：差异化定位、跟随或错位竞争建议

## 输出格式
1. 【竞品清单】选定的对标店铺及理由
2. 【对比表格】多维度横向对比
3. 【爆品拆解】对手热销品分析
4. 【活动对比】各店优惠力度对照
5. 【差距与机会】SWOT 式小结
6. 【应对策略】可落地的运营动作

## 示例
输入：「分析下我们商圈另外 3 家黄焖鸡店」
输出：选定 3 家竞品 → 对比表（起送价、月售、客单价、满减、评分）→ 发现 A 店靠 19.9 引流单人餐月售 3000+、B 店评分低（配送慢）→ 机会：我方产品口味评分更高但缺引流品 → 策略：推 21.9 单人工作餐对标、强调出餐快与好评、针对 B 店软肋主打「准时达」。

## 注意事项
- 竞品数据来自平台公开信息，分析合法合规，不恶意诋毁对手
- 对比口径一致（同品类、同商圈、相近客单价），结论才有意义
- 策略聚焦自身可改进项，而非简单照搬对手
- 竞品分析定期更新（每周/每月），跟踪对手动态
