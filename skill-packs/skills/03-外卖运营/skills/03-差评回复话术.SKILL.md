# 差评回复话术

## 适用场景
外卖店铺出现差评（口味、配送慢、漏餐、撒漏、分量不足、服务差）时，运营专员需要及时、得体地回复，既安抚当事顾客，也向潜在顾客展示负责态度，降低差评对转化和评分的影响。

## 核心能力
- 覆盖高频差评类型：口味/分量、配送时效、漏餐错餐、包装撒漏、服务态度、卫生
- 回复四要素：真诚致歉 + 说明原因 + 补偿方案 + 改进承诺
- 区分公开回复与私信沟通两套话术
- 针对不同星级（1-3 星）的语气与力度差异
- 引导顾客修改/追加评价的柔性表达
- 恶意差评的申诉与理性回应

## 输出格式
1. 【差评类型】问题归类与严重度
2. 【公开回复】可直接贴到平台的回复文案
3. 【私信话术】一对一沟通与补偿话术
4. 【补偿建议】补发/退款/赠券方案
5. 【改进动作】内部防止复发的措施

## 示例
输入：「顾客差评说送过来汤全洒了，饭也凉了，给 1 星」
输出：类型=包装撒漏 + 配送时效 → 公开回复：「非常抱歉让您收到这样的餐品！汤汁洒漏是我们包装和配送没把控好，已第一时间联系您全额退款并重新补送一份，同时升级了防洒封口，感谢您的监督」→ 私信：致歉 + 退款 + 赠 10 元券 → 改进：更换防漏汤碗 + 提醒骑手平稳配送。

## 合规红线
- 24 小时内回复，不拖延、不删除或辱骂顾客
- 回复不泄露顾客手机号、地址等隐私信息
- 补偿在权限内执行，不随意承诺无法兑现的内容
- 涉及疑似食品安全问题（异物、变质）高度重视，保留记录、主动处理，不推诿
