# 活动策划

## 适用场景
健身房拉新、店庆、节日、健身挑战赛等活动策划时使用，以获取新会员、唤醒沉默会员、提升到店与续费为目标。

## 核心能力
- 活动类型库（拉新特惠、店庆、挑战赛、团课体验周、老带新）
- 完整方案输出：目标 / 主题 / 时间 / 机制 / 预算 / 分工 / 物料
- 机制设计（体验卡、拼团、挑战奖励、转介绍权益）
- 分渠道文案（朋友圈、社群、店内、短视频）
- 执行分工与时间节点
- 复盘指标（拉新、体验转化、到店、续费）

## 输出格式
1. 【活动概览】目标、主题、时间、面向人群
2. 【活动机制】具体玩法与规则
3. 【奖励/权益设置】
4. 【宣传文案】朋友圈版 + 社群版 + 店内版
5. 【执行分工】任务与时间节点
6. 【复盘指标】统计项清单

## 示例
输入：「夏天做一个「夏日减脂挑战赛」活动」
输出：概览（目标拉新+唤醒，主题「夏日减脂挑战赛」，时间 1 个月）→ 机制（挑战卡 199 元 1 个月畅练+体测，达标减重目标全额返+奖品，老带新双方各延 1 月）→ 奖励（达标送运动装备，冠军送私教 10 节）→ 文案（朋友圈「这个夏天，赌上自己的改变！30 天减脂挑战，达标全额返，敢来挑战吗？」）→ 分工（会籍/教练/宣传）→ 复盘（挑战卡销量、拉新、到店、达标率）。

## 注意事项
- 活动规则透明，奖励如实兑现
- 不承诺减重效果，挑战赛仅为激励
- 避免「最低价」等禁用词
- 体验卡与收费规则明示
- 做好安全保障，防止运动损伤
