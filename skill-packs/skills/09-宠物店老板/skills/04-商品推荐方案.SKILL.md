# 商品推荐方案

## 适用场景
宠物店为不同宠物推荐主粮、零食、用品、保健品的搭配方案，设计关联销售话术与门店促销活动，优化选品结构与陈列。

## 核心能力
- 按画像推荐：宠物种类 / 年龄段 / 体型 / 健康状况（软便、泪痕、肥胖）匹配对应商品与推荐理由
- 关联销售组合：主粮 + 冻干零食、洗护 + 梳子、驱虫 + 环境消毒、猫砂 + 除臭剂，每组给搭配理由话术
- 选品结构建议：引流款 / 利润款 / 形象款的品类配比，新品引入评估要点
- 促销方案设计：会员日、囤粮节（满减/囤货装）、新品试吃、洗护 + 商品捆绑
- 陈列与动线建议：洗护区旁挂洗护用品、收银台旁放零食试吃
- 推荐话术：场景化讲卖点（成分、适口性、实测反馈），不硬推高价

## 输出格式
1. 【宠物画像与需求】
2. 【推荐清单】商品 / 推荐理由 / 价格带
3. 【关联搭配】组合方案与搭配话术
4. 【促销/陈列建议】（如适用）
5. 【推荐话术】可直接对客使用

## 示例
输入：「一只 1 岁肠胃敏感的法斗，主人想换粮，顺便搭点别的」
输出：推荐：低敏单一蛋白主粮（鸭肉/鱼肉配方，无谷优先），七日换粮过渡表 → 关联：益生菌（换粮期拌粮调理）+ 慢食碗（法斗进食快易吐）+ 洁齿骨（短鼻犬牙结石高发）→ 话术：「法斗肠胃本来就玻璃胃，换粮这周新旧掺着来，搭盒益生菌拌进去过渡更稳……」→ 促销建议：换粮套装（粮 + 益生菌）立减 20，试吃装先买小包看适口性。

## 注意事项
- 保健品类商品不宣称治疗功效（如「治软便、治泪痕」），定位为营养补充，疾病问题引导就医
- 推荐以宠物实际需求出发，不向客户推销过期风险品或库存积压品时隐瞒效期
- 涉及召回批次商品立即下架并主动通知已购客户
- 处方粮须提示遵医嘱使用，不自行推荐替代
