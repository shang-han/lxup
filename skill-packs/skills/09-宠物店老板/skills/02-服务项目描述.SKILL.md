# 服务项目描述

## 适用场景
宠物店梳理与上架服务项目：编写美容洗护、造型、SPA、寄养等项目的介绍文案与价目表，设计套餐组合，用于门店展示、团购平台与社群宣传。

## 核心能力
- 项目描述撰写：按「服务内容 + 流程明细 + 适用宠物 + 服务时长」结构写清每个项目，突出专业与安心
- 价目表设计：按体型（小型/中型/大型犬、猫）分级定价，洗护、精修、造型分层清晰
- 套餐组合设计：洗护月卡/季卡、洗护 + 驱虫组合、寄养 + 洗护节假日套餐
- 卖点提炼：一宠一消毒、全程可观看/视频反馈、美容师资质、接送服务
- 寄养项目规范文案：房型说明、每日护理内容、接送与探视（视频）规则、预订须知
- 平台适配：团购平台短版、门店海报版、社群长版三种长度

## 输出格式
1. 【项目清单】项目名 / 内容 / 时长 / 分级价格表
2. 【项目详情文案】可直接上墙或上架的完整描述
3. 【套餐方案】组合内容与定价逻辑
4. 【预订/服务须知】免责与注意事项条款要点

## 示例
输入：「帮我写一份猫咪洗护项目的介绍和价格」
输出：项目：猫基础洗护（剪指甲 + 清耳 + 挤肛门腺 + 两遍温水浴 + 护毛素 + 低温吹干 + 基础梳理），约 60-90 分钟 → 分级价：短毛猫 1xx / 长毛猫 1xx / 开结另计 → 卖点：全程温和保定不硬拽、一猫一消毒、可视频反馈 → 套餐：洗护 3 次卡 9 折 + 送体外驱虫 1 次 → 须知：疫苗齐全方可服务、严重打结需主人同意剃毛、应激反应明显的猫建议缩短流程。

## 注意事项
- 服务协议中明确美容风险条款（既往病史告知、应急就医授权、赔付上限），接宠时请主人签字确认
- 寄养项目须明确疫苗要求（疫苗齐全、已驱虫）、禁收情形（发情期、传染病疑似）与紧急情况处置流程
- 价格公示透明，附加收费项（开结、药浴、剃毛）事先告知并确认，避免结算纠纷
- 不夸大服务效果（如「药浴治皮肤病」），药浴仅作为护理辅助，皮肤病引导就医
