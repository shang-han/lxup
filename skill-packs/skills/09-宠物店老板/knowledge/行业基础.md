# 行业基础

## 行业概况
宠物行业市场规模超 3000 亿，猫狗消费占主导，养宠人群年轻化、情感消费特征明显，「它经济」抗周期性强。门店业务涵盖宠物美容洗护、用品零售、活体销售、寄养，部分门店延伸至医疗、训练。美容洗护是高频引流入口，用品复购是利润基础，寄养在节假日是爆发点。核心竞争力：专业服务 + 信任感 + 社群客情，线上种草（小红书/抖音）获客占比越来越高。

## 专业技能体系

### 硬技能
- 宠物洗护美容技法（剪毛/拉毛/染色）
- 常见宠物疾病识别与应急处理
- 宠物食品与用品选品（品牌/功效/年龄段）
- 活体销售合规与健康保障
- 寄养服务流程标准化

### 软技能
- 宠物主人共情沟通
- 社区宠物社群运营

## 常用术语
- 客单价：平均每次消费金额
- 美容转化率：进店客户中购买美容服务的比例
- 洗护频次：客户月均到店洗护次数
- 用品复购率：宠物食品/用品的回购比例
- 寄养入住率：寄养房间的使用率
- 活体保障期：售出宠物的健康保障期限
- 连带率：每单同时购买的商品件数

## 核心工作流程
每日：宠物接待与健康初检 → 美容洗护服务 → 商品推荐与销售 → 客户沟通与回访 → 环境与笼具消毒
每周：库存盘点与补货 → 全店卫生消毒 → 活体健康巡检 → 会员洗护提醒
每月：经营数据分析（洗护量、复购、寄养率）→ 营销活动策划 → 供应商对账

## 常见问题
Q：宠物美容风险怎么控制？
A：接宠时做健康初检（皮肤、外伤、情绪）→ 签署服务协议明确免责与赔付条款 → 购买门店责任险 → 准备应急就医预案，短鼻犬、心脏病犬等特殊犬只格外谨慎。

Q：活体销售纠纷多怎么办？
A：设定健康保障期并写入合同、提供疫苗驱虫记录、售后一周内回访、出现争议按合同条款处理，全程留痕。

Q：怎么提高用品复购？
A：建立会员档案记录宠物年龄与食品用量，按消耗周期推送补货提醒，搭配会员价与积分体系。

Q：节假日寄养订不上怎么办？
A：提前一个月开放预订并收定金，按房型分级定价，安装摄像头支持远程查看缓解宠主焦虑。
