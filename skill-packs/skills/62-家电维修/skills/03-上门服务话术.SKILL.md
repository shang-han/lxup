# 上门服务话术

## 适用场景
从客户报修电话到上门完工收款的全流程沟通，标准化服务动作与话术，提升专业形象、客户信任与好评率。

## 核心能力
- 接单询问话术：品类、品牌型号、故障现象、购买年限、地址与时间
- 引导客户「翻译」故障描述的提问技巧（不会描述时问现象：有没有声音、灯亮不亮、有没有异味）
- 到场规范话术：自我介绍、出示工牌/鞋套、询问故障复现
- 施工中沟通：发现问题实时告知、增项先确认再动工
- 收费前确认话术：先报价后维修、完工演示正常再收款
- 应对客户质疑（「这么贵」「能不能便宜」）的话术

## 输出格式
1. 【环节】当前所处服务环节
2. 【标准话术】可直接使用
3. 【关键动作】配套的服务动作（鞋套、拍照、演示）
4. 【避坑提醒】该环节容易引发纠纷的点

## 示例
输入：「客户打电话说洗衣机坏了，给我一套接单到上门的话术」
输出：接单：「您好，XX 家电维修，您家洗衣机是什么品牌型号？用了几年了？具体什么现象——是不转、不排水还是报错？方便的话拍个铭牌照片发我」→ 到场：「您好我是 XX 维修的，穿个鞋套，先给我演示下故障」→ 报价：「检查下来是 XX 问题，配件加人工一共 XXX，您看我再修？」→ 完工：现场演示一遍正常 → 收款并告知质保。

## 注意事项
- 「先报价、后维修」是铁律，客户口头同意后再动工
- 拆机前当客户面拍照记录原状，防止事后争议
- 客户家老人独自在家时，大额维修建议电话与其子女确认
- 完工带走旧件并询问是否保留，细节决定好评
