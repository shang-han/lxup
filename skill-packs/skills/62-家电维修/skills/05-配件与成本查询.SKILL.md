# 配件与成本查询

## 适用场景
维修前确认配件型号与采购渠道、估算维修成本、向客户解释费用构成，以及判断某个故障值不值得修。

## 核心能力
- 按故障列出可能涉及的配件/元器件清单（如空调主板、压缩机、电容、传感器）
- 配件分级说明：原厂件 / OEM 通用件 / 拆机件的区别与适用场景
- 采购渠道参考：品牌售后配件、电子市场、线上平台（各渠道时效与价格特点）
- 元器件级维修的成本逻辑：板级换件 vs 芯片级维修的差异
- 维修成本估算框架：配件价 + 人工 + 风险（如制冷剂、焊接工时）
- 协助判断维修经济性（配件价超过新机价格一定比例时建议换新）

## 输出格式
1. 【涉及配件清单】名称 / 规格或型号线索 / 分级
2. 【采购渠道建议】推荐渠道与到货时效
3. 【成本估算】配件 + 人工 + 其他，给区间
4. 【维修方式选择】换件 vs 板级修复
5. 【经济性判断】修 vs 换的建议

## 示例
输入：「一台 5 年的变频空调外机主板坏了，帮我分析下怎么修划算」
输出：配件：外机变频主板（原厂件约 XXX-XXX 元，需按整机型号订货）→ 渠道：品牌售后配件最稳（2-4 天）、线上按型号搜拆机板更快但风险高 → 成本：原厂板 + 人工约 XXX 区间 → 方式：主板芯片级维修便宜但考验手艺、稳定性不如换板 → 建议：5 年机龄值得修，优先原厂板，给客户报含质保的换板方案。

## 注意事项
- 变频、高端机型主板务必按完整型号订货，版本不对装不上
- 拆机件/二手板要提前告知客户并说清质保差异，别当新件报
- 制冷剂（R22/R410A/R32）价格波动大且 R32 易燃，成本与安全都要算进去
- 冷门机型配件可能断供，接单前先确认配件可得性再承诺
