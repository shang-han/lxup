# 修后使用指导

## 适用场景
维修收尾时给客户的使用注意事项与保养建议，既减少因使用不当导致的返修，也让客户感到专业贴心，促进好评与转介绍。

## 核心能力
- 按品类生成修后注意事项（空调修后观察、冰箱修后静置通电时间、洗衣机修后试机程序）
- 针对本次故障原因给预防建议（如排水泵堵塞→定期清理过滤器）
- 日常保养清单：按季节/按月/按次的维护项
- 常见错误使用纠正（空调设 16 度、冰箱塞太满、热水器常年不拔电）
- 口语化表达，适合当面说或微信发给客户
- 附质保提醒与复购引导（「有问题随时找我」）

## 输出格式
1. 【修后即刻注意】当天/当次要做什么
2. 【针对本次故障的预防】避免复发的 1-2 个习惯
3. 【日常保养清单】3-5 条
4. 【错误使用纠正】如有
5. 【一句话收尾】质保提醒 + 有问题联系

## 示例
输入：「刚给客户换了冰箱化霜传感器，修完跟他说点什么」
输出：即刻注意：通电后先空载运行 2 小时再放食物 → 预防：门封条每月擦一擦检查密封、不要长时间敞门 → 保养：冷凝器后盖半年除尘一次、食物别塞超八成满 → 纠正：冷藏别调太低、热食放凉再入箱 → 收尾：「这个部位质保 90 天，再结冰随时给我打电话」。

## 注意事项
- 指导要具体可执行（「每月清理一次过滤器」而不是「注意保养」）
- 控制在 3-5 条重点，说太多客户记不住
- 当面说 + 微信文字双份，微信版方便客户留存也加深印象
- 保养建议顺带提一句其他家电也可检修，自然的增项机会
