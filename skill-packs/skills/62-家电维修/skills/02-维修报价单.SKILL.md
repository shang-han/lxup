# 维修报价单

## 适用场景
上门检查后向客户报价、电话里给大致费用区间、客户对价格有疑虑时的费用解释。目标是报价透明、客户听得懂、减少「乱收费」质疑。

## 核心能力
- 三段式报价结构：上门/检测费 + 配件费 + 人工（维修）费
- 按品类给常见故障费用区间参考（师傅按本地行情与客户机型调整）
- 修 vs 换对比：原厂件/通用件/拆机件价格与质保差异
- 费用解释话术：把配件成本、技术难度、质保承诺讲清楚
- 客户嫌贵时的应对：拆分费用、给配件自购选项、强调质保
- 报价单附质保条款（维修部位质保期、返修免费）

## 输出格式
1. 【故障结论】查明的故障点
2. 【费用明细】上门检测费 / 配件（名称+价格）/ 人工费
3. 【合计】总费用与付款方式
4. 【方案选项】如有：原厂件方案 vs 通用件方案
5. 【修/换建议】机龄高时给客观建议
6. 【质保说明】质保范围与期限

## 示例
输入：「洗衣机不排水，检查是排水泵坏了，给客户报价」
输出：故障结论：排水泵卡滞烧毁 → 明细：上门检测费 XX（维修可抵扣）、通用排水泵配件 XX 元、更换人工 XX 元 → 合计 XXX 元 → 说明含清理排水管路 → 质保：同一故障 90 天内返修免费 → 话术：「排水泵是易损件，换完跟新的一样，比买台洗衣机划算多了」。

## 注意事项
- 先报检测规则再干活：「检查出问题报价，您不修只收上门检测费」
- 配件报价留一点浮动空间但别虚高，客户可比价，诚信是长期生意
- 主动说出质保期，这是对抗「乱收费」印象最有效的武器
- 大额维修（换主板、压缩机）建议客户考虑机龄再决定，别为了这单硬劝修
