# 行业基础

## 行业概况
家电维修是刚需本地生活服务，存量家电保有量巨大，空调、冰箱、洗衣机、热水器、电视为五大主力品类。行业格局：品牌官方售后（保内为主）+ 第三方连锁平台（啄木鸟、鲁班等）+ 个体师傅（社区口碑+平台接单）。痛点是「小病大修、乱收费」的信任问题，透明报价与质保承诺是个体师傅突围的关键。夏季空调、冬季热水器/取暖为绝对旺季。

## 专业技能体系
### 硬技能
- 常见家电故障诊断（空调/冰箱/洗衣机/热水器/电视）
- 电路板级维修与元器件替换
- 制冷系统维修（加氟/检漏/压缩机）
- 配件渠道与维修定价
- 上门服务流程标准化
### 软技能
- 客户故障描述翻译
- 维修后使用指导

## 常用术语
- 板级维修：直接更换整块电路板；芯片级维修：定位并更换板上的具体元器件
- 加氟/充注：为制冷系统补充制冷剂（R22/R410A/R32 型号不可混用）
- 检漏：用肥皂水、电子检漏仪查找制冷剂泄漏点
- 压缩机：制冷系统心脏，更换成本高，是「修还是换」的关键判断点
- 故障代码：机器自检报错（如 E1、F5），对应传感器/模块故障
- 铭牌：机身参数标签，含型号、生产日期、制冷剂类型，接单必看
- 上门费/检测费：上门检查的基础收费，通常维修后可抵扣
- 拆机件：从旧机拆下的可用配件，价格低但质保短

## 核心工作流程
接单：询问品类型号 → 了解故障现象 → 初步判断与费用预期 → 约时间
上门：出示身份 → 复现故障 → 检测定位 → 报价确认 → 维修 → 试机演示
收尾：讲解使用保养 → 收款 → 告知质保 → 引导好评
经营：配件库存管理 → 老客户维护 → 平台/社区接单渠道运营

## 常见问题
Q：客户说「你们是不是故意说坏了好多收钱」怎么回应？
A：当面检测展示故障（比如让客户看烧黑的元件、测给数据），先报价后维修，主动给质保期，透明是最好的回应。

Q：修完没几天又坏了（返修）怎么处理？
A：质保期内免费返修是行规也是招牌，态度要好、别扯皮；同时分析是使用不当还是没修彻底， latter 要反思手艺。

Q：加氟是不是万能借口？
A：制冷剂不会无故减少，缺氟必有漏点，只加氟不检漏就是糊弄，找到漏点修复才是正经维修。

Q：客户机型太老配件买不到怎么办？
A：提前确认配件可得性再承诺；实在没有就如实告知并给换新建议，收少量检测费，诚实比硬修更留客。
