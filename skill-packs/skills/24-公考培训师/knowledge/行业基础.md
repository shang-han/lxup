# 行业基础

## 行业概况
公务员考试（国考+省考）每年报名人数超 200 万，平均竞争比约 60:1，热门岗位可达数千比一。考试科目为行政职业能力测验（行测）+ 申论，部分岗位加试专业科目，笔试后进入结构化/无领导小组面试。公考培训市场规模超 300 亿元，培训师的核心能力在于题库研究、答题技巧提炼、命题趋势把握与政策解读。学员多为应届毕业生与在职备考者，备考周期紧张、应试技巧导向强，上岸率与通过率是机构口碑核心。

## 常用术语
- 行测：行政职业能力测验，客观选择题，含常识/言语/数量/判断/资料五大模块
- 申论：主观写作题，含归纳概括、提出对策、综合分析、应用文、大作文
- 进面线：进入面试的最低笔试分数线
- 报录比：岗位报名人数 / 招录人数，衡量竞争激烈程度
- 结构化面试：按固定题目与流程逐一问答的面试形式
- 无领导小组讨论：多人就议题讨论、观察综合表现的面试形式
- 调剂：未进面或放弃的考生可补报空缺岗位
- 国考/省考：中央机关及其直属机构 / 各省市地方公务员招录

## 核心工作流程
备考期：考情分析 → 知识点讲解 → 分模块练习 → 错题复盘
强化期：套题训练 → 答题技巧 → 时间分配 → 模考点评
冲刺期：全真模拟 → 查漏补缺 → 心态调整
面试期：结构化/无领导讲解 → 模拟练习 → 仪态训练 → 点评

## 常见问题
Q：哪个岗位比较好考？
A：基层岗位、偏远地区、限制条件多（专业/学历/户籍/基层经历）的岗位报录比更低、分数线更低，可结合自身条件筛选「限制多」的岗位降低竞争。

Q：行测做不完怎么办？
A：行测时间紧是常态，要先做擅长模块、放弃高耗时低正确率的题，按模块分配时间并大量限时训练形成节奏。

Q：申论怎么提分？
A：小题重在「采点得分」——精准提炼材料要点、条理清晰；大作文重在立意准确、结构规范、结合材料，多练多批改。

Q：国考和省考能同时准备吗？
A：可以，科目相通、内容互补，国考通常在前（11-12月），省考在后（3-4月联考），一套基础可覆盖多场考试。
