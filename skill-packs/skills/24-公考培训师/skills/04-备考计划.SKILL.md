# 备考计划

## 适用场景
公考培训师为不同基础（应届/在职/二战）、不同目标（国考/省考/事业单位）、不同备考周期的学员制定分阶段、可执行的备考计划，明确各模块任务、刷题量与模考节点。

## 核心能力
- 分阶段规划：基础（知识点）→ 强化（刷题）→ 冲刺（模考）三段式
- 行测分模块时间配比：根据强弱项动态分配
- 申论训练节奏：小题高频练、大作文定期写、持续积累素材
- 在职 vs 全职备考的差异化时间安排
- 刷题与错题复盘机制（错题本、限时训练）
- 模考节点与阶段测评安排
- 时政与素材积累的长期任务嵌入

## 输出格式
1. 【基本情况】目标 / 基础 / 周期 / 可投入时间
2. 【阶段划分】各阶段时间、目标、重点
3. 【行测模块安排】分模块任务与配比
4. 【申论训练安排】频次与方法
5. 【模考与复盘】关键节点
6. 【日常习惯】素材积累、时政关注

## 示例
输入：「在职备考省考，还有4个月，每天能学2小时周末多点，行测基础一般，帮我排计划」
输出：评估（在职、4 个月、行测弱）→ 阶段（基础 1.5 月、强化 1.5 月、冲刺 1 月）→ 行测模块（言语判断每天保量、数量资料重点突破、常识利用碎片时间）→ 申论（每周 2 道小题 + 1 篇大作文）→ 模考（每两周一次全真限时）→ 日常（每天看人民日报评论积累申论素材）→ 时间配比表（工作日 2h、周末 5-6h）。

## 注意事项
- 计划要可执行，在职备考尤其要利用碎片时间、留弹性
- 强弱项差异化投入，弱项优先但不忘保强项
- 限时训练贯穿始终，行测本质是速度考试
- 申论靠积累与批改，不能只看不练
- 关注报名时间节点，避免错过报考/缴费/打印准考证
