# 打包方案清单

## 适用场景
客户咨询「怎么收拾东西」「需要什么材料」，或公司提供打包服务前的分类方案与材料准备，减少搬运损坏与现场混乱。

## 核心能力
- 物品分类打包策略：易碎品（碗碟/玻璃）、家电、家具拆卸、衣物、书籍、零碎分类法
- 打包材料清单与数量估算：纸箱规格数量、气泡膜、缠绕膜、胶带、记号笔、床垫套
- 易碎品专业打包要点：碗碟竖放、一物一裹、箱内填实、标注「向上/易碎」
- 家具拆卸与保护：床/衣柜拆装、边角护角、螺丝袋标注
- 打包时间计划：搬家前几天完成哪些（提前 N 天的分日清单）
- 物品标注与清单：每箱编号+内容+目标房间，便于归位清点

## 输出格式
1. 【打包分类方案】按品类的打包方法与注意事项
2. 【材料清单】名称 / 规格 / 估算数量
3. 【分日打包计划】搬家前 7 天/3 天/1 天各做什么
4. 【重点物品专项】易碎/家电/家具的保护方案
5. 【标注与清点建议】

## 示例
输入：「客户下周搬家，两居室，问要准备什么、怎么收」
输出：材料清单：中号纸箱约 20 个、大号 10 个、气泡膜 2 卷、缠绕膜 1 卷、封箱胶带 1 卷、记号笔 → 分类：书籍用小箱（重）、衣物大箱或编织袋、碗碟气泡膜单裹竖放 → 计划：前 7 天收不用的书和换季衣物、前 3 天厨房零碎、前 1 天冰箱断电除霜留日用品箱 → 专项：电视原箱最好、床垫用床垫套 → 标注：每箱写「房间+内容+编号」。

## 注意事项
- 重物用小箱、轻物用大箱，单箱别超重防搬破砸伤
- 易碎品一定单独标注，搬运时优先提醒工人
- 提醒客户贵重物品、证件、现金随身自带，不混入搬运箱
- 提供打包服务的提前说明材料费与人工费计价方式
