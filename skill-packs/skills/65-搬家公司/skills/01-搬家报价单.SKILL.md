# 搬家报价单

## 适用场景
客户咨询搬家费用、上门或视频评估后出正式报价、解释收费构成应对「搬家怎么这么贵」的疑问。透明报价是搬家公司口碑的核心。

## 核心能力
- 收费维度拆解：起步车费 + 里程 + 楼层（有无电梯）+ 人工 + 特殊物品 + 拆装 + 距离搬运（停车点到楼门）
- 物品量评估：按户型/房间数估车次与人工（一居/两居/三居对应车型）
- 特殊物品加价：钢琴、保险柜、大鱼缸、红木家具、大件家电
- 拆装费用：床、衣柜、空调移机等分项
- 一口价 vs 计时/计件两种模式的适用与报价
- 报价说明条款：增项确认、保险、付款方式

## 输出格式
1. 【搬家概况】起止地址类型、户型、有无电梯、物品概况
2. 【费用明细】表格：项目 / 计价方式 / 数量 / 单价 / 小计
3. 【特殊物品/拆装】单独列项
4. 【合计与付款】总价或区间、定金、尾款
5. 【报价说明】有效期、增项规则、保险与赔偿

## 示例
输入：「客户两居室 5 楼无电梯搬到电梯房 12 楼，有台钢琴，怎么报」
输出：明细：4.2 米车厢起步费 + 里程费 + 两侧楼层人工搬运费（5 楼无电梯按层加价、12 楼有电梯免楼层费）+ 人工 X 名 + 钢琴专项搬运费（单独报价含保护）+ 床/衣柜拆装费 → 合计 XXX 区间 → 建议先视频看物估准 → 说明：「报价含基本保护材料，钢琴单独投保，现场无新增大件不加价」。

## 注意事项
- 报价前尽量视频看房估物，避免现场「东西比说的多」导致加价纠纷
- 楼层、有无电梯、停车距离这三个最易被忽略的加价项要问清讲明
- 给区间并说明「按实际物品量，多退少补」比报死价更稳妥
- 报价单注明含保险范围，贵重物品单独声明保价
