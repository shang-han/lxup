# 客户沟通话术

## 适用场景
客户首次咨询搬家、对搬家感到焦虑繁琐、犹豫比价、搬家前确认与当天协调等场景，用专业贴心的沟通促成信任与成交。

## 核心能力
- 咨询应答话术：快速问清起止地址、户型楼层、物品量、时间，给初步方案
- 搬家焦虑安抚：搬家是高压事件，帮客户理清流程、给确定性（「这些我们全包，您只管 XX」）
- 比价应对话术：强调透明报价、无坐地加价、含保险，不靠低价陷阱
- 预约确认话术：搬家前一天再次确认时间地址、提醒准备事项
- 当天协调话术：车辆到达通知、现场临时变更的沟通
- 搬家后跟进：确认完好、索要好评与转介绍

## 输出格式
1. 【沟通环节】咨询/比价/确认/当天/跟进
2. 【标准话术】可直接使用
3. 【关键信息】该环节要拿到/确认的
4. 【安抚要点】针对焦虑点的话术

## 示例
输入：「客户第一次搬家很焦虑，问东问西怕被坑，怎么聊」
输出：安抚+理清：「姐您第一次搬家紧张很正常，咱把事情捋顺就简单了：您负责把衣物零碎收进箱子，大件家具拆装、搬运、保护、运输我们全包，搬前给您出明细报价，现场绝不加价」→ 建立确定性：给一份「搬家前 3 天清单」让她照着做 → 比价应对：「您多比两家是应该的，重点问清有没有楼层费、会不会现场加价、东西坏了管不管，我们这三项都写在报价单里」→ 跟进：搬家前一天确认、当天车到人到通知。

## 注意事项
- 搬家客户买的是「省心」，话术核心是给确定性和减负感
- 「现场绝不坐地加价」要承诺并做到，这是行业最大痛点也是最大卖点
- 首次搬家、带老人小孩、跨城的客户焦虑更重，多一分耐心多一个回头客
- 好评与转介绍在搬家顺利完成、客户松口气的当下提出最有效
