# 行业基础

## 行业概况
搬家行业是典型的重服务、重口碑的本地生活服务，需求来自租房换房、二手房交易、公司搬迁，季节性上春秋与毕业季为旺季。行业长期痛点是「坐地加价、暴力搬运、损坏扯皮」，透明报价、规范打包与保险赔付是正规公司突围的关键。经营模式从单兵面包车向品牌化、平台化发展，客单价从几百到上千元不等，复购靠口碑与房产中介渠道。成本结构以人工和车辆为主，调度效率直接决定利润。

## 专业技能体系
### 硬技能
- 搬家报价体系（距离/楼层/物品量/特殊物品）
- 打包技法（易碎品/家电/家具拆卸/衣物/书籍分类）
- 车辆调度与路线规划
- 贵重物品保护与保险
- 工人搬运安全培训
### 软技能
- 客户搬家焦虑安抚
- 投诉与物品损坏处理

## 常用术语
- 楼层费：无电梯时按楼层收取的人工搬运费
- 平面搬运距离：货车停车点到楼门的距离，超距加收
- 一口价：看房估物后的固定总价，区别于按车/按时计费
- 保价：客户对贵重物品声明价值并付费，损坏按保价赔付
- 车型匹配：按物品量选金杯/4.2 米/6.8 米等厢式货车
- 拆装费：床、衣柜、空调等需拆解搬运再组装的费用
- 坐地加价：到现场以各种理由临时涨价，行业最大差评来源
- 出门条：小区物业开具的物品放行证明，搬家前需客户办理

## 核心工作流程
接单：咨询问清地址/户型/物品/时间 → 视频或上门估价 → 出报价单 → 确认预约
准备：派车派班组 → 路线与限行规划 → 提醒客户打包准备
执行：到场 → 打包保护 → 拆卸 → 装车 → 运输 → 卸车归位 → 组装
收尾：客户清点确认 → 收款 → 跟进好评 → 损坏复盘

## 常见问题
Q：客户总怕现场加价，怎么打消顾虑？
A：视频看房估物后出一口价报价单，写明「现场无新增大件不加价」，并把楼层、距离、拆装这些易漏项都列清，透明是最大的定心丸。

Q：物品损坏了客户要求高额赔偿怎么办？
A：先共情响应不硬顶，收集照片与保价情况判定责任，给维修/折价/保价赔付多选项；协商不成走保险或第三方调解，全程留痕，别为省钱逼出差评和投诉。

Q：怎么提高车辆和人工的利用率？
A：按区域就近派单减少空跑，长短途搭配排时段，跨城长途早出发避高峰；车型匹配别「大车拉小活」也别一车装不下跑两趟。

Q：旺季人手不够怎么保质量？
A：提前储备临时工并做搬运安全与打包培训再上岗，宁可少接单也别让生手砸了口碑，损坏赔偿和差评的代价远高于少赚的一单。
