# 行业基础

## 行业概况
服装零售行业市场规模超 4 万亿，线上线下融合加速，门店经营涵盖女装、男装、童装、运动、潮流集合等品类。行业季节性强、潮流变化快、库存压力大，是典型「靠周转和眼光吃饭」的生意。核心挑战在于选品精准度和库存控制：选对款、卖得快、压货少才能盈利。利润来自进销差价，但季末清货折价会大幅侵蚀利润。当前趋势是私域运营（朋友圈/社群/直播）成为门店重要增量，线上线下同款同价、到店体验 + 私域复购成为主流模式，单体店靠选品品味、搭配服务和熟客经营立足。

## 专业技能体系

### 硬技能
- 进货渠道开发（批发市场/工厂/尾货）
- 搭配陈列技巧（色彩/风格/场景）
- 库存周转与季末清仓策略
- 尺码管理与退换货处理
- 面料基础识别

### 软技能
- 客户体型分析与推荐
- 时尚趋势敏感度

## 常用术语
- 售罄率：销售数量占进货数量的比例，衡量款式畅销程度
- 折扣率：折后价占原价的比例，清货时的让利幅度
- 连带率：平均每单购买的件数，反映搭配销售能力
- 新旧比：新品销售占比，反映上新节奏与库存健康度
- 库龄：库存积压的天数，库龄越长越需清理
- 坪效：每平米营业面积创造的销售额
- 动销率：有销售的 SKU 占全部 SKU 的比例
- 客单价：平均每单消费金额

## 核心工作流程
每日：店铺陈列整理 → 客户接待 → 试穿搭配服务 → 成交跟进 → 私域维护
每季：选品订货 → 上新陈列 → 季中补货 → 季末促销清仓
每月：销售数据分析 → 库存盘点 → 滞销处理 → 会员回访

## 常见问题
Q：库存积压怎么办？
A：季末梯度打折清仓，畅销款搭配滞销款组合销售，跨店调拨到有需求的门店，必要时做福袋/特卖快速回款。

Q：怎么提高连带率？
A：做成套搭配推荐、关联陈列，培训导购一衣多穿话术，配合满额/买赠活动引导多件购买。

Q：怎么做好私域复购？
A：把到店顾客沉淀到微信，按偏好精准推送上新，做会员日和生日关怀，朋友圈内容种草不刷屏。

Q：选品总踩雷怎么办？
A：小批量试销验证再补货，关注动销数据及时汰换，多参考同行畅销款和潮流趋势，避免凭感觉压大货。
