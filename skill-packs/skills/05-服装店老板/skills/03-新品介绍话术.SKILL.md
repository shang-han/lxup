# 新品介绍话术

## 适用场景
服装新品上架时，导购需要一套打动顾客的介绍话术，讲清面料、版型、设计卖点和上身效果，引导顾客试穿并促成成交，也用于社群和直播的新品讲解。

## 核心能力
- 卖点提炼：面料、版型、工艺、设计细节的差异化表达
- FAB 法则：属性（Feature）→ 优势（Advantage）→ 利益（Benefit）
- 上身效果描述：显瘦、显高、提气质的具象化表达
- 适用人群与场合：帮顾客对号入座
- 试穿引导话术：降低顾客试穿心理门槛
- 异议处理：价格、款式犹豫的应对

## 输出格式
1. 【核心卖点】3 个主打卖点
2. 【FAB 话术】属性-优势-利益的完整表达
3. 【上身效果】场景化描述
4. 【试穿引导】邀请试穿的话术
5. 【异议应对】价格/款式犹豫的回应

## 示例
输入：「新到一款醋酸面料的西装外套，写介绍话术」
输出：卖点=醋酸垂感面料 + 微廓形版型 + 一粒扣设计 → FAB：「这款用的醋酸面料（属性），垂顺有光泽还不易皱（优势），您穿一整天都挺括有质感，特别显高级（利益）」→ 上身效果：微廓形藏肉、一粒扣显腰线、通勤气场足 → 试穿引导：「衣服挂着看不出来，您上身试试感受下这个垂感，不买没关系」→ 异议：「嫌贵」回应面料耐穿、质感对得起价格。

## 注意事项
- 卖点真实，不夸大面料成分与功效（如虚假标注材质）
- 话术自然口语化，避免背诵感和过度推销
- 引导试穿尊重顾客意愿，不强拉硬拽
- 价格异议靠价值塑造，而非一味贬低顾客
