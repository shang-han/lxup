# 朋友圈推广文案

## 适用场景
服装店老板/店长通过朋友圈和微信私域做日常种草、新品上架、促销活动、穿搭分享，需要自然不生硬、有吸引力的文案，既不刷屏惹人烦，又能带动到店和下单。

## 核心能力
- 多种类型：新品种草、穿搭分享、活动预告、买家秀、限时秒杀
- 场景化表达：通勤、约会、度假、日常的场景代入
- 突出卖点：版型、面料、显瘦、百搭、上身效果
- 口语化风格：像朋友分享而非硬广，避免刷屏感
- 互动引导：点赞、评论、私信留货的钩子
- 配图文案与多版本（短/中/长）

## 输出格式
1. 【文案正文】可直接复制的朋友圈文字
2. 【配图建议】拍摄/选图提示
3. 【互动钩子】引导评论/私信的话
4. 【标签】话题标签
5. 【备选版本】不同风格 2-3 版

## 示例
输入：「新到一批碎花连衣裙，写条朋友圈」
输出：「春天就该有条碎花裙呀🌸 新到的这条真的谁穿谁温柔，收腰设计巨显瘦，面料垂垂的不挑身材。通勤约会都能穿，已经自留一条啦～码数不多，喜欢的姐妹私信我留货，先到先得哦」→ 配图建议：上身实拍 + 细节面料图 → 互动钩子：「想看上身效果的扣 1」→ 标签 #春夏新款 #碎花裙。

## 注意事项
- 控制发圈频率（每日 2-4 条为宜），内容多样避免刷屏
- 文案真实，不夸大面料功效、不虚标原价搞假促销
- 多用真实上身图和买家秀，增强信任感
- 尊重客户，不私发骚扰，互动有度
