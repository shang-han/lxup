# 会员维护SOP

## 适用场景
服装店老板/店长需要一套会员维护标准流程，把到店顾客沉淀为会员，通过建档、分层、定期回访和会员日活动提升复购率和顾客忠诚度，做好私域长期经营。

## 核心能力
- 会员建档：记录身材、偏好、消费、生日等信息
- 会员分层：按消费额/频次分为普通/VIP/至尊，差异化维护
- 回访节奏：购后回访、上新提醒、生日关怀、沉睡唤醒
- 会员权益：积分、折扣、专属款、会员日
- 沉睡会员激活：流失预警与召回方案
- 社群运营：会员群的日常维护节奏

## 输出格式
1. 【会员体系】等级与权益设计
2. 【建档内容】会员档案字段
3. 【维护节奏】分阶段的回访 SOP
4. 【激活方案】沉睡会员召回
5. 【会员日活动】专属活动方案
6. 【执行要点】注意事项与话术

## 示例
输入：「帮我做一套会员维护流程，提升老客复购」
输出：会员体系（消费满额升 VIP 享 9 折 + 生日礼）→ 建档字段（尺码/风格偏好/消费记录/生日）→ 维护节奏：购后 3 天回访穿着感受 → 上新精准推送偏好款 → 生日当月专属券 → 沉睡 60 天唤醒券 → 会员日每月 1 次专属折扣 → 执行要点：推送贴合偏好不群发骚扰、回访真诚不硬推销。

## 注意事项
- 会员信息妥善保管，保护客户隐私，不泄露不滥用
- 推送精准有节制，避免频繁群发引起反感
- 维护以真诚服务为核心，不把会员当「韭菜」
- 权益承诺要兑现，失信会损害长期信任
