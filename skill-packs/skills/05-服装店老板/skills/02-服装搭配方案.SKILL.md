# 服装搭配方案

## 适用场景
顾客进店不知道买什么、怎么搭，或店主需要为新品做搭配示范时，根据顾客的身材、肤色、场合和预算，推荐整套穿搭方案，既帮顾客解决「不会搭」，也提升门店连带率。

## 核心能力
- 按身材推荐：梨形/苹果形/H 形等不同体型的扬长避短搭配
- 按场合搭配：通勤、约会、度假、休闲、正式场合
- 色彩搭配：基础色、同色系、撞色的配色逻辑，结合肤色
- 成套方案：上装 + 下装 + 外套 + 鞋包的完整组合
- 一衣多穿：单品的多种搭配提升实用感
- 预算适配：按顾客预算给出组合

## 输出格式
1. 【需求分析】身材/场合/偏好/预算
2. 【搭配方案】整套单品组合
3. 【搭配理由】为什么这样搭（扬长避短/配色）
4. 【一衣多穿】核心单品的其他搭法
5. 【备选方案】不同风格/价位的替换

## 示例
输入：「我梨形身材，下周有场约会，帮我搭一套」
输出：分析=梨形（上身瘦、胯宽）→ 方案：V 领针织衫（显上身瘦）+ 高腰 A 字伞裙（遮胯显腿长）+ 尖头浅口鞋 → 理由：上紧下松平衡比例、高腰拉长腿型、柔和色系显温柔 → 一衣多穿：伞裙也可搭白 T 走休闲风 → 备选：预算更高可换真丝衬衫款。

## 注意事项
- 搭配以顾客实际身材和喜好为准，不强推不适合的款式
- 推荐真诚，不为了连带硬塞不搭的单品
- 尊重顾客预算，提供不同价位选择
- 搭配建议结合店内现有库存，确保能落地成交
