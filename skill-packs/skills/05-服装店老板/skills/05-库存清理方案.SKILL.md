# 库存清理方案

## 适用场景
服装店季末换季或出现滞销压货时，老板/店长需要制定库存清理方案，通过打折、搭配、活动等方式加速出清、回笼资金，同时尽量保住利润和品牌调性。

## 核心能力
- 库存盘点分类：按库龄、款式、码数区分可清/可调/可留
- 清货方式库：折扣、满减、买赠、福袋、特卖会、跨店调拨
- 梯度折扣：分阶段降价，先浅折后深折
- 搭配带动：用畅销款搭滞销款组合销售
- 渠道组合：门店特卖 + 私域秒杀 + 线上尾货
- 损失测算：平衡回款速度与折价幅度

## 输出格式
1. 【库存盘点】待清库存分类清单
2. 【清理策略】分阶段的清货方式
3. 【折扣梯度】各阶段折扣力度
4. 【搭配组合】畅销带滞销的方案
5. 【渠道安排】门店/私域/线上分工
6. 【回款测算】预期回款与损失控制

## 示例
输入：「换季了还压了 200 件夏装，帮我出个清货方案」
输出：盘点分类（按款式和库龄）→ 策略三阶段：① 前 3 天会员专场 7 折 ② 第二周全场 5 折 + 买二送一 ③ 最后做 99 元 3 件福袋清尾 → 搭配：畅销 T 恤搭滞销半裙成套卖 → 渠道：门店特卖区 + 朋友圈秒杀 + 社群接龙 → 测算：预计回款 X 元、综合折扣控制在 5-6 折、两周内清完。

## 注意事项
- 折扣真实，明示原价与折后价，不搞虚假原价
- 清货也要保证商品质量，瑕疵品单独说明处理
- 把握清货节奏，避免长期打折损害品牌与正价销售
- 优先调拨到有需求的门店，减少直接折价损失
