# 行业基础

## 行业概况
保健食品市场规模超 3000 亿，涵盖维生素、蛋白粉、益生菌、鱼油、中草药滋补等品类。消费者年轻化趋势明显（朋克养生、成分党崛起）。线上渠道占比持续提升，私域运营是关键复购阵地。行业信任度曾受夸大宣传拖累，专业顾问式服务 + 合规经营是门店差异化核心。

## 常用术语
- 蓝帽子：保健食品批准标志（天蓝色帽形图案），批准文号如「国食健注 G/J」
- 保健食品 vs 药品：保健食品不能宣称疗效，不能代替药物
- 复购率：客户再次购买比例（行业核心指标）
- 客单价：平均每单金额
- 连带率：一单购买的品类数量
- 私域：门店自有客户池（微信群/个人号/小程序），可反复触达
- LTV：客户生命周期总价值
- 会销：会议营销（针对老年人场景监管严格，风险高）

## 核心工作流程
每日：客户接待 → 需求了解 → 产品介绍 → 成交 → 服用指导 → 建档
每周：社群内容运营（3-4 条科普/互动）→ 客户回访 → 库存检查
每月：会员活动 → 数据分析（复购/客单/连带）→ 选品调整 → 员工产品培训

## 常见问题
Q：客户说吃了没效果？
A：解释个体差异 + 保健品的作用周期（通常 4-12 周）+ 建议坚持并配合生活方式调整；仍不满意可协商换品或退款。

Q：怎么提高复购？
A：会员日优惠 + 积分兑换 + 耗尽节点提醒 + 定期健康回访 + 套餐周期购优惠。

Q：客户比价线上怎么办？
A：强调专业搭配服务 + 服用指导 + 售后回访 + 正品保障，服务溢价而非拼价格。

Q：社群怎么运营不冷群？
A：内容为王（七分科普三分产品），固定栏目（早安健康贴/周五问答），轻互动（打卡、话题），控制广告频次。
