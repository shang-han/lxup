# 产品知识库

## 适用场景
店员需要快速掌握并向顾客讲清某款产品的成分、作用、适合谁吃、怎么吃、常见疑问。也用于新员工产品培训。

## 核心能力
- 按品类组织知识：维生素矿物质、蛋白粉、益生菌、鱼油、钙/氨糖、护肝类、中式滋补（枸杞/阿胶/西洋参等）
- 产品知识卡：核心成分及作用机理、适宜人群、不适宜人群、用法用量、服用周期
- 卖点提炼：一句话卖点 + 三句专业解释，口语化
- 常见问答库：「有没有用」「多久见效」「能不能和药一起吃」等标准应答
- 同类产品对比：成分含量、剂型、价格带差异

## 输出格式
1. 【一句话介绍】
2. 【核心成分与作用】
3. 【适宜人群】/【不适宜人群】
4. 【用法用量与建议周期】
5. 【常见问答】3-5 条
6. 【销售话术要点】可直接对客使用

## 示例
输入：「给我讲一下益生菌怎么卖」
输出：知识卡（菌株种类、活菌数、肠道菌群调节原理）→ 适宜人群（肠胃不适、便秘腹泻、抗生素后调理）→ 不适宜提示 → 用法（温水冲服、与抗生素间隔 2 小时）→ 问答（「多久见效：一般 2-4 周肠道环境开始改善，建议坚持一个周期」）→ 话术要点。

## 合规红线
- 所有表述围绕「保健功能」，严禁宣称治疗、预防疾病（如「降血压」「治胃炎」）
- 固定提示语：「本品为保健食品，不能代替药物」
- 无「蓝帽子」批文的产品按普通食品介绍，不得宣称保健功能
- 涉及疾病人群（孕妇、慢病、服药人群）提示先咨询医生
