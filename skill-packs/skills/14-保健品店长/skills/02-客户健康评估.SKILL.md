# 客户健康评估

## 适用场景
顾客到店或线上咨询时，通过结构化问询了解其生活状态与健康诉求，识别潜在需求方向，为搭配建议提供依据。

## 核心能力
- 评估问卷设计：基本信息 / 作息 / 饮食 / 运动 / 压力 / 主要困扰 / 健康目标
- 按主诉分型：睡眠困扰、肠胃不佳、疲劳乏力、免疫力差、骨骼关节、美容抗衰、体重管理
- 输出评估报告：现状小结 + 可能原因分析 + 生活方式建议 + 营养补充方向
- 红旗识别：体重骤降、持续疼痛、长期失眠等须建议就医的信号

## 输出格式
1. 【评估摘要】一句话概括客户状态
2. 【生活方式分析】作息/饮食/运动/压力的亮点与问题
3. 【主要困扰分析】可能原因（非诊断）
4. 【改善建议】生活方式调整（优先）+ 营养补充方向
5. 【下一步】推荐了解的产品类别（衔接搭配建议技能）
6. 【就医提示】如有红旗信号，明确建议就医

## 示例
输入：「顾客 35 岁女性，长期加班，入睡困难、白天疲惫、肠胃不好」
输出：评估（压力型亚健康，睡眠+消化双困扰）→ 原因分析（作息紊乱、咖啡因、晚餐不规律）→ 建议（固定作息、睡前 1 小时离屏、晚餐提前）→ 营养方向（助眠类如 GABA/酸枣仁、益生菌调肠胃）→ 提示：若失眠持续超过一个月或伴情绪低落，建议就医。

## 合规红线
- 评估仅为健康咨询初筛，不构成医疗诊断，报告须注明
- 不解读医院检验报告下结论，可建议客户带给医生看
- 发现疑似疾病信号（红旗）一律引导就医，不推荐产品替代
- 客户健康信息仅用于本次服务，本地保存、不扩散
