# 行业基础

## 行业概况
保险代理人全国约 800 万人，行业竞争极度激烈。产品类型涵盖寿险、健康险（重疾/医疗）、意外险、车险、年金/理财险等。收入以佣金制为主，首年佣金最高、续期递减，新人前 3 个月留存率不足 30%。监管严格：金融监管总局要求销售可回溯（录音录像、可回放可追溯）、如实告知、严禁销售误导。行业正从「人海战术」转向精英化顾问模式，专业能力与服务口碑是长期生存的核心。

## 专业技能体系

### 硬技能
- 产品对比分析（重疾/医疗/寿险/意外/年金）
- 需求分析与保额测算
- 核保规则与健康告知指导
- 理赔流程与资料准备
- 合规销售与双录要求

### 软技能
- 客户风险意识唤醒
- 长期信任关系经营

## 常用术语
- 重疾险：确诊合同约定的重大疾病即按保额赔付的给付型保险，赔款用途不限
- 医疗险：报销实际发生医疗费用的报销型保险，代表产品为「百万医疗险」（高保额、有免赔额）
- 年金险：按约定定期给付生存金/养老金的理财型保险，资金锁定期长
- 等待期：投保后一段时间（通常 90-180 天）内出险不予赔付，用于防范带病投保
- 免责条款：保险公司不承担赔偿责任的情形（如酒驾、故意行为、战争等），投保时必须明确说明
- 如实告知：投保人须如实回答健康问询，隐瞒病史可能导致拒赔且不退保费
- 现金价值：保单退保时可取回的金额，投保早期远低于已交保费

## 核心工作流程
获客：缘故客户开发 → 老客户转介绍 → 社区活动 → 线上专业内容引流
面谈：需求分析（家庭结构/收入/负债）→ 产品讲解 → 方案设计与报价 → 异议处理与促成签单
服务：保单递送与条款讲解 → 续期缴费提醒 → 理赔协助 → 定期保单检视与客户维护

## 常见问题
Q：保险是不是骗人的？
A：保险是受法律保护的合同行为，理赔严格按条款执行。能否赔付关键在投保时是否如实告知、出险是否符合条款约定。保险公司受金融监管总局严格监管，偿付能力有制度保障。

Q：买了不赔怎么办？
A：先核对是否如实告知、是否符合条款赔付条件；若被拒赔，保险公司须出具拒赔通知书并说明理由。有争议可走公司投诉、12378 银行保险消费者投诉热线、调解、仲裁或诉讼渠道。

Q：新人怎么打开客户来源？
A：从缘故客户做起积累第一批服务案例，用专业和理赔协助建立口碑，再请老客户转介绍；同时通过线上科普内容建立专业形象。避免骚扰式推销和人情绑架。
