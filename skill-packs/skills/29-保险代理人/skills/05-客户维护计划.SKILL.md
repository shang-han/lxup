# 客户维护计划

## 适用场景
代理人做长期客户经营：新客成交后的服务计划、老客户年度维护节奏、节日生日问候、沉默客户激活、转介绍请求的时机与话术。保险是长期服务生意，维护质量决定转介绍与加保。

## 核心能力
- 客户分级管理：按保费规模/互动频率/家庭阶段分 A/B/C 级，匹配不同回访频率
- 设计回访节奏：成交首周（保单讲解确认）→ 首月（体验回访）→ 季度（服务触达）→ 年度（保单检视）
- 场景化问候文案：节日/生日/保单周年，有温度不带销售感
- 转介绍激活设计：最佳时机（理赔后/检视后/高满意度时）、请求话术、答谢礼仪
- 沉默客户激活：用服务型理由破冰（保单整理、理赔政策更新、健康科普），而非开口就推产品
- 输出年度客户经营日历：每月主题 + 动作清单

## 输出格式
1. 【客户分级】分级标准与对应维护策略
2. 【回访节奏】各节点时间与触达主题
3. 【问候文案】节日/生日模板 2-3 版
4. 【转介绍方案】时机、话术与答谢方式
5. 【激活策略】沉默客户触达理由与话术
6. 【年度日历】逐月维护动作清单

## 示例
输入：「刚成交一单大额重疾险，帮我做成交后一年的客户维护计划」
输出：分级：大额保单客户列 A 级 → 节奏：成交第 3 天电话确认保单条款与回访录音事项；第 2 周递送保单并做条款讲解；满 1 个月首访服务体验；第 3 个月发送健康科普 + 关怀；第 6 个月提示受益人确认；第 11 个月预约年度保单检视 → 生日文案：「X 哥生日快乐！作为您的保险服务顾问，祝您新的一岁平安健康，保单的事交给我放心。」（纯祝福不带产品）→ 转介绍：年度检视服务后自然带出「家人朋友有保单整理需求也可以找我，免费服务」→ 年度日历表逐月列明动作。

## 合规红线
- 回访频率适度，避免高频骚扰与夜间打扰
- 问候与科普内容不得含夸大收益、承诺理赔等误导性信息
- 转介绍须征得客户同意，不得向第三方泄露客户投保信息
- 不以返佣、送礼等不正当利益诱导投保或转介绍，答谢方式须符合公司与监管规定
- 涉及产品销售性质的沟通须符合可回溯管理要求
