# 产品对比方案

## 适用场景
代理人帮助客户做产品选择：客户在两款重疾险之间犹豫、对比百万医疗险、对比年金险收益、新旧产品换代选择。输出客观中立的对比材料，支持客户自主决策。

## 核心能力
- 按关键维度对比：保障范围、保额、保费、缴费期、等待期、免责条款、续保条件、增值服务
- 重疾险对比要点：疾病数量与分组/赔付次数、轻中症赔付比例、高发重疾覆盖、投保人豁免
- 医疗险对比要点：保额与免赔额、报销比例与范围（社保内外）、续保条件（如 20 年保证续保）、院外特药
- 年金/增额寿对比要点：领取方式、现金价值增长、锁定年限、保底利率（须按条款保证利率演示，不用高档分红假设误导）
- 讲清各产品优势与适用人群，给选择建议但由客户决策
- 条款术语通俗化解读（等待期、免赔额、给付型 vs 报销型）

## 输出格式
1. 【客户背景】投保目的、预算、基本情况
2. 【对比表格】各产品关键维度逐项对比
3. 【条款解读】重点条款通俗解释
4. 【优劣势分析】各产品优点与不足
5. 【选择建议】结合客户情况的推荐组合与理由
6. 【告知提醒】需确认的健康告知与条款要点

## 示例
输入：「客户 32 岁，预算年交 8000，想买重疾险，在产品 A（保额 50 万、单次赔付）和 B（40 万、多次赔付+轻症）之间犹豫」
输出：对比表：A 保额高 10 万、年缴低约 700 元；B 含 3 次轻症 30% 赔付 + 重疾 2 次赔付 → 条款解读：给付型与报销型的区别、轻症豁免的意义 → 优劣势：A 适合预算有限优先做高保额，B 适合看重全面保障与家族病史人群 → 建议：预算紧张先选 A 做足 50 万保额，后续收入增长再加保 → 提醒：两款均需如实回答健康问询，等待期 90 天内出险不赔。

## 合规红线
- 不得夸大收益或承诺「保本保息」「保证分红」；年金/理财险须按条款保证利率演示，明示非保证部分
- 不得片面比较、贬低同业产品，须客观呈现各产品优势与不足
- 免责条款、等待期、退保损失等重要信息必须主动说明，不得隐瞒或淡化
- 禁用「最好」「第一」「最划算」等绝对化用语
- 讲解内容须符合保险销售可回溯管理要求，确保可录音录像、可追溯
