# 行业基础

## 行业概况
工程资料贯穿项目建设全过程，是竣工验收与备案的必要条件，资料不合格的工程质量再好也无法交付。资料内容涵盖工程准备阶段文件、监理资料、施工记录、检验批与验收资料、竣工图等。资料员的核心能力是资料规范掌握、归档体系管理与各方沟通协调。资料工作讲究「同步形成、真实闭合、签章齐全」，突击补资料、日期逻辑矛盾、签章缺失是最常见的返工原因。

## 常用术语
- 施工组织设计：指导施工全过程的总体技术经济文件，开工前报审
- 隐蔽工程验收记录：水电、防水、钢筋等覆盖前的验收证明文件
- 检验批：按楼层、施工段、变形缝等划分的质量验收基本单元
- 分部工程：按专业性质和部位划分（地基基础、主体结构、建筑装饰装修等）
- 竣工图：依据设计变更和实际施工情况改绘的最终图纸
- 材料报验：进场材料向监理报审并附合格证、检测报告的流程
- 见证取样：在监理见证下现场取样送检的制度
- 档案移交：竣工资料整理组卷后移交城建档案馆

## 核心工作流程
开工前：开工报告 → 施工组织设计报审 → 材料进场报验制度建立 → 检验批划分方案报审
施工中：填写施工日志 → 材料报验与见证取样 → 检验批验收记录 → 隐蔽工程资料同步收集 → 试验报告归档
竣工后：编制竣工图 → 组卷编目 → 竣工验收文件成套 → 移交档案馆 → 竣工备案

## 常见问题
Q：资料总是收不齐怎么办？
A：开工就建立资料总清单和收发台账，把每份资料的形成节点绑定到施工工序，随进度同步收集，月度对照清单查缺，责任到人。

Q：甲方监理催资料很急怎么应对？
A：关键路径资料（报验、检验批、试验报告）日常即保持最新，建立快速响应清单；缺项资料明确补办时限，先交齐手的、不交半成品。

Q：资料日期对不上被退回怎么办？
A：退回的硬伤多为日期逻辑矛盾（先验收后报验、试验报告晚于施工），建立日期链条自查表，形成资料前先核对上下游日期。

Q：竣工图和施工图怎么区分处理？
A：无变更的图纸可原图加盖竣工图章；有变更处必须在图上逐处杠改并标注变更单号，保证竣工图与实体完全一致。
