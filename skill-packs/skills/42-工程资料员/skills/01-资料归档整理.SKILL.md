# 资料归档整理

## 适用场景
项目开工需要建立资料体系，施工中资料越积越乱需要梳理整改，或迎检前需要快速核查资料完整性，做到随做随收、分类清晰、查找方便。

## 核心能力
- 依据本地归档要求（参照 GB/T 50328 建设工程文件归档规范）生成分部分类资料总清单：基建文件、监理资料、施工资料、竣工图四大类
- 按单位工程 → 分部 → 分项三级建立编号规则与目录盒脊标签
- 建立资料收发台账模板：编号、名称、份数、原件/复印件、形成日期、经办人、存放位置
- 输出月度资料核查表：对照清单逐卷查缺，标注缺失项与补办责任人
- 电子资料管理：扫描命名规则、双备份与版本管理建议
- 给出迎检资料整理优先级与常见扣分项提示

## 输出格式
1. 【资料体系】分类目录树与编号规则
2. 【资料总清单】分类-资料名称-形成节点-份数-责任岗位
3. 【收发台账】台账表格模板
4. 【核查表】缺项清单与补办计划
5. 【归档要求】装订、编目、盒脊标签与电子备份规范

## 示例
输入：「项目刚开工，帮我把资料体系搭起来」
输出：资料体系（A 基建文件 / B 监理资料 / C 施工资料 / D 竣工图，施工资料按 C1 管理资料、C2 技术资料、C3 物资资料、C4 施工记录、C5 试验记录、C6 验收记录细分）→ 总清单（开工报告、施组报审、材料进场报验、检验批……逐项标明形成节点与责任人）→ 收发台账模板 → 月度核查表 → 归档要求（原件优先、碳素笔填写、扫描双备份）。

## 注意事项
- 资料与工程进度同步形成，杜绝竣工前突击补资料导致日期逻辑矛盾
- 签字盖章齐全方为有效资料，代签、漏章的资料验收时一律返工
- 隐蔽工程验收记录必须附影像资料，日期与部位可对应
- 试验报告与材料报验一一对应，合格证、检测报告批次齐全
- 资料变更走正规修改流程（杠改签字），不得涂改或使用修正液
