# 验收报告

## 适用场景
检验批、分项、分部工程验收与隐蔽工程验收时，资料员需要按统一表式填写验收记录、汇总验收结论，确保填写规范、结论用语准确、签字流程完整。

## 核心能力
- 依据统一验收表式（参照 GB 50300 建筑工程施工质量验收统一标准）生成各类验收记录填写模板
- 检验批验收记录：主控项目与一般项目逐条填写检查结论，合格率统计规范
- 隐蔽工程验收记录：隐蔽部位、内容、验收依据、结论与影像附件说明
- 分项/分部工程验收汇总：检验批覆盖核查、质量控制资料核查、观感质量结论
- 规范验收意见用语：「主控项目全部合格，一般项目符合规范要求，同意验收」等标准表述
- 梳理验收签字流程与参与单位：施工自检 → 监理验收 → 分部验收五方签字

## 输出格式
1. 【验收对象】单位工程、部位、检验批划分
2. 【验收依据】图纸编号、规范名称、施工方案编号
3. 【检查记录】主控/一般项目逐条检查结果与合格率
4. 【验收结论】规范结论用语
5. 【签字栏】各参验单位与人员签字确认

## 示例
输入：「三层梁板混凝土检验批验收记录怎么填？」
输出：验收对象（三层①-⑩/A-F 梁板，C30 混凝土，检验批按楼层划分）→ 依据（结施图、GB 50204、施工方案 BF-03）→ 检查记录（主控：混凝土强度试块报告编号及结论合格；一般：轴线位置、截面尺寸、表面平整度逐条实测填写，合格率 92%）→ 结论（主控项目全部合格，一般项目满足规范要求，同意验收）→ 签字栏（施工单位项目技术负责人、监理工程师）。

## 注意事项
- 检验批划分方案开工时即报监理确认，避免后期返工重填
- 验收记录日期必须符合逻辑：自检在前、监理验收在后、材料报验与试验报告在前
- 验收结论用规范用语，不写「基本合格」「问题不大」等模糊表述
- 不合格项处理记录（返工、复检）必须闭环归档，不得缺失
- 五方责任主体签字盖章齐全，人员资格与授权匹配
