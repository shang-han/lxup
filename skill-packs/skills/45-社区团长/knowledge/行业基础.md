# 行业基础

## 行业概况
社区团购市场规模超 5000 亿，以「线上下单 + 社区自提」模式切入居民日常生鲜与日用消费。团长是连接平台与社区居民的关键节点，负责拉新、选品、推单、分拣、售后全链路。收入来源为按销售额抽取的佣金（通常 10-20%）。核心竞争力在于社群运营能力、选品眼光与邻里服务口碑，做得好的团长本质上是「被邻居信任的采购管家」。

## 常用术语
- 开团：发起一次团购活动，上架商品供群成员下单
- 自提点：用户下单后线下取货的地点，常设在团长家中或合作小店
- 团长佣金：按团购销售额比例抽取的报酬
- 起送价：免运费或达到配送标准的最低消费金额
- 次日达：用户当日下单、次日下午到货的履约时效
- 接龙：群内以编号接龙形式统计下单的下单方式
- GMV：团购成交总金额
- 复购率：老客重复下单的比例，衡量社群黏性

## 核心工作流程
每日：选品 → 群内预热 → 发布开团 → 接单统计 → 反馈供应商
到货：收货验收 → 按订单分拣 → 通知取货 → 现场发放 → 售后处理
运营：社群互动维护 → 拉新裂变活动 → 老客回访复购 → 经营数据复盘

## 常见问题
Q：群里没人说话、变成广告群怎么办？
A：靠日常互动养群——早晚问候、红包互动、话题投票、晒单返现，控制推品频率，让群有生活气息而不只是广告。

Q：售后纠纷多，坏果破损怎么处理？
A：提前公示赔付标准（坏果包赔、质量问题退款、缺货补发），快速响应先安抚再解决，标准化的售后反而能提升信任。

Q：怎么提高老客复购？
A：做客户分层维护，首单后定时回访、复购提醒、老客专属券，记录偏好做精准推荐，把一次性客户沉淀为长期邻居。

Q：佣金收入上不去怎么办？
A：扩大社群规模（裂变拉新）+ 提升客单价（合理搭配利润品）+ 提高复购（客户维护），同时控制退货率减少损耗。
