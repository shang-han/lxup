# 团购文案生成

## 适用场景
团长每天要向社群推送数十款商品，需要快速生成有吸引力、信息完整、可直接复制到微信群的团购文案，覆盖生鲜、日用、零食、应季品等品类。

## 核心能力
- 按品类匹配文案风格：生鲜突出新鲜/产地直发，日用突出性价比，零食突出口感场景
- 自动组织「钩子开头 + 卖点 + 规格价格 + 对比锚点 + 接龙指令」结构
- 提炼 3 个核心卖点，用生活化口语而非广告腔
- 营造稀缺与从众感（限量、截单时间、已团人数）
- 一键生成接龙格式（编号 + 品名 + 规格 + 价格 + 下单方式）
- 区分预热版（提前种草）与开团版（催单转化）

## 输出格式
1. 【品名+钩子】一句话抓眼球开头
2. 【核心卖点】3 条，口语化
3. 【规格价格】规格 / 团购价 / 市场价对比
4. 【下单方式】接龙格式或链接说明
5. 【催单提示】截单时间 / 限量 / 次日达说明

## 示例
输入：「帮我写个赣南脐橙的团购文案，9.9 元 5 斤」
输出：开头「🍊爆汁预警！赣南脐橙产地直发」→ 卖点（现摘现发不打蜡 / 皮薄汁多化渣 / 5 斤够全家吃一周）→ 规格（5 斤装中果，团购价 9.9 元，超市约 19.9）→ 接龙「1. 赣南脐橙 5 斤 9.9 元」→ 催单「今晚 10 点截单，明日下午自提点取货，限量 50 份」。

## 注意事项
- 价格、规格、产地必须与实际供货一致，不得虚构原价制造虚假折扣
- 生鲜类如实说明大小/口感差异，避免「包甜」「零坏果」等无法兑现的绝对化承诺
- 截单时间与到货时间写清楚，减少售后纠纷
- 文案控制在群内一屏可读完，善用 emoji 分段但不过度
