# 活动策划

## 适用场景
团长需要扩大社群规模、提升单次团购销量、借节日做促销、或激活沉睡用户时，需要一份机制清晰、可落地执行的完整活动方案。

## 核心能力
- 活动类型库：新人入群礼、老带新裂变、满减拼团、节日大促、清仓秒杀、晒单返现
- 完整方案输出：目标 / 主题 / 时间 / 机制 / 选品 / 预算 / 执行步骤 / 复盘指标
- 裂变机制设计：邀请有礼、阶梯成团、拼团返现
- 选品搭配：引流品（低价高频）+ 利润品 + 凑单品
- 配套文案：预热、开团、催单、战报各版本
- 风险控制：库存、赔付、秩序维护预案

## 输出格式
1. 【活动概览】目标、主题、时间、面向人群
2. 【活动机制】具体玩法与优惠规则
3. 【选品建议】引流品 / 利润品 / 凑单品
4. 【执行步骤】按时间节点拆解的动作
5. 【宣传文案】预热 + 开团 + 催单版
6. 【复盘指标】参与人数、成交额、新增成员等

## 示例
输入：「我的群才 80 人，想搞个拉新活动」
输出：主题「邻里拼团·邀请有礼」→ 机制（邀请 1 位邻居入群并完成首单，双方各得 3 元无门槛券；满 100 人开整群抽奖）→ 选品（鸡蛋/纸巾引流+应季水果利润品）→ 执行（预热 2 天→活动 3 天→战报收尾）→ 预热文案「拉邻居进群一起薅羊毛，人越多越便宜～」→ 复盘指标：新增人数、首单转化率、活动 GMV。

## 注意事项
- 优惠规则要简单易懂，避免复杂计算劝退用户
- 裂变奖励设上限，控制成本，明确兑现时间
- 活动选品确保供货稳定，避免爆单后缺货失信
- 奖品/优惠券及时发放，活动承诺不打折扣
- 遵守平台规则，不做虚假抽奖或诱导分享违规操作
