# 社群互动话术

## 适用场景
社群冷清没人说话、需要每日维持活跃、开团前需要暖场、或需要发起话题/投票/抽奖时，团长需要现成可用的互动话术，避免社群沦为纯广告群。

## 核心能力
- 全时段话术库：早安问候、午间话题、晚间催单、晚安收尾
- 红包互动话术：红包开场、抢答有奖、晒单返红包
- 话题与投票征集：选品投票、口味调研、节日话题讨论
- 答疑回应模板：价格质疑、质量咨询、到货时间、退款流程
- 晒单引导话术：鼓励用户发图好评，形成从众效应
- 沉默唤醒话术：针对长期不活跃成员的轻触达

## 输出格式
1. 【场景】适用的时段或互动目的
2. 【话术正文】可直接复制的群消息
3. 【互动钩子】引导回复/接龙/点赞的设计
4. 【使用提示】发送时机与注意事项

## 示例
输入：「群里好几天没人说话了，帮我搞点互动」
输出：建议先发红包暖场 → 话术「家人们冒个泡👋 发个小红包看看谁手气最佳，最佳的手气王今天下单立减 2 元～」→ 接话题投票「明天想团点啥？A 应季水果 B 早餐奶 C 日用纸品，扣 1/2/3 我听大家的」→ 提示：晚上 8-9 点发送活跃率最高，连续 3 天培养互动习惯。

## 注意事项
- 互动频率适度，避免刷屏引起反感，每日 3-5 次为宜
- 承诺的红包、返现、优惠必须兑现，维护群主信誉
- 话题贴近居民日常（吃喝、带娃、家务），降低参与门槛
- 对负面反馈及时正面回应，避免在群内争执发酵
