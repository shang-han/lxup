# 行业基础

## 行业概况
教育培训行业规模超万亿，涵盖 K12、素质教育、职业教育等。「双减」政策后行业重塑，非学科培训 + 成人教育成新增长点。核心指标：满班率 + 续费率 + 转介绍率。合规经营（双减、广告法、资金监管、收费规范）是机构生存的前提，教学质量与服务口碑是长期竞争力。

## 常用术语
- 满班率：在班人数 / 最大容量
- 续费率：课程到期后续报比例
- 转介绍率：老带新比例
- 试听转化率：试听 → 报名比例
- 退费率：退费学员 / 总学员比例
- 课时：课程节数，收费的基本单位
- 学科类/非学科类：文化课补习 / 艺术、体育、编程等素质类培训
- 班主任/学管师：负责家长跟进与学员服务的岗位

## 核心工作流程
每日：教学巡课 → 家长沟通 → 招生咨询 → 团队管理
每周：教研活动 → 教师培训 → 数据分析
每月：招生策划 → 成本分析 → 教学评估

## 常见问题
Q：家长觉得贵怎么办？
A：拆解课时单价、对比同类、强调教学质量与服务，提供合规的课时包或分期方案。

Q：老师流失怎么办？
A：有竞争力的薪酬 + 培训晋升 + 成长空间 + 情感关怀。

Q：怎么提高续费率？
A：做好学习阶段反馈让家长看到成长，临期提前触达，提升教学质量与服务。

Q：合规经营重点注意什么？
A：遵守「双减」（周末/节假日不开展学科类培训），收费不超 3 个月或 60 课时，宣传不承诺效果、不用成绩作证明。
