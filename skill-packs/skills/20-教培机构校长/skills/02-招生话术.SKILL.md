# 招生话术

## 适用场景
招生顾问接待家长咨询（到店/电话/微信）时使用。需解答课程、收费、师资、效果等疑问，建立信任引导报名，同时严守合规红线。

## 核心能力
- 需求挖掘（孩子年龄、兴趣、目标、家长顾虑）
- 课程推荐（按需求匹配课程，讲清价值）
- 收费异议应对（价值拆解、课时说明、先试听）
- 信任建立（资质、教学理念、服务、口碑）
- 试听与报名转化引导
- 规避过度承诺与合规风险

## 输出格式
1. 【场景识别】家长问题类型
2. 【应答要点】回答核心与策略
3. 【话术文本】可直接使用的话术
4. 【转化引导】引导试听/报名的动作
5. 【合规提示】本场景的合规要点

## 示例
输入：「家长问「学了这个孩子成绩能提高吗」，怎么回？」
输出：场景为效果承诺型提问（合规高敏）→ 要点（不承诺成绩，转向素养与习惯培养）→ 话术「理解您关心孩子的成长。我们是素质类课程，更注重培养孩子的兴趣、专注力和学习习惯，这些是长期发展的基础。成绩受很多因素影响，我们无法承诺，但很多孩子变得更专注、更愿意主动学习。您可以先带孩子来试听，看看他的状态和喜好」→ 转化（邀约试听，现场感受）→ 合规提示（绝不承诺提分，不举学员成绩例子）。

## 合规红线
- 不承诺提分、通过考试、获奖、升学等效果
- 不使用学员成绩或成功案例作销售证明
- 不违反「双减」开展学科类培训（周末/节假日/寒暑假不安排学科类课程）
- 收费合规：一次性收费不超过 3 个月或 60 课时，明示退费规则
- 不使用禁用词，不夸大师资与培训效果
