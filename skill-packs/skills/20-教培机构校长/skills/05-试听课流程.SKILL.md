# 试听课流程

## 适用场景
机构组织试听课/体验课，邀约家长与孩子上门体验时使用。需要完整的邀约→接待→体验→反馈→转化流程与话术，提升试听转化。

## 核心能力
- 试听邀约与到店接待流程
- 试听课体验设计（内容、互动、时长）
- 现场观察与反馈（孩子表现、家长关注点）
- 课后沟通与转化话术
- 转化引导（权益、课时说明、从长计议）
- 规避合规风险

## 输出格式
1. 【试听概览】目标、人群、形式、时长
2. 【课前准备】邀约、物料、环境
3. 【课程流程】环节设计
4. 【课后沟通】反馈与转化话术
5. 【转化方案】权益与跟进
6. 【合规提示】注意事项

## 示例
输入：「设计一节少儿编程的试听课流程」
输出：概览（面向 6-10 岁孩子与家长，60 分钟，小班）→ 课前（电话/微信确认预约，准备教具与体验设备）→ 流程（5 分钟热身破冰→25 分钟主题体验让孩子完成一个小游戏→15 分钟作品分享→10 分钟家长答疑）→ 课后（反馈孩子课堂表现，讲解课程体系）→ 转化（当日报名赠课时，限时权益，不强迫）→ 合规提示（不承诺学习效果，收费不超 3 个月或 60 课时，退费规则明示）。

## 合规红线
- 不借试听课违规开展学科类培训
- 不承诺提分或学习效果，不用学员成绩作证明
- 收费合规，一次性收费不超 3 个月或 60 课时
- 不诱导过度消费，退费规则明示
- 保障试听过程中学员人身安全，防止意外
