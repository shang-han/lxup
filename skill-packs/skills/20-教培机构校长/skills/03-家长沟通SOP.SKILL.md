# 家长沟通SOP

## 适用场景
机构教师/班主任与家长进行日常沟通时使用。需要标准化的入学接待、学习阶段反馈、续费、投诉处理流程，提升满意度与续费率。

## 核心能力
- 入学接待 SOP（建档、课程说明、预期建立）
- 学习阶段报告设计（内容、频率、形式）
- 续费沟通流程（节点、话术、权益）
- 投诉处理流程（倾听、核实、解决、回访）
- 家校共育建议（家庭配合指导）
- 差异化沟通（新家长/老家长/流失预警家长）

## 输出格式
1. 【沟通场景】本次沟通类型
2. 【流程节点】步骤与时间安排
3. 【话术/模板参考】可直接用的沟通模板
4. 【频率与工具】沟通频次与渠道
5. 【异常处理】投诉与不满应对
6. 【效果指标】满意度、续费率

## 示例
输入：「孩子入学一个月了，怎么给家长做反馈？」
输出：场景为学习阶段报告（入学 1 个月）→ 流程（整理课堂记录→撰写报告→预约家长→沟通→跟进建议）→ 模板「您好，XX 妈妈，XX 入学一个月啦。最近上课很专注，动手能力进步明显，这次主题课的作品完成得特别主动。建议在家多鼓励他观察和尝试……」→ 频率（入学 1 个月首次，之后每 2-3 个月一次）→ 异常（家长不满先倾听记录，24 小时内回应）→ 指标（家长满意度、续费率）。

## 合规红线
- 反馈聚焦成长与习惯，不排名、不公开比较学员成绩
- 不泄露其他学员及家长个人信息
- 续费沟通合规，不诱导过度消费，收费符合监管要求
- 退费按约定及时办理，不拖延推诿
- 如实传达政策，不做违规承诺
