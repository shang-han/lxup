# 市场价格对比

## 适用场景
评估师定价与谈判时的市场行情分析：比对同款同年份车源挂牌价与成交价、参考保值率、测算收售价差空间，为收车报价和零售定价提供依据。

## 核心能力
- 可比车源筛选：按车型、年款、配置、里程、车况、区域筛出可比样本，剔除虚挂牌价
- 价格口径区分：平台挂牌价（虚高）vs 实际成交价 vs 车商收车价 vs 批售价，换算关系讲清
- 保值率应用：主流车型 1-5 年残值率参考，冷门车/新能源车的折价修正
- 定价测算表：市场零售均价 − 整备成本 − 预期毛利 − 资金成本 − 议价空间 = 收车上限
- 区域与季节修正：区域流通差价、季节因素（敞篷夏季、四驱冬季、新能源冬季续航折价）
- 比价报告呈现：给老板或客户看的对比表，含数据来源与时间

## 输出格式
1. 【目标车辆】车型年款配置里程车况
2. 【可比车源表】来源 / 报价 / 里程车况 / 可比性说明
3. 【保值率参考】该车型残值曲线定位
4. 【价格测算】各档价格推算过程
5. 【定价建议】收车价 / 零售价 / 批售价 + 依据

## 示例
输入：「2020 款 Model 3 标准续航，6 万公里，想收进来，帮我分析价格」
输出：可比车源：平台同款挂牌 3 台（11.8-13.2 万，挂牌价普遍虚 5-10%），近期本地成交约 11.x 万 → 保值率：Model 3 三年残值约 55%，该车四年处于加速贬值段 → 新能源修正：电池健康度待检（低于 85% 再砍 5000-8000）、冬季续航口碑影响流通 → 测算：零售均价 11.8 万 − 整备 3000 − 毛利 8000 − 资金周转（预计 40 天）2000 − 议价 3000 ≈ 收车上限 10.2 万 → 建议：收车 9.8-10.2 万，零售挂 11.8 万谈至 11.2 万成交。

## 注意事项
- 引用价格须注明来源与时间（平台挂牌 ≠ 成交），行情按周波动，报价前刷新数据
- 不用虚假高价做价格锚点诱导客户（「别家都卖 15 万」须有真实样本支撑）
- 新能源车须关注电池质保归属（首任车主权益是否随车转让）并计入定价
- 跨区域比价注意排放标准与迁入限制，国五及以下部分城市无法迁入
