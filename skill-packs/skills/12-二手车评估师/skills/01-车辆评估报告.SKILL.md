# 车辆评估报告

## 适用场景
二手车评估师为收车、售车或第三方检测出具结构化评估报告：记录车辆基本信息、技术状况分级、事故/水泡/火烧排查结论，给出估值区间与定价依据。

## 核心能力
- 报告结构：基本信息（车型年款、VIN、表显里程、过户次数、手续状态）→ 外观内饰 → 结构件 → 机舱底盘 → 电气功能 → 路试 → 综合评级
- 车况分级：按国标思路对覆盖件、结构件、机电系统分项评级（优/良/一般/修复/更换）
- 三非排查结论：重大事故、火烧、水泡逐项判定依据（数据 + 照片证据位）
- 里程真实性核验：维保记录、轮胎刹车磨损、内饰磨损与表显里程交叉比对
- 估值方法组合：现行市价法（同款可比车源）+ 重置成本法（残值率）+ 收益修正，给区间而非死价
- 定价建议：区分收车价、零售指导价、同行批售价三档

## 输出格式
1. 【车辆基本信息】
2. 【车况分项评级表】部位 / 状态 / 评级 / 备注
3. 【三非排查结论】逐项结论 + 依据
4. 【整备整修清单】预计费用
5. 【估值与定价建议】估值区间 / 收车价 / 零售指导价 + 依据

## 示例
输入：「评估一台 2021 款雅阁 260TURBO，表显 4.8 万公里，右前门换过」
输出：基本信息表 → 分项评级：右前门更换（覆盖件，不影响结构）、其余面原厂漆（漆膜 110-130μm）、左前轮胎磨损偏重建议更换 → 三非排查：四梁六柱原厂焊点完整、无水泡锈蚀痕迹、无火烧修复 → 里程核验：4S 维保记录连贯，磨损与里程相符 → 整备清单：轮胎 ×1、抛光打蜡，约 xxx 元 → 估值：同款近期成交 13.x-14.x 万，建议零售挂 14.x 万、收车上限 12.x 万（含整备与周转成本）。

## 注意事项
- 评估结论必须有数据与照片支撑（漆膜数值、焊点照片、记录截图），不凭经验空口下结论
- 重大事故、火烧、水泡必须如实判定并在报告中明示，严禁配合出具虚假「精品车况」报告
- 评估遵循《二手车鉴定评估技术规范》（GB/T 30323），保持第三方客观立场
- 里程存疑（疑似调表）必须在报告中注明「表显里程，真实性存疑」，不替车主背书
