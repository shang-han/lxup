# 卖车文案优化

## 适用场景
门店将收进车辆上架销售时，优化车源描述文案：二手车平台挂牌文案、朋友圈卖车文案、短视频口播脚本，提升咨询量与到店率。

## 核心能力
- 卖点提炼：从年款配置、里程、车况、整备情况、手续、价格优势中找出 3 个核心卖点
- 平台挂牌结构：标题（车型 + 年款 + 里程 + 亮点）→ 车况透明描述 → 配置亮点 → 整备与质保 → 价格与看车方式
- 信任元素植入：检测报告可查、过户次数、维保记录、门店质保承诺
- 朋友圈版本：短平快 + 真实感（收车故事、检测实拍），避免刷屏式叫卖
- 短视频脚本：3 秒钩子（「12 万开出 20 万的面子」）→ 车况实拍讲解 → 引导私信
- 文案合规自检：车况描述与实际一致，瑕疵主动披露位置设计

## 输出格式
1. 【核心卖点】3 条
2. 【挂牌文案】标题 + 正文
3. 【朋友圈版本】
4. 【瑕疵披露说明】必须写明的项
5. 【配图/视频建议】

## 示例
输入：「一台 2021 款 CR-V 两驱舒适版，3.2 万公里，一手车，帮我写挂牌文案」
输出：卖点：一手车 0 过户 / 3.2 万公里实表带维保记录 / 全程 4S 保养 → 标题：「21 款 CR-V 240TURBO 两驱｜一手 3.2 万公里｜全程 4S 记录」→ 正文：车况描述（原版原漆 90%、无事故水泡火烧、检测报告随车）+ 配置（全景天窗、Honda SENSING）+ 整备（新换轮胎、全车精洗）+ 门店承诺（7 天可退、1 年 2 万公里质保）→ 瑕疵披露：右后保险杠一处补漆如实注明 → 朋友圈：「刚到的一台 CR-V，一手小姐姐的车，维保记录厚厚一本……」→ 配图：四门两盖、发动机舱、检测报告特写。

## 注意事项
- 车况描述必须与实车一致，事故、补漆、调表嫌疑等瑕疵主动披露，严禁「精品车」虚假宣传
- 里程表述规范：非全程 4S 记录的车注明「表显里程」，不暗示保证实表
- 质保、退换承诺写清条件与期限，说到做到，承诺即合同
- 价格真实可谈，不搞虚高标价 + 「到店大砍」套路，平台规则也禁止虚假标价
