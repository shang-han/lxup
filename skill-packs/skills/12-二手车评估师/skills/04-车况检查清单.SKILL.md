# 车况检查清单

## 适用场景
评估师现场检测车辆时使用的标准化检查清单：收车检测、零售车整备前复检、帮客户购车第三方验车，确保不漏项、判定有据。

## 核心能力
- 分部位检查清单：外观覆盖件（漆膜仪逐面数值、缝隙均匀度、玻璃日期）→ 结构件（四梁六柱焊点、胶条、螺丝拆卸痕迹）→ 发动机舱（渗油、螺丝位移、线束老化）→ 内饰（磨损与里程匹配、安全带日期、点烟器锈蚀）→ 底盘（举升看纵梁、悬挂、渗油）→ 电气功能逐项 → 路试（换挡、制动、跑偏、异响）
- 三非排查要点：水泡（点烟器/座椅滑轨锈蚀、地毯霉味、保险盒泥沙）、火烧（线束熔痕、局部换件）、事故（结构件变形修复）
- 里程核验方法：维保记录 + 轮胎生产日期 + 刹车盘磨损 + 内饰磨损多维交叉
- 工具清单：漆膜仪、手电筒、OBD 诊断仪、举升机、轮胎花纹尺
- 检查记录模板：逐项打勾 + 数值记录 + 照片证据位
- 风险分级：立即终止交易项 / 压价项 / 整备项三档标注

## 输出格式
1. 【检查前准备】工具与资料（维保、出险记录）
2. 【分部位检查表】检查项 / 方法 / 判定标准 / 记录栏
3. 【三非排查专项】逐项要点
4. 【里程核验】多维比对表
5. 【结论模板】车况摘要 + 风险分级

## 示例
输入：「帮我出一份现场收车的标准检测清单」
输出：清单：①外观 13 面漆膜数值逐一记录（原厂漆 90-160μm，超 300μm 疑钣金腻子）→ ②四门两盖螺丝有无拆卸、胶条原厂焊点 → ③ABC 柱与前后纵梁（举升）→ ④机舱渗油与螺丝位移 → ⑤内饰安全带拉到底看水渍霉斑 → ⑥OBD 读故障码与里程 → ⑦路试 5 公里 → 三非专项表 → 结论模板：「覆盖件 X 处补漆、结构件无异常、无水泡火烧痕迹、表显里程与磨损相符」→ 标注压价项与整备项清单。

## 注意事项
- 检测在光线充足环境下进行，漆膜仪使用前校准，数据当场记录不凭记忆
- 结构件任一异常即按事故车处理并告知委托方，不因人情出具宽松结论
- 举升检查底盘不可省略，底盘是重大事故与水泡的照妖镜
- 受委托验车时保持独立立场，检测结论对委托方负责，不收车商好处费改口
