# 行业基础

## 行业概况
二手车年交易量超 2000 万辆，市场规模超万亿，限迁政策全面取消后全国大流通格局形成。评估师核心能力：车况鉴定 + 价格评估 + 谈判沟通。销售渠道为线上平台（瓜子/懂车帝/二手车之家）+ 线下门店 + 同行批售双轨。行业信任痛点是车况透明度，「检测标准化、车况敢承诺」是门店差异化关键。利润来源：买卖价差 + 金融保险佣金 + 整备增值。库存周转速度直接决定资金效率与盈利。

## 常用术语
- 评估准确率：评估报告与实际车况的一致性，专业信誉指标
- 成交转化率：看车客户到成交的比例
- 收车溢价：收车价低于市场零售价的利润空间
- 库存周转：车辆从收进到售出的平均天数
- 单车毛利：每辆车的平均利润
- 保值率：某车型二手残值占新车指导价的比例
- 三非车：非重大事故、非火烧、非水泡，零售车源的基本门槛

## 核心工作流程
收车：车源开发 → 现场检测 → 评估定价 → 谈判签约 → 付款过户
整备：清洗美容 → 故障检修 → 拍照摄像 → 平台上架
售车：客户咨询 → 带看讲解 → 试驾 → 谈判成交 → 过户交付 → 售后跟进

## 常见问题
Q：怎么判断事故车？
A：查维保与出险记录 + 漆膜仪全车测数值 + 检查 ABC 柱、前后纵梁、水箱框架等结构件焊点与胶条 + 底盘举升看变形修复痕迹，任一结构件损伤即按事故车定价。

Q：收车价怎么定？
A：市场零售行情价 − 整备成本 − 预期毛利 − 议价空间 − 资金与周转成本 = 收车上限，同时参考同款近期成交价与平台挂牌价交叉验证。

Q：客户总说别家报价更高？
A：拆解价格构成（过户费、整备、质保、金融方案），说明「净到手价」口径差异，用即时付款、手续全包的优势对冲虚高报价。

Q：库存积压怎么办？
A：设定库龄红线（如 45 天）分级处理：超期降价促销、转同行批售、打包处理，宁可薄利出货也不占压资金。
