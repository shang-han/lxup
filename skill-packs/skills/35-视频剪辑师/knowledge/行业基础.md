# 行业基础

## 行业概况
短视频时代视频剪辑需求爆发式增长，抖音、快手、视频号、B 站等平台内容缺口巨大，剪辑师可接单类型涵盖短视频代剪、企业宣传片、Vlog、影视综剪辑、商业广告、信息流素材等。行业呈两极分化：基础套模板剪辑被工具与 AI 替代、价格内卷，具备「剪辑 + 脚本 + 网感」复合能力的剪辑师议价能力强。自由接单与坐班并存，作品集与案例是获客核心。

## 常用术语
- 剪辑节奏：镜头切换的频率与节奏感，决定观众的情绪起伏与完播
- 转场：镜头之间的过渡方式（硬切、叠化、遮挡转场等）
- 调色：调整画面色彩与影调风格（如胶片感、青橙色调）
- 音效：背景音乐之外的环境音、动作音、转场音等
- 字幕：对白、标题、花字等文字呈现，短视频强依赖花字强调信息
- 粗剪/精剪：粗剪搭结构去废片，精剪调节奏抠细节
- 帧率：每秒画面帧数，常规 25/30fps，升格慢动作需 60/120fps 拍摄
- 关键帧：在时间线上打点控制参数（位置、缩放、透明度）变化的动画基础

## 核心工作流程
接单：需求分析（用途/平台/参考片）→ 报价 → 签约收定金 → 素材收集
制作：素材整理备份 → 粗剪 → 精剪 → 调色 → 配音配乐 → 字幕 → 特效包装
交付：内部自检 → 客户审核 → 按约定轮次修改 → 终稿交付 → 收尾款 → 工程归档

## 常见问题
Q：剪一条视频怎么收费？
A：按时长 + 难度 + 用途综合定价：简单口播/图文视频 100-500 元/条，剧情/种草短视频 500-2000 元/条，企业宣传片 2000 元起，商业广告按项目报价。用途（商用/投流）越广价格越高。

Q：客户迟迟不给素材怎么办？
A：接单时即提供素材清单与截止时间，并书面约定「素材超期交付，成片交付时间相应顺延」；开工前素材不齐不启动，避免工期被动。

Q：客户改片没完没了怎么办？
A：合同约定免费修改轮次（通常 2-3 轮），每轮意见要求客户一次性汇总确认；超出轮次按次收费，推翻风格或脚本的重剪需求重新报价。
