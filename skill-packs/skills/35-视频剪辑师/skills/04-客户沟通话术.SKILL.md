# 客户沟通话术

## 适用场景
视频剪辑师（尤其自由接单者）在项目全流程与客户的沟通：把模糊需求变成可执行 brief、报价谈判、约定修改次数、催客户交素材、交付后收尾款、应对「感觉不对」式反馈。

## 核心能力
- 需求确认话术：结构化问清用途、平台、时长、参考片（reference）、交付时间、预算，避免开工后推翻
- 报价沟通：按时长 + 难度 + 用途报价，解释「为什么这个价」，拒绝「先剪一条看看」
- 改片边界谈判：合同约定修改轮次（通常 2-3 轮），结构性推翻重剪的加价话术
- 催素材话术：列出素材清单与截止节点，超期自动顺延交付的专业表达
- 尾款催收：按节点提醒、逾期升级、保留工程文件作为协商筹码
- 差评与纠纷预防：所有承诺落文字，关键节点要客户确认后再推进

## 输出格式
1. 【场景判断】当前沟通场景与核心诉求
2. 【参考话术】2-4 条可直接复制使用的回复（标注语气强度）
3. 【沟通策略】话术背后的谈判逻辑
4. 【风险预防】该场景要同步落实的合同/文字确认项

## 示例
输入：「客户说『先剪一条试试，满意了再谈钱』，怎么回」
输出：话术（中性）：「理解您想先看效果的心情～我们标准流程是：先沟通需求 + 看参考片确认风格，然后付 50% 定金启动，初剪版带水印给您审，满意付尾款后交付无水印成片。这样您的风险是可控的——初剪不满意，我们按合同改到您认可为止」→ 策略：用流程标准化拒绝白嫖，同时给出「初剪可改」降低客户顾虑 → 风险预防：把「定金 50%、2 轮免费修改、超改按次收费」写进合同附件。

## 注意事项
- 开工前必须收到定金与明确需求，口头承诺不算数
- 修改意见请客户一次性汇总并文字确认，避免挤牙膏式改片
- 「重剪」与「修改」边界清晰：推翻脚本/风格属重剪，需重新报价
- 未结尾款不交付无水印成片和工程文件，这是最后的谈判筹码
- 全程沟通留痕，纠纷时聊天记录与确认邮件是维权依据
