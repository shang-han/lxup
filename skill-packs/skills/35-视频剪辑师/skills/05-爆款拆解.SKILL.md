# 爆款拆解

## 适用场景
视频剪辑师/短视频创作者研究对标账号与爆款视频，提炼可复用方法论时使用。解决「知道它火了但不知道为什么」「看完只会说『牛』」的问题，把感性欣赏变成可迁移的创作清单。

## 核心能力
- 多维度拆解框架：选题 → 开头钩子 → 内容结构 → 节奏剪辑 → 情绪曲线 → 互动设计 → 标题封面
- 数据维度分析：点赞/评论/转发/收藏比例反推内容特征（高收藏=干货、高转发=社交货币、高评论=争议/共鸣）
- 开头钩子归因：前 3 秒用了什么手法（悬念/冲突/利益/反常识），量化完播设计
- 剪辑节奏分析：平均镜头时长、卡点方式、信息密度与完播率的关系
- 可复用性评估：区分「不可复制的偶然因素」与「可迁移的方法要素」
- 行动转化：把拆解结论转成本账号的选题清单与拍摄模板

## 输出格式
1. 【拆解对象】视频/账号基本信息与数据表现
2. 【爆点归因】逐维度分析为什么火（选题/开头/结构/情绪/互动）
3. 【数据解读】互动比例反映的用户心理
4. 【可复用要素】3-5 条可迁移的方法，标注适用条件
5. 【行动清单】本账号可直接尝试的选题与拍法

## 示例
输入：「拆解这条 200 万赞的职场口播视频，我能学什么」
输出：数据解读：收藏 80 万（远高于点赞 40%）→ 强干货属性，用户在「存起来备用」→ 爆点归因：开头「被裁员当天我做了 3 件事，多拿了 8 万赔偿」（具体数字 + 利益钩子）；结构是「结论先行 + 三点清单 + 每点配真实对话」；节奏每 2.5 秒一个信息点无废话 → 可复用：① 开头用「具体数字 + 反常识结果」钩子 ② 清单体结构降低理解成本 ③ 引用真实对话增强可信度 → 行动清单：把「涨薪谈判」选题套同样结构重做一条。

## 注意事项
- 拆解学方法不抄内容：借鉴结构与钩子逻辑，文案与素材必须原创
- 区分偶然与必然：账号粉丝基础、投流、热点时机都是变量，别把运气当方法
- 看互动比例而非只看点赞，比例结构才反映真实用户行为
- 拆解要落到行动清单，止步于「分析得真细」没有产出
- 批量拆解（同赛道 10+ 条）找共性，单条拆解易以偏概全
