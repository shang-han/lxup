# 分镜文案

## 适用场景
视频剪辑师/编导在拍摄前规划镜头、在剪辑前梳理素材结构时使用。把文字创意翻译成逐镜头的拍摄与剪辑蓝图，让拍摄团队知道「每个镜头拍什么、怎么拍」，让剪辑知道「素材怎么组接」。

## 核心能力
- 分镜表标准化：镜号 / 景别 / 画面内容 / 运镜 / 台词 / 音效音乐 / 时长 / 备注
- 景别运用：远景交代环境、中景叙事、近景特写情绪，按叙事节奏切换
- 运镜设计：推拉摇移跟升降的适用场景与情绪表达
- 转场规划：硬切、叠化、遮挡转场、匹配剪辑的使用时机
- 声画关系：台词、环境音、BGM、音效的层次安排
- 时长控制：按目标总时长反推每镜时长，卡节奏点

## 输出格式
1. 【分镜概述】总时长、场景数、整体风格基调
2. 【分镜表】逐镜头列出：镜号 | 景别 | 画面 | 运镜 | 台词/字幕 | 音效 | 时长
3. 【转场说明】镜头之间的衔接方式
4. 【音乐建议】BGM 风格与情绪曲线
5. 【拍摄清单】由分镜反推的场景、道具、演员需求

## 示例
输入：「给 30 秒咖啡液广告写分镜」
输出：概述（30 秒、4 场景、清新快节奏）→ 分镜表：#01 特写/冰块落入杯中（慢动作 + 音效「叮」）2s → #02 近景/撕开咖啡液包装（手部特写）2s → #03 特写/咖啡液倒入冰水、云雾状扩散（升格慢镜）4s → #04 中景/主角喝一口、满足表情 + 台词「3 秒，一杯冰美式」4s → …… → 转场：#03 到 #04 用杯子遮挡转场 → 音乐：轻快电子，第 8 秒卡鼓点切换画面。

## 注意事项
- 分镜为执行服务：画面描述要具体可拍，避免「唯美大气」式模糊表述
- 每个镜头都要有存在理由，删掉对叙事无贡献的镜头
- 景别切换要有逻辑（循序渐进或刻意跳切），避免同景别硬切产生跳帧感
- 先按粗分镜与拍摄团队沟通可行性，再细化，防止拍出无法使用的素材
- 预留机动镜头（空镜、特写）便于剪辑时救场
