# 剪辑流程SOP

## 适用场景
视频剪辑师规范个人或团队的剪辑生产流程、新人培训、提升多项目并行效率时使用。把经验化的剪辑动作沉淀为可复制的标准流程，减少返工与交付事故。

## 核心能力
- 全流程 SOP 搭建：素材接收 → 素材整理 → 粗剪 → 精剪 → 调色 → 音频 → 字幕 → 特效包装 → 自检 → 输出 → 交付 → 归档
- 素材管理规范：文件夹命名（项目_日期_机位）、代理剪辑、备份三原则（本地 + 移动 + 云端）
- 粗剪/精剪标准：粗剪搭结构去废片、精剪调节奏卡点，分阶段避免边剪边改
- 技术参数规范：分辨率、帧率、码率、封装格式按平台定制（抖音 1080P/30fps、B 站高码率等）
- 自检清单：黑场、跳帧、音画不同步、错别字、音乐版权、水印位置逐项核对
- 交付与归档：成片命名规范、工程文件归档、素材保留期限

## 输出格式
1. 【流程总览】各阶段名称、目标、负责人/耗时参考
2. 【阶段细则】每阶段的动作标准与产出物
3. 【命名与目录规范】文件/文件夹/工程的命名规则示例
4. 【技术参数表】按交付平台的输出参数
5. 【交付前自检清单】可逐项勾选的检查项

## 示例
输入：「我们是个 3 人剪辑小组，帮我定一套剪辑 SOP」
输出：总览（素材整理 0.5 天 → 粗剪 1 天 → 精剪 1 天 → 调色音频 0.5 天 → 字幕包装 0.5 天 → 自检交付 0.5 天）→ 细则：素材整理阶段按「项目号_机位_卡号」建文件夹并双备份，粗剪只搭叙事骨架不加特效 → 命名规范：「P20260715_客户名_v1_初剪.mp4」，版本号只增不改 → 参数表：抖音竖版 1080×1920/30fps/15Mbps，横版 B 站 4K/60Mbps → 自检清单 12 项（音画同步、错别字、音乐授权……）。

## 注意事项
- 备份是第一纪律：素材丢失不可逆，导入后立即备份再开始剪辑
- 版本管理严格：每次修改另存版本号，不在同一文件上覆盖
- 音乐与字体版权必须核查，商用项目只用已授权素材，防止交付后被诉
- 交付前自检不可省略，低级错误（黑场、错字）最伤专业信誉
- SOP 落地后按月复盘修订，流程服务于效率而非形式主义
