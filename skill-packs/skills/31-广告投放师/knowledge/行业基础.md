# 行业基础

## 行业概况
中国数字广告市场规模超万亿，腾讯广告、巨量引擎（抖音/今日头条）、百度营销三大平台占主导，快手磁力引擎、小红书、B 站、视频号等构成补充生态。投放优化师核心能力：账户结构搭建 + 人群定向 + 出价策略 + 数据分析，ROI 是核心考核指标。行业趋势：自动化投放（程序化/UBA）、AIGC 素材批量化、隐私政策收紧（个人信息保护法、IDFA 弱化）持续抬高投放门槛。初级优化师管执行，资深优化师管策略与预算，收入与管理消耗规模及 ROI 达成挂钩。

## 常用术语
- CPM：千次展示成本，每千次广告曝光的花费，衡量流量价格
- CPC：单次点击成本，每获得一次点击的流量花费
- CTR：点击率 = 点击量/曝光量，衡量素材吸引力
- CVR：转化率 = 转化量/点击量，衡量落地页与产品承接力
- ROI/ROAS：投资回报率/广告支出回报率，电商常看 ROAS = GMV/消耗
- 消耗：广告投放总花费，管理消耗规模是优化师能力量级
- OCPM/OCPX：按转化目标智能出价投放，系统依据转化数据自动探索人群
- 冷启动：新计划/新素材转化样本不足（通常需 20-50 个转化）的模型探索期，成本波动大

## 核心工作流程
投放前：产品与卖点分析 → 人群画像与定向设计 → 素材脚本与创意制作 → 账户结构与计划搭建
投放中：出价与预算调整 → 定向与创意迭代 → A/B 测试 → 实时数据监控与异常预警
投放后：数据复盘（曝光→点击→转化→付费漏斗）→ ROI 计算 → 优化策略制定 → 下一轮投放计划

## 常见问题
Q：广告没效果怎么办？
A：按漏斗逐层诊断：曝光低查定向和出价，点击低查素材和标题，转化低查落地页与产品/客服承接。一次只动一个要素，逐一排除。

Q：ROI 怎么提高？
A：三个抓手——精准定向降获客成本，素材优化提点击率，落地页与优惠机制优化提转化率和客单价。同时关停高消耗低转化计划，把预算集中到优质计划。

Q：计划冷启动怎么度过？
A：前 20-50 个转化内不频繁调整出价与定向，给模型探索空间；备足多组素材轮换；预算给够，确认冷启动失败（成本持续超标）后及时重建计划。
