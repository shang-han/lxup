# 优化策略方案

## 适用场景
投放优化师解决具体业绩问题：转化成本持续走高、消耗跑不出量、ROI 不达标、新账户冷启动困难、优质素材衰退。需要系统性诊断 + 可执行的优化动作。

## 核心能力
- 四要素诊断框架（定向/素材/出价/落地页）定位瓶颈环节
- 定向优化：频次控制、人群收窄/放量、Lookalike 种子包更新、排除已转化与无效人群
- 素材优化：衰退判断标准、迭代方向（换钩子/换角度/换形式）、素材供给机制（每周上新数量）
- 出价与预算优化：OCPM 出价调整幅度与时机（单次 5-10%）、预算按边际 ROI 分配、分时段投放策略
- 落地页优化：首屏跳出率诊断、CTA 设计、加载速度、转化组件位置、优惠机制（定价/赠品/限时）
- 冷启动破局：转化目标设定、初始出价策略（略高于目标成本）、素材组数量、观察周期与重建标准

## 输出格式
1. 【问题定义】当前指标 vs 目标的差距
2. 【诊断结论】瓶颈环节与主因（四要素）
3. 【优化动作】按优先级排列（立即执行/本周/持续）
4. 【出价预算】出价与预算调整方案
5. 【测试安排】A/B 测试设计与成功标准
6. 【风险预案】可能的副作用与回滚方案

## 示例
输入：「电商千川投放，单均获客成本从 45 涨到 68 元，消耗还掉量了，怎么优化？」
输出：诊断：CTR 从 4.1% 降至 2.8%（素材疲劳为主），CVR 基本持平 → 瓶颈在素材，次因是定向频次过高 → 立即动作：关停曝光超 500 万的 Top2 老素材（频次已达 8+），本周上新 5 条素材（开箱实测/差评回应/使用场景剧 3 个新钩子）→ 定向：排除近 30 天已购人群，拓展相似达人互动人群 → 出价预算：新计划出价上浮 5-10% 抢冷启动量，老计划随 CTR 衰减降预算 → 测试：新钩子 A/B，成功标准为成本 <50 元且连续 3 天稳定 → 风险预案：新素材 CTR 仍 <2.5% 则转向落地页优惠机制优化（加赠品/运费险）。

## 注意事项
- 一次只改一个变量，保留观察期（至少 3 天或足够转化量），忌多动作同时上
- 出价调整忌频繁，单次幅度控制在 5-10%，避免打断模型学习
- 尊重 OCPM 模型学习周期，冷启动期不过早判死刑
- 大幅调整前备份当前设置，预留回滚方案
- 优化方案须明确预期效果与时限，不承诺「保证降本」
