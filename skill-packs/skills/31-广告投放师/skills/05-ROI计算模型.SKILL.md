# ROI计算模型

## 适用场景
投放优化师与业务方评估投放经济性：新品上线预算申请、渠道投入产出对比、盈亏平衡点计算、放量空间测算。用数字说话，支撑预算决策。

## 核心能力
- 口径区分：ROI =（收入-成本）/成本；ROAS = 收入/消耗；电商常用 ROAS（GMV/消耗），线索类用单条线索成本（CPL）
- 全成本结构测算：广告消耗 + 产品成本 + 物流履约 + 平台扣点 + 退货退款，算真实毛利口径 ROI
- 盈亏平衡线计算：盈亏平衡 ROAS = 1/毛利率，快速判断投放是否亏损
- 单客经济模型：获客成本（CPA/CAC）与用户生命周期价值（LTV），适用于教育/美业/会员等高复购行业
- 预算分配模型：按边际 ROI 递减原则在渠道/计划间分配，估算放量天花板
- 敏感性分析：乐观/中性/悲观转化情景下的利润测算表

## 输出格式
1. 【模型输入】产品售价、毛利率、消耗、转化数据
2. 【核心指标】ROAS/ROI/CPA/盈亏线计算过程
3. 【利润分析】单单/单客利润拆解
4. 【LTV延伸】考虑复购后的获客经济性（如适用）
5. 【预算建议】分配方案与放量空间判断
6. 【敏感性分析】不同转化率/成本情景下的盈亏

## 示例
输入：「电商客单价 129 元，毛利率 55%，日耗 1 万，日成交 120 单，算算亏不亏、能不能放量」
输出：日 GMV = 120×129 = 15,480 元，ROAS = 1.55 → 盈亏平衡 ROAS = 1/0.55 = 1.82 → 当前亏损：日毛利 8,514 元 < 消耗 1 万，日亏约 1,486 元 → 单均拆解：单均消耗 83 元，单均毛利 71 元，每单亏 12 元 → 改善路径：CPA 需降到 71 元（现 83），即 CVR 提升约 17% 或客单价提升 → 放量判断：ROAS 未达 1.82 前不宜放量，优先优化素材与落地页 CVR；若复购率 20%（复购单毛利 71 元），按 LTV 口径盈亏线可放宽至约 1.52，当前模型刚好打平 → 敏感性表：CPA 75/80/85 元三档下的日利润。

## 注意事项
- 模型输入须基于真实财务与投放数据，毛利率要含退货、运费等隐性成本
- 对外沟通统一 ROI/ROAS 口径（业内常把 ROAS 俗称 ROI），避免口径误会导致决策错误
- LTV 模型需要复购数据支撑，新品期用保守估计
- 边际 ROI 随规模递减，不用小预算结果线性外推大预算
- 模型是决策辅助，定期用实际数据回测修正，不承诺「保证盈利」
