# 投放文案生成

## 适用场景
投放优化师批量制作投放素材：新品上线创意储备、日常素材轮换更新、A/B 测试的文案变量组。覆盖巨量引擎/腾讯广告/百度/小红书等投放场景。

## 核心能力
- 按产品与目标人群分创意角度产出信息流标题/文案：痛点型、利益型、场景型、对比型、权威型、促销型
- 批量产出同主题多版本文案（5-10 条），支持 A/B 测试
- 短视频广告脚本结构：3 秒钩子 → 痛点/场景 → 产品解决方案 → 利益点 → 行动指令（CTA）
- 落地页文案结构：首屏钩子 → 卖点罗列 → 信任背书 → 用户证言 → 促销信息 → CTA 按钮文案
- 适配不同行业表达规范（电商/教育/本地生活/App 下载/线索收集）
- 广告法违禁词自查（极限词、虚假承诺、医疗功效暗示）

## 输出格式
1. 【投放背景】产品、目标人群、转化目标、投放平台
2. 【创意角度】3-5 个切入方向及理由
3. 【文案清单】按角度分组的多版本标题/文案
4. 【视频脚本】钩子 + 分镜口播 + CTA（如需视频素材）
5. 【落地页文案】首屏到 CTA 各模块文案
6. 【合规自查】违禁词与敏感表达检查结果

## 示例
输入：「给一款防脱洗发水写抖音信息流文案，人群 25-40 岁白领，转化目标是抖音小店下单」
输出：创意角度：痛点（掉发焦虑/枕巾落发）、场景（熬夜加班/产后脱发）、成分背书（咖啡因 + 生物素）、效果对比（发际线变化记录）→ 标题 8 条：「用了三个月，发际线冒小绒毛了」（效果呈现）「熬夜党防脱自救指南，白领必看」（场景痛点）等 → 视频脚本：钩子「枕巾上的头发比工资涨得快？」→ 痛点场景 → 产品演示 → 成分讲解 → CTA「点击小黄车，今天买 2 送 1」→ 落地页首屏文案「4 周，和掉发说再见」（配使用记录图）→ 合规自查：「生发」「根治」替换为「强韧发根」「减少断发」，效果表述标注「个体差异，效果因人而异」。

## 注意事项
- 严格遵守广告法：禁用「国家级」「最高」「最佳」等极限词，禁用虚假或引人误解的效果承诺
- 医疗、保健食品、金融、教育等强监管类目须先核实投放资质与平台审核要求
- 效果类表述须有真实依据，用户证言须真实，并标注「效果因人而异」
- 促销信息（价格、赠品、时限）必须真实、可兑现、明示限制条件
- 不用与产品无关的低俗/猎奇素材骗点击，会拉低 CVR 并触发平台处罚
