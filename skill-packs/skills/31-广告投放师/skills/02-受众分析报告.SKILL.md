# 受众分析报告

## 适用场景
投放优化师在投放前设计定向，或在账户诊断阶段分析转化人群特征：为新产品/新市场定义「广告投给谁」，并把画像翻译成平台可执行的定向设置。

## 核心能力
- 四维人群画像构建：基础属性（年龄/性别/地域/消费水平）、兴趣标签、行为特征（搜索/浏览/购买/安装行为）、心理驱动（痛点/渴望/决策因素）
- 画像转定向设置：巨量引擎/腾讯广告的兴趣行为词、自定义人群包、DMP 人群、Lookalike 拓展
- 人群分层策略：核心人群（高意向行为）→ 潜在人群（兴趣标签）→ 拓展人群（Lookalike/系统智能放量）
- 按行业差异化分析：电商（复购/客单）、本地生活（地域半径）、教育（决策人 vs 使用人分离）、B2B（职级/行业）
- 结合转化数据反推高价值人群特征，给出定向收窄/放宽建议
- 输出定向组合方案与 A/B 测试分组设计

## 输出格式
1. 【产品与目标】产品定位、转化目标、客单价
2. 【人群画像】四维目标人群描述
3. 【人群分层】核心/潜在/拓展人群定义
4. 【定向落地】各平台具体定向设置与行为兴趣词
5. 【素材适配】各层人群对应的素材角度建议
6. 【测试方案】定向 A/B 测试分组与观测指标

## 示例
输入：「0-3 岁宝宝辅食线上营，转化目标是 9.9 元体验课报名，投腾讯广告，帮我做受众分析」
输出：画像：25-35 岁新手妈妈，新一线/二线城市为主，宝宝 4-12 个月，关注辅食/育儿/母婴好物，痛点是「不会搭配、怕过敏、没时间做」，决策时段集中在宝宝午睡与晚间 → 分层：核心=近期母婴电商购买 + 搜索过「辅食食谱/高铁米粉」；潜在=关注育儿达人的 0-3 岁兴趣人群；拓展=已报名用户种子包 Lookalike 1% → 定向落地：行为兴趣词清单（辅食添加、婴儿料理机、宝宝缺铁、BLG 手指食物……），排除已报名用户 → 素材适配：核心人群投「分月龄辅食搭配表」干货素材，潜在人群投「新手妈妈踩坑」剧情素材 → 测试：3 组定向 × 2 套素材，跑 3 天看单个报名成本与报名率。

## 注意事项
- 人群分析基于公开数据与平台洞察工具，不违规采集使用个人隐私信息
- 定向设计遵守个人信息保护法与平台数据使用规范
- Lookalike/再营销人群须使用合规授权的第一方数据上传
- 避免定向歧视：不针对民族、宗教、健康状况等敏感属性设置排斥条件
- 画像是概率性判断，须用投放数据持续校准，不把假设当结论
